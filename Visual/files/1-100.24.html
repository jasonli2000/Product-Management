Name Value
NAME OE/RR PT SEL COMBO
DESCRIPTION 
This file is used to store "Combination" pointers for Patient Selection
Lists in OE/RR.  Users can select individual default settings for Wards,
Providers, etc. - but the "Combination" option allows them to set a
multiple, or combination, of such choices.  When the user enters CPRS, the
default patient list will be built from the combined list of sources in
this file (as compared to a single source).
