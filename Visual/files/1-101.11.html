XQOR WORD PROCESSING (101.11)    FILE (1)

Name Value
NAME XQOR WORD PROCESSING
DESCRIPTION 
This is a scratch file used for entering word processing fields used
during dialogs.  It allows text to be captured and edited before it is
actually passed to the database.  Entries in this file are very transient.
They are created, accepted, transferred to a destination file, then
deleted.
