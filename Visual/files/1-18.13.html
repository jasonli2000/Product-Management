WEB SERVER LOOKUP KEY (18.13)    FILE (1)

Name Value
NAME WEB SERVER LOOKUP KEY
DESCRIPTION 
This file contains lookup key names that are mapped to WEB SERVER file
entries.
 
These key-to-server mappings allow application developers to code
using key names and call the appropriate HWSC API at run-time to
resolve the server using the key name as input to the API.
 
There is also a deployment time API that can be use to registered the key
name during the installation of application.
