This file is used by the VistA Imaging DICOM Text gateway for reading
messages regarding Admission, Transfer, Discharge, Patient demographic
changes, and Radiology order processing.  The DICOM gateway will route the
messages to commercial PACS vendors or modality worklist interfaces.
 
Fileman should not be used to enter information into this file. Entries
are made into this file from the MAGD SEND ORM, MAGD SEND ORU, and MAGD
DHCP-PACS ADT EVENTS protocols.
 
Maintenance on this file is performed from the DICOM gateway.
