This file is used to indicate that a correction for a historical
problem with certain Fuji CR images should be applied when
performing measurements on those images.  Entries should be added
to this file to define precisely which CR Models/Software versions
need to have this correction applied.  The problem was that the
Fuji CR image header stored incorrect values in the newer Imager
Pixel Spacing field (0018,1164), whereas the older field (Pixel
Spacing--0028,0030) still carried the correct value.
Note that the problem could appear in Fuji CR images, or images
that may be identified as coming from DeJarnette ImageShare CR
devices.
