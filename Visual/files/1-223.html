NAME DENTAL CLASS I-VI (AMIS)
DESCRIPTION 
This  information is used to report the number of staff class I-VI treatment
cases authorized, the number of staff class I-VI cases pending initiation of
treatment and the number of staff class I-VI treatment cases pending 
completion.  This information is entered by the Dental Service monthly.
