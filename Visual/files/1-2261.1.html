NAME ASISTS DOL ANATOMICAL LOCATION CODES
DESCRIPTION 
This file contains a list of terms and codes used to describe
the body part affected by the injury.  This file contains codes
defined by the DOL (Department of Labor) and MUST NOT be changed
or deleted.  Modifications to this file may cause the rejection
of CA1/CA2 claims submitted to DOL.
