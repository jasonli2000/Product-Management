EDI MESSAGES (364.2)    FILE (1)

Name Value
NAME EDI MESSAGES
DESCRIPTION 
This file contains the messages that are sent electronically back to the
site relating to EDI processing.  These include, but are not limited to,
status messages, error and warning messages found in the EDI transmission
cycle, insurance company updates, batch summaries and statistics.
These messages are stored by message type and options exist that scan this
file and display the messages to the appropriate users and allow each one
to be dealt with and resolved if necessary.
