Name Value
NAME NCPDP PROCESSOR APPLICATION
DESCRIPTION 
This Integrated Billing (IB) file was created for the e-Pharmacy Project.
It is maintained centrally.
Never maintain locally.
 
A NCPDP Processor Application identifies an electronic interface
application associated with the NCPDP PROCESSOR File (366.01).
A NCPDP Processor Application is uniquely identified by its' name.
 
Per VHA Directive 10-93-142, this file definition should not be modified.
