Name Value
NAME STATION NUMBER (TIME SENSITIVE)
DESCRIPTION 
The purpose of this file is to allow DHCP software flexibility and
reliability when the station number of a medical center changes or when
one or more stations have merged into one station.
 
Adding or modifing entries in this file may affect many DHCP applications.
Control of entry into this file should be carefully controlled by the
IRM Service Chief.
