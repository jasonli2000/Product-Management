ADT/HL7 TRANSMISSION (39.4)    FILE (1)

Name Value
NAME ADT/HL7 TRANSMISSION
DESCRIPTION 
This file will be used to keep track of HL7 transmissions sent to external
systems.  An entry in this file will be created for each transmission
sent, when an acknowledgement is returned the entry will be deleted.  This
file will contain transmitted records and records not accepted by other
systems.
 
The file is currently maintained by the VIC software modules and should
not be altered.
 
Per VHA Directive 10-93-142, this file definition should not be modified.
