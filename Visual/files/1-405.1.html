Name Value
NAME FACILITY MOVEMENT TYPE
DESCRIPTION 
This file contains all allowable admission, discharge, transfer,
specialty change, check-in lodger, and check-out lodger movements.
These are site determined and site maintained.  However, we STRONGLY
recommend that when updating movement types, you utilize the 'Edit
Bed Control Movement Types' option to ensure all data is properly
updated.
