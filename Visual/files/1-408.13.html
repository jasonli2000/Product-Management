INCOME PERSON (408.13)    FILE (1)

Name Value
NAME INCOME PERSON
DESCRIPTION 
This file contains demographic information (name, social security number,
date of birth, and sex) for non-veteran relations of patients seen at this
facility.  This information is collected for use in the means test and
income screening modules of MAS.  This data will also be used for
IVM (Income Verification Matching) with the IRS (Internal Revenue Service).
