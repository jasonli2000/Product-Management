SD WL CLINIC LOCATION (409.32)    FILE (1)

Name Value
NAME SD WL CLINIC LOCATION
DESCRIPTION 
This file contains the list of Clinics that can be used by an
institution in the Wait List (Sch/PCMM) package. The clinic must be an 
active clinic in the Hospital Location file. Each entry in this file will 
contain a wait list activation date and inactivate date.
