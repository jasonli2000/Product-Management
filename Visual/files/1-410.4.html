SUB-CONTROL POINT (410.4)    FILE (1)

Name Value
NAME SUB-CONTROL POINT
DESCRIPTION 
This file contains the names of sub-control points used by Control Point
Clerks and Control Point Officials to sub-divide the funds allocated to
them by Fiscal Service.  Entries in this file are established by entering
a new sub-control point name in the Sub-Control Point field of the
Control Point Activity file (#410).
