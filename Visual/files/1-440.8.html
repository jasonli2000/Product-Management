PRCH AFC CHARGE TRANSMISSION LOG (440.8)    FILE (1)

Name Value
NAME PRCH AFC CHARGE TRANSMISSION LOG
DESCRIPTION 
This file contains a record of each transmission batch from the Credit
Card System of Purchase Card Charges and Purchase Card Demographic Changes
to be posted to IFCAP files (#440.6 and #440.5, respectively).  This file
is the source for the report Daily Charge Transmission Log [PRCH DAILY
CHARGE TRANS LOG].
