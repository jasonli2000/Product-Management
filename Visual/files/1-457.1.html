TOUR OF DUTY (457.1)    FILE (1)

Name Value
NAME TOUR OF DUTY
DESCRIPTION 
This file contains a list of all Tours of Duty utilized by the
station. Each tour can contain up to seven time segments which
may optionally have a Special Tour Indicator to indicate
special processing. Further, a tour may be available to all
T&L Units or may be associated with a list of T&L Units which
may utilize the tour.
