NAME VOLUNTARY MASTER
DESCRIPTION 
This file is a master file which contains all the information needed for
completing VA Form 10-5390, and all the information found on the alpha
listing produced by the DPC, Austin, Tx.  This file also has some additional
fields to include the volunteers full date of birth, telephone number, and next
of kin information.
