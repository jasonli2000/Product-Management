A&SP PATIENT (509850.2)    FILE (1)

Name Value
NAME A&SP PATIENT
DESCRIPTION 
The A&SP PATIENT file (#509850.2) contains identifying, demographic, and 
other clinical information for all patients seen in the Audiology and 
Speech Pathology clinics.
  
The entries in this file are pointers to the PATIENT file (#2) used by 
MAS.  For this reason, a patient must already be present in DHCP File #2 
before he can be added as an Audiology/Speech Pathology patient.
  
Data in this file SHOULD NOT be altered through the use of VA FileMan; 
input should take place ONLY through the QUASAR menu options.
  
Per VHA Directive 10-93-142, this file definition SHOULD NOT be modified.
