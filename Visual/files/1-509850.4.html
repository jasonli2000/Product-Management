A&SP PROCEDURE CODE (509850.4)    FILE (1)

Name Value
NAME A&SP PROCEDURE CODE
DESCRIPTION 
The A&SP PROCEDURE CODE file (#509850.4) contains pointers to a small 
subset of CPT-4 codes which pertain to Audiology and Speech Pathology.
Entries in this file are pointed to by the PROCEDURE CODE field in the 
A&SP CLINIC VISIT file (#509805.6).
  
Before using the QUASAR package, the supervisor must enter cost data for 
each procedure code in the A&SP PROCEDURE CODE file. This can be done 
using the Enter Cost Information for Procedures option.
  
This file arrives at the VAMC with data.  This file SHOULD NOT be altered 
through the use of VA FileMan; input should take place ONLY through the 
QUASAR menu options.
  
Per VHA Directive 10-93-142, this file definition SHOULD NOT be modified.
