CDR ACCOUNT (509850)    FILE (1)

Name Value
NAME CDR ACCOUNT
DESCRIPTION 
The CDR ACCOUNT file (#509850) contains all of the cost accounts used to 
calculate and print the Cost Distribution Report (RCS 10-0141) for 
Audiology and Speech Pathology (Cost Center 228).
  
Each patient visit contains a pointer to one of these accounts for 
distribution of workload to appropriate cost accounts.
  
This file arrives at the VAMC with data.  This file SHOULD NOT be altered 
through the use of VA FileMan; input should take place ONLY through the 
QUASAR menu options.
  
Per VHA Directive 10-93-142, this file definition SHOULD NOT be modified.
