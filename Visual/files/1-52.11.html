NAME PATIENT NOTIFICATION (Rx READY)
DESCRIPTION 
This file holds information to be used for the Prescription Ready
display and also data to be used to collect work load and waiting
time information.  Ideally, as the pharmacy receives the Rx
from the patient, the information will be entered into this file.
within the outpatient pharmacy package.
