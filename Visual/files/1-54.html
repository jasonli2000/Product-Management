RX CONSULT (54)    FILE (1)

Name Value
NAME RX CONSULT
DESCRIPTION 
This file holds the expansion of the number codes that represent the
additional warnings/consultations that may be needed for a particular
drug.  The DRUG file informally points to this file, in much the same
way that it points to the Medication Instruction file.  The field that
points to this file is WARNING LABELS and it essentially requires that
the numbers entered here be valid entries in this file.
 
These numbers are printed on the labels with the idea that the pharmacy
would attach the appropriate warning stickers to the bottle.
 
Data storage moved to PS(54 from DIC(54 with version 7 of Outpatient
Pharmacy.
