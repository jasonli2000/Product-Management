Name Value
NAME UNIT DOSE PICK LIST STATS
DESCRIPTION 
  Contains medication amounts and costs for the Unit Dose package.  Data
is entered into this file when pick lists are filed away, and when pre-
exchange units, extra units dispensed, and returns are entered through the
package.  Most of the cost reports gather their data from this file.
