OUTPATIENT CLINIC SORT GROUP (59.8)    FILE (1)

Name Value
NAME OUTPATIENT CLINIC SORT GROUP
DESCRIPTION 
This file will be used to store the clinic sort groups.  The sort groups
will be used to sort action/information profiles.  The current sort for
profiles is by patient or clinics (a clinic or all clinics).  This third
sort will allow sites to sort by specific clinic group.
