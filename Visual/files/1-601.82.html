MH RULES (601.82)    FILE (1)

Name Value
NAME MH RULES
DESCRIPTION 
This file contains the rules used in administering an interview, survey or 
test. A rule is an action that is performed when a specified question is 
answered in a specified manner. An example of a rule would be to enter 
'Not applicable' to the Question, "Are you pregnant?" if the patient has 
had a hysterectomy. Rules are complex logic based entries that enables 
the user to create complex scripts.
