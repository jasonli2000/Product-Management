DSM (627.7)    FILE (1)

Name Value
NAME DSM
DESCRIPTION 
The DSM (#627.7) file holds the DSM-III codes originally stored in the
DSM3 (#627) file, DSM-III-R codes originally stored in the DSM-III-R
(#627.5) file, and DSM-IV diagnostic codes.  In addition, it will hold
all codes created be DSM versions following IV.  Each entry is unique
to a DSM version (even if entries are identical) and is identified
by the DSM VERSION field.
