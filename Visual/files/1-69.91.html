LR ROUTINE INTEGRITY CHECKER (69.91)    FILE (1)

Name Value
NAME LR ROUTINE INTEGRITY CHECKER
DESCRIPTION 
   This file contains routine size (^%ZOSF("SIZE")) AND ROUTINE BIT
SIZE (^LRNITEG) FOR EXPORTED LAB PACKAGES BY VERSION NUMBER.
   CAN BE USED TO IDENTIFY DIFFERENCE BETWEEN VERSION'S ROUTINE AND
TO DOUBLE CHECK PATCH APPLICATIONS.
