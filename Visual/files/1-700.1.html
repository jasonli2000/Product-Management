PFT PREDICTED VALUES (700.1)    FILE (1)

Name Value
NAME PFT PREDICTED VALUES
DESCRIPTION 
This file holds the set of Predicted Value and Correction formulas for
use by the PFT report.  The formulas are selected from the PFT FORMULA
file (700.2).   There are 2 sets present (Male and Female).  The
Male and Female sets can be changed through the Pulmonary Managers' menu.
