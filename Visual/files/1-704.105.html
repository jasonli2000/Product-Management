TERM_UNIT_PAIR (704.105)    FILE (1)

Name Value
NAME TERM_UNIT_PAIR
DESCRIPTION 
NOTICE: This file is maintained exclusively by the CP Terminology group 
and must not be modified or edited in any way. Subsequent releases of CP 
Terminology may create lasting errors in the CliO data store if this file 
is changed.
 
This file matches observation(s) with available units for the observation,
and the range settings for each pair.
 
NOTE: 
Observations are presented as entries in the TERM File (#704.101). For 
example the observation "TRACTION WEIGHT" could be identified by the TERM 
File (#704.101), TERM ID (field #.01) 
"{0C913807-C678-4E94-BD20-B8B2EBE697BE}".
