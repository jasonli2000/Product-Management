TERM_RANGE_CHECK (704.107)    FILE (1)

Name Value
NAME TERM_RANGE_CHECK
DESCRIPTION 
This file maintains value ranges for related observations/terms.
 
An example could be a TERM RANGE CHECK entry that would result in an 
observed TEMPERATURE of 80 DEGREES FAHRENHEIT for a MALE between 55-150 
years of AGE, triggering a "CRITICAL LOW" response by the Clinical 
Flowsheets application.
