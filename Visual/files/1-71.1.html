Name Value
NAME MAJOR RAD/NUC MED AMIS CODES
DESCRIPTION 
This file contains the valid AMIS codes, descriptions and weighted work
units as assigned by Radiology Service in VACO. The data in this file is
used in the compilation of various workload reports including the AMIS
Report.
 
New entries and modifications to existing entries is prohibited without
approval from Radiology Service VACO.
 
 Data Storage
 ------------
 The data for the 'MAJOR RAD/NUC MED AMIS CODES' file is stored in the
^RAMIS(71.1, global. At the present time this file is very static. However,
the global should be journaled.  It should also be translated if the
operating system supports this function.
