EC EVENT CODE SCREENS (720.3)    FILE (1)

Name Value
NAME EC EVENT CODE SCREENS
DESCRIPTION 
This file contains information that is used to screen procedures
based on a unique location, DSS unit, category, and procedure.
Information in this file should only be entered or edited
through the Event Code Screen (Create) option. Access
to this option is restricted to the Event Capture Management
Menu.
 
One important note, information related to the event code
screen cannot be deleted; however, it may be inactivated
using the Inactivate Event Code Screens option within the
Event Capture software.
 
This file should NOT be modified directly using VA FileMan.
This information is created and edited using the Event Capture
Management Menu options.
