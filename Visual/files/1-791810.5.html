Name Value
NAME ROES MENU
DESCRIPTION 
The ROES Menu file contains the memus for ROES. Each menu is assigned to a
Service and is designed to restrict such things as product groups and
disability codes used with the menu or for the Service. All data in this
file is distributed with ROES and should not be edited by the local
facility.
