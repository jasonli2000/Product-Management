Name Value
NAME HEARING AID COMPONENTS
DESCRIPTION 
This file contains hearing aid components that may be ordered with custom
hearing aids from the Denver Distribution Center through the DDC
Remote Order/Entry System. Each hearing aid in the Remote Inventory
Product File may have associated with it a component or components from
this file. This file is exported by the DDC with data. The file will be
updated via patch if components are added or inactivated. The file should
not be edited at the local facility.
