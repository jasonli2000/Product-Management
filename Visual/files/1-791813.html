Name Value
NAME REMOTE ORDER/ENTRY PARAMETER
DESCRIPTION 
The Remote Order/Entry Parameter File contains a number of fields
that give the ADPAC's at each local facility the ability to customize
portions of the system for use at their stations. Definitions of each
parameter and recommendations for the settings to be used are included in
the Parameters section of the ROES V2.0 Technical Manual.
 
Data definitions are distributed with the ROES. No data is distributed.
