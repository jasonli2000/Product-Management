REMINDER ORDER CHECK ITEMS GROUP (801)    FILE (1)

Name Value
NAME REMINDER ORDER CHECK ITEMS GROUP
DESCRIPTION 
This file contains a list of Orderable Items and Pharmacy Items that should
have a reminder order check rule(s). The Orderable Item will only be
evaluated when the user places the order in CPRS. An Order Check Items Group
can contain a list of Orderable\Pharmacy Items and a list of Rules.
