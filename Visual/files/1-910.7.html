ESP SELECTABLES (910.7)    FILE (1)

Name Value
NAME ESP SELECTABLES
DESCRIPTION 
The ESP SELECTABLES file contains colors and makes of vehicles.  It
is pointed to by the ESP POLICE REGISTRATION LOG file (#910.2) color
and make fields.  This file is also pointed to by the ESP OFFENSE
REPORT file (#912) hair, skin, and eye color fields.  These fields
are screened so only appropriate choices can be selected.
