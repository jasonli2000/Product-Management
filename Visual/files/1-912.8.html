ESP CRIME TYPES (912.8)    FILE (1)

Name Value
NAME ESP CRIME TYPES
DESCRIPTION 
The ESP CRIME TYPES file contains types of criminal activity which
fall under each of the thirteen Crime Categories within ESP CRIME
CATEGORIES file (#912.7).  These crime types have been identified by
the VA Police policy managers (VACO) and no additional crime types
are to be added by the local sites.
You CANNOT add entries to this file.
