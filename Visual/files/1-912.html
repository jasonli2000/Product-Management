ESP OFFENSE REPORT (912)    FILE (1)

Name Value
NAME ESP OFFENSE REPORT
DESCRIPTION 
The ESP OFFENSE REPORT file contains the offense report information.
This file points to the ESP MASTER NAME INDEX file (#910).  Once a
case is closed, the offense report becomes the case history.  The
offense report cannot be edited once a case is closed, but can be
reopened for editing by the appropriate authority.
