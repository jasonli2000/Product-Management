BIOCHEM ABNORMALITY FOR CONTRAST MEDIA (9)    ORDER CHECKS (100.8)

Name Value
NAME BIOCHEM ABNORMALITY FOR CONTRAST MEDIA
DESCRIPTION 
Triggered by selection of Imaging orderable items. CPRS checks
to see if the imaging orderable item uses a contrast media. If so and the
patient's BUN and/or creatinine results are abnormal or critical, the order
check occurs.
