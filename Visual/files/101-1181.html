PXCA DATA EVENT (1181)    PROTOCOL (101)

Name Value
NAME PXCA DATA EVENT
ITEM TEXT PCE Device Interface Module's Data Event
DESCRIPTION 
This is the event point invoked by PCE Device Interface Module when it
has not found any errors in the data passed to it.  This makes the data
available to other users of the data including users of any Local data
that may be included.
TYPE extended action
CREATOR USER,ONE
