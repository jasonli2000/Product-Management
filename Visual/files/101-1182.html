Name Value
NAME PXK VISIT DATA EVENT
ITEM TEXT VISIT RELATED DATA
DESCRIPTION 
This is a Protocol that PIMS can hook onto to find the data that
was collected by PCE using List Manager,Scanning etc.
 
PIMS has developed a protocol, SDAM PCE EVENT, which will use the visit
related data to do an auto-checkout.
