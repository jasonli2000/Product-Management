NAME PXK SDAM TO V-FILES
ITEM TEXT SDAM to V-Files
SYNONYM
  1. SYNONYM:   PR
DESCRIPTION 
Populates the V-Provider file based on the Provider identified
by MAS clerks in the PIMS Check-out process.
  
The V-Provider entry will only be created if the PIMS Check-out process
has the Visit entry that the V-Provider entry should be related to.
 
This protocol should be attached to the SDAM APPOINTMENT EVENTS Protocol.
