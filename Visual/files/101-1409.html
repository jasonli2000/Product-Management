NAME PXCE EDIT V-FILE
ITEM TEXT Edit an Item
DESCRIPTION 
This action allows you to edit an entry for a selected encounter.
 
You may edit one item, several items separated by commas, or a range of
items separated by a hyphen.
 
You may also edit entries by entering the number associated with the item
at the "Select Item(s)" prompt.
