PXCE ENCOUNTER EDIT (1410)    PROTOCOL (101)

Name Value
NAME PXCE ENCOUNTER EDIT
ITEM TEXT Encounter
DESCRIPTION 
This action displays encounter information related to the encounter,
including the Encounter Date and Time, the Check Out Date and Time,
the Evaluation and Management code, the Service Connected Status, Agent
Orange Exposure, Ionizing Radiation Exposure, and Persian Gulf Exposure.
 
If this is a main encounter, you may edit the Check Out Date and Time, the
Evaluation and Management code and Service Connected.
TYPE action
CREATOR USER,ONE
PACKAGE PCE PATIENT CARE ENCOUNTER
EXIT ACTION D INIT^PXCEAE
ENTRY ACTION S PXCEFIEN=+^TMP("PXCEAEIX",$J,1) D EN^PXCEVFIL($S($P(^AUPNVSIT(+PXCEFIEN,0),"^",7)="E":"HIST",1:"VST"))
