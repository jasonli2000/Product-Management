Name Value
NAME PXCE HOSPITAL LOCATION VIEW
ITEM TEXT View by Clinic
DESCRIPTION 
This action allows you to change the display of appointments or
encounters based on a clinic.  For example, if your current list
includes all appointments for a specified date range for the selected
patient, and you want to display all appointments for a specific clinic,
you may use this action to change the display to include appointments or
encounters for the desired clinic.
