NAME PXCE SDAM EXPAND
ITEM TEXT Expand Appointment
DESCRIPTION 
This action allows you to display all the patient demographics and
appointment event log items that have been entered for a selected patient
appointment. This action displays information from the Scheduling package
and not from PCE.
TYPE action
CREATOR USER,ONE
PACKAGE PCE PATIENT CARE ENCOUNTER
EXIT ACTION S:'VALMCC VALMBCK="R"
