NAME PXCE GMPL OE DATA ENTRY
ITEM TEXT Patient Problem List
DESCRIPTION 
This protocol can be added to the PCE menu protocols, PXCE MAIN MENU and 
PXCE SDAM MENU, to allow the user of PCE's user interface to go to 
Problem List without leaving the interface.
