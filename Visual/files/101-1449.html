NAME SDAM PCE EVENT
ITEM TEXT PROCESS PCE EVENT DATA
DESCRIPTION 
This protocol is the event handler attached
to the PXK VISIT DATA EVENT protocol.
 
The protocol processes scheduled appointment
check out data made available by this PCE
event point. PCE currently obtains this check out
data from MCCR data capture pilots and also a
manual entry module within the PCE package.
