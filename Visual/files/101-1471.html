RA CANCEL (1471)    PROTOCOL (101)

Name Value
NAME RA CANCEL
ITEM TEXT Rad/Nuc Med exam cancellation
DESCRIPTION 
This protocol is triggered whenever a Radiology/Nuclear Medicine exam is
cancelled.  It executes code that creates an HL7 ORM message consisting of
PID, ORC, OBR and OBX segments.  The message contains all relevant
information about the exam, including procedure, time of cancellation,
procedure modifiers, patient allergies and clinical history.
TYPE event driver
CREATOR USER,ONE
