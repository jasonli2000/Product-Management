GMPL EDIT MENU (1501)    PROTOCOL (101)

Name Value
NAME GMPL EDIT MENU
ITEM TEXT Edit Problem Values
DESCRIPTION 
This protocol is for use with the List Manager utility, to display
the current editable values of the selected problem entry in a list
format for editing.
TYPE menu
CREATOR USER,ONE
ITEM
  1. ITEM:   GMPL EDIT REMOVE
    MNEMONIC:   RM
    SEQUENCE:   11
  2. ITEM:   GMPL EDIT SAVE
    MNEMONIC:   SC
    SEQUENCE:   13
  3. ITEM:   GMPL EDIT NEW NOTE
    MNEMONIC:   CM
    SEQUENCE:   1
    DISPLAY NAME:   Additional Comments
  4. ITEM:   GMPLX BLANK2
    SEQUENCE:   5
  5. ITEM:   VALM QUIT
    MNEMONIC:   Q
    SEQUENCE:   3
    DISPLAY NAME:   Quit w/o Saving Changes
PACKAGE PROBLEM LIST
HEADER D INIT^GMPLEDIT:$G(GMPREBLD),KEYS^GMPLEDT3,SHOW^VALM S:XQORM("B")="Quit"&$$EDITED^GMPLEDT2 XQORM("B")="Save Changes and Exit"
MENU PROMPT Select Item(s):
COLUMN WIDTH 40
MNEMONIC WIDTH 4
