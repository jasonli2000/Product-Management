NAME GMPL USER PREFS
ITEM TEXT Preferred View of Problem List
DESCRIPTION 
This menu contains actions allowing a user to change his/her preferred
view of patient problem lists.  A set of services may be defined here
that will be used as a default screen when displaying patient problem
lists for this user; the view may be changed dynamically within the
Problem List application through the "Change View" action, but
it will not be stored as a new default unless updated here.
