GMPL LIST MENU (1503)    PROTOCOL (101)

Name Value
NAME GMPL LIST MENU
ITEM TEXT List Commonly Seen Problems
DESCRIPTION 
This protocol is for use with the List Manager utility, to display
the user's preferred list of commonly seen problems to facilitate
selection and addition to the patient's problem list.
TYPE menu
CREATOR USER,ONE
ITEM
  1. ITEM:   GMPLX BLANK1
    SEQUENCE:   3
  2. ITEM:   GMPL LIST CLU
    MNEMONIC:   AD
    SEQUENCE:   1
  3. ITEM:   VALM QUIT
    MNEMONIC:   Q
    SEQUENCE:   11
    DISPLAY NAME:   Quit to Problem List
PACKAGE PROBLEM LIST
HEADER W:$D(GMPLGRP) !!,"Rebuilding menu ..." D BUILD^GMPLMENU:$D(GMPLGRP),KEYS^GMPLMENU,SHOW^VALM S:XQORM("B")="Quit" XQORM("B")="Quit to Problem List" K GMPLGRP
