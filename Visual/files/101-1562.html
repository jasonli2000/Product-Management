GMPL VIEW RESTORE (1562)    PROTOCOL (101)

Name Value
NAME GMPL VIEW RESTORE
ITEM TEXT Preferred View
DESCRIPTION 
This action will replace the currently specified view with the
user's pre-defined preferred view.
TYPE action
CREATOR USER,ONE
PACKAGE PROBLEM LIST
ENTRY ACTION S X=$$VIEW^GMPLX1(DUZ) W:'$L(X) !,"You have no preferred view defined.",! Q:'$L(X) S:X'=GMPLVIEW("VIEW") GMPREBLD=1,GMPLVIEW("VIEW")=X K X S:GMPLVIEW("PROV") GMPLVIEW("PROV")=0,GMPREBLD=1 S:GMPLVIEW("ACT")'="A" GMPLVIEW("ACT")="A",GMPREBLD=1
