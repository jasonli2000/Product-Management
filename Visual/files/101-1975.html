GMTS HS BY PATIENT & DATE (1975)    PROTOCOL (101)

Name Value
NAME GMTS HS BY PATIENT & DATE
ITEM TEXT Range of Dates Patient Health Summary
SYNONYM
  1. SYNONYM:   HS PRINT RANGE OF DATES
DESCRIPTION 
Generates Health Summaries of a specified pre-defined Health Summary
Type for multiple patients.  After patients are selected.  The user can
pick a date range.  Data for summaries is based on the date range.  This 
date range overrides time limits for componenets which allow this option
TYPE action
CREATOR USER,ONE
EXIT ACTION K GMRANGE
