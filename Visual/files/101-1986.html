DESCRIPTION 
This protocol hangs off of the DGPM MOVEMENT EVENTS Protocol. It generates
HL7 ADT messages based on what action has fired the DGPM MOVEMENT EVENTS
Protocol, such as: Admissions; Discharges, or Transfers.
