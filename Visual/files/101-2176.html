Name Value
NAME RA TCP ORU
ITEM TEXT TCP Client
DESCRIPTION 
This protocol is used in conjunction with the RA RPT event point protocol.
The HL7 package uses this protocol to send rad/nuc med report (ORU)
messages to TCP/IP recipients.  This protocol should be entered in the
ITEM multiple field of the RA RPT protocol if this messaging scenerio is
used in a facility.  This is part of the file set-up to enable message
flow when a rad/nuc med report is verified.
