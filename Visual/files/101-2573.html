PSJ LM INTERVENTION DELETE (2573)    PROTOCOL (101)

Name Value
NAME PSJ LM INTERVENTION DELETE
ITEM TEXT Delete Pharmacy Intervention
DESCRIPTION 
This option is used to delete an intervention from the APSP INTERVENTION
file (#32.4).  You may only delete an intervention that was entered on
the same day.
TYPE action
CREATOR USER,SEVENTEEN
PACKAGE INPATIENT MEDICATIONS
EXIT ACTION S VALMBCK="R"
ENTRY ACTION D FULL^VALM1,^PSODELI
TIMESTAMP 2013-02-12 10:39:04
