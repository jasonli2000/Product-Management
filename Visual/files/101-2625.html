NAME GMRC MGR ACTIONS
ITEM TEXT CONSULTS MANAGEMENT ACTIONS
DESCRIPTION 
This protocol contains specialized actions that an ADPAC or manager
of the consult/request package can take to remedy various situations
that arise in day-to-day usage of the consults package. The menu
for these actions should only be given to the consults ADPAC or similar
manager since these actions can cause trouble for the consults package
if anyone is allowed to use them.

CREATOR USER,ONE
