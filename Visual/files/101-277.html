NAME OR GUA DC GEN ORD ON TRANS
ITEM TEXT DC Generic Orders on Transfer
DESCRIPTION 
This protocol will discontinue all generic orders for a patient when the
patient is transferred.  The protocol should be set up as an item on the
DGPM MOVEMENT EVENTS protocol.  It will check to see if the movement is
a transfer and then discontinue all generic orders (orders created by the
Order Entry/Results Reporting package itself).  The actual DC process is
tasked, to reduce the wait for the person doing the transfer.
