NAME RA PSCRIBE ORU
ITEM TEXT TCP Client
DESCRIPTION 
This protocol is used in conjunction with the RA RPT 2.3 event protocol.
The HL7 package uses this protocol to send rad/nuc med report (ORU)
messages to TCP/IP recipients.  This protocol should be entered in the
SUBSCRIBERS multiple field of the RA RPT 2.3 protocol if this messaging
scenario is used in a facility.  This is part of the file set-up to
enable message flow when a rad/nuc med report is verified.
