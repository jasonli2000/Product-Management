RA TALKLINK ORM (2826)    PROTOCOL (101)

Name Value
NAME RA TALKLINK ORM
ITEM TEXT TCP Client
DESCRIPTION 
This protocol is used in conjunction with the RA REG 2.3, RA EXAMINED 2.3,
and RA CANCEL 2.3 event protocols.  It is used by the VISTA HL7 package to
send ORM messages to TCP/IP recipients.  This protocol should be entered
in the SUBSCRIBERS multiple field of those event point protocols if this
type of messaging scenerio is used at a facility.  This is part of the
file set-up to enable HL7 message flow when exams are registered, cancelled,
and when they reach the status flagged as 'examined' by the site.
