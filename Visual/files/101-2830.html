NAME RA TALKLINK TCP REPORT
ITEM TEXT Client for TalkStation TCP rpt
DESCRIPTION 
Subscriber protocol for sending report to VISTA Radiology/Nuclear
Medicine.  This protocol is used by the HL7 package to process messages
sent to VISTA from a COTS voice recognition unit using TCP/IP for message
flow.  This protocol should be entered in the SUBSCRIBERS multiple of the
RA TALKLINK TCP SERVER RPT protocol.
