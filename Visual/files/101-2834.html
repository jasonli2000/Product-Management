RA REG 2.3 (2834)    PROTOCOL (101)

Name Value
NAME RA REG 2.3
ITEM TEXT Rad/Nuc Med exam registered for HL7 v2.3 message
DESCRIPTION 
This protocol is triggered whenever a Radiology/Nuclear Medicine exam is
registered.  It executes code that creates an HL7 ORM version 2.3 message
consisting of PID, ORC, OBR, and OBX segments.  The message containes all
relevant information about the exam, including procedure, time of
registration, procedure modifiers, patient allergies, and clinical
history.

SENDING APPLICATION RA-VOICE-SERVER
TRANSACTION MESSAGE TYPE ORM
EVENT TYPE O01
VERSION ID 2.3
RESPONSE PROCESSING ROUTINE Q
