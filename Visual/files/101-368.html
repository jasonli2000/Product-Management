GMTS HS BY PATIENT (368)    PROTOCOL (101)

Name Value
NAME GMTS HS BY PATIENT
ITEM TEXT Patient Health Summary
SYNONYM
  1. SYNONYM:   HS PRINT BY PATIENT
DESCRIPTION 
Generates a Health Summary of a specified pre-defined Health Summary Type for
a specified patient.
TYPE action
CREATOR USER,ONE
PACKAGE HEALTH SUMMARY
ENTRY ACTION W ! S DIC("B")=$P($G(^GMT(142,+$G(^DISV(+$G(DUZ),"^GMT(142,")),0)),U),GMTSTYP=0 S:'$D(DFN) DFN=$S($D(ORVP):$P(ORVP,";"),1:"") D SELPT^GMTS:+DFN'>0,SELTYP^GMTS Q:GMTSTYP'>0 D HSOUT^GMTS
