WII ADD ENTRY (4306)    PROTOCOL (101)

Name Value
NAME WII ADD ENTRY
ITEM TEXT Add Adm/Dischg to list
SYNONYM
  1. SYNONYM:   WII
DESCRIPTION 
This protocol is a list manager action item located on the WII REVIEW ADT 
EVENTS option.  Its function allows users to manually add an entry into 
the WII ADMISSIONS DISCHARGES file (#987.5) in the case that an admission 
or discharge did not get collected by the weekly back ground job (WII 
BUILD ADT EVENTS).
TYPE action
CREATOR USER,SEVENTEEN
PACKAGE WOUNDED INJURED ILL WARRIORS
EXIT ACTION S VALMBCK="R" D CLEAN^VALM10,INIT^WIILM
ENTRY ACTION D ADD^WIILM02
