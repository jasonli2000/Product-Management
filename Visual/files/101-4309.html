WII PENDING RBCK (4309)    PROTOCOL (101)

Name Value
NAME WII PENDING RBCK
ITEM TEXT Roll Back Status to Pending
SYNONYM
  1. SYNONYM:   WII
DESCRIPTION 
This protocol is a list manager action item located on the WII LM STS 2 
MENU.  Its function allows users to roll back entries that have been 
selected to transmit to the repository but have yet been sent.
TYPE action
CREATOR USER,SEVENTEEN
PACKAGE WOUNDED INJURED ILL WARRIORS
