XOBW ASSOCIATE FILTER SERVER (4438)    PROTOCOL (101)

Name Value
NAME XOBW ASSOCIATE FILTER SERVER
ITEM TEXT Filter Server
DESCRIPTION 
This protocol allows the user to limit the list of web server lookup
key entries that are displayed by the key manager.
 
The user specifies text that is used as a filter against the beginning
characters of the server name. Only keys with matching associated 
web server names will be listed in lookup key manager utility.
 
Also, this protocol is used to clear a server filter if one is currently 
being applied.
