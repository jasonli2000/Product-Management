NAME MD DGPM PATIENT MOVEMENT
ITEM TEXT CliO DGPM patient movement interface
DESCRIPTION 
This Protocol is an interface to the DGPM Movement Event Protocol that 
will process ADT Events for patient that are admitted to the hospital and 
will call the code in CliO to process that movement.
