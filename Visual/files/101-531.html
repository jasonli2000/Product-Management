DESCRIPTION 
This is a very important protocol for activating the Procedure Request
order capabilities in the Add orders process.  This is the protocol which
should be performed when "Procedure..." is selected from the "Add New
Orders menu.  This protocol should only be added for clinician use when
the procedures which may be ordered without requiring a consult are
defined in the "GMRCRM REQUEST TYPES" protocol menu.
