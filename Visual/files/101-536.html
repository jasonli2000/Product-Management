NAME GMRCRM REQUEST TYPES
ITEM TEXT Procedures which may be ordered directly w/o consult
DESCRIPTION 
THIS PROTOCOL NAME SHOULD NOT BE ALTERED.  The menu items are the only
site modifiable information for this protocol menu.
  
This is the list of procedures, treatments or other service requests which
do not require a consult.
  
This menu represents the items which are selectable when the "Procedures..."
entry is selected on the Add Orders menu in OE/RR.
Procedures should be added to this menu as clinicians identify things they
order from a service which they don't require a consult for.
