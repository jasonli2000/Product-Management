NAME ORS GRAPH LABS
ITEM TEXT Graph Lab Tests
DESCRIPTION 
This action type protocol allows graphing of lab results by collection
sample, and test for any time interval and # of data points.
TYPE action
CREATOR USER,ONE
PACKAGE ORDER ENTRY/RESULTS REPORTING
ENTRY ACTION S LRCW=8,LRFLAG="",LRCTRL=0,(LRNSET,N)=80 D RANGE^ORPRS01() Q:$G(OREND) S ORSSTRT(+XQORNOD)=+ORSSTRT,ORSSTOP(+XQORNOD)=+ORSSTOP D L2^LRDIST4 S:$D(LRSPEC) ORLRSPEC(+XQORNOD)=LRSPEC S:'$D(LRTEST) OREND=1 I X="^" S DUOUT=1 S:X="^^" DIROUT=1
TIMESTAMP 1999-04-16 15:22:35
METHOD
  1. METHOD:   20
    METHOD ACTION:   S DFN=+ORVP,LRSS="CH",LRCW=8,LRFLAG="",LREDT=ORSSTRT(XQORNOD),LRSDT=ORSSTOP(XQORNOD),LRSPEC=ORLRSPEC(+XQORNOD) W:$E(IOST)="C" @IOF D OERR^LRDIST4 D:$G(ORSI)=$G(ORSCPAT) CLEAN^LRDIST4
