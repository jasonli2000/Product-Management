SDPP PATIENT PROFILE DISPLAY ENROLLMENTS (664)    PROTOCOL (101)

Name Value
NAME SDPP PATIENT PROFILE DISPLAY ENROLLMENTS
ITEM TEXT Enrollments
SYNONYM
  1. SYNONYM:   DE
DESCRIPTION 
This action will allow the user to display all enrollments or enrollments
for a date range.
 
It will also allow the user to select a specific clinic to display.

PACKAGE SCHEDULING
