IBDF EDIT FORM (750)    PROTOCOL (101)

Name Value
NAME IBDF EDIT FORM
ITEM TEXT Edit Form
SYNONYM
  1. SYNONYM:   EF
DESCRIPTION 
This protocol calls the list manager to display an encounter form. There
is a menu of actions that allows the form description to be edited.
TYPE action
CREATOR USER,ONE
PACKAGE AUTOMATED INFO COLLECTION SYS
EXIT ACTION I $G(IBFASTXT) S VALMBCK="Q"
ENTRY ACTION D EDITFORM^IBDF6
MENU PROMPT Edit Form:
