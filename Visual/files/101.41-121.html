OR GXMOVE DISCHARGE (121)    ORDER DIALOG (101.41)

Name Value
NAME OR GXMOVE DISCHARGE
DISPLAY TEXT Discharge Patient
TYPE dialog
DISPLAY GROUP M.A.S.
SIGNATURE REQUIRED ORES
PACKAGE ORDER ENTRY/RESULTS REPORTING
ITEMS
  1. SEQUENCE:   1
    ITEM:   OR GTX EVENT
    REQUIRED:   YES
    INDEX:   C
    HELP MESSAGE:   Enter the movement you wish to order for this patient.
    ASK ON CONDITION:   N X S X=$S($G(OREVT):0,1:1) I X ;uneditable if defined
    SCREEN:   I $S($P(^(0),U,2)="D":1,'$P(^(0),U,12):0,$P($G(^ORD(100.5,+$P(^(0),U,12),0)),U,2)="D":1,1:0)
    DEFAULT:   S:$G(OREVT) Y=+OREVT
    ORDER TEXT SEQUENCE:   1
    FORMAT:   8~8
  2. SEQUENCE:   2
    ITEM:   OR GTX MOVEMENT TYPE
    DISPLAY TEXT:   Discharge Type:
    ASK ON ACTION:   C
    HELP MESSAGE:   Enter the type of discharge for this patient.
    SCREEN:   I $P(^(0),U,2)=3,$P(^(0),U,4) ;Type=Discharge, Active
    ORDER TEXT SEQUENCE:   2
  3. SEQUENCE:   3
    ITEM:   OR GTX FREE TEXT 1
    DISPLAY TEXT:   Instructions:
    ASK ON ACTION:   C
    HELP MESSAGE:   Enter any special instructions for this order, up to 240 characters.
    ORDER TEXT SEQUENCE:   3
    LEADING TEXT:   Instructions:
DESCRIPTION 
This allows a patient to be discharged, as a generic order (does not
interact with MAS).
