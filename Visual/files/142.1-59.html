PROBLEM LIST ACTIVE (59)    HEALTH SUMMARY COMPONENT (142.1)

Name Value
NAME PROBLEM LIST ACTIVE
PRINT ROUTINE ACTIVE;GMTSPLST
EXTERNAL/EXTRACT ROUTINE
  1. EXTERNAL/EXTRACT ROUTINE:   GETLIST;GMPLHS(DFN,STA)
ABBREVIATION PLA
DESCRIPTION 
This component lists all known active problems for a patient. Information
displayed: ICD data (based on ICD Text Display parameter), provider
narrative (unless Provider Narrative Display parameter is set to NO), date
of onset, date last modified, the responsible provider, and all active
comments.
DEFAULT HEADER NAME Active Problems
ICD TEXT APPLICABLE yes
PROVIDER NARRATIVE APPLICABLE yes
PREFIX GMPL
