PRCB FCP USER COPY (10005)    OPTION (19)

Name Value
NAME PRCB FCP USER COPY
MENU TEXT Copy Control Point User and Profiles
UPPERCASE MENU TEXT COPY CONTROL POINT USER AND PR
DESCRIPTION 
This option will copy users and their profiles from an existing Fund Control
Point to a new one that has been created, but has no users assigned.
 
IMPORTANT NOTE: This option should only be used to populate control
point users for new control points in support of the FY04 Omnibus
Appropriations Law.  For the option to work properly, it is essential
that no individuals are currently assigned to the control point.  If
users exist in thedestination control point, nothing will be copied.
