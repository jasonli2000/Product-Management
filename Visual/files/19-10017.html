SR VIEW HISTORICAL REPORTS (10017)    OPTION (19)

Name Value
NAME SR VIEW HISTORICAL REPORTS
X ACTION PRESENT YES
MENU TEXT Make Reports Viewable in CPRS
UPPERCASE MENU TEXT MAKE REPORTS VIEWABLE IN CPRS
DESCRIPTION 
This option allows Operation Reports, Nurse Intraoperative Reports,
Anesthesia Reports and Procedure Reports (Non-O.R.) for historical cases
to be moved into TIU as "electronically unsigned" to make them viewable
within the CPRS Surgery tab.
CREATOR USER,ONE
TYPE run routine
PACKAGE SURGERY
E ACTION PRESENT YES
EXIT ACTION D EXIT^SROVAR
ENTRY ACTION D ^SROVAR
XQUIT EXECUTABLE Q
