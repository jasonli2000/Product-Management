XUSEC UP ARROW TERM REPORT (10051)    OPTION (19)

Name Value
NAME XUSEC UP ARROW TERM REPORT
MENU TEXT Up Arrow Delimited Termination Report
UPPERCASE MENU TEXT UP ARROW DELIMITED TERMINATION
DESCRIPTION 
The report produced by this option is a up arrow(^) delimited termination 
report that can be used is a spread sheet.  It is suggested that the 
report be sent to an HFS device.  If the screen is used, it will require
132 column width.
CREATOR USER,ONE
TYPE run routine
PACKAGE KERNEL
E ACTION PRESENT YES
ENTRY ACTION S XUSECU=""
ROUTINE EP1^XUSECAD
