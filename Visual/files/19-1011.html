Name Value
NAME LRAC DIAG
MENU TEXT Diagnostic routine for Lab Reports file (64.5)
UPPERCASE MENU TEXT DIAGNOSTIC ROUTINE FOR LAB REP
LOCK LRLIASON
DESCRIPTION 
This options checks the Lab Reports file for erroneous entries and
gives fatal and warning messages when problems are found.  Problems
reported by this option can be fixed by editing the appropriate fields
in the Lab Reports file or if necessary by the LRAC XREF option (LRAC
XREF re-cross-references selected indexes).
CREATOR POSTMASTER
TYPE run routine
PACKAGE LAB SERVICE
ROUTINE LRACDIAG
