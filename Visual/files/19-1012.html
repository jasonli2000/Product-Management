LRAC XREF (1012)    OPTION (19)

Name Value
NAME LRAC XREF
MENU TEXT Re-cross-reference indexes in LAB REPORTS file
UPPERCASE MENU TEXT RE-CROSS-REFERENCE INDEXES IN
LOCK LRLIASON
DESCRIPTION 
This option will allow the user to re-cross-reference selected
indexes from the Lab Reports file.
  1.  The "A" index contains the reference ranges, critical ranges,
      and units from file 60.  This index needs to be re-cross-
      referenced any time changes are made to these fields in file
      60, if those changes need to be reflected on the cumulative report.
  2.  The "AC" index contains the pointers to the Lab Data file (63).
      This index may need to be re-cross-referenced if it is found
      that the cross-reference contains entries for tests in the
      cumulative that don't have data names in file 63.
  3.  The "AR" index contains for each site/specimen the test on
      each minor header of the cumulative.  This index may have
      to be re-cross-referenced if it is found that a given
      test and site/specimen is located in more than one place
      on the report.
 
The option LRAC DIAG should be used to diagnose when these indexes
need to be re-cross-referenced.
