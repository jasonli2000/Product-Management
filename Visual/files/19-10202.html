XU SID STARTUP (10202)    OPTION (19)

Name Value
NAME XU SID STARTUP
MENU TEXT Startup PROD check
UPPERCASE MENU TEXT STARTUP PROD CHECK
DESCRIPTION 
This option should run at every startup to check if the current SID 
matches the stored SID.
To do this the option needs to be in the Option Scheduling file with the 
SPECIAL QUEUEING field set to Startup Persistent.
CREATOR USER,ONE
TYPE run routine
PACKAGE KERNEL
DELEGABLE NO
ROUTINE CHECK^XUPROD
