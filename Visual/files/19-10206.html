SD-PAIT REPAIR (10206)    OPTION (19)

Name Value
NAME SD-PAIT REPAIR
MENU TEXT SD-PAIT Last Run Repair
UPPERCASE MENU TEXT SD-PAIT LAST RUN REPAIR
DESCRIPTION 
This option will allow repair of a PAIT that was interrupted either by a system
problem or an error. The error cause has to be addressed before this repairing
process takes place and a new transmission starts.
CREATOR USER,ONE
TYPE run routine
E ACTION PRESENT YES
ENTRY ACTION S SDR=+$O(^SDWL(409.6,99999999),-1) I $P($G(^SDWL(409.6,SDR,0)),"^",7)?7N1".".N W !,"NO ENTRY TO REPAIR!",! S SDR="" H 1 Q
ROUTINE RPAIT^SDRPA04(SDR)
