ECX SUR VOL REPORT (10215)    OPTION (19)

Name Value
NAME ECX SUR VOL REPORT
X ACTION PRESENT YES
MENU TEXT SUR Volume Report
UPPERCASE MENU TEXT SUR VOLUME REPORT
DESCRIPTION 
This option creates a report that lists volumes for all surgical 
cases that would be generated by the Surgery extract (SUR) for
transmission to AAC for review.
