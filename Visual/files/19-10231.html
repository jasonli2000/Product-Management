DGFFP ALPHA LIST (10231)    OPTION (19)

Name Value
NAME DGFFP ALPHA LIST
MENU TEXT Fugitive Felon Alpha Report
UPPERCASE MENU TEXT FUGITIVE FELON ALPHA REPORT
DESCRIPTION 
This option generates an alpha list of patients with the Fugitive Felon
Flag currently set in the Patient File.  This report will display
the patients name, SSN Last 4, the date the flag was set, and who
entered the flag.
CREATOR USER,ONE
TYPE action
