NAME WV PAP SMEAR SNOMED CODES
MENU TEXT Link Pap Smear with SNOMED Codes
UPPERCASE MENU TEXT LINK PAP SMEAR WITH SNOMED COD
DESCRIPTION 
Use this option to link the Pap Smear procedure with SNOMED codes used in 
the Lab package to identify a lab test as a pap smear. When the results of
a lab test are verified, the SNOMED codes used in that test will be
compared to the SNOMED codes identified here to determine if the test was 
a pap smear.
