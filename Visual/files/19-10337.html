RMPR INV PRINT ALL BARCODE (10337)    OPTION (19)

Name Value
NAME RMPR INV PRINT ALL BARCODE
MENU TEXT Print All Barcode in a Location
UPPERCASE MENU TEXT PRINT ALL BARCODE IN A LOCATIO
LOCK RMPRMANAGER
DESCRIPTION 
This option prints barcode of all items in a specific location for a 
station.  In order to use this option, user must have an RMPRMANAGER 
key.  Be sure to have enough labels before using this option, since it 
will print labels for all items in a given station.
CREATOR USER,ONE
TYPE run routine
PACKAGE PROSTHETICS
ROUTINE PB^RMPRPIYZ
