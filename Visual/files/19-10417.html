XTPM COMPLETE A NON-KIDS PATCH (10417)    OPTION (19)

Name Value
NAME XTPM COMPLETE A NON-KIDS PATCH
MENU TEXT Mark a Non-Kids Patch as Complete
UPPERCASE MENU TEXT MARK A NON-KIDS PATCH AS COMPL
DESCRIPTION 
This option allows the completion of a non-kids patch, which is defined 
as not having any KIDs installs but rather a .zip or .exe or other type 
patch.  An example of an 'other' patch is a DBA patch.
CREATOR USER,ONE
TYPE edit
PACKAGE TOOLKIT
DIC {DIC} XPD(9.9,
DIC(0) AEMQ
DIC(S) I $P(^(0),U,10)=1
DIE XPD(9.9,
DR {DIE} [XTPM COMPLETE NON-KIDS PATCH]
