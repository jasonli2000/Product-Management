LRBLSF (1042)    OPTION (19)

Name Value
NAME LRBLSF
MENU TEXT Edit number of lines in a label
UPPERCASE MENU TEXT EDIT NUMBER OF LINES IN A LABE
OUT OF ORDER MESSAGE out-of-order
DESCRIPTION 
The number of lines from the top of one label to the top of the next
 label.  This is used as the default value when printing various labels.
