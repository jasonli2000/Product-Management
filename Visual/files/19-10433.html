PRCH NIF UPDATE REPORT (10433)    OPTION (19)

Name Value
NAME PRCH NIF UPDATE REPORT
MENU TEXT Item Descriptions After NIF Update
UPPERCASE MENU TEXT ITEM DESCRIPTIONS AFTER NIF UP
DESCRIPTION 
This option displays the pre-NIF and current values of the Short
Description (#.05) and the Description (#.1) fields, along with
the most recent Short Description edit date and the Inactive Flag.
The user has a choice of up to 5 sort keys to specify the sort orders
at run time.
CREATOR USER,ONE
TYPE run routine
PACKAGE IFCAP
ROUTINE PRT^PRCHITM2
