Name Value
NAME IBT QUICK REV CODING STAT
X ACTION PRESENT YES
MENU TEXT Quick Review Coding Status Report
UPPERCASE MENU TEXT QUICK REVIEW CODING STATUS REP
DESCRIPTION 
This report allows the MCCF staff to select a specific patient to see 
the status of the claim so they know at a glance whether or not it is 
ready for billing.
 
The report input variable is: Patient Name
Report is sorted by ENCOUNTER DATE/TIME
The following fields are to printed on this report:
1.      Patient Social Security Number
2.      Outpatient  Encounter Location
3.      Encounter Date and Time 
4.      Reason Not Billable
5.      Billable Findings Type
CREATOR USER,ONE
TYPE print
PACKAGE INTEGRATED BILLING
E ACTION PRESENT YES
EXIT ACTION K DFN
ENTRY ACTION N DIC S DIC="^DPT(",DIC(0)="AEMNQ" D ^DIC S:Y<1 XQUIT=1 S:Y>0 DFN=+Y
DIC {DIP} IBT(356,
L. 0
FLDS [IBT QUICK REV CODING STAT]
BY [IBT QUICK REV CODING STAT]
