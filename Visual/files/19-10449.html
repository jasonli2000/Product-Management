HLO MESSAGE STATISTICS (10449)    OPTION (19)

Name Value
NAME HLO MESSAGE STATISTICS
MENU TEXT HLO MESSAGE STATISTICS
UPPERCASE MENU TEXT HLO MESSAGE STATISTICS
DESCRIPTION 
This option is a report that displays by period the counts of messgages sent
and received via HLO.  The counts are sub-categorized by application and
message type.
CREATOR USER,SEVENTEEN
TYPE action
Short Menu Text HLO MESSAGE STATISTICS REPORT
DISPLAY OPTION? YES
PACKAGE HEALTH LEVEL SEVEN
E ACTION PRESENT YES
