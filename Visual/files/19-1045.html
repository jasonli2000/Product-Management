Name Value
NAME LRUMD
MENU TEXT Print/display preselected lab tests
UPPERCASE MENU TEXT PRINT/DISPLAY PRESELECTED LAB
DESCRIPTION 
 This option prints/displays preselected lab tests selected by the lab or
 or the user.  If no user selection the default is lab selection.
 Users can keep their own lists of patients to print which can be
 edited upon entering the option.
 This option is intended to be used by a physician who wants a quick
 list of preselected lab tests for his patients from one date to another
 and is not intended for chart copy.
