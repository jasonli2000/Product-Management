PRCEC PRINT PO FROM 1358 (10458)    OPTION (19)

Name Value
NAME PRCEC PRINT PO FROM 1358
MENU TEXT Print Obligated 1358s
UPPERCASE MENU TEXT PRINT OBLIGATED 1358S
DESCRIPTION 
This option allows the user to enter a date range and obtain a list of 
purchase orders from obligated 1358's with a dollar value of $0 and 
higher. If vendor and contract information was entered when the 1358's
were created, then the report will contain that information as well.
