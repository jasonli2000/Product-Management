ORWORB REMOVE NON-OR (10478)    OPTION (19)

Name Value
NAME ORWORB REMOVE NON-OR
MENU TEXT GUI Remove Button Enabled for Non-OR Alerts
UPPERCASE MENU TEXT GUI REMOVE BUTTON ENABLED FOR
DESCRIPTION 
This option allows you to set the flag to indicate if a non-OR alert can
be deleted without processing via the GUI alert Remove button. Enter the
Alert identifier (XQAID) or a portion of the Alert identifier for each
type of alert you want to be able to remove in the CPRS GUI via the Remove
button. For example, to remove NOIS alerts enter FSC.  You may enter as
many alert identifiers as desired.  Please note that most alert
identifiers include information specific to the patient or instance which
triggered the alert. For this parameter you should only enter the portion
of alert identifier which is consistent for all alerts of this type.  For
example, a NOIS alert's identifier might actually look like
FSC-M,275546.  Entering the entire alert identifier in this parameter will
only allow removal of this specific alert.  However if the parameter value
is FSC, all NOIS alerts can be removed.
