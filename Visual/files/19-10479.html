ORW ALLERGY EIE (10479)    OPTION (19)

Name Value
NAME ORW ALLERGY EIE
MENU TEXT GUI Mark Allergy Entered in Error
UPPERCASE MENU TEXT GUI MARK ALLERGY ENTERED IN ER
DESCRIPTION 
The OR ALLERGY ENTERED IN ERROR parameter controls who can mark
an existing allergy as entered in error from the cover sheet.  Set
the appropriate level to "yes" if you wish to allow users to mark
allergies as entered in error.  This parameter can be set at
the system, division, class, and user level.
CREATOR USER,ONE
TYPE action
E ACTION PRESENT YES
ENTRY ACTION D EDITPAR^XPAREDIT("OR ALLERGY ENTERED IN ERROR")
