MAG DICOM CORRECT VALIDATE (10514)    OPTION (19)

Name Value
NAME MAG DICOM CORRECT VALIDATE
MENU TEXT Validate DICOM Correct Information
UPPERCASE MENU TEXT VALIDATE DICOM CORRECT INFORMA
DESCRIPTION 
This menu option invokes the program that validates
(and, if necessary, corrects) the table that describes
all images that are waiting for corrections to be applied.
