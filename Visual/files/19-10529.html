DG INVALID STATE/COUNTY REPORT (10529)    OPTION (19)

Name Value
NAME DG INVALID STATE/COUNTY REPORT
MENU TEXT Invalid State/Inactive County Report
UPPERCASE MENU TEXT INVALID STATE/INACTIVE COUNTY
DESCRIPTION 
The report lists PATIENT file records whose address fields
contain invalid states and/or inactive counties.  The user
may run the report for 1) US and US Possessions Only,
2) Foreign Addresses Only, or 3) Both.
CREATOR USER,ONE
TYPE run routine
ROUTINE EN^DGRSTBAD
