XUS NPI SIGNON CHECK (10548)    OPTION (19)

Name Value
NAME XUS NPI SIGNON CHECK
MENU TEXT NPI Signon Check
UPPERCASE MENU TEXT NPI SIGNON CHECK
DESCRIPTION 
This option is used to check at a user's signon whether the user needs to 
enter an NPI value.  If so, a message to that effect is displayed to the 
user.
CREATOR USER,ONE
TYPE action
PACKAGE KERNEL
E ACTION PRESENT YES
ENTRY ACTION D SIGNON^XUSNPIED
