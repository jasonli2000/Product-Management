XUS NPI PROVIDER SELF ENTRY (10554)    OPTION (19)

Name Value
NAME XUS NPI PROVIDER SELF ENTRY
MENU TEXT PROVIDER NPI SELF ENTRY
UPPERCASE MENU TEXT PROVIDER NPI SELF ENTRY
DESCRIPTION 
This option provides the ability for a Provider to enter his/her own NPI 
value and effective date.  It is intended to be attached to the XU COMMON 
menu and checks for whether the user selecting it has the need to enter 
an NPI value.
CREATOR USER,ONE
TYPE run routine
PACKAGE KERNEL
ROUTINE USEREDIT^XUSNPIED
