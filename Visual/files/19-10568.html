BPS REOPEN CLOSED CLAIM (10568)    OPTION (19)

Name Value
NAME BPS REOPEN CLOSED CLAIM
MENU TEXT Re Open CLOSED Claim
UPPERCASE MENU TEXT RE OPEN CLOSED CLAIM
LOCK BPS MANAGER
DESCRIPTION 
This option allows the user to Re Open a claim that has been previously 
REJECTED and Closed using the ECME Close Claim option. Once a claim is Re 
Opened, the user may resubmit the claim to the payer for payment.
CREATOR USER,ONE
TYPE run routine
ROUTINE EN1^BPSREOP
