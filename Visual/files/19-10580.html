NAME SDWL NON-REMOVAL REASON RPT
MENU TEXT EWL Non-Removal Reason Report
UPPERCASE MENU TEXT EWL NON-REMOVAL REASON REPORT
DESCRIPTION 
This report lists patients who were on a Wait List for a particular
Clinic or Specialty and were not removed from the Wait List when an
appointment was made for that Clinic or Specialty.  The report displays
the patient's name and the last 4 of the patient's Social Security
Number along with the Originating Date, Non-Removal Date and Non-Removal
Reason.  The user's name and EWL record Status is also included.
