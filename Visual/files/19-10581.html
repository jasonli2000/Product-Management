SDWL WAIT TIME STATISTICS (10581)    OPTION (19)

Name Value
NAME SDWL WAIT TIME STATISTICS
MENU TEXT EWL Wait Time Statistics
UPPERCASE MENU TEXT EWL WAIT TIME STATISTICS
DESCRIPTION 
This report provides a number of wait time statistics for each patient
listed in the report.  Stats include the number of days elapsed from
the Originating Date to the Scheduled Date of Appointment, from the 
Originating Date to the Desired Date of Appointment, from the Scheduled
Date of Appointment to the Desired Date of Appointment, from the
Originating Date to the Date of Disposition, and from the Originating
Date to the actual Date the appointment was made.
