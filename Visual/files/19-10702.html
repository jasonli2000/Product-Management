QACI REPRINT ERRORS REPORT (10702)    OPTION (19)

Name Value
NAME QACI REPRINT ERRORS REPORT
MENU TEXT Reprint Migration/Pre-Migration Data Errors Report
UPPERCASE MENU TEXT REPRINT MIGRATION/PRE-MIGRATIO
DESCRIPTION 
Reprints the error reports showing data that cannot migrate due to errors.
The same error report is generated in the pre-migration phase, and when
the data is moved to the staging area.
CREATOR USER,ONE
TYPE run routine
ROUTINE ENREPRNT^QACI2E
