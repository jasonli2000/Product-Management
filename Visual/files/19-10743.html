SD WAIT LIST OPEN CLOSED ENTRY (10743)    OPTION (19)

Name Value
NAME SD WAIT LIST OPEN CLOSED ENTRY
X ACTION PRESENT YES
MENU TEXT Open Closed EWL Entry
UPPERCASE MENU TEXT OPEN CLOSED EWL ENTRY
DESCRIPTION 
This option will allow the user to manually open every type of closed
entry. The system shall enter the code 'O' - Other in the Reopen Entry
reason field, and the user must enter a required comment in the Reopen
Comment field.
CREATOR USER,ONE
TYPE run routine
E ACTION PRESENT YES
