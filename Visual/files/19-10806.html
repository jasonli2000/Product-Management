XU SID EDIT (10806)    OPTION (19)

Name Value
NAME XU SID EDIT
MENU TEXT Edit Logical/Physical Mapping
UPPERCASE MENU TEXT EDIT LOGICAL/PHYSICAL MAPPING
DESCRIPTION 
This option lets you edit the two fields that are used in the SID code to 
map a logical name in the Cache cpf file to the Physical name that is 
returned by a $ZU(12,"") call.
CREATOR USER,ONE
TYPE run routine
PACKAGE KERNEL
DELEGABLE YES
ROUTINE EDIT^XUPROD
