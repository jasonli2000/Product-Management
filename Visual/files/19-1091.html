Name Value
NAME LR ACC CONTROLS
X ACTION PRESENT YES
MENU TEXT Edit controls added to the accessions each day
UPPERCASE MENU TEXT EDIT CONTROLS ADDED TO THE ACC
LOCK LRSUPER
DESCRIPTION 
This option allows the user to specify what accessions are to have
what tests.  These are placed in the system at the beginning of each day.
Be aware that the accession will not be added if the number to be
assigned is already in use.
