LEX CSV ICP QUERY (10939)    OPTION (19)

Name Value
NAME LEX CSV ICP QUERY
X ACTION PRESENT YES
MENU TEXT ICD-9 Procedure Code Set Query
UPPERCASE MENU TEXT ICD-9 PROCEDURE CODE SET QUERY
DESCRIPTION 
This option displays a single versioned entry from the ICD 
Operations/Procedure file #80.1 based on a date provided 
by the user.  The date may be a future date.
CREATOR USER,SEVENTEEN
TYPE run routine
E ACTION PRESENT YES
EXIT ACTION K ^TMP("LEXQIP",$J),^TMP("LEXQIPA",$J),^TMP("LEXQIPO",$J)
ENTRY ACTION K ^TMP("LEXQIP",$J),^TMP("LEXQIPA",$J),^TMP("LEXQIPO",$J)
ROUTINE EN^LEXQIP
