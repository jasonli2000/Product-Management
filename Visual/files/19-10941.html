LEX CSV MOD QUERY (10941)    OPTION (19)

Name Value
NAME LEX CSV MOD QUERY
X ACTION PRESENT YES
MENU TEXT CPT Modifier Code Set Query
UPPERCASE MENU TEXT CPT MODIFIER CODE SET QUERY
DESCRIPTION 
This option displays a single versioned entry from the CPT
Modifier file #81.3 based on a date provided by the user. 
The date may be a future date.
CREATOR USER,SEVENTEEN
TYPE run routine
E ACTION PRESENT YES
EXIT ACTION K ^TMP("LEXQCM",$J),^TMP("LEXQCMO",$J)
ENTRY ACTION K ^TMP("LEXQCM",$J),^TMP("LEXQCMO",$J)
ROUTINE EN^LEXQCM
