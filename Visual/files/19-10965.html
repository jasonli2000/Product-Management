VBEC BB COMPONENTS DISABLE (10965)    OPTION (19)

Name Value
NAME VBEC BB COMPONENTS DISABLE
X ACTION PRESENT YES
MENU TEXT Disable VistA Blood Bank components
UPPERCASE MENU TEXT DISABLE VISTA BLOOD BANK COMPO
DESCRIPTION 
This option will allow the user to disable VistA Blood Bank options,
files, and fields. The program automatically disables all options, files,
and fields necessary so that users cannot enter or edit data in the VistA
Blood Bank application.  This option should be run just before starting
the data conversion.
CREATOR USER,SEVENTEEN
TYPE run routine
E ACTION PRESENT YES
EXIT ACTION K VBECECHO
ENTRY ACTION S VBECECHO=""
ROUTINE EN^VBECDC19(1)
