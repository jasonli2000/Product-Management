NAME PSA RET DRG MENU
MENU TEXT Return Drug Credit Menu
UPPERCASE MENU TEXT RETURN DRUG CREDIT MENU
DESCRIPTION 
This option provides sites with a means to improve internal control of
credits received for returned drugs. It will track drugs returned and
their status over time as they are picked up by the contracting company.
It will also provide summary and detailed reports of all return drugs.
