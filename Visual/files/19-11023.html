SDRR CONVERT ENTRIES (11023)    OPTION (19)

Name Value
NAME SDRR CONVERT ENTRIES
MENU TEXT Convert Retired Providers/Clinic Recalls
UPPERCASE MENU TEXT CONVERT RETIRED PROVIDERS/CLIN
LOCK SDRR MANAGER
DESCRIPTION 
This option will allow an approved user to convert Recall providers
and clinics.  If a provider leaves and there are Clinic Recalls already 
scheduled, you can convert these recalls to the new provider.
