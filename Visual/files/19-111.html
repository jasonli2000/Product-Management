XU FINDUSER (111)    OPTION (19)

Name Value
NAME XU FINDUSER
MENU TEXT Find a user
UPPERCASE MENU TEXT FIND A USER
DESCRIPTION 
This option will find a user that is currently signed on to the
system in this UCI group. If the user is on this CPU it will also
show the menu path.  Uses the "CUR" X-ref on the sign-on log file.
