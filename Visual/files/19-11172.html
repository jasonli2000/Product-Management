NAME PSO RX LIST
X ACTION PRESENT YES
MENU TEXT Prescription List for Drug Warnings
UPPERCASE MENU TEXT PRESCRIPTION LIST FOR DRUG WAR
DESCRIPTION 
This report provides the ability to identify all patients who have 
been dispensed a particular medication(s). 
             
Prescription List for Drug Warnings [PSO RX LIST] option uses a sort
template PSO DRUG WARNINGS to prompt for user input. Users are prompted
for inclusion or exclusion of deceased Patients, Divisions and a range of
Finish Dates. Print templates PSO DRUG WARNINGS and PSO DRUG WARNINGS
HEADER produce a report in a delimited output format suitable for import
to a spreadsheet. Report output will be delimited by a caret (^).
CREATOR USER,SEVENTEEN
TYPE print
PACKAGE OUTPATIENT PHARMACY
E ACTION PRESENT YES
EXIT ACTION K PSOSHOW,PSOEXC
ENTRY ACTION S PSODWFL=1 D MARGIN^PSORLST K DIR S DIR("A")="EXCLUDE DECEASED PATIENTS (Y/N) ",DIR(0)="SA^Y:YES;N:NO",DIR("B")="Y" D ^DIR Q:$D(DUOUT)!$D(DIRUT) S PSOEXC=$S(Y="Y":1,1:0)
DIC {DIP} PSRX(
L. 0
FLDS [PSO DRUG WARNINGS]
BY [PSO DRUG WARNINGS]
DIS(0) S PSOSHOW=$S('$G(PSOEXC):1,$G(PSOEXC)&$G(^DPT(+$P(^PSRX(D0,0),U,2),.35)):0,1:1) S PSODED=$$PSODED^PSORLST(D0) I PSOSHOW
DIOBEG D EXCMSG^PSORLST
