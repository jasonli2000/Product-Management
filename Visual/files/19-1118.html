LRDCOM (1118)    OPTION (19)

Name Value
NAME LRDCOM
MENU TEXT Audit of deleted/edited comments
UPPERCASE MENU TEXT AUDIT OF DELETED/EDITED COMMEN
LOCK LRLIASON
DESCRIPTION 
This option provides an audit of all comments that are deleted/edited from
"CH" subscripted results.  The report may be for a single patient or 
for all patients.  The time range of the report is entered by the user.  
The time range corresponds to the collection dates of specimens.
CREATOR POSTMASTER
TYPE run routine
PACKAGE LAB SERVICE
ROUTINE LRDCOM
