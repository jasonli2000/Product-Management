NAME XQSMD SET PREFIX
MENU TEXT Specify Allowable New Menu Prefix
UPPERCASE MENU TEXT SPECIFY ALLOWABLE NEW MENU PRE
DESCRIPTION 
Permits the user to give another user an Allowable New Menu Prefix for
purposes of Secure Menu Delegation, so that the Delegate may build new
menus from the options which have been delegated.
