SD CLINIC EDIT LOG (11216)    OPTION (19)

Name Value
NAME SD CLINIC EDIT LOG
MENU TEXT Clinic Edit Log Report
UPPERCASE MENU TEXT CLINIC EDIT LOG REPORT
DESCRIPTION 
Clinic Edit Log Report will contain a list of users that made changes to 
Clinic Locations for a specific time frame.
 
The following fields are currently being audited for the HOSPITAL 
LOCATION file (#44) via Set up a Clinic [SDBUILD], Inactivate a clinic 
[SD INACTIVATE], and Reactivate a Clinic [SD REACT] options.
 
-Name (#.01)
-Stop Code (#8)
-Non-Count (#2502)
-Credit Stop Code (#2503)
-Inactivate Date (#2505)
-Reactivate Date (#2506)
CREATOR USER,SEVENTEEN
TYPE run routine
PACKAGE SCHEDULING
ROUTINE EN^SD44AUDI
