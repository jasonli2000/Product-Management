FB ARCH REMINDER DELAY (11310)    OPTION (19)

Name Value
NAME FB ARCH REMINDER DELAY
MENU TEXT ARCH Clinical Reminder Due Delay
UPPERCASE MENU TEXT ARCH CLINICAL REMINDER DUE DEL
LOCK FB ARCH
DESCRIPTION 
This option will modify the PROJECT ARCH REMINDER DELAY field #38 in the
FEE BASIS SITE PARAMETERS file #161.4 for this site. The value represents 
the number of days that the Project ARCH Clinical Reminder will become 
due again after the patient has either declined or refused Project ARCH 
services. The default value is 1 but a site can edit the number of days 
up to 180.
