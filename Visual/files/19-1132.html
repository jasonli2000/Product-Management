LRBLSDPL (1132)    OPTION (19)

Name Value
NAME LRBLSDPL
MENU TEXT Delete a user's patient list
UPPERCASE MENU TEXT DELETE A USER'S PATIENT LIST
LOCK LRBLSUPER
DESCRIPTION 
Allows the blood bank supervisor to delete a user's patient
 list for the transfusion & hematology data option.  This
 is necessary when the last printing of the report for
 the user was never completed or the queued report was
 not deleted from the tasked list.
CREATOR POSTMASTER
TYPE run routine
PACKAGE LAB SERVICE
ROUTINE EN3^LRBLU
