XUSSPKI UPN SET (11470)    OPTION (19)

Name Value
NAME XUSSPKI UPN SET
MENU TEXT ePCS Set SAN from PIV Card
UPPERCASE MENU TEXT EPCS SET SAN FROM PIV CARD
DESCRIPTION 
This only needs to be run once for a user at a site.
It sets the Subject Alternative Name (SAN) field in file #200
from the PIV Smart Card.  This is used with the DEA e-sig to be
sure the right certificate is selected from the PIV card.
CREATOR USER,SEVENTEEN
TYPE Broker (Client/Server)
PACKAGE KERNEL
DELEGABLE NO
RPC
  1. RPC:   XUS PKI GET UPN
  2. RPC:   XUS PKI SET UPN
