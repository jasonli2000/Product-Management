Name Value
NAME XU EPCS EXP DATE
MENU TEXT Print DEA Expiration Date Null
UPPERCASE MENU TEXT PRINT DEA EXPIRATION DATE NULL
DESCRIPTION 
This option is for the DEA ePCS project and will print all active users with a 
DEA# and DEA Expiration Date is unpopulated.  This option prints the 
Name, DEA# and DEA Expiration Date.
CREATOR USER,SEVENTEEN
TYPE print
PACKAGE KERNEL
DIC {DIP} VA(200,
L. 0
FLDS [XU EPCS XDATE PRINT]
BY .01
