XU EPCS PSDRPH (11473)    OPTION (19)

Name Value
NAME XU EPCS PSDRPH
MENU TEXT Print PSDRPH Key Holders
UPPERCASE MENU TEXT PRINT PSDRPH KEY HOLDERS
DESCRIPTION 
This option is for the DEA ePCS project and will print all active users holding 
the PSDRPH key. This report will sort by division and within division it 
sorts by name.  This report will print name, duz, person who assigned key
and date assigned.
CREATOR USER,SEVENTEEN
TYPE print
PACKAGE KERNEL
DIC {DIP} VA(200,
L. 0
FLDS [XU EPCS PSDRPH PRINT]
BY [XU EPCS PSDRPH SORT]
DIS(0) I $$ACTIVE^XUSER(D0)
