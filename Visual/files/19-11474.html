XU EPCS SET PARMS (11474)    OPTION (19)

Name Value
NAME XU EPCS SET PARMS
MENU TEXT Print Setting Parameters Privileges
UPPERCASE MENU TEXT PRINT SETTING PARAMETERS PRIVI
DESCRIPTION 
This option is for the DEA ePCS project and will print all active users holding 
the XUEPCSEDIT key. This report will identify individuals responsible 
for setting the parameters.
CREATOR USER,SEVENTEEN
TYPE print
PACKAGE KERNEL
DIC {DIP} VA(200,
L. 0
FLDS [XU EPCS SET PARMS PRINT]
BY [XU EPCS SET PARMS SORT]
DIS(0) I $$ACTIVE^XUSER(D0)
