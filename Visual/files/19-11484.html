XU EPCS DISUSER PRIVS (11484)    OPTION (19)

Name Value
NAME XU EPCS DISUSER PRIVS
MENU TEXT Print DISUSER Prescribers with Privileges
UPPERCASE MENU TEXT PRINT DISUSER PRESCRIBERS WITH
DESCRIPTION 
This option is for the DEA ePCS project and will print all DISUSER users who
have privileges to any of the SCHEDULEs II through V and who have a DEA# or 
VA#.  This option prints the NAME, SCHEDULESs, VA#, DUZ, TERMINATION DATE
and DEA#.
CREATOR USER,SEVENTEEN
TYPE print
PACKAGE KERNEL
DIC {DIP} VA(200,
L. 0
FLDS [XU EPCS DISUSER PRIVS PRINT]
BY [XU EPCS DISUSER2 SORT]
DIS(0) I $$VUSER2^XUEPCSRT(D0,0)
