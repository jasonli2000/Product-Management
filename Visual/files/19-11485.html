XU EPCS DISUSER XDATE EXPIRES (11485)    OPTION (19)

Name Value
NAME XU EPCS DISUSER XDATE EXPIRES
MENU TEXT Print DISUSER DEA Expiration Date Expires 30 days
UPPERCASE MENU TEXT PRINT DISUSER DEA EXPIRATION D
DESCRIPTION 
This option is for the DEA ePCS project and will print all DISUSER users with a 
DEA# and where the DEA Expiration Date will expire within 30 days.  This 
option prints the Name, DEA# and DEA Expiration Date.
CREATOR USER,SEVENTEEN
TYPE print
PACKAGE KERNEL
DIC {DIP} VA(200,
L. 0
FLDS [XU EPCS DISUSER XDATE2 PRINT]
BY [XU EPCS DISUSER SORT]
DIS(0) I $$XT30^XUEPCSRT(D0,0)
