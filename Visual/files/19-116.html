XUXREF-2 (116)    OPTION (19)

Name Value
NAME XUXREF-2
MENU TEXT Show Users with a Selected primary Menu
UPPERCASE MENU TEXT SHOW USERS WITH A SELECTED PRI
DESCRIPTION 
This option generates a listing of those users who have a selected
option as their primary menus option or as a secondary menu option.
It does NOT show all users who might have access to a particular
option.  It only looks at Primary (sign-on) menus and top-level
secondary menu options.
CREATOR USER,ONE
TYPE run routine
ROUTINE XQ32
