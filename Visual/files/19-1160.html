Name Value
NAME LRAPLBL
MENU TEXT Anatomic pathology labels
UPPERCASE MENU TEXT ANATOMIC PATHOLOGY LABELS
DESCRIPTION 
Allows labels to be printed for surgical path, cytopath and electron
microscopy from one number to another within a year.  If there is more
than one specimen for an accession, a separate label will print for
each specimen.
