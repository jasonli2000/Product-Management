NAME LRAPDES
MENU TEXT Enter/edit lab description file
UPPERCASE MENU TEXT ENTER/EDIT LAB DESCRIPTION FIL
DESCRIPTION 
Specimen description for an entry in this file will be
 entered as the gross description for an accession if
 the SPECIMEN name entered at log-in time exactly matches
 the LAB DESCRIPTION NAME.
 THe specimen description for an entry in this file will be
 entered as the microscopic description for an accession if a
 '*' preceeds the exact match in the LAB DESCRIPTION FILE when
 an entry is made at the microscopic description prompt.
