LRAC FORCE (1194)    OPTION (19)

Name Value
NAME LRAC FORCE
MENU TEXT Force cumulative data to Permanent Page
UPPERCASE MENU TEXT FORCE CUMULATIVE DATA TO PERMA
LOCK LRSUPER
DESCRIPTION 
This option will identify patients who have been inactive in laboratory 
for the specified period of time in the GRACE PERIOD FOR INACTIVITY field
of the Laboratory Site File and force their lab data onto a permanent
cumulative page, making the data eligible for archiving.
CREATOR POSTMASTER
TYPE run routine
PACKAGE LAB SERVICE
ROUTINE QUEUE^LRACF
