LR INTEGRITY SINGLE (1201)    OPTION (19)

Name Value
NAME LR INTEGRITY SINGLE
MENU TEXT Check a single routine size
UPPERCASE MENU TEXT CHECK A SINGLE ROUTINE SIZE
DESCRIPTION 
  This option is used to check a routine against data stored in the
LR ROUTINE INTEGRITY CHECKER file (69.91). After the file has been loaded
using LR INTEGRITY LOAD option, use of this option will check for integrity
of that routine. The size and ASCII value of the selected routine will be
compared to the date in the file. A message will then display the value(s)
of the selected routine and indicate if the routine has changed.
   The routine are stored by version numbers.
CREATOR POSTMASTER
TYPE run routine
PACKAGE LAB SERVICE
ROUTINE SING^LRNITEG
