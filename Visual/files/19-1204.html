NAME LROPTLST
MENU TEXT Listing of Laboratory Menus/Options
UPPERCASE MENU TEXT LISTING OF LABORATORY MENUS/OP
DESCRIPTION 
This option is used to provide a listing of options for a given Menu.
The option will print by menu all of the options for that menu, including
any sub menus. This listing can be used as a checklist of options tested.
The listing will provide the the following:
  Menu Text                 Option Name (Program name if any)
       The description of the option from the option file #19
 
         ---------------------------------------------
 
Generally LRMENU option is entered; however any menu can be used.
This option is a simplified version of diagram menus.
Very helpful when checking off options when testing a new release, or
when comparing local menus to distributed versions.
