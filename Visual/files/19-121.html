XQ XUTL $J NODES (121)    OPTION (19)

Name Value
NAME XQ XUTL $J NODES
MENU TEXT Clean old Job Nodes in XUTL
UPPERCASE MENU TEXT CLEAN OLD JOB NODES IN XUTL
DESCRIPTION 
This option will cause old job related nodes which may remain in the
XUTL("XQ",$J, and UTILITY($J, globals to be removed.  An old job node is one
which was started two days prior to the current day, irrespective of the time
of day.
 
This option is provided for those systems in which the job numbers are
normally not reused, in which case leaving the system in an abnormal
manner (mainly halting from programmer's mode) may leave an accumulation
of these nodes.
 
The "CUR" X-ref of the sign-on log can get left defined in a similar manner
and this routine will clean this up also.
CREATOR USER,ONE
TYPE run routine
