LROW IMMED COLLECT (1258)    OPTION (19)

Name Value
NAME LROW IMMED COLLECT
MENU TEXT Fast lab test order (IMMEDIATE COLLECT)
UPPERCASE MENU TEXT FAST LAB TEST ORDER (IMMEDIATE
DESCRIPTION 
 This option is used by the wards to request immediate collection
of a test specimen by the laboratory. This option is similar to the
'Fast lab test order (ROUTINE)' except that these orders are not for
scheduled laboratory collection times. These orders are placed for
irregular collection times.
  The laboratory has set certain time frames and days of the week which
it will allow the user to select from. If the requested times are
not allowed, the user must make a PHONE CALL TO THE LAB TO ARRANGE
SPECIMEN COLLECTION.
 
NOTE: IT IS RECOMMENDED THAT THE ORDERING PERSON ALWAYS CALL THE
LABORATORY TO CONFIRM THE RECEIPT OF THE PLACED ORDER.
