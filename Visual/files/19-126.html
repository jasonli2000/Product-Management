XU OPTION QUEUE (126)    OPTION (19)

Name Value
NAME XU OPTION QUEUE
MENU TEXT One-time Option Queue
UPPERCASE MENU TEXT ONE-TIME OPTION QUEUE
DESCRIPTION 
This option allows any option that is in the Option file with the
SCHEDULING RECOMMENDED field set to yes, to be set up for one-time
queuing.
CREATOR USER,ONE
TYPE run routine
ROUTINE XUTMOPT
