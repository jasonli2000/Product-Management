LR WKLD STATS ON ACC AREA (1282)    OPTION (19)

Name Value
NAME LR WKLD STATS ON ACC AREA
MENU TEXT Turn on workload stats for accession area
UPPERCASE MENU TEXT TURN ON WORKLOAD STATS FOR ACC
LOCK LRLIASON
DESCRIPTION 
  This option turns on workload data collection for a specific
accession area. If you wish to begin collection workload data for
this accession area answer YES.
(field WORKLOAD ON, #10, in the ACCESSION file, #68)
   NOTE:  YOU MUST ALSO HAVE (FIELD #17) WKLD STATS ON IN THE 
LABORATORY SITE FILE (#69.9) ANSWERED YES.   BOTH FIELDS MUST
BE ANSWER 'YES' FOR DATA COLLECTION TO BE ACTIVATED.
