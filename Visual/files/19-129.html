XUSER-CLEAR-ALL (129)    OPTION (19)

Name Value
NAME XUSER-CLEAR-ALL
MENU TEXT Clear all users at startup
UPPERCASE MENU TEXT CLEAR ALL USERS AT STARTUP
DESCRIPTION 
This option is to be setup in the SPECIAL QUEUING to run when ever
Tankman starts up. It will go thru and clear all users signed-on
from the multiple sign-on restriction.
CREATOR USER,ONE
TYPE run routine
ROUTINE XUS6
INDEPENDENTLY INVOCABLE NO
