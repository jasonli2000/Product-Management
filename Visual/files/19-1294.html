NAME LR WKLD LMIP 1
MENU TEXT PHASE 1: Move data from 64.1 to 67.9.
UPPERCASE MENU TEXT PHASE 1: MOVE DATA FROM 64.1 T
LOCK LRSUPER
DESCRIPTION 
  This option performs the first step in producing your monthly report
for LMIP and management. This option's function is to extract from your
workload file (#64.1) ^LRO(64.1, which contains your daily workload, and 
roll up the data in a condensed form. The condensed data is stored in the 
LAB MONTHLY WORKLOADS file (#67.9)  ^LRO(67.9 ).
 This option should be used two days after the last day of the month
you wish to collect data for. Refer to your package documentation for more 
information.
