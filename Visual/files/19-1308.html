LRCAPF (1308)    OPTION (19)

Name Value
NAME LRCAPF
MENU TEXT Etiology WKLD Codes (Force)
UPPERCASE MENU TEXT ETIOLOGY WKLD CODES (FORCE)
LOCK LRSUPER
DESCRIPTION 
 This option is used to automatically add WKLD codes into the
Etiology File (#61.2). A sort template must be created using the
standard Fileman option. This option will utilize the selected
template and then add WKLD codes.
 This feature can be used to speed up implementation of WLDK module.
It also can be used to change existing/added WKLD codes.
CREATOR POSTMASTER
TYPE run routine
PACKAGE LAB SERVICE
ROUTINE LRCAPF
