LR ROLLOVER (1310)    OPTION (19)

Name Value
NAME LR ROLLOVER
MENU TEXT Rollover Accession (Manual)
UPPERCASE MENU TEXT ROLLOVER ACCESSION (MANUAL)
DESCRIPTION 
 This option should be used if the task option LRTASK ROLLOVER does
not run for some reason. This option should not be the normal method
for transferring incomplete accession from the previous day to the
present day's accession file.
  If if becomes necessary to use this option, there has probably been
a problem with TASK MANAGER. IRM Service should be notified if this 
option is required to move accession onto day's list.
 
 NOTE: This option can be ran multiple times with out harm. It checks
to see if the DATE ROLLOVER LAST RUN field is set correctly before 
actually preforming the rollover function.
CREATOR POSTMASTER
TYPE run routine
PACKAGE LAB SERVICE
E ACTION PRESENT YES
ENTRY ACTION W !!,"NOTIFY IRM SERVICE OF A POTENTIAL TASK MAN PROBLEM",!!,$C(7)
ROUTINE LROLOVER
