DIINQUIRE (1332)    OPTION (19)

Name Value
NAME DIINQUIRE
MENU TEXT Inquire to File Entries
UPPERCASE MENU TEXT INQUIRE TO FILE ENTRIES
DESCRIPTION 
This option is used to display all the data for a group of specified
entries in a file.  This is useful for a quick look at a small number
of entries.  Use the Print File Entries option for larger numbers
of entries.
CREATOR USER,ONE
TYPE action
DELEGABLE NO
E ACTION PRESENT YES
ENTRY ACTION D INQ^DII
