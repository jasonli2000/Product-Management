NAME XQSMD LIMITED FM OPTIONS
MENU TEXT Limited File Manager Options (Build)

LOCK XQSMDFM
DESCRIPTION 
This option permits a user to build limited PRINT, EDIT, or INQUIRE
options.  The user must have one or more ALLOWABLE MENU PREFIX, and
the XQSMDFM key.  To build an edit option the user must have an
edit template to use.  To build a print option, the user must
have both a sort template and a print template.  For an inquire
option the user may have a print template, or can specify the file to use.
The option is entered among the user's delegated options, so that they
may be included on a menu built under the Secure Menu Delegation and/or
given to other users as secondary options.  The name of the option is
restricted to the namespace designated in the ALLOWABLE MENU PREFIX
followed by a 'Z', and then identifying text.
