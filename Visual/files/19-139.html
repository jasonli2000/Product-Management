XQSMD COPY USER (139)    OPTION (19)

Name Value
NAME XQSMD COPY USER
MENU TEXT Copy One Users Menus and Keys to others
UPPERCASE MENU TEXT COPY ONE USERS MENUS AND KEYS
DESCRIPTION 
This option is used to copy the primary menu and secondary menu options
from one user to one or more others.  The only options transferred will
be those which have been delegated to the current user.  Likewise, keys
which are held by the selected user may be copied to the receipient
users if the current user has been delegated this capacity as well.
 
This option provides an application coordinator the ability to produce
users which have the capabilities of a current user, e.g., a ward
clerk, scheduling clerk, etc.
CREATOR USER,ONE
