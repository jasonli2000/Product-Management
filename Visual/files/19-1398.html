LR WKLD CODE EDIT PRINT (1398)    OPTION (19)

Name Value
NAME LR WKLD CODE EDIT PRINT
MENU TEXT Edit or Print WKLD CODES
UPPERCASE MENU TEXT EDIT OR PRINT WKLD CODES
DESCRIPTION 
 This option will allow the user to edit or print activated workload
codes. A code automatically becomes activated the first time it is used
by the system to capture workload.
 
 The option allows the user to select a WKLD CODE LAB SECTION to limit
the list of WKLD CODES for printing or editing.
 
If the user selects the edit function, a selection menu of possible fields
is displayed. Multiple fields maybe selected but one at a time. The edit
selection does not allow all as a valid entry.
  During the edit process, the user may select a starting WKLD CODE name
or enter the beginning characters. The option will find the next matching
code in the file. Names are stored in mixed character format. Therefore
to select a specific code to begin editing, upper and lower characters
must be entered.
  Before the user is allowed to edit the requested fields, a display of
current values are presented. The user is prompted with fields to edit
in the order they were originally entered.
 
The user is able to jump to any field selected.
 
The user is able to print the list to home screen. If another device is
selected for printing, the report will be queued to that device.
