XQAB LIST LOW USAGE OPTS (142)    OPTION (19)

Name Value
NAME XQAB LIST LOW USAGE OPTS
MENU TEXT Low Usage Alpha/Beta Test Options
UPPERCASE MENU TEXT LOW USAGE ALPHA/BETA TEST OPTI
DESCRIPTION 
 
This option is available for obtaining a listing of options which are in
a package which is under alpha or beta testing and which have low levels
of utilization since the last installation of the package.  An option
with low utilization is any option in the package namespace(s) with zero to
five accesses.
CREATOR USER,ONE
TYPE run routine
ROUTINE LOW^XQABLIST
