XQAB ACTUAL OPTION USAGE (143)    OPTION (19)

Name Value
NAME XQAB ACTUAL OPTION USAGE
MENU TEXT Actual Usage of Alpha/Beta Test Options
UPPERCASE MENU TEXT ACTUAL USAGE OF ALPHA/BETA TES
DESCRIPTION 
 
This option is available for listing actual usage of options within a package
which is in alpha or beta testing.  This will list only those options which
have been accessed one or more times since the last installation of the
package.
CREATOR USER,ONE
TYPE run routine
ROUTINE ACTUAL^XQABLIST
