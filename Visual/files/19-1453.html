ENWO-STATS-EMP (1453)    OPTION (19)

Name Value
NAME ENWO-STATS-EMP
MENU TEXT Incomplete Work Orders by Employee
UPPERCASE MENU TEXT INCOMPLETE WORK ORDERS BY EMPL
DESCRIPTION 
List of incomplete work requests by assigned technician.  User is
prompted to select the ENGINEERING EMPLOYEE of interest, from among
those who belong to the chosen shop.  If you simply press <RETURN>
instead of selecting a technician, the system will then allow you to
enter the word 'NOT' and thereby produce a list of incomplete
work orders that are not assigned to anyone.
CREATOR USER,ONE
TYPE run routine
ROUTINE E^ENWOST
