X ACTION PRESENT YES
MENU TEXT Request Electronic Work Order
UPPERCASE MENU TEXT REQUEST ELECTRONIC WORK ORDER
DESCRIPTION 
Entry of Engineering work requests by non-Engineering personnel. Intent
is to reduce volume of written work orders and to reduce telephone calls
to Engineering Work Control desk. Assumes that AEMS/MERS has been
integrated with CORE systems so that all DHCP users have access to the
Engineering Package.
