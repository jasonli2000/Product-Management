ENWOEDIT-WARD (1507)    OPTION (19)

Name Value
NAME ENWOEDIT-WARD
MENU TEXT Edit Electronic Work Order
UPPERCASE MENU TEXT EDIT ELECTRONIC WORK ORDER
DESCRIPTION 
Permits non-Engineering users to edit work requests which were input via
the 'Electronic Work Order Request' option. Once work orders have been
transferred by Engineering Service to a working shop they may no longer
be edited via this option, but users may check their status.
CREATOR POSTMASTER
TYPE run routine
ROUTINE WRDEDT^ENWARD
