NAME ENBCEE RM
MENU TEXT Bar Code Labels by Specific Location (ROOM)
UPPERCASE MENU TEXT BAR CODE LABELS BY SPECIFIC LO
DESCRIPTION 
Prints bar code labels for each piece of equipment in a specified
room. Sites must be using the Space File (#6928) in order to profit
from using this option.
