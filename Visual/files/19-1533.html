NAME ENBCPM MGR
MENU TEXT Bar Coded PMI Functions
UPPERCASE MENU TEXT BAR CODED PMI FUNCTIONS
DESCRIPTION 
Driver for bar coded Preventive Maintenance functions. Will prompt you
to either (1) download a data acquisition program to a bar code reader,
(2) upload collected data from a bar code reader, or (3) restart
processing of a previously uploaded data set.
