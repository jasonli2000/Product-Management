XQAB AUTO SEND (156)    OPTION (19)

Name Value
NAME XQAB AUTO SEND
MENU TEXT Send Alpha/Beta Usage to Developers
UPPERCASE MENU TEXT SEND ALPHA/BETA USAGE TO DEVEL
DESCRIPTION 
This option is set up to automatically send to the developing ISC's
information on the usage of options in those packages currently
in test status.
 
If it is selected manually, it will send a similar message(s) as well.
CREATOR USER,ONE
TYPE run routine
ROUTINE DOMAIL^XQABLIST
