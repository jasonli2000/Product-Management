XUFILESETDELETE (160)    OPTION (19)

Name Value
NAME XUFILESETDELETE
MENU TEXT Delete Users' Access to a Set of Files
UPPERCASE MENU TEXT DELETE USERS' ACCESS TO A SET
OUT OF ORDER MESSAGE ACCESS DISABLED
DESCRIPTION 
This option deletes the access to files held by one or more users.
You may enter an individual file number, a range of numbers, and/or
a list of file numbers.
CREATOR USER,ONE
TYPE run routine
DELEGABLE NO
ROUTINE XUDEL^XUFILE
