ENEQINV7 (1602)    OPTION (19)

Name Value
NAME ENEQINV7
X ACTION PRESENT YES
MENU TEXT Non-EIL Inventory
UPPERCASE MENU TEXT NON-EIL INVENTORY
DESCRIPTION 
This option generates the Non-EIL Inventory report. Equipment that belongs
to the user-specified CMR and is not required to be accountable by VHA
Directive will be printed on this report.
 
See the CMR Inventory (EIL) report to print accountable equipment that
belongs to a CMR.

TYPE run routine
PACKAGE ENGINEERING
E ACTION PRESENT YES
EXIT ACTION K ENNXEXP
ENTRY ACTION S ENNXEXP=1
