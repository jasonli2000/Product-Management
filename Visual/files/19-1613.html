DESCRIPTION 
This reprot will assist sites to prepare for installation of patch
EN*7*25. Patch EN*7*25 will use the TYPE ENTRY, CAPITALIZED, CMR, and
CATEGORY STOCK NUMBER fields to automatically populate new data fields.
This reprot lists equipment which may be of interest to FIxed Assets so
sites can ensure that the data which will be used in the patch EN*7*25
conversion is appropriate.
 
Equipment meeting the following criteria will be listed on the report:
 TYPE OF ENTRY = "NX"
 ACQUISITION VALUE greater than 4999.99
 DISPOSITION DATE = ""
CREATOR POSTMASTER
TYPE print
PACKAGE ENGINEERING
DIC {DIP} ENG(6914,
L. 0
FLDS [ENXKFAP]
