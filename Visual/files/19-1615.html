ENFA VALCHK SINGLE (1615)    OPTION (19)

Name Value
NAME ENFA VALCHK SINGLE
MENU TEXT FAP Validity Check For A Single Item (FA Document)
UPPERCASE MENU TEXT FAP VALIDITY CHECK FOR A SINGL
DESCRIPTION 
This option will perform a validity check on an entry in the Equipment
file to determine if a FA Document (code sheet) can be created for that
entry.
CREATOR POSTMASTER
TYPE action
PACKAGE ENGINEERING
E ACTION PRESENT YES
ENTRY ACTION S ENBAT("XMT")=0 D EQUIP^ENFAXMTM K ENBAT("XMT")
ROUTINE SINGLE^ENFAXMTM
