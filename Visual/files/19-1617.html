NAME ENFAR2
MENU TEXT Summary of Capitalized NX Equipment Assets
UPPERCASE MENU TEXT SUMMARY OF CAPITALIZED NX EQUI
DESCRIPTION 
This option generates a summary report of all capitalized NX equipment
assets in the Engineering Equipment Inv. file. Equipment items with a
disposition date are excluded from this report. The output will contain
the item count and asset value by station, fund, and standard general
ledger. This report may take some time to run.

TYPE run routine
