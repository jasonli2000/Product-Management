NAME ENFAR4
MENU TEXT Capitalized Equipment List for Station
UPPERCASE MENU TEXT CAPITALIZED EQUIPMENT LIST FOR
DESCRIPTION 
This option generates a list of capitalized equipment sorted by category
stock number and CMR for a specified station. Equipment must be on a CMR
in order to be included in this report. This report can be used to
reconcile the equipment file with the LOG I corporate database. The report
is similar to the Nonexpendable Property Locator Listing (Report 051) from
LOG I.
