XUFILESINGLEADD (165)    OPTION (19)

Name Value
NAME XUFILESINGLEADD
X ACTION PRESENT YES
MENU TEXT Single file add/delete for a user
UPPERCASE MENU TEXT SINGLE FILE ADD/DELETE FOR A U
OUT OF ORDER MESSAGE ACCESS DISABLED
DESCRIPTION 
This option is used to add or delete the access a user has for
a single file.
CREATOR USER,SEVENTEEN
TYPE edit
EXIT ACTION K V,W,C,DI,DISYS,DQ,%X,%Y,DLAYGO
DIC {DIC} VA(200,
DIC(0) AEMQ
DIC(S) I $P($$ACTIVE^XUSER(+Y),"^",2)'="",$P($$ACTIVE^XUSER(+Y),"^",2)'="TERMINATED"
DIE VA(200,
DR {DIE} [XUFILESINGLE]
