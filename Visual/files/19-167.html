XUFILEDELETE (167)    OPTION (19)

Name Value
NAME XUFILEDELETE
MENU TEXT Take away All access to a File
UPPERCASE MENU TEXT TAKE AWAY ALL ACCESS TO A FILE
OUT OF ORDER MESSAGE ACCESS DISABLED
DESCRIPTION 
This option deletes the access all users hold to a particular file.
This does not include those users with programmer access, i.e., DUZ(0)="@".
CREATOR USER,ONE
TYPE run routine
ROUTINE DELF^XUFILE1
