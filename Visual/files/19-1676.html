NAME ENFAR7
MENU TEXT Excessed Equipment Report
UPPERCASE MENU TEXT EXCESSED EQUIPMENT REPORT
DESCRIPTION 
This option generates a report of FA Documents (code sheets) which were
transmitted to FAP for standard general ledger 1524 (excess). This list is
intended to identify equipment assets which should have been picked up in
the FMS excess account by Fiscal Service.
 
It is recommended that this option be scheduled to automatically print on
a monthly basis (via TaskManager). If automatically queued, the report
will include FA Documents transmitted within the last 30 days or so.
CREATOR USER,ONE
TYPE run routine
ROUTINE ENFAR7
