ENFA VALCHK BY STATION (1679)    OPTION (19)

Name Value
NAME ENFA VALCHK BY STATION
MENU TEXT FAP Validity Check by Station (FA Documents)
UPPERCASE MENU TEXT FAP VALIDITY CHECK BY STATION
DESCRIPTION 
This option will perform validity checks on equipment belonging to a
station to determine if FA Documents (code sheets) can be created for
those equipment entries. Only capitalized (CAPITALIZED field = YES)
equipment which is on a CMR (CMR field not blank) will be considered.
Equipment which is already established in Fixed Assets will not be
checked.
CREATOR POSTMASTER
TYPE action
PACKAGE ENGINEERING
E ACTION PRESENT YES
ENTRY ACTION S ENBAT("XMT")=0 D STN^ENFAXMTM K ENBAT("XMT")
