XUFILERANGEASSIGN (168)    OPTION (19)

Name Value
NAME XUFILERANGEASSIGN
X ACTION PRESENT YES
MENU TEXT Assign/Delete a File Range
UPPERCASE MENU TEXT ASSIGN/DELETE A FILE RANGE
OUT OF ORDER MESSAGE ACCESS DISABLED
DESCRIPTION 
This option allows the user to assign to user or delete from a user
a file range.  This range is used when creating a new file.   If the
range is present, then the user can only create new files whose numbers
are within the range.
CREATOR USER,ONE
TYPE edit
EXIT ACTION K %X,%Y,DI,DISYS,DQ,V,W
DIC {DIC} VA(200,
DIC(0) AEMQ
DIE VA(200,
DR {DIE} [XUFILERANGE]
