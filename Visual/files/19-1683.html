HL EDIT SITE PARAM (1683)    OPTION (19)

Name Value
NAME HL EDIT SITE PARAM
MENU TEXT Non-DHCP Application Parameter Enter/Edit
UPPERCASE MENU TEXT NON-DHCP APPLICATION PARAMETER
DESCRIPTION 
This option is used to enter/edit parameters associated with other (non-DHCP
or DHCP) applications with which the DHCP system will communicate using
the HL7 protocol.
CREATOR USER,ONE
TYPE run routine
