HL PRINT PENDING TRANS (1696)    OPTION (19)

Name Value
NAME HL PRINT PENDING TRANS
MENU TEXT Awaiting/Pending Transmission Report (non-TCP)
UPPERCASE MENU TEXT AWAITING/PENDING TRANSMISSION
DESCRIPTION 
This option will print/display a list of entries from the HL7 Transmission
file that have a status of 'Pending Transmission' or 'Awaiting
Acknowledgement'.  The user is prompted for the date range for the report.
CREATOR USER,ONE
TYPE run routine
PACKAGE HEALTH LEVEL SEVEN
ROUTINE PLPT^HLUOPT
