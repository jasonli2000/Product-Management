XULIST (17)    OPTION (19)

Name Value
NAME XULIST
MENU TEXT List Terminal Types
UPPERCASE MENU TEXT LIST TERMINAL TYPES
DESCRIPTION 
This option is used to print a list of the various terminal types known
to the system.
CREATOR USER,ONE
TYPE print
DIC {DIP} %ZIS(2,
L. 0
FLDS .01,1,3,2,4,18,19
BY .01
DHD I/O TERMINAL TYPES
