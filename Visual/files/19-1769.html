ORLP ADD CLINIC (1769)    OPTION (19)

Name Value
NAME ORLP ADD CLINIC
MENU TEXT Clinic (Add to list)
UPPERCASE MENU TEXT CLINIC (ADD TO LIST)
DESCRIPTION 
This option lets you add patients to a clinic list that is
already defined, or build a new patient list by clinic.
After you have selected a clinic, you can add or delete patients
to further customize your list.
