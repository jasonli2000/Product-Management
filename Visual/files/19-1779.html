ORLP DELETE PATIENT (1779)    OPTION (19)

Name Value
NAME ORLP DELETE PATIENT
MENU TEXT Remove Patient(s) from Patient List
UPPERCASE MENU TEXT REMOVE PATIENT(S) FROM PATIENT
DESCRIPTION 
You can remove patients from a list that you have built by patient, ward
or clinic. 
This is convenient if you want to add all of the patients from a
ward or clinic except for one or two.
CREATOR USER,ONE
TYPE action
E ACTION PRESENT YES
ENTRY ACTION D DEL^ORLP00
