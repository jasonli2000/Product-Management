NAME ORLP TEAM MENU

MENU TEXT Team List Mgmt Menu
UPPERCASE MENU TEXT TEAM LIST MGMT MENU
DESCRIPTION 
Options on this menu allow you to create team patient lists.  You can
create or add autolinks to a team list, review existing team lists, remove
patients or users from team lists, delete autolinks from an existing team
list or delete an entire team list.
