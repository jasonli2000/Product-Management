XUERTRP PRINT ERRS (186)    OPTION (19)

Name Value
NAME XUERTRP PRINT ERRS
MENU TEXT Interactive Print of Error Messages
UPPERCASE MENU TEXT INTERACTIVE PRINT OF ERROR MES
DESCRIPTION 
This option provides for an interactive print of the first N occurrences
of an error (where N is user selectable) over the specified date range.

TYPE run routine
