DG BED AVAILABILITY (1894)    OPTION (19)

Name Value
NAME DG BED AVAILABILITY
MENU TEXT Bed Availability
UPPERCASE MENU TEXT BED AVAILABILITY
DESCRIPTION 
This option produces a listing of available beds.  The user may select
a specific ward and view the beds available for that ward on the terminal
the request was initiated from or may enter the word "ALL" and output of
listing of available beds by ward to a specific terminal/printer.
CREATOR POSTMASTER
TYPE run routine
