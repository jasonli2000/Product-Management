XUSERREPRINT (190)    OPTION (19)

Name Value
NAME XUSERREPRINT
MENU TEXT Reprint Access agreement letter
UPPERCASE MENU TEXT REPRINT ACCESS AGREEMENT LETTE
DESCRIPTION 
This option will allow a site manager to reprint the computer
access agreement letter.  It will not reprint the access code
on the letter.
CREATOR USER,ONE
TYPE run routine
ROUTINE REPRINT^XUSERNEW
