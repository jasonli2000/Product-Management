Name Value
NAME DG INPATIENT LIST
MENU TEXT Inpatient Listing
UPPERCASE MENU TEXT INPATIENT LISTING
DESCRIPTION 
This option is used to generate a listing of current inpatients.
The list can be generated in several formats, i.e., with ward
breakout, DRG breakout, etc.
