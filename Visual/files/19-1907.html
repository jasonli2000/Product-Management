DG REGISTER PATIENT (1907)    OPTION (19)

Name Value
NAME DG REGISTER PATIENT
MENU TEXT Register a Patient
UPPERCASE MENU TEXT REGISTER A PATIENT
DESCRIPTION 
This option is used to process an application for care.  The user
may also use this option to establish or edit a specific patient's
data base.  After an application for care has been processed the
patient should either be dispositioned from care using the DISPOSITION
option or the registration should be deleted using the DELETE
REGISTRATION option.
