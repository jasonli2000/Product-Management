NAME DG DEVICE SELECTION
X ACTION PRESENT YES
MENU TEXT Device Selection
UPPERCASE MENU TEXT DEVICE SELECTION
DESCRIPTION 
This option is used to enter or change the default output devices
to which the 10-10 report and associated documents (routing slips,
prescription profiles) should be automatically printed upon
completion of an application for care.  If a multi-divisional
facility the output devices may differ for each division.
 
This option is also used to enter the default output device to print
the AMIS code sheets generated for the segments 334-341, 345&346 and
401-420. The code sheet printer default must be defined for the user
to be asked "".
