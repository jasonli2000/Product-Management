DG PTF OPEN CLOSED RECORD (1935)    OPTION (19)

Name Value
NAME DG PTF OPEN CLOSED RECORD
MENU TEXT Open Closed PTF Record
UPPERCASE MENU TEXT OPEN CLOSED PTF RECORD
LOCK DG PTFREL
DESCRIPTION 
This option is used to reactivate PTF records that have been
closed but not released by MIS, so that changes can be made to 
those records.

TYPE run routine
