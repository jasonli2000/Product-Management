NAME DG PTF OPEN RECORD OUTPUT
MENU TEXT Unreleased PTF Record Output
UPPERCASE MENU TEXT UNRELEASED PTF RECORD OUTPUT
DESCRIPTION 
This option prints out the PTF records that have been closed out
by the coding clerks but have not yet been released.
