NAME DG G&L RECALC
MENU TEXT Recalculate G&L Cumulative Totals
UPPERCASE MENU TEXT RECALCULATE G&L CUMULATIVE TOT
DESCRIPTION 
This option is intended to update the Cumulative totals after past
patient movements have been recorded.  This eliminates the need for
reprinting numerous G&L sheets.
