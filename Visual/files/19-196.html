XQAB ERROR LOG SERVER (196)    OPTION (19)

Name Value
NAME XQAB ERROR LOG SERVER
MENU TEXT Handle Alpha/Beta Errors Logged at Sites
UPPERCASE MENU TEXT HANDLE ALPHA/BETA ERRORS LOGGE
DESCRIPTION 
 
This SERVER option is used to store data sent by the 'XQAB ERROR LOG XMIT'
option back to the developing site (usually an ISC).  As a server to which
the mail messages containing data on the types and frequencies of errors
associated with an application package in alpha or beta test, this option
starts a routine which processes the message contents and stores the data
in file 8991.5 (^XTV(8991.5,...).  The contents of the file may be
processed using several options or by the use of File Manager directly.
The file contains data on the type of error, the routine involved, the
option which was in use at the time of the error, the date, and the number
of errors for that date, by site (and if multiple error traps are used at
a site, by the vol,uci).
CREATOR USER,ONE
TYPE server
ROUTINE XQABELOG
SERVER ACTION RUN IMMEDIATELY
SUPRESS BULLETIN YES, SUPRESS IT
SERVER REPLY NO REPLY (DEFAULT)
