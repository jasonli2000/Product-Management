DG G&L RECALCULATION AUTO (1961)    OPTION (19)

Name Value
NAME DG G&L RECALCULATION AUTO
MENU TEXT Auto-recalculation of G&L Statistics
UPPERCASE MENU TEXT AUTO-RECALCULATION OF G&L STAT
DESCRIPTION 
This option uses the G&L CORRECTIONS file as to determine what the
oldest correction on file is for which the census hasn't been
updated.  All census data from that date forward will be updated to
reflect accurate totals.  This option should be queued to run every
night.
CREATOR POSTMASTER
RESTRICT DEVICES? .5
TYPE run routine
PACKAGE REGISTRATION
ROUTINE AUTO^DGPMBSAB
