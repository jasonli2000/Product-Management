DG CONSISTENCY UPDATE (1981)    OPTION (19)

Name Value
NAME DG CONSISTENCY UPDATE
MENU TEXT Update Inconsistency File
UPPERCASE MENU TEXT UPDATE INCONSISTENCY FILE
LOCK DG CONSISTENCY
DESCRIPTION 
This option scans through all entries in the INCONSISTENT DATA file
to see if the inconsistencies identified indeed still exist.  If not
the entry will be deleted from the file.  This option should be run
periodically to insure that any data cleaned up by means other than
those options which call up the consistency checker, i.e., fileman,
is removed from the file.
CREATOR POSTMASTER
TYPE run routine
PACKAGE REGISTRATION
ROUTINE DGRPCU
