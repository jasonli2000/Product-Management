DG PTF DRG INDEX (1997)    OPTION (19)

Name Value
NAME DG PTF DRG INDEX
MENU TEXT DRG Index Report
UPPERCASE MENU TEXT DRG INDEX REPORT
DESCRIPTION 
This option generates the DRG INDEX Report which sorts and lists patient
names according to their DRG.  It may be run for all current inpatients or
for discharged patients within a discharge date range.
CREATOR POSTMASTER
TYPE run routine
