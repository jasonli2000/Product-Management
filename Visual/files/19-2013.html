DG MEANS TEST DELETE (2013)    OPTION (19)

Name Value
NAME DG MEANS TEST DELETE
MENU TEXT Delete a Means Test
UPPERCASE MENU TEXT DELETE A MEANS TEST
LOCK DG MTDELETE
DESCRIPTION 
This option is used to delete means test data which may have been 
inadvertantly entered.  Under normal circumstances only individual dates
of test may be deleted using this option.  The exception is non-veterans.
All means tests found for a non-veteran may be deleted.
  
This option invokes the means test event driver.
   o Deletion of a means test will cause any associated changes to that 
      means test to also be deleted from the MEANS TEST CHANGES file.
   o If another means test remains on file for the same patient, and the
     status of the remaining means test differs from the one which was
     deleted, a 'Means Test Category Change' is entered into the MEANS
     TEST CHANGES file.  This can be viewed using the View Means Test
     Editing Activity option.
CREATOR POSTMASTER
TYPE action
PACKAGE REGISTRATION
E ACTION PRESENT YES
ENTRY ACTION S DGMTYPT=1 D EN^DGMTDEL
