DG SCHED ADMIT STATS (2024)    OPTION (19)

Name Value
NAME DG SCHED ADMIT STATS
MENU TEXT Scheduled Admission Statistics
UPPERCASE MENU TEXT SCHEDULED ADMISSION STATISTICS
DESCRIPTION 
This option is used to generate statistics for a specified date range of
patients scheduled for admission versus those which were cancelled.  A
subtotal of the reasons for cancellation is also provided on this report.
Report is broken into divisions (where applicable) and ward/treating
specialty within each division.
