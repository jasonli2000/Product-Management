DG DISPOSITION TIME STATS (2029)    OPTION (19)

Name Value
NAME DG DISPOSITION TIME STATS
MENU TEXT Disposition Time Processing Statistics
UPPERCASE MENU TEXT DISPOSITION TIME PROCESSING ST
DESCRIPTION 
This option generates a reports of average processing time for patients
registered during a specified timeframe.  Any patients not dispositioned
during that timeframe are not included in the report.  The user can
elect to generate a listing of those "undispositioned" registrations
at the end of the report.
CREATOR POSTMASTER
TYPE run routine
PACKAGE REGISTRATION
ROUTINE DGDIST
