DG BULLETIN LOCAL (2030)    OPTION (19)

Name Value
NAME DG BULLETIN LOCAL
MENU TEXT Bulletin Selection
UPPERCASE MENU TEXT BULLETIN SELECTION
DESCRIPTION 
This option is used to define the local mailgroup to which you want
certain notifications made.  If you do not wish to utilize a
specific notification type DO NOT enter a mailgroup name otherwise
select from the available listing the mailgroup you want notified.
