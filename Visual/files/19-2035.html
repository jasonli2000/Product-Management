DG PTF 099 TRANSMISSION (2035)    OPTION (19)

Name Value
NAME DG PTF 099 TRANSMISSION
MENU TEXT 099 Transmission
UPPERCASE MENU TEXT 099 TRANSMISSION
LOCK DG PTFTRANS
DESCRIPTION 
This option is used when it is necessary to reopen a previously
transmitted PTF record and transmit an 099 deletion transaction to
the Austin DPC to delete the DPC's master record.
 
Also, this option will allow the user to transmit a 'free-form' 099
to delete an Austin master record for an entry with bad control data
not represented in the DHCP database.
 
This bad control data could have resulted from keypunch error or a
change in a patient's social security number or admission date and/or
time.
