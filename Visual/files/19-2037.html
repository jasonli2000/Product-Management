DG SECURITY ENTER/EDIT (2037)    OPTION (19)

Name Value
NAME DG SECURITY ENTER/EDIT
MENU TEXT Enter/Edit Patient Security Level
UPPERCASE MENU TEXT ENTER/EDIT PATIENT SECURITY LE
LOCK DG SENSITIVITY
DESCRIPTION 
This option permits holders of the DG SENSITIVITY key to assign
a security level to a patient. A patient can either be sensitive
or non-sensitive.
 
If the security level for a patient is changed from sensitive to
non-sensitive a bulletin will be sent notifying the Station Security
Officer of the change.
