DG SECURITY PURGE PATIENTS (2038)    OPTION (19)

Name Value
NAME DG SECURITY PURGE PATIENTS
MENU TEXT Purge Non-sensitive Patients from Security Log
UPPERCASE MENU TEXT PURGE NON-SENSITIVE PATIENTS F
LOCK DG SECURITY OFFICER
DESCRIPTION 
This option permits holders of the DG SECURITY OFFICER key
to purge patients from the DG SECURITY LOG file if the security
level of the patient is non-sensitive.

TYPE run routine
