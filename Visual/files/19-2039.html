DG SECURITY PURGE LOG (2039)    OPTION (19)

Name Value
NAME DG SECURITY PURGE LOG
MENU TEXT Purge Record of User Access from Security Log
UPPERCASE MENU TEXT PURGE RECORD OF USER ACCESS FR
LOCK DG SECURITY OFFICER
DESCRIPTION 
This option permits holders of the DG SECURITY OFFICER key to
purge the access log for a select patient, or all patients during
a specified date range. However, any access made to a sensitive
patient record must be kept in the log at least 30 days.
