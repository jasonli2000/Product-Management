DG RELEASE COMMENTS (2042)    OPTION (19)

Name Value
NAME DG RELEASE COMMENTS
MENU TEXT Transmit/Generate Release Comments
UPPERCASE MENU TEXT TRANSMIT/GENERATE RELEASE COMM
LOCK DG SUPERVISOR
DESCRIPTION 
This option is used to send comments concerning the installation of
ADT/Scheduling to the developers.  The comments may be generated
either as a mailman message (which will be automatically forwarded
to the developers) or as a letter which you may mail to the
address indicated.  Network mail must be up in order to transmit
message via mailman.

TYPE run routine
