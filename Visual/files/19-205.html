XUTM REQ (205)    OPTION (19)

Name Value
NAME XUTM REQ
MENU TEXT Requeue Tasks
UPPERCASE MENU TEXT REQUEUE TASKS
DESCRIPTION 
Allows user to requeue own tasks; user may modify the device that the
task is to be run on and also the time that the task is to be run.
Holders of the ZTMQ key selecting this option may requeue any tasks,
and may also modify the task's priority and partition size.
CREATOR USER,ONE
TYPE run routine
ROUTINE XUTMR
