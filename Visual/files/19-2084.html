DG GECS CODE EDIT (2084)    OPTION (19)

Name Value
NAME DG GECS CODE EDIT
MENU TEXT Code Sheet Edit
UPPERCASE MENU TEXT CODE SHEET EDIT
DESCRIPTION 
This option allows the user to edit code sheets created for
AMIS segments 167, 223, 290, 334-341, 345-346 and 401-420.
When editing the code sheet the first field displayed is a
status field code sheets typically will have one of the
following statuses:
 
 -CODE SHEET RETAINED IN FILE!
 -CODE SHEET READY FOR BATCHING!
 -CODE SHEET READY FOR TRANSMISSION IN BATCH {batch #}!
 -CODE SHEET TRANSMITTED IN BATCH {batch #} on {date transmitted}!
 
The status field can not be edited, it is only displayed. The
remaining fields pertain directly to the code sheet being
edited. These are followed by the option to print, mark
the code sheet for batching, delete and edit the code sheet.
CREATOR POSTMASTER
TYPE action
PACKAGE REGISTRATION
E ACTION PRESENT YES
ENTRY ACTION S GECSSYS="MAS" D EDIT^GECSE1
