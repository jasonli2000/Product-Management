DG GECS CREATE (2085)    OPTION (19)

Name Value
NAME DG GECS CREATE
MENU TEXT Create a Code Sheet
UPPERCASE MENU TEXT CREATE A CODE SHEET
DESCRIPTION 
This option will allow the user to create code sheets for the AMIS
segments 167 and 290. The option can also be used to create
code sheets for the other types of MAS segments (223, 334-341,
345-346 and 401-420) but typically the code sheets for these
segments should be built using the 'Generate a Code Sheet'
option.
 
If you are a multi-divisional facility the information for
AMIS segments 334-341 and 345-346 should be combined for
all divisions and a code sheet should be created, for each segment,
for the primary division containing the combined statistics
for all divisions.
CREATOR POSTMASTER
TYPE action
PACKAGE REGISTRATION
E ACTION PRESENT YES
ENTRY ACTION S GECSSYS="MAS" D EN1^GECSA
