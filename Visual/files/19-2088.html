DG GECS PURGE (2088)    OPTION (19)

Name Value
NAME DG GECS PURGE
MENU TEXT Purge Transmission Records/Code Sheets
UPPERCASE MENU TEXT PURGE TRANSMISSION RECORDS/COD
DESCRIPTION 
This option allows the user to purge code sheets created
for AMIS segments 167, 223, 290, 334-341, 345-346 and
401-420. The user is first asked the number of days to
retain the code sheets. The date, based upon the number
of days selected, is then determined and all code sheets
and batches created prior to this date are purged.
CREATOR POSTMASTER
TYPE action
PACKAGE REGISTRATION
E ACTION PRESENT YES
ENTRY ACTION S GECSSYS="MAS" D ^GECSPURG
