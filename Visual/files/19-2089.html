DG GECS READY FOR BATCH LIST (2089)    OPTION (19)

Name Value
NAME DG GECS READY FOR BATCH LIST
MENU TEXT Code Sheets Ready for Batching
UPPERCASE MENU TEXT CODE SHEETS READY FOR BATCHING
DESCRIPTION 
This option allows the user to print the code sheets ready for
batching. This report contains the code sheet ID, scheduled
transmission date, station number, batch type and transaction
type/segment.
CREATOR POSTMASTER
TYPE action
PACKAGE REGISTRATION
