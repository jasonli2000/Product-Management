DG GECS RETRANSMIT (2091)    OPTION (19)

Name Value
NAME DG GECS RETRANSMIT
MENU TEXT Mark Batch for Retransmission
UPPERCASE MENU TEXT MARK BATCH FOR RETRANSMISSION
DESCRIPTION 
This option allows the user to mark any of the MAS
Code Sheet batches which have already been transmitted;
so that they can be retransmitted.
CREATOR POSTMASTER
TYPE action
PACKAGE REGISTRATION
E ACTION PRESENT YES
ENTRY ACTION S GECSSYS="MAS" D RETRAN^GECSE2
