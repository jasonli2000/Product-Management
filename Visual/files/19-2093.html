DG GECS TRANSMIT (2093)    OPTION (19)

Name Value
NAME DG GECS TRANSMIT
MENU TEXT Transmit Code Sheets
UPPERCASE MENU TEXT TRANSMIT CODE SHEETS
DESCRIPTION 
This option will allow the user to transmit any of the MAS
Code Sheet batches which have not yet been transmitted or
that have been marked for retransmission. Code Sheets are
transmitted to Austin.  Members of the AMD
mail group will receive the confirmation and summary
messages from Austin.
CREATOR POSTMASTER
TYPE action
PACKAGE REGISTRATION
E ACTION PRESENT YES
ENTRY ACTION S GECSSYS="MAS" D ^GECSTRAN
