NAME DG GECS USER MENU
MENU TEXT MAS Code Sheet User Menu
UPPERCASE MENU TEXT MAS CODE SHEET USER MENU
DESCRIPTION 
This menu will allow users to generate, create and keypunch
MAS Code Sheets. The menu also contains options to edit and review
the code sheets as well as delete and print code sheets.
 
The option to 'Generate a Code Sheet' can be used to generate the
AMIS 223, 334-341, 345-346 and 401-420 segments. The 'Create a Code
Sheet' option can be used to create the AMIS 167 and 290 segments.
The 'Key Punch a Code Sheet' option is typically used for prior data
corrections.
