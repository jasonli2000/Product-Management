DG GECS GENERATE (2096)    OPTION (19)

Name Value
NAME DG GECS GENERATE
MENU TEXT Generate a Code Sheet
UPPERCASE MENU TEXT GENERATE A CODE SHEET
DESCRIPTION 
This option will allow a user to generate code sheets for
AMIS segments 334-341, 345-346 and 401-420. The AMIS
information is pulled from a select month/year for which the
AMIS has already been generated. Code sheets will only
be generated for balanced AMIS segments.
 
If you are a multi-divisional facility the information for
AMIS segments 334-341 and 345-346 are combined for all
divisions and a code sheet is created, for each segment,
for the primary division containing the combined statistics
for all divisions.
CREATOR USER,ONE
TYPE run routine
PACKAGE REGISTRATION
ROUTINE DGGECS
