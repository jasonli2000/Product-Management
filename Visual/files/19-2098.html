NAME DG GECS BATCH MENU
MENU TEXT MAS Code Sheet Batch Menu
UPPERCASE MENU TEXT MAS CODE SHEET BATCH MENU
DESCRIPTION 
This menu contains all the MAS batch type options which
are used to coordinate and group all the MAS AMIS Code
Sheets so that they can be transmitted.
 
Options on this menu include a report of code sheets
ready for batching, an option to batch the code sheets,
an option to edit the priority and batch number of a code
sheet and an option to mark code sheets for rebatching.
