DGPT CENSUS STATUS REPORT (2102)    OPTION (19)

Name Value
NAME DGPT CENSUS STATUS REPORT
MENU TEXT Census Status Report
UPPERCASE MENU TEXT CENSUS STATUS REPORT
DESCRIPTION 
This options prints a report of all active admissions as of
11:59pm for a specified census date.  The report will indicate the status
of the census record for each admission.  This report uses the
data stored in the CENSUS WORKFILE.
 
This workfile is an extract of data from the admission data for the
patients who were inpatients as of 11:59 on the census date.  This
workfile makes the printing of this report much faster than otherwise
would be possible.  Without this workfile, it would be neccessary
to scan all admissions prior to the census date to determine if
the admission was still active on the census date.
 
NOTE:  The first time the 'Census Status Report' is run for a
       particular census date, the CENSUS WORKFILE is automatically
       generated.
 
The user is allowed to select one, many or all divisions and
one census status.
 
The report is sorted by:
         1) division
         2) then status
         3) then ward on census date
         4) then patient name.
