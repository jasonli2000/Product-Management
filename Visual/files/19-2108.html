DGPT CENSUS INQUIRE (2108)    OPTION (19)

Name Value
NAME DGPT CENSUS INQUIRE
MENU TEXT Inquire Census Record
UPPERCASE MENU TEXT INQUIRE CENSUS RECORD
DESCRIPTION 
This option allows the user to print the CENSUS record for an
admission.
 
When a PTF record is 'closed for census' a snap shot of the PTF
record is taken.  This snap shot is the census record that is
transmitted to Austin and is used by this option.
 
If the PTF record has not yet been 'closed for census' then the
PTF record is displayed/printed since the census record will not
exist.
