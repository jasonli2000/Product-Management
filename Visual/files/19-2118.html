DGPT CENSUS 099 TRANSACTION (2118)    OPTION (19)

Name Value
NAME DGPT CENSUS 099 TRANSACTION
MENU TEXT 099 Transmission for Census Record
UPPERCASE MENU TEXT 099 TRANSMISSION FOR CENSUS RE
LOCK DG PTFTRANS
DESCRIPTION 
This option allows the user to send a 099 transaction for a
census record.  When received in Austin, this 099 will result
in deletion of the master record.
 
At the end of this option, the census record will be re-opened
(ie, deleted) from the PTF file in the DHCP database.
 
Also, this option will allow the user to transmit a 'free-form'
099 to delete an Austin master record for an entry with
bad control data not represented in the DHCP database.
 
This bad control data could have resulted from keypunch
error or a change in a patient's social security number or
admission date and/or time.
