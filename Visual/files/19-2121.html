DGPT SET XMIT FLAG (2121)    OPTION (19)

Name Value
NAME DGPT SET XMIT FLAG
MENU TEXT Set Transmit Flag on Movements
UPPERCASE MENU TEXT SET TRANSMIT FLAG ON MOVEMENTS
LOCK DG PTFSUP
DESCRIPTION 
This option allows the supervisor to flag(or unflag) a 501
or a 535 movement so it will not be transmitted to Austin.
 
Only PTF records that have more than twenty-five 501 movements or
more than twenty-five 535 movements can be selected.
 
PTF records with more than 25 of either type of movement can not
be closed or transmitted.  Some of the movements must be flagged
not to be transmitted.  This is an Austin DPC limitation.
 
The situation, where more than twenty-five 501's or 535's
are present in a PTF record, is very rare.
