MENU TEXT Reasons for Lodging Entry/Edit
UPPERCASE MENU TEXT REASONS FOR LODGING ENTRY/EDIT
DESCRIPTION 
This option should be used to add new reasons for lodging.  Examples
include distance too far to travel and lack of transportation.  When
checking in a lodger patient, a reason for lodging will be required.
