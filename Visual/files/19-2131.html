DGPM WARD OOS EDIT (2131)    OPTION (19)

Name Value
NAME DGPM WARD OOS EDIT
MENU TEXT Edit Ward Out-of-Service Dates
UPPERCASE MENU TEXT EDIT WARD OUT-OF-SERVICE DATES
DESCRIPTION 
Use this option to place a ward out-of-service or return a ward to service.
If you are placing a ward out-of-service indefinately, do not answer the
'RETURN TO SERVICE' prompt.  You will be asked how long you wish to show
statistics for this ward on the G&L.  If you do not answer these questions,
the data will always show on the G&L.
CREATOR POSTMASTER
TYPE run routine
