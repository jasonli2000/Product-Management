DGPM BED ENTRY/EDIT (2132)    OPTION (19)

Name Value
NAME DGPM BED ENTRY/EDIT
MENU TEXT Add/Edit Beds
UPPERCASE MENU TEXT ADD/EDIT BEDS
DESCRIPTION 
Use this option to add a new bed or edit an existing bed.  You will be
asked which ward or wards will be able to assign the bed.  You will NOT
be able to delete a bed, but you will be able to remove all wards from
being able to assign.  To inactivate a bed for a period of time, use the
'Bed Out-of-Service Date Enter/Edit' option.
