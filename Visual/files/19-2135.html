DG ELIG ID FORMAT EDIT (2135)    OPTION (19)

Name Value
NAME DG ELIG ID FORMAT EDIT
MENU TEXT ID Format Enter/Edit
UPPERCASE MENU TEXT ID FORMAT ENTER/EDIT
DESCRIPTION 
This option allows IRM personnel to enter/edit patient
identification formats.  These formats are used to set the
various patient id fields in the PATIENT file, such as
PRIMARY LONG ID and PRIMARY SHORT ID.
 
The standard format is the 'VA STANDARD'. This format is the
Social Security Number format.  All eligibility codes in the
ELIGIBILITY CODE file will be associated with this format as
a result of the MAS v5 post-init.
 
Whenever the patient's SSN or PRIMARY ELIGIBILITY CODE fields
are updated the 'VA STANDARD' format will be used to set the
ID fields.
 
The site should not have to add any other formats nor should
they have to modify the 'VA STANDARD' format.
 
The only exception are those sites running the VA/DOD sharing
agreement software being developed by the Dallas ISC.
 
If the site feels it needs to add formats or modify the 
'VA STANDARD' format, IRM staff should first call their local
support ISC.
