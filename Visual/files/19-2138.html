NAME DG ELIG PRI ELIG RESET
MENU TEXT Primary Eligibility ID Reset (All Patients)
UPPERCASE MENU TEXT PRIMARY ELIGIBILITY ID RESET (
DESCRIPTION 
This option will set/reset the id's associated with each
patient's PRIMARY ELIGIBILITY CODE.  This utility is
called by the version 5 conversion clean-up routine.
 
It will create the necessary entry in the PATIENT
ELIGIBILITIES multiple corresponding to each patient's PRIMARY 
ELIGIBILITY CODE.  It will then update all the ID fields.
 
This option can be executed multiple times with no harmful
effects.
 
This option should be executed during non-peak hours.  When
the job has completed, a Mailman message will be sent to
the user.
