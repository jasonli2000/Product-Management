NAME DG ELIG MAINTENANCE
MENU TEXT Eligibility/ID Maintenance Menu
UPPERCASE MENU TEXT ELIGIBILITY/ID MAINTENANCE MEN
DESCRIPTION 
This menu contians utility options that allow the site to
maintain their ELIGIBILITY CODE file and patient ID formats.
 
The primary reason for having different ID formats and then
associating them with a eligibilities is to help meet DOD/Sharing
agreement requirements.
 
For most sites each eligibility needs to simply be assoicated with the
VA STANDARD format.  This association is first accomplished during
the post-init of MAS v5.  As a result of this post-init,
most site will not require any further action.
 
This is an IRM menu and should be used with care.
Some options are very cpu intensive and should not be
executed during peak hours.
