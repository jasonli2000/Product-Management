NAME DGPT TOOLS MENU
MENU TEXT Utility Menu
UPPERCASE MENU TEXT UTILITY MENU
DESCRIPTION 
This menu contains options that allow the user to send special
transactions to Austin, such as 099's and 150's. There are also options to
delete PTF's and establish PTF's for an existing admission. An option to
modify the effective date and active/inactive flag of a facility suffix is
also included in this menu.
