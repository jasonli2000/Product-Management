XUTM ERROR PURGE TYPE (216)    OPTION (19)

Name Value
NAME XUTM ERROR PURGE TYPE
MENU TEXT Purge Error Log Of Type Of Error
UPPERCASE MENU TEXT PURGE ERROR LOG OF TYPE OF ERR
DESCRIPTION 
This option provides a way to clean Task Manager's log of errors.  The
site manager can enter a string, and this option will then delete every
entry in the log that contains that string.  For example, if the site
manager enters "<UNDEF>" then every error that contains this string
will be deleted.  Two other examples are "ZTSK+3^XQ1:4" and "E".  The
first removes all errors that occurred on the line and command indicated
while the second removes ALL errors whose $ZE value contains an "E".
Note that this does not edit the system's main error log, only Task
Manager's Error Log in the ^%ZTSCH global.
CREATOR USER,ONE
TYPE run routine
ROUTINE TYPE^XUTMKE
