XUTM ERROR LOG CLEAN RANGE (217)    OPTION (19)

Name Value
NAME XUTM ERROR LOG CLEAN RANGE
MENU TEXT Clean Error Log Over Range Of Dates
UPPERCASE MENU TEXT CLEAN ERROR LOG OVER RANGE OF
DESCRIPTION 
This option allows the site manager to remove from Task Manager's error
log all entries that occur on or between two dates.  The site manager
enters the two dates, FIRST THE EARLIER DATE AND THEN THE LATER DATE,
and then the option removes the appropriate entries.
CREATOR USER,ONE
TYPE run routine
ROUTINE RANGE^XUTMKE
