DG CO-PAY TEST DELETE (2172)    OPTION (19)

Name Value
NAME DG CO-PAY TEST DELETE
MENU TEXT Delete a Copay Exemption Test
UPPERCASE MENU TEXT DELETE A COPAY EXEMPTION TEST
LOCK DG MTDELETE
DESCRIPTION 
This option is used to delete co-pay test data which may have been 
inadvertantly entered.  Under normal circumstances only individual dates of
test may be deleted using this option.  The exception is non-veterans.  All
co-pay tests found for a non-veteran may be deleted.  
  
This option invokes the means test event driver.
CREATOR POSTMASTER
TYPE action
E ACTION PRESENT YES
ENTRY ACTION D ON^DGMTCOU I Y S DGMTYPT=2 D EN^DGMTDEL
