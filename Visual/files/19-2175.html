DG CO-PAY TEST VIEW EDITING (2175)    OPTION (19)

Name Value
NAME DG CO-PAY TEST VIEW EDITING
MENU TEXT View Copay Exemption Test Editing Activity
UPPERCASE MENU TEXT VIEW COPAY EXEMPTION TEST EDIT
DESCRIPTION 
This option allows the user to view all changes made to a particular co-pay 
test for a patient.  Some of the displayed information includes date of 
change, user who made change and the type of change.  If a change involves 
a change in the co-pay test status, the previous and current values are 
displayed.  If a co-pay test is deleted, any associated changes are also 
deleted.
