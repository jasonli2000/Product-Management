SDCHART (2187)    OPTION (19)

Name Value
NAME SDCHART
X ACTION PRESENT YES
MENU TEXT Chart Request
UPPERCASE MENU TEXT CHART REQUEST
DESCRIPTION 
This option is used to request a patient's chart for review without
actually scheduling the patient for an appointment.  It would be used,
for instance, in a case where a physician has ordered lab tests for
a particular patient and wishes to review those results say in one
week but not see the patient for another two weeks.  No appointment
is scheduled for the patient and no appointment letters are generated.
The patient will appear on the File Room list though but the appointment
time will not be indicated since there is none.
