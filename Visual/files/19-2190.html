MENU TEXT Check-in/Unsched. Visit
UPPERCASE MENU TEXT CHECK-IN/UNSCHED. VISIT
DESCRIPTION 
This option allows scheduling of an unanticipated appointment. If you want
to keep statistics on when a patient arrives for an appointment, this option
will allow you to record a patient's arrival time for the appointment.
