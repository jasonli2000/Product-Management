XUTM ERROR SCREEN LIST (220)    OPTION (19)

Name Value
NAME XUTM ERROR SCREEN LIST
MENU TEXT List Error Screens
UPPERCASE MENU TEXT LIST ERROR SCREENS
DESCRIPTION 
This option allows the user to see the error screens that are currently
in place.  An error screen is a string of characters.  Any error that
Task Manager logs will be checked against the list of error screens.
If an error occurs whose $ZE string contains a screen as a substring,
then that error will not be logged.  Some error screens count the number
of errors that they prevent from being logged.
CREATOR USER,ONE
TYPE run routine
ROUTINE SCLIST^XUTMKE1
