SDCANCEL (2200)    OPTION (19)

Name Value
NAME SDCANCEL
X ACTION PRESENT YES
MENU TEXT Cancel Clinic Availability
UPPERCASE MENU TEXT CANCEL CLINIC AVAILABILITY
DESCRIPTION 
This option allows the user to cancel a clinic's availability for
either a whole day or portion's of a day.  If several portions's
of a day are to be cancelled the user must cancel one portion first,
re-enter the Cancel Clinic Availability option to cancel the second,
etc.
