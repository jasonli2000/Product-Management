X ACTION PRESENT YES
MENU TEXT Remap Clinic
UPPERCASE MENU TEXT REMAP CLINIC
DESCRIPTION 
This option is used to remap a clinic.  It is used primarily after the
entry of a new holiday or deletion of an existing holiday.  It basically
updates the pattern to insure that it knows that the holiday now exists
or no longer exists.  It must be run for each clinic which meets on the
date of the holiday.
