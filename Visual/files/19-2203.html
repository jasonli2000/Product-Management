SDDISCHARGE (2203)    OPTION (19)

Name Value
NAME SDDISCHARGE
X ACTION PRESENT YES
MENU TEXT Discharge from Clinic
UPPERCASE MENU TEXT DISCHARGE FROM CLINIC
DESCRIPTION 
This option is used to discharge a patient from a clinic in which he/she
is enrolled.  Once discharged from the clinic the patient would have to
either be re-enrolled or scheduled for a consultation if further visits
are necessary.
CREATOR POSTMASTER
TYPE action
PACKAGE SCHEDULING
E ACTION PRESENT YES
EXIT ACTION D SDCD^SDKILL
ENTRY ACTION K SDFN,SDCLN D ^SDCD
