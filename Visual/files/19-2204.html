NAME SDCLINLIST
X ACTION PRESENT YES
MENU TEXT Clinic List (Day of Week)
UPPERCASE MENU TEXT CLINIC LIST (DAY OF WEEK)
DESCRIPTION 
This option generates a listing of clinics by the day of week they meet.
If a particular clinic meets Monday, Wednesday and Friday it will appear
on each days listing.
