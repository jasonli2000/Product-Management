XUTM ERROR SCREEN REMOVE (223)    OPTION (19)

Name Value
NAME XUTM ERROR SCREEN REMOVE
MENU TEXT Remove Error Screens
UPPERCASE MENU TEXT REMOVE ERROR SCREENS
DESCRIPTION 
Allows the user to remove error screens.  Error screens provide the
system manager with a way to prevent certain errors from being logged.
Task Manager traps every error caused by its own code or by the code
of the tasks it runs, but the only errors logged in the system error log
and in Task Manager's error log are those errors whose $ZE values do
not contain an error screen as a substring.  The system manager can decide
whether to count the number of errors screened out.
CREATOR USER,ONE
TYPE run routine
ROUTINE SCREM^XUTMKE1
