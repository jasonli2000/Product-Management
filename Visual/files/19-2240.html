SDADDEDIT (2240)    OPTION (19)

Name Value
NAME SDADDEDIT
MENU TEXT Add/Edit Stop Codes
UPPERCASE MENU TEXT ADD/EDIT STOP CODES
DESCRIPTION 
This option should be used to add/edit standalone encounters which were not
captured through scheduled or unscheduled visits or through registration.

This option should not be used in lieu of the Make Appointment or
Registration option.
