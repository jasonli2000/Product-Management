SDACS CGEDIT (2259)    OPTION (19)

Name Value
NAME SDACS CGEDIT
MENU TEXT Edit Computer Generated Appointment Type
UPPERCASE MENU TEXT EDIT COMPUTER GENERATED APPOIN
DESCRIPTION 
Option should be used to edit appointment types which have a value
of 'Computer Generated' to a valid appointment type so that
the clinic will receive credit.
   
This option is only needed if other packages (such as Radiology v 4.0)
capture stop code workload.  Refer to the User manual, Volume II,
Appendix, for further documentation.
CREATOR POSTMASTER
TYPE run routine
ROUTINE EDIT^SDACSCG
