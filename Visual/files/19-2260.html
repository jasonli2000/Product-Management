SDACS CGMENU (2260)    OPTION (19)

Name Value
NAME SDACS CGMENU
MENU TEXT Computer Generated Menu
UPPERCASE MENU TEXT COMPUTER GENERATED MENU
DESCRIPTION 
This menu contains options that allow the user to do the following:
       1) identify those 'stop code addition' encounters added by other
          packages to the Outpatient Encounter file
 
       2) identify those encounters added by other packages which still
          have an invalid appointment type of 'Computer Generated' and
 
       3) to resolve the appointment type for encounters with a
          'Computer Generated' appointment type.
  
This menu option should be used when a package is installed (such as radiology
and lab) that will automatically capture workload by passing encounter
information to PCE.
