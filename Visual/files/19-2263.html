SDAM APPT UPDATE (2263)    OPTION (19)

Name Value
NAME SDAM APPT UPDATE
MENU TEXT Appointment Status Update
UPPERCASE MENU TEXT APPOINTMENT STATUS UPDATE
DESCRIPTION 
This option will update the appointment status for a date range
of oupatient encounters specified by the user. If an encounter has not
been checked-in, no-showed or cancelled, then it will be given the
status of 'NO ACTION TAKEN'.  Such encounters will not be used
as part of NPCDB workload reporting.
 
Note: Inpatient encounters and encounters for 'non-count' clinics are
      not updated since they are not reported to Austin for workload
      credit.
 
A mailman message will be sent to the mail group specified in the
APPT UPDATE MAIL GROUP (#215) of the scheduling parameters.
 
The bulletin will show  the following:
         o  clinic
         o  number of encounters flagged as 'NO ACTION TAKEN'
         o  the total number of encounters
         o  percentage flagged as 'no action taken'
