NAME SDAM APPT MGT
X ACTION PRESENT YES
MENU TEXT Appointment Management
UPPERCASE MENU TEXT APPOINTMENT MANAGEMENT
DESCRIPTION 
This option allows the user to view appointments and take appropriate
actions against those appointments, such as check-in and cancel.
 
The user is allowed to view/act on appointments either by patient or by
clinic.
 
When a patient is selected, 'all' appointments for the patient for a
date range are displayed.  The date range used is determined by using
today's date and the new APPT SEARCH THRESHOLD parameter, specified
using the 'Scheduling Parameters' option:
     o lower limit:  today - parameter
     o upper limit:  today + 999 days
 
When a clinic is selected, the user is prompted for a beginning and
ending date. This date range is used to search for appointments in
the clinic with a status of 'no action taken'.
 
 
Note: The user can use the barcodes produced by the Record Tracking
      system, that are on the patient's medical record, to identify a
      patient while using this option.
