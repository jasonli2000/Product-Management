SDACS CGBUPDATE (2265)    OPTION (19)

Name Value
NAME SDACS CGBUPDATE
MENU TEXT Batch Update Comp Gen Appt Type for C&Ps
UPPERCASE MENU TEXT BATCH UPDATE COMP GEN APPT TYP
DESCRIPTION 
This option should be exercised after visits which have an appointment type
of 'Computer Generated' have been reviewed AND any possible C&P visits
which are not C&P related have been selectively edited to reflect the
appropriate appointment type.
  
Invoking this option will cause automatic update of all visits which
have a computer generated appointment type due to a possible C&P visit
and meet the selected parameters, to a Compensation & Pension appointment
type.
  
This option is only needed if other packages (such as Radiology v 4.0)
capture stop code workload.  Refer to the User manual, Volume II,
Appendix, for further documentation.
CREATOR POSTMASTER
TYPE run routine
ROUTINE SDACSCGB
