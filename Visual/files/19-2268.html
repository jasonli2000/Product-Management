SDAM APPT UPDATE VIEW (2268)    OPTION (19)

Name Value
NAME SDAM APPT UPDATE VIEW
MENU TEXT View Appointment Status Update Date (Single Date)
UPPERCASE MENU TEXT VIEW APPOINTMENT STATUS UPDATE
DESCRIPTION 
This option allows the user to view a single day in the Appointment Status
Update Log File.  The default for the Appointment Date is LAST//, allowing
the user to select the last date the update process was run.
 
Displayed is the Appointment Date, Date Last Update Started, Update Method,
User, Date Last Update Completed.
