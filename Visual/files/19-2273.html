SDAM RPT RETROACTIVE LIST (2273)    OPTION (19)

Name Value
NAME SDAM RPT RETROACTIVE LIST
MENU TEXT Retroactive Visits List
UPPERCASE MENU TEXT RETROACTIVE VISITS LIST
DESCRIPTION 
This option provides output of appointments and/or standalone encounters
that have been entered past the date transmission for workload credit is
allowed.
 
The user will be prompted for a date range.  This date range will be
used to look at the visit dates.
 
The user will be allowed to select one/many/all division(s).  It will
also allow the use to find the input either by stop code or clinic.
 
If the stop code is selected, it will allow the selection of one/many
all stop codes.  It will look at all clinics in specified division
that match the stop codes, then look at the standalones for the specified
stop code.
 
If the clinic is selected, it will allow the selection of one/many/all
clinic(s).  It will look at the clinics specified, then look at the
add/edits for any associated clinics that match the specified clinic.
 
The user will then receive an output of the following information:
 
      o Appointment Date/Time
      o Date Entered
      o Date Transmitted
     *o Clinic
      o Patient
      o Patient ID
      o Type (either APPT or ADD/EDIT)
      o User who entered appointment or add/edit
 
*Clinic name will only appear for add/edits when an associated clinic
has been specified.
