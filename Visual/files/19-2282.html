MENU TEXT Add/Edit Suffix Effective Date

OUT OF ORDER MESSAGE OUT OF ORDER, DO NOT USE THIS OPTION!!!
LOCK DG PTFSUP
DESCRIPTION 
This option allows the user to edit the effective date multiple of the
Facility Suffix (#45.68) file.  An effective date may be added, edited, or
deleted (this holds true for the ACTIVE? field within the multiple as
well). A status of the suffix, effective date(s), and active flag(s) will
be displayed prior and after editing.  This option is locked with the DG
PTFSUP security key.
