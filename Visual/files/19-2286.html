IB AUTHORIZE BILL GENERATION (2286)    OPTION (19)

Name Value
NAME IB AUTHORIZE BILL GENERATION
MENU TEXT Authorize Bill Generation
UPPERCASE MENU TEXT AUTHORIZE BILL GENERATION
LOCK IB AUTHORIZE
DESCRIPTION 
This option allows the user to perform final review of 
information contained in a billing record. The user is
then able to authorize the generation of the bill and
the release of the information to Fiscal.
CREATOR POSTMASTER
TYPE run routine
PACKAGE INTEGRATED BILLING
ROUTINE 2^IBCMENU
