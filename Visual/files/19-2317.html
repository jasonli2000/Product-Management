IB FILER START (2317)    OPTION (19)

Name Value
NAME IB FILER START
MENU TEXT Start the Integrated Billing Background Filer
UPPERCASE MENU TEXT START THE INTEGRATED BILLING B
DESCRIPTION 
This option will task the IB background filer to run whether or not a
filer is currently running.  This option can be used when a filer
job has terminated unexpectedly and won't restart itself.  This will
force a filer to start running.
CREATOR POSTMASTER
TYPE run routine
PACKAGE INTEGRATED BILLING
ROUTINE START^IBEFUTL
