XPD VERIFY BUILD (233)    OPTION (19)

Name Value
NAME XPD VERIFY BUILD
MENU TEXT Verify a Build
UPPERCASE MENU TEXT VERIFY A BUILD
DESCRIPTION 
This option verifies the contents of a build.  It checks that every
component that is listed in the build still exists on your system.  You
should use it before you export a package.
CREATOR USER,ONE
TYPE run routine
ROUTINE VER^XPDE
