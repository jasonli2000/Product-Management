XPD TRANSPORT PACKAGE (235)    OPTION (19)

Name Value
NAME XPD TRANSPORT PACKAGE
MENU TEXT Transport a Distribution
UPPERCASE MENU TEXT TRANSPORT A DISTRIBUTION
DESCRIPTION 
This option uses entries in the Build file to create a Transport Global to 
export packages.  If you choose an HFS file as an output device, the
Transport Global is written to the HFS file, creating a distribution.  If
you don't choose a device, the Transport Global is written to the ^XTMP
global on your system.
