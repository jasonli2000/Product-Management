IB PURGE/FIND BILLING DATA (2361)    OPTION (19)

Name Value
NAME IB PURGE/FIND BILLING DATA
MENU TEXT Find Billing Data to Archive
UPPERCASE MENU TEXT FIND BILLING DATA TO ARCHIVE
LOCK XUMGR
DESCRIPTION 
This option may be used to identify records to be archived and purged
from the following files:
 
  #350  INTEGRATED BILLING ACTION
  #351  CATEGORY C BILLING CLOCK
  #399  BILL/CLAIMS
 
Entries which are selected to be archived and then purged are placed
into a Search (Sort) template.
CREATOR POSTMASTER
TYPE run routine
PACKAGE INTEGRATED BILLING
ROUTINE FIND^IBP
