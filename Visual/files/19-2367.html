IB PURGE LIST LOG ENTRIES (2367)    OPTION (19)

Name Value
NAME IB PURGE LIST LOG ENTRIES
MENU TEXT List Archive/Purge Log Entries
UPPERCASE MENU TEXT LIST ARCHIVE/PURGE LOG ENTRIES
LOCK XUMGR
DESCRIPTION 
This option may be used to list all of the log entries in the
IB ARCHIVE/PURGE LOG file, #350.6.  All entries in the file are
listed, in the order that they were added to the file.
CREATOR POSTMASTER
TYPE run routine
PACKAGE INTEGRATED BILLING
