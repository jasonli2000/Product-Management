IB SITE DEVICE SETUP (2372)    OPTION (19)

Name Value
NAME IB SITE DEVICE SETUP
MENU TEXT Select Default Device for Forms
UPPERCASE MENU TEXT SELECT DEFAULT DEVICE FOR FORM
DESCRIPTION 
This option allows associating devices as the default answer when printing
forms.  This is used to enter the default device for AR for follow-up
activity.
