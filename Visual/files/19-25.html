XUSERREACT (25)    OPTION (19)

Name Value
NAME XUSERREACT
MENU TEXT Reactivate a User
UPPERCASE MENU TEXT REACTIVATE A USER
DESCRIPTION 
Use this option to reactivate a user who has a
TERMINATION DATE.  You will be asked to enter his ACCESS CODE.  If you
give him an access code (or if he still has one), his TERMINATION DATE
will be removed.
CREATOR USER,ONE
TYPE run routine
ROUTINE REACT^XUSERNEW
