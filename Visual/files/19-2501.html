PRCP POST ISSUE BOOK ORDER (2501)    OPTION (19)

Name Value
NAME PRCP POST ISSUE BOOK ORDER
MENU TEXT Post Issue Book Order
UPPERCASE MENU TEXT POST ISSUE BOOK ORDER
DESCRIPTION 
This option should be used by the warehouse to post an issue book
distribution order.  When posting the order, the quantity on-hand
and quantity due-out in the warehouse is adjusted, and the
quantity on-hand and due-in in the primary is adjusted.  At completion
of posting, the FMS and ISMS code sheets are automatically created and
transmitted to Austin.
