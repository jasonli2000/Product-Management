NAME PRCPW ADJUST MENU
MENU TEXT Adjust Inventory Quantity Menu
UPPERCASE MENU TEXT ADJUST INVENTORY QUANTITY MENU
DESCRIPTION 
This menu contains the options which will adjust inventory item
quantities on-hand.  It also contains the option to move item
quantities to and from non-issuable status, and the report to
track items which are in non-issuable.
