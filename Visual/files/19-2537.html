PRCP ISMS BALANCE TASKMAN (2537)    OPTION (19)

Name Value
NAME PRCP ISMS BALANCE TASKMAN
MENU TEXT Taskman Sends ISMS Balance Update
UPPERCASE MENU TEXT TASKMAN SENDS ISMS BALANCE UPD
DESCRIPTION 
This option can be set up through TaskManager to automatically send
the ISMS Balance Update Transaction at a specified time.
 
This ISMS transaction will send a snapshot of the current warehouse
inventory balances to ISMS.  ISMS will use the balances contained in
this transaction to overlay the current ISMS balances.
