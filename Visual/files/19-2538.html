NAME PRCP CC ENTER/EDIT DEFINITION
MENU TEXT Enter/Edit/Create A Case Cart
UPPERCASE MENU TEXT ENTER/EDIT/CREATE A CASE CART
DESCRIPTION 
The Enter/Edit/Create A Case Cart option will allow the user to
define which items and quantity should be used in assembling the
case cart.  The user can also enter special instructions or remarks
for the case cart.
