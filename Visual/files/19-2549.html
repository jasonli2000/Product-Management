UPPERCASE MENU TEXT SURGERY SUPPLY ORDERING MENU
DESCRIPTION 
This is the main menu for the surgery users.  This menu will allow
surgery users to tie operation codes to specific case carts and
instrument kits.  It will also allow the surgery user to quickly
complete orders to SPD for scheduled surgical procedures.
