PRCP QUANTITY DISTRIBUTION (2559)    OPTION (19)

Name Value
NAME PRCP QUANTITY DISTRIBUTION
MENU TEXT Quantity Distribution Report
UPPERCASE MENU TEXT QUANTITY DISTRIBUTION REPORT
DESCRIPTION 
The Quantity Distribution Report will print a comparison between the
normal stock level (entered by the user) and the calculated average
stock distributed for the past year.  This report will help in making
adjustments to the stock levels (temporary, emergency, and normal).
The report will break out the quantity distributed by month which will
help in determining usage trends for items.
 
The report will print the item master number, description, nsn (for
whse), unit per issue, standard reorder point, optional reorder point,
temporary stock level, emergency stock level, and normal stock level.
 
The warehouse will have the option to select specific item nsn's to
display.  The report will sort by nsn.
 
The primary will have the option to select specific Master Item numbers
to display.  The report will sort by description.
 
The secondary will have options similar to the primary but the report will
display recipients instead of distribution points.
CREATOR USER,ONE
TYPE run routine
