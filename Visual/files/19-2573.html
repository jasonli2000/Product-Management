PRCP DIST ORDER LIST TO/FROM (2573)    OPTION (19)

Name Value
NAME PRCP DIST ORDER LIST TO/FROM
MENU TEXT List Distribution Orders To/From Inventory Points
UPPERCASE MENU TEXT LIST DISTRIBUTION ORDERS TO/FR
DESCRIPTION 
The List Distribution Order To/From Inventory Points will display the
distribution orders which have not been posted from the primary to the
secondary inventory point.  This option will allow the primary to print
the picking tickets for all orders which have been released but have
not had a picking ticket previously printed.  The user has the option
to print the orders with or without a breakout of the items on the
order.
CREATOR USER,ONE
