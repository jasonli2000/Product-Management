PRCHPC VEN EDIT (2639)    OPTION (19)

Name Value
NAME PRCHPC VEN EDIT
MENU TEXT Vendor File Edit
UPPERCASE MENU TEXT VENDOR FILE EDIT
DESCRIPTION 
This function allows A&MM Service to maintain a file of Vendors.
For each vendor data is specified for the address, phone number,
account number, method of business, contract number, type of
ownership and special factors.
This file is maintained by A&MM Service.
