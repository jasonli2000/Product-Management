XPD CONVERT PACKAGE (266)    OPTION (19)

Name Value
NAME XPD CONVERT PACKAGE
MENU TEXT Convert Loaded Package for Redistribution
UPPERCASE MENU TEXT CONVERT LOADED PACKAGE FOR RED
DESCRIPTION 
This option will change a Package that was loaded on your system, to
a Package that can be transported in a Distribution.  The loaded Package
will be unloaded from your system and deleted from the Install file. A
Build file entry will be created for this Package. You can use the
'Transport a Distribution' option to create a new Distribution with this
Package.
CREATOR USER,ONE
TYPE run routine
ROUTINE EN2^XPDIU
