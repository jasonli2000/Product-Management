PRCHPC ASSIGN REQUEST (2660)    OPTION (19)

Name Value
NAME PRCHPC ASSIGN REQUEST
MENU TEXT Assign a Request to Purchasing Agent
UPPERCASE MENU TEXT ASSIGN A REQUEST TO PURCHASING
LOCK PRCHASSIGN
DESCRIPTION 
This option allows a A&MM supervisor to assign a 2237 Request to a particular
purchasing agent. This option is locked with security key PRCHASSIGN.
