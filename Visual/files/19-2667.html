Name Value
NAME PRCHOUT PA STATS
MENU TEXT Purchasing Agents Statistics
UPPERCASE MENU TEXT PURCHASING AGENTS STATISTICS
DESCRIPTION 
This option prints the number of purchase orders, total line counts, 
total dollar values, and the average of these totals by purchasing agents 
for a given time period.
CREATOR USER,ONE
TYPE print
DIC {DIP} PRC(442,
L. 0
FLDS [PRCH TOT AMT]
BY [PRCHOUT PA STATS]
DIS(0) N NDUZ S NDUZ=$P($G(^PRC(442,D0,1)),"^",10) I $P($G(^VA(200,+NDUZ,400)),"^")'="",$P($G(^VA(200,+NDUZ,400)),"^")'=1
