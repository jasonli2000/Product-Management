PRCHPC PO REMOVE 2237 (2693)    OPTION (19)

Name Value
NAME PRCHPC PO REMOVE 2237
MENU TEXT Remove 2237 from P.O.
UPPERCASE MENU TEXT REMOVE 2237 FROM P.O.
DESCRIPTION 
This option will remove the line items associated with a 2237 from a Purchase
Order. The line items can then be transferred onto another Purchase Order.
