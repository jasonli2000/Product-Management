PRCHPM CS TRANSMIT (2701)    OPTION (19)

Name Value
NAME PRCHPM CS TRANSMIT
MENU TEXT Transmit Code Sheets to Austin (LOG/GSA/DLA)
UPPERCASE MENU TEXT TRANSMIT CODE SHEETS TO AUSTIN
LOCK PRCHPM CS TRANSMIT
DESCRIPTION 
This option allows the user to transmit 'Batched and Printed' LOG, GSA or
DLA code sheets to Austin.  Code sheets may be released by 'Batch' or
by 'Transmission'.  Appropriate information about the release, such as
date/time released and person who did the release, are recorded automatically.
CREATOR USER,ONE
TYPE run routine
ROUTINE TRANSMIT^PRCFALOG
