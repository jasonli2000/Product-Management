PRCHPM CS DELETE FROM BATCH (2732)    OPTION (19)

Name Value
NAME PRCHPM CS DELETE FROM BATCH
MENU TEXT Delete Code Sheet from Printed Batch (LOG/GSA/DLA)
UPPERCASE MENU TEXT DELETE CODE SHEET FROM PRINTED
LOCK PRCHPM CS TRANSMIT
DESCRIPTION 
This option allows the user to remove a code sheet from a batch prior to
transmission and it adjusts the Batch Header Code Sheet accordingly.  This
option must be used first when moving a code sheet from one batch to
another.
CREATOR USER,ONE
TYPE run routine
ROUTINE DELETE^PRCFALOG
