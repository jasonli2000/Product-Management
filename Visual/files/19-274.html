XMSEND (274)    OPTION (19)

Name Value
NAME XMSEND
X ACTION PRESENT YES
MENU TEXT Send a Message
UPPERCASE MENU TEXT SEND A MESSAGE
DESCRIPTION 
Enables the user to send MailMan messages.  Messages may be sent
to other users or groups of users.  The sender of the message may
request a confirmation message, to be sent to him when each recipient
reads the message.
CREATOR USER,ONE
HELP FRAME XM-U-M-SEND
TYPE run routine
PACKAGE MAILMAN
E ACTION PRESENT YES
EXIT ACTION D CHECKOUT^XM
ENTRY ACTION D CHECKIN^XM
ROUTINE SEND^XMJMS
