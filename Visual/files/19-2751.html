PRCB FCP ADD/EDIT (2751)    OPTION (19)

Name Value
NAME PRCB FCP ADD/EDIT
MENU TEXT Add/Edit Control Point
UPPERCASE MENU TEXT ADD/EDIT CONTROL POINT
DESCRIPTION 
This function allows the Fiscal service to add Control Points
and to edit those existing Control Points.  Data to be
specified includes Control Point Name, controlling service,
ALD code, Control Point official, cost centers and text
to describe the control point.
