PRCFA OBLIGATION PROCESSING (2792)    OPTION (19)

Name Value
NAME PRCFA OBLIGATION PROCESSING
MENU TEXT Obligation Processing
UPPERCASE MENU TEXT OBLIGATION PROCESSING
DESCRIPTION 
This function is used to assign an obligation number to a service
request.  The request is transferred to Fiscal/Accounting from
A&MM service after they have reviewed and confirmed the
order.
After the document has been obligated, the option updates the Control
Point records, when possible, and prints the completed 2138 in A&MM.
