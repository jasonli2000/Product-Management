PRCFA PO BUILD LIST (2802)    OPTION (19)

Name Value
NAME PRCFA PO BUILD LIST
MENU TEXT Build List of POs Printed in Fiscal by Date
UPPERCASE MENU TEXT BUILD LIST OF POS PRINTED IN F
DESCRIPTION 
User is allowed to enter a Beginning and Ending date and time.
The computer will build a list of all Purchase Orders previously
printed in Fiscal (from A&MM) within the date/time range, and then
will print the list on the device selected.
CREATOR USER,ONE
TYPE run routine
