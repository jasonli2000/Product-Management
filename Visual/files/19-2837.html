PRCFA RR NOT PROC BY FISCAL (2837)    OPTION (19)

Name Value
NAME PRCFA RR NOT PROC BY FISCAL
MENU TEXT List Receiving Reports not Processed by Fiscal
UPPERCASE MENU TEXT LIST RECEIVING REPORTS NOT PRO
DESCRIPTION 
Print a report showing all receiving reports that have been signed
by the Warehouse person, but have not been processed by Fiscal.
CREATOR USER,ONE
TYPE print
E ACTION PRESENT YES
ENTRY ACTION I '$D(PRC("SITE")) S %F="S" D ^PRCFSITE S:'$D(PRC("SITE")) XQUIT=""
DIC {DIP} PRC(442,
L. 0
FLDS [PRCHOUT RR NOT PROC.BY FISCAL]
BY [PRCHOUT RR NOT PROC.BY FISCAL]
DIS(0) I $D(^PRC(442,D0,0)),$E(^(0),1,3)=PRC("SITE"),"^1^7^8^"[("^"_$P(^(0),U,2)_"^")
