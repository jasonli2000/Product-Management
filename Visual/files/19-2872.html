PRCFD BULLETIN DUE (2872)    OPTION (19)

Name Value
NAME PRCFD BULLETIN DUE
MENU TEXT Send CI's Due Bulletin
UPPERCASE MENU TEXT SEND CI'S DUE BULLETIN
DESCRIPTION 
This option will generate a bulletin to CP Officials and clerks 
to notify them that a CI is due in Fiscal service on
a specific date.  The date selected will be for those Invoices due in 7 days
from the date this option is run.
 
This option will notify the service ONE time only.
