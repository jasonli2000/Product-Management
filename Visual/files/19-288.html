XMBANNER (288)    OPTION (19)

Name Value
NAME XMBANNER
MENU TEXT Banner Edit
UPPERCASE MENU TEXT BANNER EDIT
DESCRIPTION 
Enables a user to edit his "banner" message for MailMan.
When messages are addressed to users with banners, their banners
may be displayed.
CREATOR USER,ONE
HELP FRAME XM-U-P-BANNER
TYPE run routine
PACKAGE MAILMAN
ROUTINE BANNER^XMVVITA
