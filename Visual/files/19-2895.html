Name Value
NAME PRCEC RECALC 1358 BALANCE
MENU TEXT Recalculate 1358 Balance
UPPERCASE MENU TEXT RECALCULATE 1358 BALANCE
DESCRIPTION 
    This option will allow recalculation of the 1358 balance for an
obligation. It will only allow one control point to be done at a time.
It will check to see if the user is allowed access to that control
point. This is a control point option.
