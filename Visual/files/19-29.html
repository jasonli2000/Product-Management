Name Value
NAME XUXREF
MENU TEXT List Options by Parents and Use
UPPERCASE MENU TEXT LIST OPTIONS BY PARENTS AND US
DESCRIPTION 
This option produces a cross reference listing of all options, showing
their parents on the menu tree, detecting bad pointers, and showing
which options have no parents.  It also shows the use(s) of the option as
a primary menu option, a secondary menu option, or a tasked option, or
a combination of these.
CREATOR USER,ONE
HELP FRAME XQXREF
TYPE run routine
ROUTINE XQ31
