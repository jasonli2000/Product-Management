PRCD DOC REQUIRED DATA (2943)    OPTION (19)

Name Value
NAME PRCD DOC REQUIRED DATA
MENU TEXT Required Fields Edit
UPPERCASE MENU TEXT REQUIRED FIELDS EDIT
DESCRIPTION 
This option allows the user to edit the Required Field file settings,
which allow IFCAP to determine the fields that must be prompted for during
various processes that collect data for FMS documents.  See also, "Required
Fields List".
