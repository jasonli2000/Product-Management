PRC GECS STACK RETRANSMIT (2953)    OPTION (19)

Name Value
NAME PRC GECS STACK RETRANSMIT
MENU TEXT Retransmit Stack File Document
UPPERCASE MENU TEXT RETRANSMIT STACK FILE DOCUMENT
LOCK PRCFA SUPERVISOR
DESCRIPTION 
This option will retransmit the document located in the stack file.
The option should be used to retransmit those documents which have not
been received (no confirmation message).  Retransmitting received
documents may lead to rejects.
CREATOR USER,ONE
TYPE action
PACKAGE IFCAP
E ACTION PRESENT YES
ENTRY ACTION S GECSSYS="FINANCIAL MANAGEMENT" D STACRETR^GECSCALL
