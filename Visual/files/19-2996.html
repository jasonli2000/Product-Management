PRCFA RETRANSMIT BATCH (2996)    OPTION (19)

Name Value
NAME PRCFA RETRANSMIT BATCH
MENU TEXT Retransmit Code Sheets Batch to Austin
UPPERCASE MENU TEXT RETRANSMIT CODE SHEETS BATCH T
LOCK PRCFA TRANSMIT
DESCRIPTION 
This option allows a user to retransmit a batch to Austin.  It
must be determined by the user that the original transmission failed or a
duplicated transmission could occur.
