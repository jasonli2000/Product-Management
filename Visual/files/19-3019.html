Name Value
NAME PRCAC TR ADM
MENU TEXT Add an Administrative Cost
UPPERCASE MENU TEXT ADD AN ADMINISTRATIVE COST
DESCRIPTION 
This option adds an administrative cost to the  accounts receivable.
The cost could be marshal fee, court cost, IRS Loc. cost, Credit Rep. cost
, Dept. of Motor Vehicles Loc. cost, Consumer Rep. Agency cost, etc.
These costs are accrued to the administrative cost balance automatically.
