Name Value
NAME PRCAC DCDOJ REFER
MENU TEXT Refer to RC/DOJ
UPPERCASE MENU TEXT REFER TO RC/DOJ
DESCRIPTION 
This option is used to refer the selected bills to Regional Counsel or the
Department of Justice.  This option should not be used for Third Party
Bills which are electronically transferred.  The referral process consists
of creating the appropriate AR Transaction and updating the bill
information.
 
If the bill is currently referred, the user may cancel the referral,
using this option.
