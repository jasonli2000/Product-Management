UPPERCASE MENU TEXT EDIT FORM LETTERS
DESCRIPTION 
The Supervisor can use this option to edit the text of a form letter.
Great caution should be taken NOT to alter the contents of the
window functions contained in vertical bars - '|  |'. Formatting
of the printed letter will be adversely affected.
