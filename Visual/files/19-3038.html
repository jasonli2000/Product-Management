Name Value
NAME PRCAA SET/AUDIT NEW BILL
MENU TEXT Audit/Set up a New Accounts Receivable
UPPERCASE MENU TEXT AUDIT/SET UP A NEW ACCOUNTS RE
DESCRIPTION 
This menu has options that allow the AR Clerk to audit bills and
establish new accounts. An account for the debtor must exist before
the audit of a bill can be completed.  The system will prompt for
account information if necessary.
