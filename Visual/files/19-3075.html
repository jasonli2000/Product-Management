PRCAC DCDOJ REREFER (3075)    OPTION (19)

Name Value
NAME PRCAC DCDOJ REREFER
MENU TEXT Re-Refer to RC/DOJ
UPPERCASE MENU TEXT RE-REFER TO RC/DOJ
DESCRIPTION 
Use this option to refer bills that are currently referred to the
Department of Justice or Regional Counsel.  This option should not be
used for Third Party Bills that are referred electronically.
This option will create a Reestablish to RC/DOJ AR Transaction  
and update the appropriate AR File 430 fields.
