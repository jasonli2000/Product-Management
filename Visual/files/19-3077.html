PRCAC DCDOJ WAIVER (3077)    OPTION (19)

Name Value
NAME PRCAC DCDOJ WAIVER
MENU TEXT Waived by RC/DOJ
UPPERCASE MENU TEXT WAIVED BY RC/DOJ
DESCRIPTION 
Use this option to apply the transaction for an account that is being
waived by the Regional Counsel or Department of Justice.  The account
should have already been referred to the RC/DOJ for this option to
function.
This option should not be used for Third Party Bills that are referred
electronically.
CREATOR USER,ONE
TYPE run routine
PACKAGE ACCOUNTS RECEIVABLE
ROUTINE A10^RCWROFF
