PRCAC DCDOJ RETURNED (3080)    OPTION (19)

Name Value
NAME PRCAC DCDOJ RETURNED
MENU TEXT Returned by RC/DOJ
UPPERCASE MENU TEXT RETURNED BY RC/DOJ
DESCRIPTION 
Use this option to record the date returned by RC/DOJ for an account that
was referred.  This option should be used for recording accounts that are
returned by RC/DOJ because the debt has been cleared as a result of
corrective award action, or accounts that are returned and will
be cleared by appropriate write-off because of death, bankruptcy, etc.
This option should not be used for Third Party bills that are 
electronically referred.
