PRCAX CO-PAY EXEMPTION REPORT (3129)    OPTION (19)

Name Value
NAME PRCAX CO-PAY EXEMPTION REPORT
MENU TEXT Medication Co-pay Exemption Report
UPPERCASE MENU TEXT MEDICATION CO-PAY EXEMPTION RE
DESCRIPTION 
This option produces a report showing the medication co-pay exemption
adjustments applied to the system for a given time period selected by the
user.  The report will display the patient name, patient id, bill number,
transaction number, and transaction amount for the adjustment.
