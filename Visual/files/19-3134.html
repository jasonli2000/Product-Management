Name Value
NAME PRCA LIST ALL BILLS
MENU TEXT List All Bills
UPPERCASE MENU TEXT LIST ALL BILLS
DESCRIPTION 
This option will list all bills for the service of the person that
has generated the report.  The report will display information,
such as: Form Type, Original Amount, Current Balance, Debtor, and
Bill # sort by Status.
