PRCAK ARCHIVE MARK PRINT (3141)    OPTION (19)

Name Value
NAME PRCAK ARCHIVE MARK PRINT
MENU TEXT Detailed Report of Pending Archive Records
UPPERCASE MENU TEXT DETAILED REPORT OF PENDING ARC
DESCRIPTION 
This option will print a report of all the AR records that are
flagged for Archive. (The status is 'PENDING ARCHIVE').  This report
includes the Bill no., Debtor name, category, the previous status
before it was marked for Archive, the balance and the date of the last
activity.
 
A total of entries marked as ARCHIVE will print at the end of the
report.  This total may be very useful in determining the amount of
systems activity the archive processes will create.
