NAME PRCAE PR STATEMENT
MENU TEXT Reprint Patient Statements
UPPERCASE MENU TEXT REPRINT PATIENT STATEMENTS
DESCRIPTION 
This option will reprint Patient Statements for the selected date.
Enter a patient range in print order to have only the statements in that
sequence reprint or do not select a range to reprint all statements for
the selected date.
 
NOTE:  This option will not reprint Follow-up Letters.
