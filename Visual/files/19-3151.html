PRCAE REPRINT UB (3151)    OPTION (19)

Name Value
NAME PRCAE REPRINT UB
MENU TEXT Reprint UB Letters
UPPERCASE MENU TEXT REPRINT UB LETTERS
DESCRIPTION 
This option allows you to reprint a UB Letter for the selected
date.  You may enter a range of bills to print (print order range)
or have all the UB Letters reprint for that date by not selecting
a bill to start or end the sort.
CREATOR POSTMASTER
TYPE action
