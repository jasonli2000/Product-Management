Name Value
NAME PRCAK AR SUPERVISOR
MENU TEXT Archive Menu
UPPERCASE MENU TEXT ARCHIVE MENU
DESCRIPTION 
This option allows the AR Supervisor access to the Archive menu.
The Archive process should be coordinated between IRM and Fiscal.
The options available will allow Fiscal service to list all
bills that are pending Archive and also remove the bill from the
list before IRM archives the bill.
