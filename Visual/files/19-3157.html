NAME PRCA DISC LIST
X ACTION PRESENT YES
MENU TEXT Statement Discrepancy Listing
UPPERCASE MENU TEXT STATEMENT DISCREPANCY LISTING
DESCRIPTION 
This option will list all the AR Debtors whose accounts do not
balance. Use this option to generate a list of debtors who are
not receiving statements because of balance discrepancies.
 
This option takes awhile to run. You should queue this report
to a printer.
