IBT MONTHLY AUTO GEN AVE BILL (3204)    OPTION (19)

Name Value
NAME IBT MONTHLY AUTO GEN AVE BILL
MENU TEXT Auto-Build Average Bill Amounts
UPPERCASE MENU TEXT AUTO-BUILD AVERAGE BILL AMOUNT
DESCRIPTION 
This option will calculate the number of bills and the average bill
amounts for a month and store the data in the CLAIMS TRACKING UNBILLED
AMOUNTS file (356.19).  This data will then be used by the scheduled
option Auto-Build Unbilled Amounts Report (IBT MONTHLY AUTO GEN UNBILLED)
to generate the unbilled amounts data that needs to be reported by the 3rd
work day of the month.
 
Queue this option to run once monthly.  Sites may choose the date it
should run but it is suggested that it run after the 15th of the month
when user activity is low  (i.e. November 19, 1994 @ 2:00am).  No device
is necessary, the results are stored and a completion mail message is sent
to the mail group specified in the IB SITE PARAMETERS file.
CREATOR POSTMASTER
TYPE run routine
PACKAGE INTEGRATED BILLING
ROUTINE AUTO^IBTUBAV
*RESCHEDULING FREQUENCY 1M
SCHEDULING RECOMMENDED YES
