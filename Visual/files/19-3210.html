IBT SEND TEST UNBILLED MESS (3210)    OPTION (19)

Name Value
NAME IBT SEND TEST UNBILLED MESS
MENU TEXT Send Test Unbilled Amounts Bulletin
UPPERCASE MENU TEXT SEND TEST UNBILLED AMOUNTS BUL
DESCRIPTION 
This option allows for sending of a test mail message to the mail group to
receive the Unbilled Amounts messages.  Using this prior to reporting
problems can assist sites in determining whether the mail groups are set
up correctly. The mail group to get the message should be specified in
field UNBILLED MAIL GROUP (6.25) in the IB SITE PARAMETERS file (350.9).
