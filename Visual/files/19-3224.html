PSNVFY (3224)    OPTION (19)

Name Value
NAME PSNVFY
MENU TEXT Verify Matches
UPPERCASE MENU TEXT VERIFY MATCHES
DESCRIPTION 
Using this option, you can verify, one at a time, the unverified drugs that
have been matched using the option "Rematch / Match Single Drugs". The
unverified matches are stored in the National Drug Translation file. It
is not necessary to verify all of the matches at once. It will also allow
you to rematch a drug if you do not like the match. It is a requirement of
the software that all matched drugs be verified. The same person should
not match the drugs and then verify the matches.
CREATOR USER,ONE
TYPE run routine
ROUTINE PSNVFY
