Name Value
NAME PSNMRG
MENU TEXT Merge National Drug File Data Into Local File
UPPERCASE MENU TEXT MERGE NATIONAL DRUG FILE DATA
DESCRIPTION 
This option merges the following data from National Drug File into local
drug file: the pointer to VA Generic file, VA Product Name, pointer to
VA Product file, pointer to Package Size and Type, and VA Drug Classification.
This option will also print a report listing any drugs in the National Drug
Translation File that were not merged. There must be at least one matched
and verified drug in the National Drug Translation file for the merge to
work. The printout of the "error report" will require 80 columns.
CREATOR USER,ONE
TYPE run routine
ROUTINE PSNMRG
