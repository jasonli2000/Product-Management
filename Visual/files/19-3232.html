PSNSTCL (3232)    OPTION (19)

Name Value
NAME PSNSTCL
MENU TEXT Allow Unmatched Drugs To Be Classed
UPPERCASE MENU TEXT ALLOW UNMATCHED DRUGS TO BE CL
LOCK PSNMGR
DESCRIPTION 
This option allows unmatched drugs to be classed. You may class one drug at
a time by generic name or internal number in your local file or you may
choose to go through a "looping" process alphabetically for all drugs that
do not have an inactive date and have not been merged. You may not change
the class of a merged drug. You must use the option "Rematch / Match
Single Drugs" to change the class of a merged drug.
CREATOR USER,ONE
TYPE run routine
ROUTINE PSNSTCL
