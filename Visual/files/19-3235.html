PSNVIEW (3235)    OPTION (19)

Name Value
NAME PSNVIEW
MENU TEXT Inquire To Local Drug File
UPPERCASE MENU TEXT INQUIRE TO LOCAL DRUG FILE
DESCRIPTION 
This option allows you to look-up into your local drug file by Generic
Name or VA Drug Class Code. If you choose VA Drug Class Code, it will
sort by the least expensive Price Per Dispense Unit. It will display
the Generic Name, VA Drug Class Code, Price Per Dispense Unit, the
Message field, and Synonyms not marked as quick codes.
CREATOR USER,ONE
TYPE run routine
ROUTINE PSNVIEW
