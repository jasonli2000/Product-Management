PSO PNDEL (3254)    OPTION (19)

Name Value
NAME PSO PNDEL
MENU TEXT Auto-delete from Suspense
UPPERCASE MENU TEXT AUTO-DELETE FROM SUSPENSE
DESCRIPTION 
This option deletes printed Rx's from the suspense file if they were
printed over the site specified number of days to remain in the suspense
file. Once this job is queued, it is then set to run every 7 days.
CREATOR USER,ONE
TYPE run routine
PACKAGE OUTPATIENT PHARMACY
