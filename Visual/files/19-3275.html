PSO AMIS (3275)    OPTION (19)

Name Value
NAME PSO AMIS
MENU TEXT AMIS Report
UPPERCASE MENU TEXT AMIS REPORT
DESCRIPTION 
This option lists prescription statistics which are required
by the VA Central Office. If you are a multidivisional site,
the print device will report each division's statistics on a
separate page with the grand totals on the last page. This report
must be printed on a 132 column printer.
