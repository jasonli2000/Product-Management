NAME PSO ACTION PROFILE
MENU TEXT Action Profile (132 COLUMN PRINTOUT)
UPPERCASE MENU TEXT ACTION PROFILE (132 COLUMN PRI
DESCRIPTION 
The action profile is intended to be used in the clinics and areas outside
of the pharmacy to get this information to the doctor.  It can be printed
by patient(s) or clinic(s) and is sorted accordingly.  The option is placed
here to allow the pharmacy to have access for general exposure, as it is
intended to be run by the clinics, etc., primarily.  
This printout must be printed on a 132 column printer.
