PSO INTERVENTION PRINTOUT (3334)    OPTION (19)

Name Value
NAME PSO INTERVENTION PRINTOUT
MENU TEXT Print Pharmacy Intervention
UPPERCASE MENU TEXT PRINT PHARMACY INTERVENTION
LOCK PSORPH
DESCRIPTION 
This option is used to obtain a captioned printout of Pharmacy
Interventions for a certain date range.  It will print out on
normal width paper and can be queued to print at a later time.
CREATOR USER,ONE
HELP FRAME PSO PRINT INTERVENTIONS
TYPE action
PACKAGE OUTPATIENT PHARMACY
E ACTION PRESENT YES
