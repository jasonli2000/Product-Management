XMUT-REC-FIND (335)    OPTION (19)

Name Value
NAME XMUT-REC-FIND
MENU TEXT Find Messages for User
UPPERCASE MENU TEXT FIND MESSAGES FOR USER
DESCRIPTION 
Use this option to find all the messages that a user has ever had and has
not terminated from.  This list of messages can be used as input into two
other processes:  Deliver Found Messages into User's IN Basket, and List
Messages Found.
CREATOR USER,ONE
HELP FRAME XM-I-D-RECOVER MSGS-FIND
TYPE run routine
ROUTINE A^XMUT1
