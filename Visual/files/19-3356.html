PSO AUTOCANCEL1 (3356)    OPTION (19)

Name Value
NAME PSO AUTOCANCEL1
MENU TEXT Autocancel Rx's on Admission
UPPERCASE MENU TEXT AUTOCANCEL RX'S ON ADMISSION
DESCRIPTION 
This option should be set to run at a time between 5:30pm and 11:30 (as
convenient for the site). It will check all admissions for T-3, and if 
the patient is still in the hospital, will cancel all his active Rx's,
delete all those in a non-verified status, and delete those in the 
RX SUSPENSE file.
