PSO MGMT MONTHLY TYPE OF RX (3382)    OPTION (19)

Name Value
NAME PSO MGMT MONTHLY TYPE OF RX
MENU TEXT Type of Prescriptions Filled
UPPERCASE MENU TEXT TYPE OF PRESCRIPTIONS FILLED
DESCRIPTION 
This option prints the monthly Type of Prescriptions Filled Management
Report. It contains the number of prescriptions filled by fee and staff
physicians, new and refill prescriptions,  prescriptions sent by mail or 
dispensed at the window,  prescriptions filled by participating pharmacies,
and investigational drug prescriptions.
CREATOR USER,ONE
TYPE run routine
PACKAGE OUTPATIENT PHARMACY
E ACTION PRESENT YES
ENTRY ACTION S RUN=2
ROUTINE PSOMGMRP
