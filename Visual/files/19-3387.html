Name Value
NAME PSO MGMT RPT DAILY COMPILE
MENU TEXT Initialize Daily Compile
UPPERCASE MENU TEXT INITIALIZE DAILY COMPILE
DESCRIPTION 
This option recompiles the management data for a specific date range and
tasks a job to run every day at 1:00am to compile the previous day's
Management Report data.
