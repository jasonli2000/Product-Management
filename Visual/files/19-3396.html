PRCH ALL STATUS AMEND TO PO (3396)    OPTION (19)

Name Value
NAME PRCH ALL STATUS AMEND TO PO
X ACTION PRESENT YES
MENU TEXT All Status Amendment to PO
UPPERCASE MENU TEXT ALL STATUS AMENDMENT TO PO
LOCK PRCH TRANSACTION COMPLETE
DESCRIPTION 
This option allows the user to amend a Purchase Order even if the Purchase
Order is in a status of Transaction Complete.   
This option is locked with the Security Key PRCH TRANSACTION COMPLETE.
CREATOR USER,ONE
TYPE run routine
E ACTION PRESENT YES
