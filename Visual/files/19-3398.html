DESCRIPTION 
This option will allow FISCAL to review VENDOR REQUESTS prior to being
sent to Austin.  This option was added to reduce the number of VRQs
going to Austin.  It is expected that the user reviewing the 
VRQs will check with the VENDOR file in FMS, get any information
available, and edit the vendor locally rather than sending a VRQ to
Austin.
