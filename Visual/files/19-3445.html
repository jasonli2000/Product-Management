GMRA PRINT-EXCEPT BY DATE (3445)    OPTION (19)

Name Value
NAME GMRA PRINT-EXCEPT BY DATE
MENU TEXT Print All FDA Exceptions within a D/T Range
UPPERCASE MENU TEXT PRINT ALL FDA EXCEPTIONS WITHI
DESCRIPTION 
This option allows the user to select a date/time range and then displays
all reactions that:
 1) are not entered-in-error,
 2) are observed drug reactions,
 3) are marked as 'signed-off' by the originator of the entry,
 4) have an origination date/time within the date/time range selected by
    the user, and
 5) do not have the minimum amount of FDA data needed.
     
The FDA data needed is:
 a) date/time of the reaction,
 b) patient,
 c) date/time the reaction was reported,
 d) person who reported the reaction,
 e) the reaction itself,
 f) the suspected agent, and
 g) at least one of the following questions must be answered
    1) Patient died?
    2) Reaction treated with RX drug?
    3) Life Threatening illness?
    4) Required hospitalization?
    5) Prolonged Hospitalization?
    6) Resulted in permanent disability?
    7) Patient recovered?
    8) Is this event a Congenital Anomaly?
    9) Did this event require intervention to prevent impairment?
    10) Required ER/MD visit?
