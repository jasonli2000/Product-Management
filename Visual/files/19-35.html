XQALERT BY USER DELETE (35)    OPTION (19)

Name Value
NAME XQALERT BY USER DELETE
MENU TEXT Purge Alerts for a User
UPPERCASE MENU TEXT PURGE ALERTS FOR A USER
LOCK XQAL-DELETE
DESCRIPTION 
This option is provided to permit users holding the correct key to delete
alerts for another user.  This is intended for when a user has been
inactive for a period of time (e.g., vacation, etc.) and has accumulated a
number of alerts which should not need processing.
CREATOR USER,ONE
TYPE run routine
ROUTINE USERDEL^XQALERT
