MENU TEXT Inactivate Records Menu
UPPERCASE MENU TEXT INACTIVATE RECORDS MENU
DESCRIPTION 
If a record is going to be taken out of service it is inactivated so
it is not requested, charged out, etc.
This menu includes the options for Record Retirement tasks.
The options are ordered as a guide for the work flow.  Pull lists
are generated, printed, managed, perpetual records created and 
then the record retirement pull lists are 'charged out' when the
records are transferred to the appropriate facility.
