PSJU EWG (3618)    OPTION (19)

Name Value
NAME PSJU EWG
MENU TEXT Ward Groups
UPPERCASE MENU TEXT WARD GROUPS
DESCRIPTION 
  Oft times a Pharmacy 'ward' will actually encompass several MAS wards.
Therefore, this package uses WARD GROUPS.  This option allows the user to
name their ward groups and select which MAS wards fall under each one.  If
there is a ward that stands alone, a ward group must be defined (probably
with the same name), with that ward under it.
  When a new WARD GROUP is entered, it's TYPE is asked (PHARMACY and MAS
are the choices).  This field is NOT editable.  No two ward groups marked
as PHARMACY may contain the same ward, while MAS ward groups may be
arranged and rearranged, as desired.  Only PHARMACY ward groups are select-
able at the PICK LIST options, while any ward group is selectable at the
other print options.
  The parameters that determine how pick lists for a ward group should print
are now editable through this option, allowing the site to make such
determinations by ward group.
