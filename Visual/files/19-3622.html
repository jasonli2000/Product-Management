PSJU VBW (3622)    OPTION (19)

Name Value
NAME PSJU VBW
MENU TEXT Non-Verified/Pending Orders
UPPERCASE MENU TEXT NON-VERIFIED/PENDING ORDERS
DESCRIPTION 
  Allows the user to verify (or take other actions) on non-verified orders
for patients.  If the user is a pharmacist, this option will show all
orders not verified by a pharmacist for a ward group, ward, or single
patient, as the user chooses.  If the user is a nurse, this option will
show all orders not verified by a nurse.  If the user is ward staff,
this option will show patients with orders not verified by a nurse,
but not allow any action to be taken.
