PSJU INQ DRUG (3626)    OPTION (19)

Name Value
NAME PSJU INQ DRUG
MENU TEXT Dispense Drug Look-Up
UPPERCASE MENU TEXT DISPENSE DRUG LOOK-UP
DESCRIPTION 
  Allows any user to view any item in the Drug file (#50).  The user will be
shown the fields pertaining to the Unit Dose package.  This option is
a "view" option only, and affects no data.
