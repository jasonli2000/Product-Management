PSJU CPDD (3647)    OPTION (19)

Name Value
NAME PSJU CPDD
MENU TEXT Edit Patient's Default Stop Date
UPPERCASE MENU TEXT EDIT PATIENT'S DEFAULT STOP DA
LOCK PSJU PL
DESCRIPTION 
  Allows a pharmacist (or pharmacy technician) to edit the selected
patient's Default Stop Date.  If the corresponding site parameter is
turned on, this date is used as a default stop date value when orders
are entered or renewed for the patient.  The entry and renewal processes
may change this date.
