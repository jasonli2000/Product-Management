PSJI INDIVIDUAL SUSPENSE (3655)    OPTION (19)

Name Value
NAME PSJI INDIVIDUAL SUSPENSE
X ACTION PRESENT YES
MENU TEXT Individual Order Suspension (IV)
UPPERCASE MENU TEXT INDIVIDUAL ORDER SUSPENSION (I
DESCRIPTION 
  This option allows the user to suspend labels for a specific active order.
These labels will then be available for release by the IV room for doses
due within the present coverage interval, or for the next interval if the 
Ward list for that batch has already been run.
CREATOR USER,ONE
TYPE action
E ACTION PRESENT YES
EXIT ACTION K FIL,I1,ND,ORIFN,PSGLMT,PSIVUP,PSIVX,PSJIVORF,PSJORL,UL80
