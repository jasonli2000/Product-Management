PSJI PURGE ORDERS (3657)    OPTION (19)

Name Value
NAME PSJI PURGE ORDERS
X ACTION PRESENT YES
MENU TEXT Purge Expired Orders (IV)
UPPERCASE MENU TEXT PURGE EXPIRED ORDERS (IV)
OUT OF ORDER MESSAGE TEMPORARILY UNAVAILABLE
DESCRIPTION 
  This option will allow the user to purge all expired and DC'ed IV orders 
that are at least 30 days old.  Any orders that are purged from the database
CANNOT be retrieved. Orders which have an expiration date following the date
selected will remain on file.  Orders which have been inactive for less than
30 days cannot be purged.  Because of the large number of orders entered in 
the IV package, the Purge option should be run at least once a month, to 
prevent decreased performance caused by compiling orders which are no longer
needed.
CREATOR USER,ONE
TYPE action
E ACTION PRESENT YES
EXIT ACTION K %T,CAO,D,DFN,DR,N,ON,ORETURN,ORPV,ORSTOP,ORSTRT,ORSTS,P,P17,PSGTOL,PSGTOO,PSGUOW,PSIV,PSIVREA,PSJS,TN,WCA,Y
ENTRY ACTION D ^PSIVXU Q:$D(XQUIT) D ^PSIVPGE
