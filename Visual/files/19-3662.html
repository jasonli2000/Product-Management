PSJI LBLS (3662)    OPTION (19)

Name Value
NAME PSJI LBLS
X ACTION PRESENT YES
MENU TEXT Scheduled Labels (IV)
UPPERCASE MENU TEXT SCHEDULED LABELS (IV)
DESCRIPTION 
  This option prints labels for a particular scheduled manufacturing
time.  Usually a manufacturing list has been run prior to the printing
of the labels.  If this has been done, the labels will print immediately
and in the same order as on the manufacturing list.  The use of this option
also causes workload counts in the IV STATS file.
CREATOR USER,ONE
TYPE action
E ACTION PRESENT YES
EXIT ACTION K I1
ENTRY ACTION D ^PSIVXU Q:$D(XQUIT) D ^PSIVLBL1,ENIVKV^PSGSETU
