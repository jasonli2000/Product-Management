PSJI UP (3671)    OPTION (19)

Name Value
NAME PSJI UP
MENU TEXT Update Daily Ward List (IV)
UPPERCASE MENU TEXT UPDATE DAILY WARD LIST (IV)
DESCRIPTION 
  This option allows the user to update the Ward List.  He/she can
change the number of labels needed, DC an order, renew it or put
it on hold.  One is allowed to jump within a ward to any patient using
"^Patient Name".  "^" alone means end this update.
CREATOR USER,ONE
TYPE action
E ACTION PRESENT YES
ENTRY ACTION D ^PSIVXU Q:$D(XQUIT) D ^PSIVUWL
