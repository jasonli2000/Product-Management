DESCRIPTION 
  This option is designed for use by nursing staff to allow access to
several inpatient functions.  The user is allowed to perform drug inquiries,
view inpatient profiles, place orders, print labels, update and print ward
lists, print manufacturing lists, perform suspense functions, enter returned
and destroyed IV data, and change inpatient sites.
