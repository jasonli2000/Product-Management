PSJI SUSREP (3685)    OPTION (19)

Name Value
NAME PSJI SUSREP
X ACTION PRESENT YES
MENU TEXT Reprint Labels from Suspense (IV)
UPPERCASE MENU TEXT REPRINT LABELS FROM SUSPENSE (
DESCRIPTION 
This option will allow suspended labels that have been printed
to be reprinted.  Each time labels are printed from suspense they
are assigned a batch number.  When a "batch" of labels have to be
reprinted, that batch number is entered and the labels will be
reprinted.
CREATOR USER,ONE
TYPE action
E ACTION PRESENT YES
EXIT ACTION K %T,Y,ZTSK
ENTRY ACTION D ^PSIVXU Q:$D(XQUIT) D ^PSIVSUS1
