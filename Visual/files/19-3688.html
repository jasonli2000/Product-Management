PSJI SUSLBDEL (3688)    OPTION (19)

Name Value
NAME PSJI SUSLBDEL
X ACTION PRESENT YES
MENU TEXT Delete Labels from Suspense (IV)
UPPERCASE MENU TEXT DELETE LABELS FROM SUSPENSE (I
DESCRIPTION 
  Allows labels to be deleted from suspense.  This option allows deletion
of labels for orders which have been placed on suspense.  The order is not
affected by this option, only the labels are deleted.
CREATOR USER,ONE
TYPE run routine
EXIT ACTION K I1,PSJORL
ROUTINE PSIVLBDL
