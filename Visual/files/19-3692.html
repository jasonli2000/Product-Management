DESCRIPTION 
  This option generates a report which captures the total IV dispensing
cost for a specific patient.  The user enters the range of dates for
which data is to be captured, and the name of the selected patient.  The
report first lists the patient name, SSN, DOB, ward, weight, diagnosis, 
and the date range selected.  For each drug ordered, the drug name, amount  
dispensed, destroyed, recycled, or cancelled is listed, along with the 
total cost of each drug and a grand total for the selected patient.
