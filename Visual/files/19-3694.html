NAME PSJI MANAGEMENT REPORTS
X ACTION PRESENT YES
MENU TEXT Management Reports (IV)
UPPERCASE MENU TEXT MANAGEMENT REPORTS (IV)
DESCRIPTION 
  This menu contains options which allow the printing of reports from data
compiled by the IV Medications module.  Access to this option is restricted
to users with the "PSJI MGR" security key.  Those reports requiring 132
column paper are so noted in the name of the report.  If no paper width
is indicated, the standard 80 column width is assumed.  At the device
prompt, you may either accept the default printing device by pressing
the return key, or enter the name of the device where the report should
print.
