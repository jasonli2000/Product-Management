PSXR AUTO TRANSMIT (3815)    OPTION (19)

Name Value
NAME PSXR AUTO TRANSMIT
MENU TEXT Setup Auto-transmission
UPPERCASE MENU TEXT SETUP AUTO-TRANSMISSION
LOCK PSXAUTOX
DESCRIPTION 
This options schedules or unschedules the CMOP data transmission
to run automatically in the background. The user is asked to enter a
re-scheduling frequency, in hours. The job automatically re-schedules to 
run for the time interval selected. The job will continue to run 
for the selected time intervals until it is unscheduled by the user.
