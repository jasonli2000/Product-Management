XMBLOBSEND (384)    OPTION (19)

Name Value
NAME XMBLOBSEND
X ACTION PRESENT YES
MENU TEXT Send Multimedia Message
UPPERCASE MENU TEXT SEND MULTIMEDIA MESSAGE
OUT OF ORDER MESSAGE MailMan is no longer used to send IMAGING files.
LOCK XMBLOB
DESCRIPTION 
This option is used to send MailMan multimedia messages.  Messages may be
sent to other users or groups of users.  The sender of the message may
request a confirmation message, to be sent to him when each recipient
reads the message.
CREATOR USER,ONE
HELP FRAME XM-U-M-SEND
TYPE run routine
E ACTION PRESENT YES
EXIT ACTION D UNLOCK^XM K XMMENU
ENTRY ACTION K XMLOAD,XMDF S XMMENU(0)="XMUSER" D LOCK^XM
ROUTINE BLOB^XMA2B
