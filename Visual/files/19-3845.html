PSX COST DRUG COST BY DRUG (3845)    OPTION (19)

Name Value
NAME PSX COST DRUG COST BY DRUG
MENU TEXT Drug Cost by Drug Report
UPPERCASE MENU TEXT DRUG COST BY DRUG REPORT
DESCRIPTION 
This option prints a report of Drug Cost by drug. The report lists the
drug name, number of original fills, number of refills, total fills, total
cost, and average cost per fill. The report can be printed for all
facilities or a specific facility and all divisions or a specific division.
If the facility has more than one division and all divisions are
selected, a summary sheet prints. The summary report contains
a total line for each division and a grand total line for the facility.
