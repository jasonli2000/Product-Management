PSX COST PURGE (3847)    OPTION (19)

Name Value
NAME PSX COST PURGE
MENU TEXT Purge Cost Data
UPPERCASE MENU TEXT PURGE COST DATA
LOCK PSXCOST
DESCRIPTION 
This option deletes data from the CMOP COST STATS file (#552.5) . The data
is always deleted from the oldest date in the file to the date specified
by the user. Data for the three most recent complete months must remain in
the file.
CREATOR USER,ONE
TYPE run routine
PACKAGE CMOP
ROUTINE PSXCSTPG
