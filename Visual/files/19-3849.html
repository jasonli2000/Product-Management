PSX COST DRUG COST BY FAC (3849)    OPTION (19)

Name Value
NAME PSX COST DRUG COST BY FAC
MENU TEXT Drug Cost by Facility Report
UPPERCASE MENU TEXT DRUG COST BY FACILITY REPORT
DESCRIPTION 
This option prints a summary report of Drug Cost by facility and division. 
The report lists the division, total original fills, total refills,
total fills, total cost, and average cost per fill. It prints all drugs
for all facilities or a specific facility, all divisions or a specific 
division.
