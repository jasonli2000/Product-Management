PSX COST HIGH DRUG COST REPORT (3850)    OPTION (19)

Name Value
NAME PSX COST HIGH DRUG COST REPORT
MENU TEXT High Cost Rx Report
UPPERCASE MENU TEXT HIGH COST RX REPORT
DESCRIPTION 
This option prints the High Drug Cost report for all facilities or a
specific facility. The report lists the prescription number,
drug name, quantity dispensed, drug cost, and total drug cost
for each prescription. The CMOP MASTER DATABASE file (#552.4) is searched
for prescriptions with the total cost greater than or equal to a
dollar amount specified by the user.
