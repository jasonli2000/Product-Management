PSX COST DATE RANGE COMPILE (3851)    OPTION (19)

Name Value
NAME PSX COST DATE RANGE COMPILE
MENU TEXT Date Range Compile/Recompile Cost Data
UPPERCASE MENU TEXT DATE RANGE COMPILE/RECOMPILE C
LOCK PSXCOST
DESCRIPTION 
This option compiles or recompiles cost data for a date range specified by
the user. Before the data for the specified date range is compiled, the
data is deleted from the CMOP COST STATS file (#552.5).
CREATOR USER,ONE
TYPE run routine
PACKAGE CMOP
ROUTINE RECOM^PSXCST
