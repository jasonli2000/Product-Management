PSD NAOU INV GROUP SORT (3891)    OPTION (19)

Name Value
NAME PSD NAOU INV GROUP SORT
MENU TEXT Sort NAOUs in Inventory Group
UPPERCASE MENU TEXT SORT NAOUS IN INVENTORY GROUP
DESCRIPTION 
This option allows the user to change the order in which the NAOUs
will sort within an inventory group.  NAOUs should sort in the order
that they will be visited.
