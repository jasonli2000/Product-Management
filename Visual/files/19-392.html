XMMGR-MAIL-GRP-COORDINATOR (392)    OPTION (19)

Name Value
NAME XMMGR-MAIL-GRP-COORDINATOR
MENU TEXT Mail Group Coordinator's Edit
UPPERCASE MENU TEXT MAIL GROUP COORDINATOR'S EDIT
DESCRIPTION 
This option allows a mail group coordinator to edit the mail groups that
he or she is the coordinator of (and no others).  It does not allow edit of
remote recipients.
 
Some mail group coordinators might be given the corresponding option,
XMMGR-MAIL-GRP-COORD-W/REMOTE.
CREATOR USER,ONE
TYPE run routine
PACKAGE MAILMAN
ROUTINE LCOORD^XMVGROUP
