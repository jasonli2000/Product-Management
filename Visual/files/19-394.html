Name Value
NAME XMEDIT-DOMAIN-VALIDATION#
MENU TEXT Validation Number Edit
UPPERCASE MENU TEXT VALIDATION NUMBER EDIT
DESCRIPTION 
This option allows sites to edit the validation numbers so that they can be
synchronized.  The sender and the receiver must edit the numbers concurrently. 
At S1.DOMAIN.EXT they edit S2.DOMAIN.EXT's validation number.  At S2.DOMAIN.EXT they edit
S1.DOMAIN.EXT's validation number.  The same number is used for input at both
sites.
