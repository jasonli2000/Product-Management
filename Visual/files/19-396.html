XMMGR-DIRECTORY-ALL (396)    OPTION (19)

Name Value
NAME XMMGR-DIRECTORY-ALL
MENU TEXT Remote Directory from all Domains
UPPERCASE MENU TEXT REMOTE DIRECTORY FROM ALL DOMA
DESCRIPTION 
This option schedules tasks to run in background to update MailLink
directory ^XMD(4.2997).  Use this option cautiously because it sends
requests to all the Domains for soliciting remote user directories.
CREATOR USER,ONE
TYPE run routine
PACKAGE MAILMAN
ROUTINE ALL^XMDIRQST
