PSJV MAN CREATE PD (4046)    OPTION (19)

Name Value
NAME PSJV MAN CREATE PD
MENU TEXT Create Primary Drug (Manual)
UPPERCASE MENU TEXT CREATE PRIMARY DRUG (MANUAL)
DESCRIPTION 
  This allows the user to create primary drugs and tie them to current
entries in the drug file.  Primary drugs are entries without a strength
or dose form, which are used in entering Medication orders by non-pharmacy
personnel.
CREATOR POSTMASTER
TYPE run routine
PACKAGE PATIENT DATA EXCHANGE
