VAQ (EDIT) MAX LIMITS (4097)    OPTION (19)

Name Value
NAME VAQ (EDIT) MAX LIMITS
X ACTION PRESENT YES
MENU TEXT Edit maximum limits for automatic processing
UPPERCASE MENU TEXT EDIT MAXIMUM LIMITS FOR AUTOMA
LOCK VAQ EDIT FILE
DESCRIPTION 
 
This option is used to edit the maximum time and occurrence limits that a
site is willing to give automatic processing to.  Limits entered for each
data segment will be compared against the limits contained in a request.
If a requested data segment's limits are over the maximum allowed, the
request will require user intervention.
