DVB HREQ-STATUSBYPAT (4178)    OPTION (19)

Name Value
NAME DVB HREQ-STATUSBYPAT
MENU TEXT Status of HINQ by Patient
UPPERCASE MENU TEXT STATUS OF HINQ BY PATIENT
LOCK DVBHINQ
DESCRIPTION 
This option gives information about entries in the suspense file
by patient name.  The suspense file status, the eligibility status
(from the station's patient file), and the HINQ response message, if
a response has been received, will be displayed for each patient
selected.
CREATOR POSTMASTER
TYPE run routine
ROUTINE FIND^DVBHQUS
