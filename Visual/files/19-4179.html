DVB HSUSP-VIEWFILE (4179)    OPTION (19)

Name Value
NAME DVB HSUSP-VIEWFILE
MENU TEXT View the HINQ Suspense File
UPPERCASE MENU TEXT VIEW THE HINQ SUSPENSE FILE
DESCRIPTION 
This option lists HINQ suspense file entries on the screen.  Entries
are displayed in order according to the last status update date and
time, beginning with the most recent.  Only entries whose last status
update date is on or after the last network workday will be shown.
For each entry displayed, the following information is given:  patient's
name, SSN, suspense file status, last status update date, and the names
and times of users who requested an entry for the patient.
