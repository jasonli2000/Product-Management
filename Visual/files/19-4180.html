DVB HSUSP-ENTERREQ (4180)    OPTION (19)

Name Value
NAME DVB HSUSP-ENTERREQ
MENU TEXT Enter a Request in the HINQ Suspense File
UPPERCASE MENU TEXT ENTER A REQUEST IN THE HINQ SU
DESCRIPTION 
This option allows HINQ inquiry requests to be entered into the 
HINQ suspense file for later transmission to the VBA computer.
Entries may be made in the suspense file by users who do not
have the required security to transmit requests to the VBA
computer.  Requests should also be entered in the suspense file
when the VBA computer is not available as during non-VBA 
computer workday hours or when the network is disabled because
of a communication problem.
 
This option may be made available during the Registration options:
Register Patient and Load/Edit Patient Data by turning the USE
HINQ INQUIRY parameter on.  It can also be placed stand alone in
the ADT menu.
 
If the patient's eligibility status in the station's patient file
is 'Verified', this will be indicated when an entry is made
in the suspense file.
