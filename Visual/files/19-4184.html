DVB HNET-WORK (4184)    OPTION (19)

Name Value
NAME DVB HNET-WORK
MENU TEXT Network Enable/Disable
UPPERCASE MENU TEXT NETWORK ENABLE/DISABLE
DESCRIPTION 
This option enables/disables the options which allow the user to
transmit HINQ requests, 'PROCESS THE HINQ SUSPENSE FILE' and 
'GENERATE A HINQ REQUEST'.
 
The network is disabled automatically during VBA computer non-workday
hours.  It should also be disabled if there are known problems with
the communication link as indicated by failure of the HINQ
transaction test.  Network error messages also indicate a 
transmission problem so the network should be disabled.
 
This option merely disables the transmitting functions and not
actually the DHCP-IDCU link.
