DVB HSUSP-PURGEFILE (4185)    OPTION (19)

Name Value
NAME DVB HSUSP-PURGEFILE
MENU TEXT Purge Suspense File (retain 30 days)
UPPERCASE MENU TEXT PURGE SUSPENSE FILE (RETAIN 30
DESCRIPTION 
This suspense file utility function allows users who are holders
of the HINQ key to delete entries in the suspense file.  All entries
whose last status update date is before the last 30 days will be
deleted.
CREATOR POSTMASTER
TYPE action
E ACTION PRESENT YES
ENTRY ACTION D ASK^DVBHQPU Q:'$D(DVBSTOP) K DVBSTOP D 30^DVBHQPU
*DEVICE FOR QUEUED JOB OUTPUT 89
