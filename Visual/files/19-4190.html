DVB HREQ-INDIVHREQ (4190)    OPTION (19)

Name Value
NAME DVB HREQ-INDIVHREQ
X ACTION PRESENT YES
MENU TEXT Individual HINQ Request
UPPERCASE MENU TEXT INDIVIDUAL HINQ REQUEST
LOCK DVBHINQ
DESCRIPTION 
This option allows the user to access the VBA computer
directly and request a HINQ.  Access to VBA is through the
DHCP-IDCU network.  The HINQ response
will return within 20 seconds formatted on the screen
the same as the HINQ response mail message. The response
will also be sent to the IN basket of the requestor, the
HINQ mail group, and users with the DVBHINQ key who have
requested a HINQ in the Suspense file.
CREATOR POSTMASTER
TYPE action
E ACTION PRESENT YES
EXIT ACTION K DVBSTOP
ENTRY ACTION D EN^DVBHQTM D:'$D(DVBSTOP) ^DVBHQDL
