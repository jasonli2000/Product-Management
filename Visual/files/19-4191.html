DVB HSUSP-PROCESSFILE (4191)    OPTION (19)

Name Value
NAME DVB HSUSP-PROCESSFILE
X ACTION PRESENT YES
MENU TEXT Process the HINQ Suspense File
UPPERCASE MENU TEXT PROCESS THE HINQ SUSPENSE FILE
LOCK DVBHINQ
DESCRIPTION 
This option queues a TaskMan job which logs onto the remote
VBA computer via the DHCP-IDCU and generates
HINQ requests for all 'P'ending entries in the suspense file.
 
HINQ responses will go to the person who requested the Batch
process, members of the DVBHINQ mail group, and requestors
in the Suspense file with the DVBHINQ key
CREATOR POSTMASTER
TYPE action
E ACTION PRESENT YES
EXIT ACTION K DVBSTOP
ENTRY ACTION D EN^DVBHQTM I '$D(DVBSTOP) W:'$P(^DVB(395,1,"HQ"),U,5) !,"Batch processing disabled " I $P(^("HQ"),U,5) D ^DVBHQDB
