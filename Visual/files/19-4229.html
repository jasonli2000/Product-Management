DVBA RE-GENERATE 21-DAY CERTIF (4229)    OPTION (19)

Name Value
NAME DVBA RE-GENERATE 21-DAY CERTIF
MENU TEXT Manual 21-day Certificate Processing
UPPERCASE MENU TEXT MANUAL 21-DAY CERTIFICATE PROC
DESCRIPTION 
This option is only used in case the Task Manager ever fails.  It is used
to ENTIRELY reprocess a date.  Only one date at a time can be reprinted
using this option.
