DVBA RO REPRINT 21-DAY CERT (4231)    OPTION (19)

Name Value
NAME DVBA RO REPRINT 21-DAY CERT
MENU TEXT Reprint a 21-day Certificate for the RO
UPPERCASE MENU TEXT REPRINT A 21-DAY CERTIFICATE F
DESCRIPTION 
This option will reprint any one day of certificates for the RO.  The only
prerequisite is that the RO know what day the certificate was originally
printed.
CREATOR POSTMASTER
TYPE run routine
PACKAGE AUTOMATED MED INFO EXCHANGE
