DVBA C ADD ADDITIONAL EXAM (4246)    OPTION (19)

Name Value
NAME DVBA C ADD ADDITIONAL EXAM
MENU TEXT Add an Exam to an Existing Request
UPPERCASE MENU TEXT ADD AN EXAM TO AN EXISTING REQ
DESCRIPTION 
Allows adding another exam to an existing request provided
that it has a status which will allow addition.  Those that
are transcribed, cancelled , completed or transferred in
will not be allowed to have exams added.
CREATOR POSTMASTER
TYPE run routine
