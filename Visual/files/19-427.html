Name Value
NAME XTSUMBLD
MENU TEXT Build an 'NTEG' routine for a package
UPPERCASE MENU TEXT BUILD AN 'NTEG' ROUTINE FOR A
DESCRIPTION 
This option gets a package namespace from the package file and
then a list of routines from the user.  It then builds a 
<namespace>NTEG routine that has a checksum for each of the
routines.  This routine can be run to see if there has been
any change to a routine since the NTEG routine was built.
CREATOR POSTMASTER
TYPE run routine
