DENTPERS RE-RELEASE (4377)    OPTION (19)

Name Value
NAME DENTPERS RE-RELEASE
MENU TEXT Personnel Entry (type 4) Edit/Re-Release
UPPERCASE MENU TEXT PERSONNEL ENTRY (TYPE 4) EDIT/
OUT OF ORDER MESSAGE DAS turned off
DESCRIPTION 
This option should be used to enable re-release of a Personnel entry (and
all related Non Cinical Time entries) that has been rejected by Austin.
This option merely enables the re-release of an entry.  The next time
Personnel data is released using the Review/Release option, this entry
can be included in the transmission to Austin.
CREATOR POSTMASTER
TYPE run routine
ROUTINE PRR^DENTE2
