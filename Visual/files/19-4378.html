DENTADM RE-RELEASE (4378)    OPTION (19)

Name Value
NAME DENTADM RE-RELEASE
MENU TEXT Class I-VI Admin Entry (type 3) Edit/Re-Release
UPPERCASE MENU TEXT CLASS I-VI ADMIN ENTRY (TYPE 3
OUT OF ORDER MESSAGE DAS turned off
DESCRIPTION 
This option should be used to enable re-release of a Class I-VI Admin entry
that has been rejected by Austin.  This option merely enables the re-release
of an entry.  The next time Class I-VI Admin data is released using the
Review/Release option, this entry can be included in the transmission
to Austin.

TYPE run routine
