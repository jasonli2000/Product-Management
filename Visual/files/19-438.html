XTVR COMPARE (438)    OPTION (19)

Name Value
NAME XTVR COMPARE
MENU TEXT Routine Compare - Current with Previous
UPPERCASE MENU TEXT ROUTINE COMPARE - CURRENT WITH
DESCRIPTION 
This option is used to compare one or more current routines to previous
versions which have been recorded using the 'Update with current routines'
option.  Differences between the current version and the indicated number
of prior versions will be noted.
CREATOR POSTMASTER
TYPE run routine
ROUTINE XTVRC2
