XTVG UPDATE (440)    OPTION (19)

Name Value
NAME XTVG UPDATE
MENU TEXT Accumulate Globals for Package
UPPERCASE MENU TEXT ACCUMULATE GLOBALS FOR PACKAGE
DESCRIPTION 
This option is used to accumulate the current globals for a package for
comparison with subsequent versions.  The global data is accumulated for
the ^DIC(fn,0,  the ^DD(fn,  nodes where fn is an included file number),
and the Edit, Print, and Sort Templates for the file(s) indicated as
related to the package in the VERIFIER PACKAGE file.
CREATOR POSTMASTER
TYPE run routine
ROUTINE XTVGC1
