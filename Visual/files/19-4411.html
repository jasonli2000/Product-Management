SDOQM PM NIGHTLY JOB (4411)    OPTION (19)

Name Value
NAME SDOQM PM NIGHTLY JOB
MENU TEXT Nightly job for PM data extract
UPPERCASE MENU TEXT NIGHTLY JOB FOR PM DATA EXTRAC
DESCRIPTION 
This option collects the next avaialable appointment data for
selected clincis and transmits the data to a central database.  This option
should be scheduled to run nightly.  Currently, the Performance Management
Office is requiring the colletion if this data for the month of September.
The option will need to manually scheduled, and will unschedule itself after
the end of September.
CREATOR USER,ONE
TYPE run routine
PACKAGE SCHEDULING
ROUTINE EN^SDOQMP
SCHEDULING RECOMMENDED YES
