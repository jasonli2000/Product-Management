XTMOVE-IN (447)    OPTION (19)

Name Value
NAME XTMOVE-IN
MENU TEXT Bring in Sent Routines
UPPERCASE MENU TEXT BRING IN SENT ROUTINES
LOCK XUPROGMODE
DESCRIPTION 
This option is only to be used after invoking the XTMOVE option (Move
Routines across Volume Sets).  When in the destination UCI/Volume Set,
this option will install the routines that were previously sent.
CREATOR POSTMASTER
TYPE run routine
PACKAGE KERNEL
ROUTINE IN^%ZTMOVE
