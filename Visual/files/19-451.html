XTVR MOST RECENT CHANGE DATE (451)    OPTION (19)

Name Value
NAME XTVR MOST RECENT CHANGE DATE
MENU TEXT Last Routine Change Date Recorded
UPPERCASE MENU TEXT LAST ROUTINE CHANGE DATE RECOR
DESCRIPTION 
This option will list the most recent date on which a change was
recorded for the selected routine(s).  The date piece of the first
line of the routine and version and patch information are also
displayed.  The version number may be changed and the routine
recorded after that date, but the last change date recorded will be
that date involving a change in more than the second (version number)
line.
 
The Update Routine.. option must have been used one or more times to
record the routine and changes to the routine.
CREATOR POSTMASTER
TYPE run routine
PACKAGE KERNEL
ROUTINE XTVRC1A
