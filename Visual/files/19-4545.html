NAME PRSA TD EDIT
MENU TEXT Enter/Edit Tour of Duty
UPPERCASE MENU TEXT ENTER/EDIT TOUR OF DUTY
DESCRIPTION 
This option allows the user to create a new tour or to select an
existing tour. The user may enter the data for the new tour or edit
the data on the existing tour.
 
The user will be asked for the tour description, length of meal time,
whether the meal time is subject to premium pay, whether the tour
spans two calendar days, whether the tour is available to all time
and leave units and the start/stop times for each segment of the tour.
