NAME PRSA TD DISP
MENU TEXT Display Tour of Duty
UPPERCASE MENU TEXT DISPLAY TOUR OF DUTY
DESCRIPTION 
This option will prompt the user to select an established tour of
duty. It will display the tour description, length of meal time,
whether meal time is subject to premium pay, whether the tour spans
two calendar days, whether the tour is available to all time and leave
units, and the start/stop times for each segment of the tour.
 
This option merely displays the tour information. You may not edit
the tour with this option.
