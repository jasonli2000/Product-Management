Name Value
NAME PRSA OT REQ
MENU TEXT Enter OT and CT/CH Requests
UPPERCASE MENU TEXT ENTER OT AND CT/CH REQUESTS
DESCRIPTION 
This option allows the user to enter a request for an employee to
work overtime or compensatory time/credit hours.  Select an employee
and enter date the work is to be performed, type of time worked,
number of hours to be worked, time and leave unit where the work
will be performed, and the justification for making this request. 
For compensatory time/credit hours the justification should include
whether the requests is for comp time or credit hours.
