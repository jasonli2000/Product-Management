XTLKMODPARK (455)    OPTION (19)

Name Value
NAME XTLKMODPARK
MENU TEXT Delete Entries From Look-up
UPPERCASE MENU TEXT DELETE ENTRIES FROM LOOK-UP
LOCK XTLKZMGR
DESCRIPTION 
This option is used to delete entries out of the 'Local Lookup' file.
There can not be any Keywords Shortcuts or Synonyms associated with the
file to be deleted, in order to do this.
CREATOR POSTMASTER
TYPE run routine
ROUTINE DD^XTLKEFOP
