Name Value
NAME PRSA LV REQ
MENU TEXT Leave Request
UPPERCASE MENU TEXT LEAVE REQUEST
DESCRIPTION 
This option allows an employee to make a request for leave (i.e.,
APPLICATION FOR LEAVE, SF 71). The employee must be entered in both
the PAID EMPLOYEE (#450) and NEW PERSON (#200) files and the social
security numbers must match.
