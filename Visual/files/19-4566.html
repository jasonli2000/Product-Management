PRSA OT CAN (4566)    OPTION (19)

Name Value
NAME PRSA OT CAN
MENU TEXT Cancel OT and CT/CH Requests
UPPERCASE MENU TEXT CANCEL OT AND CT/CH REQUESTS
DESCRIPTION 
This option allows the user to cancel overtime and
compensatory/credit hours requests. After selecting the employee,
the user specifies a date to begin searching for all such requests.
That employee's requests are listed and the user selects the one to
cancel.  Note that compensatory time and credit hours are both coded
under CT.
