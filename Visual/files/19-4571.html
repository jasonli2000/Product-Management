PRSA OT LST (4571)    OPTION (19)

Name Value
NAME PRSA OT LST
MENU TEXT Display OT and CT/CH Requests
UPPERCASE MENU TEXT DISPLAY OT AND CT/CH REQUESTS
DESCRIPTION 
This option allows a user to list overtime and compensatory/credit
hours requests for one or more employees. The user enters a 'begin'
date and an 'end' date. All such requests for the employee(s) chosen
will be displayed. The date, number of hours, type of request and
the justification for the request are listed.  Note that
compensatory time and credit hours are both coded under CT, so it
may be necessary for timekeepers to distinguish the type of request
(CT or CH) in the justification, when entering the request.
