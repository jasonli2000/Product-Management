NAME PRSA LV LST
MENU TEXT List Leave Requests
UPPERCASE MENU TEXT LIST LEAVE REQUESTS
DESCRIPTION 
This option allows the user to list leave requests for one or all
employees. The user enters a 'begin' date and an 'end' date. All
leave requests for the employee(s) chosen will be displayed. The
list may be sorted by employee name or date of leave request. The
date, time, type of leave and any remarks are listed.
