Name Value
NAME PRSA TD LIST
MENU TEXT List Tours by T&L
UPPERCASE MENU TEXT LIST TOURS BY T&L
DESCRIPTION 
The user is prompted to select a particular Time and Leave Unit (T&L)
or ALL Time and Leave Units. If a particular T&L is selected, then all
the tours associated with it are displayed. If ALL T&Ls are selected,
then all Tours of Duty will be displayed along with the T&Ls associated
with the tour.
 
The tour number, start/stop times, number of hours in the tour,
start/stop times of the various segments of the tour, any special
indicators for each time segment and the T&L(s) associated with
the tour are displayed.
