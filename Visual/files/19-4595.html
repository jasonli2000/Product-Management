PRSA TPE PPE (4595)    OPTION (19)

Name Value
NAME PRSA TPE PPE
MENU TEXT List Prior Pay Period Exceptions
UPPERCASE MENU TEXT LIST PRIOR PAY PERIOD EXCEPTIO
DESCRIPTION 
This option will allow a Timekeeper to list all unresolved timecard exceptions
from prior pay periods for any T&L assigned to them.  The employee's name, time
and leave unit, date of the exception and a short message are displayed.
CREATOR USER,ONE
TYPE run routine
ROUTINE TK^PRSATPG
