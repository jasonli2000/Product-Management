Name Value
NAME PRSA TH
MENU TEXT Transmission History
UPPERCASE MENU TEXT TRANSMISSION HISTORY
DESCRIPTION 
This option will provide a summary of information concerning a selected
pay period including the number of records transmitted, the number of mail
messages sent, the number of mail messages acknowledged from Austin, the
clerk who transmitted the records and the date/time of data transmission
to Austin.
