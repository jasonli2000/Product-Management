Name Value
NAME PRSA PAY PPC
MENU TEXT List/Clear Prior Pay Period Corrections
UPPERCASE MENU TEXT LIST/CLEAR PRIOR PAY PERIOD CO
DESCRIPTION 
This option allows Payroll to display or list the Prior Pay Period
Corrections that have been approved by the supervisor. If performed
on a CRT, the Payroll Clerk can indicate that the entry has been
cleared and that appropriate corrective OLDE actions have been
taken.
