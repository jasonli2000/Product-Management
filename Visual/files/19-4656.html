PRSE-EE-CLAS-INFO (4656)    OPTION (19)

Name Value
NAME PRSE-EE-CLAS-INFO
MENU TEXT Enter/Edit Class Information
UPPERCASE MENU TEXT ENTER/EDIT CLASS INFORMATION
DESCRIPTION 
This option allows the user to create and edit different types of classes
(e.g., mandatory training, continuing education, miscellaneous inservices,
and unit/location inservices) for registration and attendance.  Entries
are stored in the PRSE Program/Class file (#452.1), and the PRSE Education
Registration file (#452.8).
