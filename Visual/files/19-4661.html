PRSE-OLDE (4661)    OPTION (19)

Name Value
NAME PRSE-OLDE
MENU TEXT OLDE Training Coding Report
UPPERCASE MENU TEXT OLDE TRAINING CODING REPORT
DESCRIPTION 
This option prints a report of class information in OLDE coding order.
The report may be printed for all or selected employees in a specific
service or the entire facility.  A date range is also prompted for.  The
report is sorted by social security number.
