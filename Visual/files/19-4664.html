PRSE-EE-EMP (4664)    OPTION (19)

Name Value
NAME PRSE-EE-EMP
MENU TEXT Modify Past Class Information
UPPERCASE MENU TEXT MODIFY PAST CLASS INFORMATION
DESCRIPTION 
This option allows a user to edit or delete all records in the Student
Education Tracking file (#452) for a specific class of a given date/time.
There are four pieces of data (i.e., Training Class, Program/Class
Supplier, Training Type, and Length) changes allowed and the system
will automatically update all student records that are associated
with this class.
CREATOR POSTMASTER
