NAME XUSEREDITSELF
MENU TEXT Edit User Characteristics
UPPERCASE MENU TEXT EDIT USER CHARACTERISTICS
DESCRIPTION 
This option allows the user to edit certain attributes as defined
by the Kernel Site Parameter file.  At most, the user can edit
their own VERIFY CODE, AUTO-MENU, TYPE-AHEAD, and NICKNAME.  This
can vary in your site.
A user may also edit the 'SUBTYPE' of their device.
