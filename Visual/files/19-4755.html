NURAAM-MD-UNCBAT (4755)    OPTION (19)

Name Value
NAME NURAAM-MD-UNCBAT
MENU TEXT Batch Run of Unclassified Midnight Patients
UPPERCASE MENU TEXT BATCH RUN OF UNCLASSIFIED MIDN
DESCRIPTION 
This is an optional report, that the site may or may not wish to queue to run.  This option should be queued at least 2 hours after the queuing of the NURAAM-ACU option.
This option will print out the unclassified patients not counted by the NURAAM-ACU.
CREATOR USER,ONE
TYPE run routine
PACKAGE NURSING SERVICE
E ACTION PRESENT YES
ENTRY ACTION S NURTYPE=1
ROUTINE EN3^NURAAU2
