NAME GMRY NURSHIFT
MENU TEXT Shift Starting Hour and Other Parameters
UPPERCASE MENU TEXT SHIFT STARTING HOUR AND OTHER
DESCRIPTION 
The shift starting hours stored in the GMRY NUR SHIFT/OTHER file (#126.95)
are used to calculate the patient's total intake/output for night, day and 
evening shifts, respectively.  The data type for hours can be entered 
"0800", "1600" and "2400".
