Name Value
NAME GMRV CHANGE V/M PARAMETERS
X ACTION PRESENT YES
MENU TEXT Change Date/Time Taken

OUT OF ORDER MESSAGE GMRV*5*19 - shutdown Vitals CHUI
DESCRIPTION 
This option allows the user to enter vitals/measurements (on the same
patient) for a different Date/Time.
