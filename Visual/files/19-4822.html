GMRV VMSITE (4822)    OPTION (19)

Name Value
NAME GMRV VMSITE
MENU TEXT Enter/Edit Vitals Qualifiers
UPPERCASE MENU TEXT ENTER/EDIT VITALS QUALIFIERS
OUT OF ORDER MESSAGE GMRV*5*19 - shutdown Vitals CHUI
DESCRIPTION 
This option is used by the application coordinator to create and edit
hospital-wide qualifiers associated with vital signs and other patient
measurements captured in the Vitals/Measurements application.  Data is
stored in the GMRV Vital Qualifier (#120.52) file.
