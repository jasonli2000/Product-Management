Name Value
NAME NURCPE-VIT-ERROR
MENU TEXT Edit a Vital/Measurement Entered in Error
UPPERCASE MENU TEXT EDIT A VITAL/MEASUREMENT ENTER
OUT OF ORDER MESSAGE NUR*4*40 - shutdown Vitals CHUI
DESCRIPTION 
This option allows users to correct errors in vitals/measurements.  A new 
record is created, and the old record is marked as entered in error.
