OUT OF ORDER MESSAGE NUR*4*40 - shutdown Vitals CHUI
DESCRIPTION 
This menu contains the following options:
1. Enter/Edit Vitals Qualifierrs option is used to configure qualifiers in
   the GMRV Vital Qualifier file (#120.52) for vital types, blood
   pressuree, for example.
2. Change Default Qualifiers for Temp./Pulse is used to change the default
   qualifier for temperature and pulse.
3. Display Vitals Category/Qualifier Table is used to print the categories
   /qualifiers table for each vital type.
