XTVR RESTORE PREV ROUTINE (487)    OPTION (19)

Name Value
NAME XTVR RESTORE PREV ROUTINE
MENU TEXT UNDO Edits (Restore to Older Version of Routine)
UPPERCASE MENU TEXT UNDO EDITS (RESTORE TO OLDER V
DESCRIPTION 
This option is used to restore a routine back to a previous version which is
available in the previous version edits shown by the Routine Compare option.
The user must specify a routine name to be used for the restored routine that
is not currently used (so that no current routine is destroyed as a result of
saving the newly restored routine).  After checking that the restored routine
is the desired version, the user may rename it as desired.
