DESCRIPTION 
Permits the service to enter and modify demographic information specific to
ward/units, clinics, and other nursing assignment locations in the NURS
LOCATION FILE (#211.4).  This option also allows editing of the Budgeted
FTEE information contained in the NURS POSITION CONTROL FILE (#211.8).
