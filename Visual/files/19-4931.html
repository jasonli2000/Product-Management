Name Value
NAME NURSE-PR-MIDEF
MENU TEXT Mandatory Deficiency Report by Unit/Svc. Category
UPPERCASE MENU TEXT MANDATORY DEFICIENCY REPORT BY
DESCRIPTION 
This option generates a fiscal or calendar year mandatory training class
deficiency report, by nursing location, or service category, in an 80 or
132 column reporting format. This report also contains the anniversary
date which is the date by which the employee is scheduled to reattend
the class.
