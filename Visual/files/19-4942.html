NURS-COMP (4942)    OPTION (19)

Name Value
NAME NURS-COMP
MENU TEXT Position Control/Staff File Integrity Report
UPPERCASE MENU TEXT POSITION CONTROL/STAFF FILE IN
DESCRIPTION 
This option searches the Position Control File (#211.8) for staff who are
not in the NURS Staff File (#210). If any is found, a listing is displayed
to the terminal or can be queued to a printer. The user may selectively
re-enter any of the staff with all required data.
CREATOR USER,ONE
TYPE run routine
Short Menu Text COMPARE F-211.8 AGAINST F-210
PACKAGE NURSING SERVICE
ROUTINE NURSCEP
