YSCL RETRANSMIT DATA (4945)    OPTION (19)

Name Value
NAME YSCL RETRANSMIT DATA
MENU TEXT Retransmit Clozapine Roll-up Data
UPPERCASE MENU TEXT RETRANSMIT CLOZAPINE ROLL-UP D
DESCRIPTION 
Use this option to retransmit data to the national clozapine data
base when the weekly transmission report fails.  This option will
ask for an ending date for data collection. Use the previous Tuesday
as the ending date.

Place this option on the secondary menu of the person responsible
for maintaining the Clozapine Roll-up software.
CREATOR USER,ONE
TYPE run routine
ROUTINE REXMIT^YSCLTST2
