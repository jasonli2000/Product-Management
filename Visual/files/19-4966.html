QAP FIX RESPONSE (4966)    OPTION (19)

Name Value
NAME QAP FIX RESPONSE
MENU TEXT Fix a Survey Response
UPPERCASE MENU TEXT FIX A SURVEY RESPONSE
LOCK QAP MANAGER
DESCRIPTION 
This option will fix a survey response which may have exited abnormally as
in when a job was killed or a system crash.
 
Such abnormal exits will leave the survey without proper information to
allow the participant to restart and complete it.
CREATOR POSTMASTER
TYPE run routine
PACKAGE SURVEY GENERATOR
ROUTINE QAPFIX
