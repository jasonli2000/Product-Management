GMRG-PURGE PT DATA (4982)    OPTION (19)

Name Value
NAME GMRG-PURGE PT DATA
MENU TEXT Purge GMR Text File Data
UPPERCASE MENU TEXT PURGE GMR TEXT FILE DATA
DESCRIPTION 
The purge GMR Text File Data [GMRG-PURGE PT DATA] option allows the site
to purge data from the GMR Text (124.3) and NURS Care Plan (216.8) files.
This option should only be assigned to IRM personnel, and prior to its
running, IRM should notify Nursing and figure out how many months of data
is to be retained.
