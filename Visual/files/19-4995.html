PXTT EDIT TREATMENT (4995)    OPTION (19)

Name Value
NAME PXTT EDIT TREATMENT
X ACTION PRESENT YES
MENU TEXT Treatments Add/Edit
UPPERCASE MENU TEXT TREATMENTS ADD/EDIT
DESCRIPTION 
This option allows a user to create a new Treatment or edit a Treatment
that was originally created at your site. Treatments that are
distributed with the PCE package can be inactivated using the PCE
"Activate/Inactivate TAble Items" menu.
 
Once a treatment is defined in this table, it should not be deleted if
there is any encounter form defintions which may be referencing the
treatment, or if there is any patient encounter data which may be
referencing the treatment historically.  Use the "Inactive Flag" field to
make a treatment "INACTIVE" for selection in the Encounter form definition
process and the PCE encounter data entry process.
 
If treatments are to be included on the clinic's encounter form, the
treatments must first be defined in the Treatment file using this option.
 
The scanning of encounter forms with the AICS package will provide PCE
with patient treatment information which is stored in the V TREATMENT
file.
CREATOR USER,ONE
TYPE edit
DISPLAY OPTION? NO
E ACTION PRESENT YES
EXIT ACTION K PXTDUZ,PXNAT,PXB
ENTRY ACTION D ZERO^PXTTU1("^AUTTTRT(",$G(PXNAT))
DIC {DIC} AUTTTRT(
DIC(0) QAMEZL
DIC(S) S PXTDUZ=+$P($$SITE^VASITE,U,3) I $S($G(PXNAT):1,(+$G(Y)>100000):1,1:0)
DR{DDS} .01
DDSFILE 9999999.17
DIE AUTTTRT(
DR {DIE} .01;.04;8801
NO UP-ARROW EXIT AND BACK JUMPING OK
