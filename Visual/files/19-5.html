XUEDITOPT (5)    OPTION (19)

Name Value
NAME XUEDITOPT
X ACTION PRESENT YES
MENU TEXT Edit options
UPPERCASE MENU TEXT EDIT OPTIONS
DESCRIPTION 
This option is used to create the building blocks of the menu system:
the options. Each option should have an internal name, menu text, a
description and a type.  Depending on its type, other fields are filled in.
CREATOR USER,ONE
HELP FRAME XQEDIT
TYPE edit
DELEGABLE NO
E ACTION PRESENT YES
EXIT ACTION K DLAYGO D KICK^XQ7
ENTRY ACTION S DLAYGO=19
DIC {DIC} DIC(19,
DIC(0) AELMQ
DIC(A) Select OPTION to edit:
DIE DIC(19,
DR {DIE} [XUEDITOPT]
