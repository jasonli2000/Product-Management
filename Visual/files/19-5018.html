DESCRIPTION 
This is the menu for the person that is the ADPAC for PCE.  It includes
all of the user interface options as well as the options for file
maintenance.
 
The data entry options may be assigned to clerk and/or clinican menus as
needed.
 
The HS and Report parameter edit and print options are for the coordinator
to manage fields for site specific preferences/definitions in the Health
Summary and PCE Reports.
