Name Value
NAME PX REPORT PARAMETER EDIT
MENU TEXT PCE Report Parameter Edit
UPPERCASE MENU TEXT PCE REPORT PARAMETER EDIT
DESCRIPTION 
This option is used to define parameters that will be used by the PCE
Report Module.  The user will need to identify which clinics are viewed by
clinicians as being Emergency Room clinics.  The user will also need to
identify the lab test names that are used by your site to identify the
following types of Lab tests: Glucose, Cholesterol, LDL Cholesterol, and
HBA1C.
 
To get a print out of current definitions in the PCE Parameters fields for
these fields, use the PCE HS/RPT Parameters Print.
