PXTT COPY EDUCATION TOPIC (5044)    OPTION (19)

Name Value
NAME PXTT COPY EDUCATION TOPIC
X ACTION PRESENT YES
MENU TEXT Education Topic Copy
UPPERCASE MENU TEXT EDUCATION TOPIC COPY
DESCRIPTION 
This option allows the user to copy an existing education topic into a new
education topic entry in the Educatin Topics file (#9999999.09). The
original education topic to be copied is selected first.  If the topic is
prefixed with "VA-", the "VA-" will be stripped off the name
automatically. The new name must be unique.  If the name is not unique,
the user must enter a unique name for the new education topic.
CREATOR USER,ONE
TYPE run routine
EXIT ACTION K PXNAT
ROUTINE PXTTEDC
