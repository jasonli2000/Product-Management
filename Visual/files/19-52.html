XQRESTRICT (52)    OPTION (19)

Name Value
NAME XQRESTRICT
MENU TEXT Restrict Availability of Options
UPPERCASE MENU TEXT RESTRICT AVAILABILITY OF OPTIO
DESCRIPTION 
This option is used to assign various sorts of restrictions to options.
The following fields can be set, for individual options or groups of
options: LOCK, PROHIBITED TIMES, SPECIFY DEVICES, PRIORITY, and OUT OF ORDER.
CREATOR USER,ONE
