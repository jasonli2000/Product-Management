FBCH REOPEN REQUEST (5211)    OPTION (19)

Name Value
NAME FBCH REOPEN REQUEST
MENU TEXT Reconsider a Denied Request
UPPERCASE MENU TEXT RECONSIDER A DENIED REQUEST
LOCK FBAASUPERVISOR
DESCRIPTION 
This option is restricted to the holder of the 'FBAASUPERVISOR' key.
It allows the supervisor to reconsider a previously denied request.
There is an audit on the Legal and Medical Entitlement fields.
CREATOR POSTMASTER
TYPE run routine
ROUTINE FBCHREQ2
