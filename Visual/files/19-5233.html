FBCNH CANCEL 7078 (5233)    OPTION (19)

Name Value
NAME FBCNH CANCEL 7078
MENU TEXT Cancel Authorization Entered in Error
UPPERCASE MENU TEXT CANCEL AUTHORIZATION ENTERED I
LOCK FBAASUPERVISOR
DESCRIPTION 
This option should be used when an authorization has been set up and
has been determined that it was entered in error.  Once cancelled,
a user can re-enter the correct authorization.
