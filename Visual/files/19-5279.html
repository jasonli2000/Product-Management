FBCH CENSUS REPORT (5279)    OPTION (19)

Name Value
NAME FBCH CENSUS REPORT
MENU TEXT Civil Hospital Census Report
UPPERCASE MENU TEXT CIVIL HOSPITAL CENSUS REPORT
DESCRIPTION 
This output provides a listing for all Civil Hospital patients
for a user specified census date.  Output includes the vendor name and
Participation Code, veteran specifics such as DOB and ID, as well as
the Primary Service Area and the Authorization To Date.
CREATOR POSTMASTER
TYPE run routine
E ACTION PRESENT YES
ENTRY ACTION S FBI=6
ROUTINE FBCNHCEN
