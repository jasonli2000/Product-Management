FB VENDOR/VETERAN PAYMENTS (5285)    OPTION (19)

Name Value
NAME FB VENDOR/VETERAN PAYMENTS
X ACTION PRESENT YES
MENU TEXT Payment Listing for Vendor/Veteran
UPPERCASE MENU TEXT PAYMENT LISTING FOR VENDOR/VET
DESCRIPTION 
This option will prompt the user to select the vendor calling, and the
veteran information is requested upon.  It will then call in to list
manager to display a listing of payments for all programs in the order
of date service provided (with the most recent displaying first).
CREATOR POSTMASTER
TYPE action
E ACTION PRESENT YES
EXIT ACTION K DFN,FBV
ENTRY ACTION K XQORS,VALMEVL D ^FBPHON,EN^VALM("FBPHONE MENU")
