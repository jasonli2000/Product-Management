RA COMPILE TEMPLATES (5296)    OPTION (19)

Name Value
NAME RA COMPILE TEMPLATES
MENU TEXT Template Compilation
UPPERCASE MENU TEXT TEMPLATE COMPILATION
DESCRIPTION 
     This option will compile all input and print templates which are
currently compiled.  It is advised that all Radiology/Nuclear Medicine
users be off the system during the compilation process.  Compilation
should also taken place when system activity is at a minimum.
CREATOR USER,ONE
TYPE run routine
PACKAGE RADIOLOGY/NUCLEAR MEDICINE
ROUTINE RACMPLE
