RA EXAM/STATUS ITYPE MISMATCH (5297)    OPTION (19)

Name Value
NAME RA EXAM/STATUS ITYPE MISMATCH
MENU TEXT Imaging Type Mismatch Report
UPPERCASE MENU TEXT IMAGING TYPE MISMATCH REPORT
DESCRIPTION 
This report will print each case where the imaging type of the visit does
not match the imaging type of the current exam status.  These cases should
be edited to complete as soon as possible to correct the exam status.  The
mismatch condition should only exist on cases not yet completed at the
time Version 4.5 is installed.  If not corrected, the mismatches will lead
to inaccurate counts on workload reports that depend on the examination
status parameters to determine which statuses are included in which
reports.
