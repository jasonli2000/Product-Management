RA EDITPT (5300)    OPTION (19)

Name Value
NAME RA EDITPT
X ACTION PRESENT YES
MENU TEXT Edit Exam by Patient
UPPERCASE MENU TEXT EDIT EXAM BY PATIENT
DESCRIPTION 
This function can be used to edit exams for patients by patient name.  The
fields edited are the same as in the 'Case No. Exam Edit' function,
however, this function allows selection of a patient, displays open cases
for the patient, then asks you to select one of the patient's registered
exams to edit.  This function should be used to re-edit fields that have
already been entered, but need correction.  Only fields with previously
entered data will appear for re-edit. The Status Tracking functions should
be used to enter case data and insure that exams progress to a 'Complete'
status.
 
The user can use this function to edit an exam that has been
completed, however, the user must have the 'manager' key to edit
a 'completed' exam.
 
Once an exam is edited and in the 'COMPLETE' status, the associated
request will display a message as follows: "** Note: Request has been
changed by Radiology/Nuclear Medicine**".
