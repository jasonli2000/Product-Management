RA FLASH (5309)    OPTION (19)

Name Value
NAME RA FLASH
X ACTION PRESENT YES
MENU TEXT Duplicate Flash Card
UPPERCASE MENU TEXT DUPLICATE FLASH CARD
DESCRIPTION 
This function allows the user to print additional flash cards
or exam labels for a previous exam.  The user can duplicate 
between 0 and 20 additional flash cards or exam labels.
  
This function is often used because of a printer malfunction
occurring during the original printing of the flash card or exam
label.
CREATOR USER,ONE
TYPE run routine
E ACTION PRESENT YES
EXIT ACTION K RADUPSCN
ENTRY ACTION S RADUPSCN=""
ROUTINE DUP^RAEDCN
