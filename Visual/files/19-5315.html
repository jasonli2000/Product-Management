NAME RA SYSDIV
MENU TEXT Division Parameter Set-up
UPPERCASE MENU TEXT DIVISION PARAMETER SET-UP
DESCRIPTION 
The 'Division Parameter Set-Up' function allows the
coordinator to define division parameters.
 
The module is designed to handle multiple divisions
within a medical center.  While most medical centers have only
one division, there are a number that are multi-divisional.  As
a result, it was necessary to structure the entire system around
the division concept.
 
However, single division medical centers will not notice this
in the everyday execution of the module.  Only in the initial set-up
will there be any reference to 'division' for these single division
sites.
 
Each division will have its own set of parameters defining such
division-wide criteria as, should the exam 'requested date' be asked
and are impressions required on reports.
