MENU TEXT Classification Enter/Edit
UPPERCASE MENU TEXT CLASSIFICATION ENTER/EDIT
DESCRIPTION 
This function allows the coordinator to enter/edit
personnel and their corresponding classifications.
 
There are currently four classifications the personnel can be
placed under. They are as follows:
 
      1) Technologist
      2) Interpreting Resident
      3) Interpreting Staff
      4) Clerk
