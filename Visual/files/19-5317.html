RA FILMEDIT (5317)    OPTION (19)

Name Value
NAME RA FILMEDIT
MENU TEXT Film Type Entry/Edit
UPPERCASE MENU TEXT FILM TYPE ENTRY/EDIT
DESCRIPTION 
This function allows the coordinator to edit existing
film types and enter new film types.
 
These film types will be the allowable choices the technologist can
choose from during the 'Status Tracking of Exams' and 'Case No. Exam Edit'
functions to indicate what films were used during a procedure.
