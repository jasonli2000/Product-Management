NAME RA MAINTENANCE
MENU TEXT Utility Files Maintenance Menu
UPPERCASE MENU TEXT UTILITY FILES MAINTENANCE MENU
DESCRIPTION 
The 'Utility Files Maintenance Menu' provides the
coordinator with many functions to enter/edit package utility
files.  All of the files are 'day-one' files. That is, these files
need to be initialized before the module is placed in a
production mode.
 
When the system is first loaded, many of these files
will have data.  However, the coordinator must review this data to
insure that the data meets his/her site's needs and to modify or add
file entries accordingly.
