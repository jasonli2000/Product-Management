RA MAJORAMIS (5322)    OPTION (19)

Name Value
NAME RA MAJORAMIS
MENU TEXT Major AMIS Code Entry/Edit
UPPERCASE MENU TEXT MAJOR AMIS CODE ENTRY/EDIT
DESCRIPTION 
This function allows the coordinator to add new AMIS
codes and edit existing AMIS codes.
 
When the system is first installed, all necessary entries and
data associated with each entry will be part of the module.
 
This function should only be used if the AMIS weights are changed
by VACO or if new codes are added by VACO.
CREATOR USER,ONE
TYPE run routine
ROUTINE 3^RAMAIN
