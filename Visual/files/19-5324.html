RA COMPEDIT (5324)    OPTION (19)

Name Value
NAME RA COMPEDIT
MENU TEXT Complication Type Entry/Edit
UPPERCASE MENU TEXT COMPLICATION TYPE ENTRY/EDIT
DESCRIPTION 
This function allows the coordinator to enter/edit
complication types that may be associated with an exam.  The
user can indicate an exam and a specific complication using the
'Case Number Edit' option.
 
The most commonly used complication type is a contrast material
reaction complication.  In the case of this complication, the
patient's record in the Allergy Tracking System
will be automatically updated to indicate
the patient has had a contrast material reaction.  As a result,
everytime the patient is registered for a procedure that may involve
contrast material, the receptionist will be warned and he/she should
then take the appropriate action.
 
When the system is initialized, two complications are loaded into
the module.  They are 'Contrast Material Reaction' and 'No Complication'.
Other site specific entries can be added using this function.
 
Finally, this field can be specified in criteria used when searching
the database using VA FileMan.
