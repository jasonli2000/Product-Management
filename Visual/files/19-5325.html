RA FLASHFORM (5325)    OPTION (19)

Name Value
NAME RA FLASHFORM
MENU TEXT Label/Header/Footer Formatter
UPPERCASE MENU TEXT LABEL/HEADER/FOOTER FORMATTER
DESCRIPTION 
This function allows the coordinator to design site
specific formats for the following:
 
      1) Flash Cards
      2) Exam Labels
      3) Jacket Labels
      4) Report Headers
      5) Report Footers
 
After a format has been entered or edited, the system will display
a test version of the format.
 
The 'Imaging Locations' file uses this file to obtain the format
specifications for the following location parameters:
 
      1) Default Flash Card Format
      2) Default Exam Label Format
      3) Default Jacket Label Format
      4) Default Report Header Format
      5) Default Report Footer Format
 
  These parameters are set using the 'Location Parameter Set-Up'
function.
