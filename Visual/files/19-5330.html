NAME RA SYSLOC
MENU TEXT Location Parameter Set-up
UPPERCASE MENU TEXT LOCATION PARAMETER SET-UP
DESCRIPTION 
The 'Location Parameter Set-Up' function allows the
coordinator to define location parameters.
 
Within a specific division, there may be a number of physical
locations where a procedure can be performed.  For
example, a division may have its own main department on
the second floor and  a satellite location on the
first floor at an outpatient clinic area.
 
The module is designed to handle multiple locations within a
division.  Each location will have its own set of parameters
defining such location-specific criteria as printer devices for
flash cards and jacket labels.
 
A location may only be associated with one division.
