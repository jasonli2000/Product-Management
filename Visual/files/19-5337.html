RA EDITCN (5337)    OPTION (19)

Name Value
NAME RA EDITCN
X ACTION PRESENT YES
MENU TEXT Case No. Exam Edit
UPPERCASE MENU TEXT CASE NO. EXAM EDIT
DESCRIPTION 
This function allows the user to edit exams for patients by selecting
either the case number or the patient's name.  If the case number doesn't
exist or is inactive, the system will indicate so with an error message.
This function should be used to re-edit fields that have already been
entered but need to be corrected. Only fields where data has previously
been entered will appear for re-edit.  The Status Tracking options should
be used to enter data and insure that exams progress to a status of
'Complete'.
 
Once an exam is edited and in the 'COMPLETE' status, the associated
request will display a message as follows: "** Note: Request has been
changed by Radiology/Nuclear Medicine**".
CREATOR USER,ONE
TYPE run routine
