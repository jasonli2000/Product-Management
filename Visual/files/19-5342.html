NAME RA PROFSORT
X ACTION PRESENT YES
MENU TEXT Exam Profile (selected sort)
UPPERCASE MENU TEXT EXAM PROFILE (SELECTED SORT)
DESCRIPTION 
This function allows the user to list a patient's exam profile.
The profile can be sorted by either exam procedure, exam status,
or exam date order.
  
If the user selects a specific exam, the module switches control
to the 'View Exam by Case No.' function, which will display
very detailed information about the exam, such as the requesting
physician and the technologist who performed the exam.
