RA DELINQUENT (5348)    OPTION (19)

Name Value
NAME RA DELINQUENT
MENU TEXT Delinquent Status Report
UPPERCASE MENU TEXT DELINQUENT STATUS REPORT
DESCRIPTION 
This option allows the supervisor to generate a listing of
examination reports with a status considered delinquent. The only statuses
considered to be delinquent are those designated through the 'Exam Status
Entry/Edit' option.
CREATOR USER,ONE
