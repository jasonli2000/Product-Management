NAME RA WKLRES
MENU TEXT Resident Report
UPPERCASE MENU TEXT RESIDENT REPORT
DESCRIPTION 
This option allows the supervisor to generate a report of examinations for
interpreting resident physicians. If the supervisor has access to more
than one division or imaging type, s/he is prompted to select the
divisions and imaging types to include in the report. The supervisor
is prompted to select the interpreting resident physicians to include in
the report. The supervisor may select one, many or all interpreting
resident physicians. The supervisor is prompted to enter a date range for
the exams to be included in the report. The exam status values that will
be included in the report are displayed by imaging type.
 
The report contains the following information: procedure and number of
examinations. A 'summary' report groups all examinations together by
interpreting resident physician for each imaging type and division. A
'detailed' report gives information for each individual procedure
performed by each interpreting resident physician.
