Name Value
NAME RA WKLSTAFF
MENU TEXT Staff Report
UPPERCASE MENU TEXT STAFF REPORT
DESCRIPTION 
This option allows the supervisor to generate a report of examinations for
interpreting staff physicians. If the supervisor has access to more than
one division or imaging type, s/he is prompted to select the divisions
and imaging types to include in the report. The supervisor is prompted to
select the interpreting staff physicians to include in the report. The
supervisor may select one, many or all interpreting staff physicians. The
supervisor is prompted to enter a date range for exams to be included in
the report. The exam status values that will be included in the report
will be displayed by imaging type.
 
The report contains the following information: procedure and number of
exams. A 'summary' report groups all examinations together by interpreting
staff physician for each imaging type and division. A 'detailed' report
gives information for each individual procedure performed by each
interpreting staff physician.
