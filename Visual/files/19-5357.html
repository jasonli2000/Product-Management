NAME RA WKLROOM
MENU TEXT Camera/Equip/Rm Report
UPPERCASE MENU TEXT CAMERA/EQUIP/RM REPORT
DESCRIPTION 
This option allows the supervisor to generate a report of workloads for
camera/equipment/rooms. If the supervisor has access to more than one
division or imaging type, s/he is prompted to select the divisions and
imaging types to include in the report. The supervisor is prompted to
select the camera/equipment/rooms to include in the report. The supervisor
may select one, many or all camera/equipment/rooms. The supervisor is
prompted to enter a date range for the exams to be included in the report.
The exam status values that will be included in the report will be
displayed by imaging type.
 
The report contains the following information: procedure, number of
examinations, weighted work units, and percent of total weighted work
units.  A 'summary' report groups all examinations together by
camera/equipment/rooms for each imaging type and division. A 'detailed'
report gives information for each individiual procedure performed by each
camera/equipment/rooms.
