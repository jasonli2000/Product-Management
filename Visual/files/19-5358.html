RA WKLPROCEDURE (5358)    OPTION (19)

Name Value
NAME RA WKLPROCEDURE
MENU TEXT Detailed Procedure Report
UPPERCASE MENU TEXT DETAILED PROCEDURE REPORT
DESCRIPTION 
This option allows the supervisor to generate a workload report
for each 'Detailed Procedure' completed for one imaging type only. If
the supervisor has access to more than one division, s/he is prompted to
select the divisions to include in the report. One, many or all divisions
may be selected. The user is prompted for an imaging type. One imaging
type may be selected. If the user has access to one imaging type only,
then that imaging type will be selected automatically. The user is then
prompted to enter a date range. The examination status values that will
be included in the report are displayed to the user.
 
The report consists of the following information for each AMIS category:
procedure, number of examinations In and Out, total number of
examinations, percent of exams, weighted work units, and percent of
weighted work units.
