RA PURGE (5362)    OPTION (19)

Name Value
NAME RA PURGE
MENU TEXT Purge Data Function
UPPERCASE MENU TEXT PURGE DATA FUNCTION
DESCRIPTION 
This option allows IRM to purge rad/nuc med data from the system.  The
following types of data can be purged:
 
       1) Activity Logs
       2) Status Tracking Times
       3) Reports (but not impressions)
       4) Clinical Histories
       5) Order Data
 
 
For each of the above, the user is first asked how many days of on-line
data should be saved. The user is then asked when he wants this job to
run. It should not be executed during normal working hours.
 
Finally, this routine will produce a large amount of global kills
and as a result, journaling may need to be disabled during this process.
Otherwise, many journal tapes may be produced.
CREATOR USER,ONE
TYPE run routine
ROUTINE SCH^RAPURGE
