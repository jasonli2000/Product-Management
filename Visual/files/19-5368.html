RA STATRPT (5368)    OPTION (19)

Name Value
NAME RA STATRPT
MENU TEXT Status Time Report
UPPERCASE MENU TEXT STATUS TIME REPORT
DESCRIPTION 
This option allows the supervisor to generate a 'Status Tracking
Statistics' Report. For each status change within each division, the report
lists the original status, the updated status, minimum time to make the
status change, maximum time to make the status change, and the average time
to make the status change for an associated procedure.
CREATOR USER,ONE
TYPE run routine
DISPLAY OPTION? YES
ROUTINE RASTAT
