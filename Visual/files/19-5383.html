NAME RA LWKLWARD
MENU TEXT Ward Report
UPPERCASE MENU TEXT WARD REPORT
DESCRIPTION 
This option allows the supervisor to generate a listing of ward workloads.
The supervisor is prompted to generate a 'detailed' or 'summary' report.
If the supervisor has access to more than one division or imaging type,
s/he is prompted to select the divisions and imaging types to include in
the report. The supervisor is prompted to select the wards to include in
the report. The supervisor may select one, many or all wards. The
supervisor is prompted to enter a date range for the exams to be included
in the report. The exam status values that will be included in the report
will be displayed by imaging type.
 
The reports are listed by division, imaging type within the division, and
then by ward within the imaging type. The report contains the following
information: procedure, number of examinations, weighted work units, and
percent of total weighted work units. A 'summary' report groups all
examinations together by ward for each imaging type and division. A
'detailed' report gives information for each individual procedure
performed within the ward.
