Name Value
NAME RA LWKLBEDSEC
MENU TEXT PTF Bedsection Report
UPPERCASE MENU TEXT PTF BEDSECTION REPORT
DESCRIPTION 
This option allows the supervisor to generate a listing of PTF Bedsection
workloads. The supervisor is prompted to generate a 'detailed' or
'summary' report. If the supervisor has access to more than one division
or imaging type, s/he is prompted to select the divisions and imaging
types to include in the report. The supervisor is prompted to select the
bedsections to include in the report. The supervisor may select one, many
or all bedsections. The supervisor is prompted to enter a date range for
the exams to be included in the report. The exam status values that will
be included in the report will be displayed by imaging type.
 
The reports are listed by division, imaging type within the division, and
then by bedsection within the imaging type. The report contains the
following information: procedure, number of examinations, weighted work
units, and percent of total weighted work units. A 'summary' report groups
all examinations together by bedsection for each imaging type and
division. A 'detailed' report gives information for each individual
procedure performed within the bedsection.
