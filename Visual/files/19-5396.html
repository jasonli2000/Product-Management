RA REPRINT (5396)    OPTION (19)

Name Value
NAME RA REPRINT
MENU TEXT Draft Report (Reprint)
UPPERCASE MENU TEXT DRAFT REPORT (REPRINT)
DESCRIPTION 
This option should only be given to those users in the department
who need to reprint a draft report. For example, the transcriptionist
should have access to this option, but ward clerks should not. Instead, the
ward clerks should be given access to the 'Select Report to Print by
Patient' option.
