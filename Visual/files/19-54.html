XQHELP-ASSIGN (54)    OPTION (19)

Name Value
NAME XQHELP-ASSIGN
MENU TEXT Assign Editors
UPPERCASE MENU TEXT ASSIGN EDITORS
DESCRIPTION 
Allows the author of a help frame to assign editors.  A help frame is 
editable thru '^E' by the author, the editors, and anyone holding the
key XUAUTHOR.
CREATOR USER,ONE
HELP FRAME XQHELP-ASSIGNEDITOR
TYPE run routine
ROUTINE EN1^XQ61
