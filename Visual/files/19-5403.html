NAME RA BTCH
MENU TEXT Batch Reports Menu
UPPERCASE MENU TEXT BATCH REPORTS MENU
DESCRIPTION 
This menu contains options that allow the user to maintain the
batches associated with his/her user access code. Functions include
the following:
 
       1) Create a batch 
       2) Delete Printed Batches
       3) List Reports in a Batch
       4) Print a Batch of Reports
       5) Remove/Add Reports from a Batch
       6) Verify Batch
CREATOR USER,ONE
