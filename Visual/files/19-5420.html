RA IMGLOG (5420)    OPTION (19)

Name Value
NAME RA IMGLOG
MENU TEXT Imaging Type Activity Log
UPPERCASE MENU TEXT IMAGING TYPE ACTIVITY LOG
DESCRIPTION 
This printout lists the activity log for all imaging types in the
'TYPE OF IMAGING' file (#79.2).
 
The types of activities listed are the following:
 
       1) When imaging type parameters were changed
       2) When data was scheduled to be purged
       3) When data was actually purged
       4) When the on-line data criteria was modified
CREATOR USER,ONE
TYPE run routine
ROUTINE 5^RASITE
