Name Value
NAME RA PAST
MENU TEXT Enter Last Past Visit Before VISTA
UPPERCASE MENU TEXT ENTER LAST PAST VISIT BEFORE V
DESCRIPTION 
This option allows the user to enter the last visit to the department for
a patient.
 
Many file rooms are divided by date. By entering the last exam date, it
will allow the file room clerk to lookup the patient's last visit by using
the patient demographic display option.
