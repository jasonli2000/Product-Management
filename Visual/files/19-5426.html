NAME RA RPTDIST
MENU TEXT Distribution Queue Menu
UPPERCASE MENU TEXT DISTRIBUTION QUEUE MENU
DESCRIPTION 
This menu contains the options which allow the user to print reports
sorted for distribution to the wards, clinics and file rooms.
 
Other options on the menu include printing the activity logs for the
various distribution queues and displaying a report's print status.
