RA RPTDISTQUE (5427)    OPTION (19)

Name Value
NAME RA RPTDISTQUE
MENU TEXT Print By Routing Queue
UPPERCASE MENU TEXT PRINT BY ROUTING QUEUE
DESCRIPTION 
This option allows the user to print the reports for the respective
distribution queues. The user is prompted for the routing queue, division,
imaging type, sort sequence and unprinted or reprint reports. The
reports are then printed (preceded and followed by the queue banner). The
header of each report includes the distribution queue name.
CREATOR USER,ONE
TYPE run routine
ROUTINE 1^RARTST1
