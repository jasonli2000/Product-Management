RA RPTDISTSINGLEWARD (5428)    OPTION (19)

Name Value
NAME RA RPTDISTSINGLEWARD
MENU TEXT Individual Ward
UPPERCASE MENU TEXT INDIVIDUAL WARD
DESCRIPTION 
This option allows the user to print the reports for
a select ward. The user is prompted for ward, division, sort
sequence (Terminal Digits, SSN, Patient) and unprinted or
reprint reports. The reports are then printed preceded and
followed by the Ward Reports queue banner. The header of each
report includes "***** Ward Distribution *****".
CREATOR USER,ONE
TYPE run routine
ROUTINE 2^RARTST1
