RA RPTDISTSINGLECLINIC (5429)    OPTION (19)

Name Value
NAME RA RPTDISTSINGLECLINIC
MENU TEXT Single Clinic
UPPERCASE MENU TEXT SINGLE CLINIC
DESCRIPTION 
This option allows the user to print the reports
for a select clinic. The user is prompted for clinic, division,
sort sequence (Terminal Digits, SSN, Patient) and unprinted or
reprint reports. The reports are then printed preceded and
followed by the Clinic Reports queue banner. The header of
each report includes "***** Clinic Distribution *****".
CREATOR USER,ONE
TYPE run routine
ROUTINE 3^RARTST1
