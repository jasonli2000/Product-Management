RA DISTP (5431)    OPTION (19)

Name Value
NAME RA DISTP
MENU TEXT Report Distribution Lists
UPPERCASE MENU TEXT REPORT DISTRIBUTION LISTS
DESCRIPTION 
This option allows the user to generate a report which
contains information corresponding to each distribution
queue. The report contains the distribution queue name,
the top of page message, the category of reports included
in the queue and the inactivation date.
CREATOR USER,ONE
TYPE run routine
ROUTINE 16^RAMAINP
