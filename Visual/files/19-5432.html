RA RPTDISTPURGE (5432)    OPTION (19)

Name Value
NAME RA RPTDISTPURGE
MENU TEXT Distribution Queue Purge
UPPERCASE MENU TEXT DISTRIBUTION QUEUE PURGE
DESCRIPTION 
This option allows IRM to schedule the purging
of the distribution queue files. The user is prompted
for a date. Any report printed prior to the specified
date is purged from the Report Distribution file and the
activity log in the Report Distribution Queue file.
Output for the purge contains the date the
purge is started and finished.
 
Again, only information printed prior to the scheduled date
is purged.
CREATOR USER,ONE
TYPE run routine
ROUTINE PURGE^RARTST
