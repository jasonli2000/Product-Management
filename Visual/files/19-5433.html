RA RPTDISTREBUILD (5433)    OPTION (19)

Name Value
NAME RA RPTDISTREBUILD
MENU TEXT Rebuild Distribution Queues
UPPERCASE MENU TEXT REBUILD DISTRIBUTION QUEUES
DESCRIPTION 
This option allows IRM to rebuild the distribution queues.
The user is prompted for a date and any reports verified
on or after the specified date are placed in the Report
Distribution file. Depending on the category of the report and
the requirements of the distribution queue, there is an entry
in the Report Distribution file for each report and the
corresponding queue. In other words, if a report has a category
of outpatient and the Clinic Reports Queue includes outpatient
reports and the File Room Queue includes outpatient reports, there
will be two entries in the Report Distribution file. This option
will print the time the rebuild began and ended, the total number
of reports on file and the total number used to rebuild the
distribution file.
CREATOR USER,ONE
TYPE run routine
ROUTINE RESET^RARTST
