RA RPTDISTAUTOPURGE (5434)    OPTION (19)

Name Value
NAME RA RPTDISTAUTOPURGE
MENU TEXT Autopurge of Distribution Queues
UPPERCASE MENU TEXT AUTOPURGE OF DISTRIBUTION QUEU
DESCRIPTION 
This option allows the Site Manager to schedule the purging of
the distribution queue files as a background job. It is
recommended this option be scheduled for an off hour. This
option must be scheduled via Fileman or Kernel. The option
must be scheduled for a specific date and a device
must be specified for the output. The scheduling frequency
recommended is daily. The scheduled date is used when
purging. Any report printed prior to the scheduled date,
is purged from the Report Distribution file and the activity
log in the Report Distribution Queue file .
Output for the purge contains the date the purge is
started and finished.
 
Again, only information printed prior to the scheduled date
is purged.
CREATOR USER,ONE
TYPE run routine
