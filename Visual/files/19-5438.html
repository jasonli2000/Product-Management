RA RPTDISTLISTUNPRINTED (5438)    OPTION (19)

Name Value
NAME RA RPTDISTLISTUNPRINTED
MENU TEXT Unprinted Reports List
UPPERCASE MENU TEXT UNPRINTED REPORTS LIST
DESCRIPTION 
This option allows the user to generate a report which contains
information about all the unprinted reports in the Report
Distribution file. The report contains the day/case #, patient
name, social security number, date verified, ward/clinic and
routing queue.
CREATOR USER,ONE
TYPE run routine
ROUTINE 5^RARTST1
