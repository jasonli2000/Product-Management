RA RPTDISTLISTCLINIC (5439)    OPTION (19)

Name Value
NAME RA RPTDISTLISTCLINIC
MENU TEXT Clinic Distribution List
UPPERCASE MENU TEXT CLINIC DISTRIBUTION LIST
DESCRIPTION 
This option allows the user to generate a report which contains
information about the reports in the clinic distribution queue.
The report can be generated for all clinics or a select clinic.
It can also be generated for 'PRINTED' or 'UNPRINTED' reports.
If the report is generated for 'PRINTED' reports, the user is
prompted for a date range. The report then contains the day/
case #, patient name, social security number,
date printed, printed by and clinic. If the report is generated
for 'UNPRINTED' reports, the report contains day/case #, patient
name, social security number, date verified and clinic.
CREATOR USER,ONE
TYPE run routine
ROUTINE 6^RARTST1
