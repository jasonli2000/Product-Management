RA RPTDISTPRINTSTATUS (5441)    OPTION (19)

Name Value
NAME RA RPTDISTPRINTSTATUS
MENU TEXT Report's Print Status
UPPERCASE MENU TEXT REPORT'S PRINT STATUS
DESCRIPTION 
This option allows the user to inquire about the print status of
a specific report. The print status can only be checked for
verified reports. The inquiry lists the reports' day/case #,
patient name, social security number, procedure, date verified,
routing queue, date printed, printed by and ward/clinic.
CREATOR USER,ONE
TYPE run routine
DISPLAY OPTION? YES
ROUTINE 8^RARTST1
