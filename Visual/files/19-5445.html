RA ORDERCANCEL (5445)    OPTION (19)

Name Value
NAME RA ORDERCANCEL
MENU TEXT Cancel a Request
UPPERCASE MENU TEXT CANCEL A REQUEST
DESCRIPTION 
This option allows the user to cancel a request.
If the user is not a department employee s/he
will only be able to cancel pending requests. If the user
is a department employee, s/he will be able to cancel
pending, held and scheduled requests. When an exam is
cancelled, a reason must be entered for the cancellation.
If a reason is not entered, the request is not cancelled.
CREATOR USER,ONE
TYPE run routine
ROUTINE 3^RAORD
