RA ORDERLOG (5447)    OPTION (19)

Name Value
NAME RA ORDERLOG
MENU TEXT Log of Scheduled Requests by Procedure
UPPERCASE MENU TEXT LOG OF SCHEDULED REQUESTS BY P
DESCRIPTION 
This option allows the user to generate a list of scheduled requests by
procedure.  The list includes the following information: procedure,
patient name, social security number, patient location, requesting
location if it was different than the current patient location, scheduled
time of examination and urgency. Before printing a report, the option will
prompt for imaging locations and allows selection of only the location(s)
to which the user has access through Personnel Classification. The
location(s) selected must also be within the current sign-on imaging type.
Requests for which an imaging location has not been entered will print on
every report under location "UNKNOWN" regardless of which locations are
selected.
 
If a date range in the past is specified, the user will also be asked if
only "no-show's" should appear on the log.  If the response is 'Yes', then
only requests with a Scheduled Date in the past that are still in the
'Scheduled' status will appear on the report.  (If the patient had shown
up, registration of the exam would have caused the request status to go to
'Active', so the log assumes that if a request is not yet active, the
patient didn't show up.)
