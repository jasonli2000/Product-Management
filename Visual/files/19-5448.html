MENU TEXT Detailed Request Display
UPPERCASE MENU TEXT DETAILED REQUEST DISPLAY
DESCRIPTION 
This option allows the user to see a 'detailed' report on a requested 
examination for a particular patient. The following information is
displayed: the request status, urgency, procedure, date exam was
ordered, requestor, and patient location. If the request was 'cancelled'
or placed on 'hold', the reason will be displayed. If the requested exam
is in an examination status, that status will also be displayed. If 
information has been recorded in the status tracking log for the request,
the user will also have the option to view the log.
