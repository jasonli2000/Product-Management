RA ORDERPRINTPAT (5452)    OPTION (19)

Name Value
NAME RA ORDERPRINTPAT
X ACTION PRESENT YES
MENU TEXT Print Selected Requests by Patient
UPPERCASE MENU TEXT PRINT SELECTED REQUESTS BY PAT
DESCRIPTION 
This option allows the user to print any request for a
selected patient. The request contains the department location, patient
demographics, run date, urgency, request and clinical history.
CREATOR USER,ONE
TYPE run routine
E ACTION PRESENT YES
EXIT ACTION K RAOPT("ORDERPRINTPAT")
ENTRY ACTION S RAOPT("ORDERPRINTPAT")=""
ROUTINE 9^RAORD
