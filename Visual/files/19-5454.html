RA TRANSCRIP REPORT (5454)    OPTION (19)

Name Value
NAME RA TRANSCRIP REPORT
MENU TEXT Transcription Report
UPPERCASE MENU TEXT TRANSCRIPTION REPORT
DESCRIPTION 
This option allows you to print the number of lines transcribed for a date
range. The user is prompted to select the divisions to include in the
report. The user may select one, many or all divisions. Next, the user is
prompted to select the transcriptionists to include in the report. The
user may select one, many or all transcriptionists. If the user does not
want all the transcriptionists to appear in the report, then s/he may
choose which persons will appear. The choices are limited to those persons
who are classified as Radiology/Nuclear Medicine users AND have actually
entered report text in a Radiology/Nuclear Medicine report. Last, the user
is prompted to enter a date range.
 
The report lists the name of the transcriptionist, number of lines
transcribed, and the number of reports written for each division selected.
