RA EXAMSTATUS MASS OVERRIDE (5455)    OPTION (19)

Name Value
NAME RA EXAMSTATUS MASS OVERRIDE
MENU TEXT Mass Override Exam Status
UPPERCASE MENU TEXT MASS OVERRIDE EXAM STATUS
LOCK RA MGR
DESCRIPTION 
This option is used by the supervisor to override one or more
statuses of exams to complete.  The user is asked to select the statuses
to override and then asked for a cutoff date.  The cutoff date must be
at least 60 days prior to the current date.  A printed report is
generated of all the exams whose status is overridden to complete.
Due to the volume of records which may be affected by this option, it is
recommended that the ADPAC carefully consider the division parameters
setup in the Rad/Nuc Med Division file.  The fields in question are: Track
Exam Status Changes (.11) and Track Request Status Changes (.119).
