RA PROCMSGEDIT (5456)    OPTION (19)

Name Value
NAME RA PROCMSGEDIT
MENU TEXT Procedure Message Entry/Edit
UPPERCASE MENU TEXT PROCEDURE MESSAGE ENTRY/EDIT
DESCRIPTION 
This option is used to enter, edit and delete messages related to
procedures.  These messages are used to alert the requestor at the time they
enter their request of special requirements relating to the procedure.
