RA ORDERPENDING (5458)    OPTION (19)

Name Value
NAME RA ORDERPENDING
MENU TEXT Pending/Hold Rad/Nuc Med Request Log
UPPERCASE MENU TEXT PENDING/HOLD RAD/NUC MED REQUE
DESCRIPTION 
This option will print a list of requests in the 'pending' or 'hold'
status for the imaging location(s) chosen. The user is prompted for the
request status (pending or hold) and for the imaging location(s).
 
The output will give patient name, last 4 digits of patient ID number,
procedure, patient location and date ordered sorted by request date. The
report contains an updated patient location with the requesting location
printing below the patient location if the requesting location is
different from the patient location.
 
If a request exists in the date range selected by the user that does not
have a value for imaging location, then a separate page is printed listing
that request. The imaging location will be labelled as "Unknown" in the
header of the page. This will identify orders that have not been submitted
to an imaging location.
CREATOR USER,ONE
TYPE run routine
DISPLAY OPTION? YES
