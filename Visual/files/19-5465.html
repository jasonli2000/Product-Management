RA SIGN-ON MSG (5465)    OPTION (19)

Name Value
NAME RA SIGN-ON MSG
MENU TEXT RAD/NM SIGN-ON MSG
UPPERCASE MENU TEXT RAD/NM SIGN-ON MSG
DESCRIPTION 
Inform the user if he/she has any imaging reports to pre-verify or verify.
This option should be entered as an "ITEM" on the XU USER SIGN-ON option.
When physicians classified as Rad/Nuc Med Residents or Staff within
the Radiology/Nuclear Medicine package sign on to the system, a routine
finds the total number of Radiology/Nuclear Medicine reports that are
waiting for this physician to review.  If the interpreting physician's
name is entered in one of the Resident fields of the exam record, the
message will state 'You have n reports to pre-verify'.  If the
interpreting physician's name is entered in one of the Staff fields of the
exam record, the message will state 'You have n reports to verify'.
