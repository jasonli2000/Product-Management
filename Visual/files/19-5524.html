ECTP CRED/PRIV SUM (5524)    OPTION (19)

Name Value
NAME ECTP CRED/PRIV SUM
MENU TEXT Credentials Summary
UPPERCASE MENU TEXT CREDENTIALS SUMMARY
DESCRIPTION 
This option produces a credential summary for the individual
specified.  Also included is a listing of privileges and
procedures currently assigned to a practitioner.
Data comes from the 'Credential Tracking' package.
