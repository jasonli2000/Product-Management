ECTP T&L ID (5526)    OPTION (19)

Name Value
NAME ECTP T&L ID
MENU TEXT Identify T&L for Services
UPPERCASE MENU TEXT IDENTIFY T&L FOR SERVICES
DESCRIPTION 
This option allows the user to display all T&L units, display local
services, and thereby match the T&L units to the correct service.
The option allows the input and editing of T&L data for local services.
Finally, the user may display the list of T&L units input for services
in order to verify accuracy.
