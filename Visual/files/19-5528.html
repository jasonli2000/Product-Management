ECTP T&L 1 SERVICE (5528)    OPTION (19)

Name Value
NAME ECTP T&L 1 SERVICE
MENU TEXT PAID Data for One Service
UPPERCASE MENU TEXT PAID DATA FOR ONE SERVICE
DESCRIPTION 
This option produces a report for a selected service.  The user may
choose the pay period/date range.  For the selected time-frame,
the following PAID totals will be displayed: annual leave, sick leave,
leave without pay, authorized absence, earned comp time, used comp
time, unscheduled regular time, and over time.
