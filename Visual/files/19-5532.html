Name Value
NAME ECTF CPO
MENU TEXT Fund Control Point Officials
UPPERCASE MENU TEXT FUND CONTROL POINT OFFICIALS
DESCRIPTION 
This option displays data from the 'Fund Control Point' File.
Data is sorted by station name and control point.  Fields
displayed include: control point, controlling service, and
control point manager.
