ECTQ RPT SEMI ANNUAL (5542)    OPTION (19)

Name Value
NAME ECTQ RPT SEMI ANNUAL
MENU TEXT Summary of Occurrence Screening
UPPERCASE MENU TEXT SUMMARY OF OCCURRENCE SCREENIN
DESCRIPTION 
This option calls the QA entry point PSM0^QAOSXTRN to produce the
Summary of Occurrence Screening Semi-Annual Report.  The report is
the same as attachment A in circular 10-88-71 supplement number 2.
Users may use the 'List QA Screens' option to determine the meaning
of the screen code displayed on this report.
CREATOR POSTMASTER
TYPE run routine
PACKAGE INTERIM MANAGEMENT SUPPORT
E ACTION PRESENT YES
ENTRY ACTION S X="QAOSXTRN" X ^%ZOSF("TEST") E S XQUIT=1 W !,X," routine does not exist on your system!,",!
ROUTINE PSM0^QAOSXTRN
