ECTQ RPT ADVERSE FINDINGS (5543)    OPTION (19)

Name Value
NAME ECTQ RPT ADVERSE FINDINGS
MENU TEXT Adverse Findings
UPPERCASE MENU TEXT ADVERSE FINDINGS
DESCRIPTION 
This option calls the QA entry point PAD0^QAOSXTRN to produce the
Adverse Findings report.  This report does not display names or codes
of the involved practitioner.  The report displays occurrences with
care level findings of two or three.  Fields shown include: patient
identification, occurrence date, screen code, status, and level.
Users may use the 'List QA Screens' option to determine the meaning
of the screen code displayed on this report.
CREATOR POSTMASTER
TYPE run routine
PACKAGE INTERIM MANAGEMENT SUPPORT
E ACTION PRESENT YES
ENTRY ACTION S X="QAOSXTRN" X ^%ZOSF("TEST") E S XQUIT=1 W !,X," routine does not exist on your system!,",!
ROUTINE PAD0^QAOSXTRN
