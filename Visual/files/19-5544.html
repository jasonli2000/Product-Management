MENU TEXT Bedsection Workload
UPPERCASE MENU TEXT BEDSECTION WORKLOAD
DESCRIPTION 
The user may choose to print this report for a single selected
AMIS segment, or for all segments in the AMIS 334-341 file.
If all segments are chosen, the user may select the AMIS
month/year date range for the report.  A separate page prints
for each AMIS month/year selected.  For each segment, the
following totals are shown: operating beds at the end of the month,
discharges, and patient days of care.  The discharge total is
comprised of 'DEATHS, BO AND ABO' plus 'DISCHARGE TO OPT/NSC'
plus 'DISCHARGES NOT OPT/NSC' plus 'TRANSFER OUT'.
Alternatively, the user may choose to print the report for a
single segment.  After choosing the segment, the user may
select the AMIS month/year date range.  The same fields will
be displayed for the selected AMIS segment.
