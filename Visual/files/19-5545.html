NAME ECTS AMIS 334
MENU TEXT AMIS 334-341 Hospital Inpatient Data
UPPERCASE MENU TEXT AMIS 334-341 HOSPITAL INPATIEN
DESCRIPTION 
When a station loads the new MAS version 5, this option will print the
Hospital Inpatient AMIS from that package.  Until version 5 is loaded,
the option presents, for a user selected month/year, the following
fields:
Total admissions, transfers in, changes in bedsection, deaths, BO and
ABO, discharge to OPT/NSC, discharges not OPT/NSC, transfer out,
changes in bedsection, bed occupants EOM, absent bed occupant EOM,
patient days of care, days of auth absence <96 hrs, wait list in
hospital, wait list not in hospital, wait list service connected,
scheduled admis OPT/NSC status, operating beds EOM, female patients
remaining EOM, 1 day hemodialysis inpatients, and patient days of
care >45.  These fields are displayed by AMIS segment, and within
segment, by division.
