ECTS AMIS 345 (5546)    OPTION (19)

Name Value
NAME ECTS AMIS 345
MENU TEXT AMIS 345-346 NHCU/DOM Patient Data
UPPERCASE MENU TEXT AMIS 345-346 NHCU/DOM PATIENT
DESCRIPTION 
When a station loads the new MAS version 5, this option will print the
AMIS 345-346 from that package.  Until version 5 is loaded,
the option presents, for a user selected month/year, the following
fields:
Admis after rehosp >30 days, admissions--all others, transfers to
similar facil, from ASIH, discharges, deaths, transfers out-similar
facil, to ASIH, bed occupant, absent bed occupants, ASIH, female
bed occupants remaining, discharges-ASIH, died-ASIH, patient days of
care, days of auth abs <96hrs, operating beds EOM, waiting list
EOM (SC), and waiting list EOM (NSC).  These fields are displayed by
AMIS segment, and within segment by division.
