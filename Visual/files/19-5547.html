MENU TEXT AMIS 401 - Patient Care by Vet. Category
UPPERCASE MENU TEXT AMIS 401 - PATIENT CARE BY VET
DESCRIPTION 
Data for this report is pulled from the AMIS Segment file.  For a
selected date range, sorted by division, the report displays the
following fields:  patient classification category, number of
applications for care, inpatient hospital care received,
outpatient care received, and the acceptance rate.
Acceptance rate is calculated by this formula:
((hospital care received + NHCU care received + domiciliary care
received + outpatient care received) divided by the total number
of applications for care).
