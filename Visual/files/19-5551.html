NAME ECTD LAB STATS
MENU TEXT Laboratory Workload
UPPERCASE MENU TEXT LABORATORY WORKLOAD
DESCRIPTION 
This report allows the user to specify the range of accession areas
and the date range.  For the selected areas and dates, the report
displays the total number of samples or specimens, and
the total number of tests performed.  There is a subcount for each
accession area, and a total for all areas for the date range.
