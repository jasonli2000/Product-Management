ECTP LOCAL SERVICES (5559)    OPTION (19)

Name Value
NAME ECTP LOCAL SERVICES
MENU TEXT Identify Local Services from National File
UPPERCASE MENU TEXT IDENTIFY LOCAL SERVICES FROM N
DESCRIPTION 
Use this option to indicate which of the services from the National
Service file are 'local' services.  The option allows four choices:
(1) Enter and/or edit the station's services; (2) List all services
from the National Service file; (3) List the services from the National
Service file which have been identified as 'local' station services;
and (4) Edit a single service.
