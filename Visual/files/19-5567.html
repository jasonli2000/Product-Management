ECTF ACT REC (5567)    OPTION (19)

Name Value
NAME ECTF ACT REC
MENU TEXT Accounts Receivable
UPPERCASE MENU TEXT ACCOUNTS RECEIVABLE
DESCRIPTION 
This option displays data from the 'Accounts Receivable' file.  For a
selected fiscal year, the data is sorted by category.  Data elements
displayed include category, principal balance, and total paid principal.
