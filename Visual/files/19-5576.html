ECTW UD SCT (5576)    OPTION (19)

Name Value
NAME ECTW UD SCT
X ACTION PRESENT YES
MENU TEXT Total Cost per Service
UPPERCASE MENU TEXT TOTAL COST PER SERVICE
DESCRIPTION 
This option is taken from the Inpatient Meds - Unit Dose module.
This report prints the total cost of dispensed medications for
ward services (medicine, surgery, etc.) over a user specified
time frame.  The report provides the total number of units
dispensed, the total cost of those units dispensed, and the average
cost of each unit dispensed on a service by service basis.
