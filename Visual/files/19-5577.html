NAME ECTW LRCAPS
MENU TEXT CAP Statistics Summary
UPPERCASE MENU TEXT CAP STATISTICS SUMMARY
DESCRIPTION 
This option is taken from the Laboratory software.  The report is a
summary of CAP statistics by user selectable station, lab section and
date.  It gives counts and totals of lab procedures by lab section.
Counts are also given for tests on inpatients and outpatients, stat
tests, repeats, QC and standards.  Unit weights (UW) are calculated
by the weight defined in the CAP file (#64) times the specimen count.
This report requires 132 columns for display/printout.
