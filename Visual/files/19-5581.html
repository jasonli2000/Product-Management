NAME IVM MAIN MENU
MENU TEXT IVM Main Menu
UPPERCASE MENU TEXT IVM MAIN MENU
DESCRIPTION 
This menu contains all options available with the IVM (Income Verification
Match) software.  These option are grouped in three menus: an upload
menu, an outputs menu, and an IRM menu.  The upload menu contains
options to upload demographics, SSN, and insurance information.  This
menu is locked with the 'IVM UPLOAD' key.  The outputs menu contains
various reports and inquiries.  The IRM menu contains options for
purging, controlling parameters, and viewing transmission statistics.
This menu is locked with the 'IVM SYS' key.
