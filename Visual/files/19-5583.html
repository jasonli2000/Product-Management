IVM UPLOAD DEM (5583)    OPTION (19)

Name Value
NAME IVM UPLOAD DEM
MENU TEXT Demographics Upload
UPPERCASE MENU TEXT DEMOGRAPHICS UPLOAD
LOCK IVM UPLOAD
DESCRIPTION 
This option is used to upload demographic information which has
been transmitted to the facility from the IVM Center.  This information
was collected during the Means Test verification process by IVM
personnel.
 
The demographic information is classified as either 'uploadable' or
'non-uploadable.'  Uploadable information may automatically be loaded
into the DHCP database upon review.  Non-uploadable information must
be manually entered into DHCP if the information is judged to be
accurate.  Either type of data may be rejected and purged from
the IVM database where it is temporarily stored.
