IVM SYS TRANSMISSION REPORT (5589)    OPTION (19)

Name Value
NAME IVM SYS TRANSMISSION REPORT
MENU TEXT IVM Transmission Report
UPPERCASE MENU TEXT IVM TRANSMISSION REPORT
LOCK IVM SYS
DESCRIPTION 
This report will allow the user to display a report of Income Verification
Match transmissions for a single date or a date range.
 
If a single date is chosen, the report will include: total number of
transmissions, total number of retransmissions, total Category A's with
insurance, total Category A's without insurance, and total Category C's
without insurance.
 
If a date range is chosen, the report will include: total number of
days for the date range, total number of transmissions, average number of
transmissions per day, average number of retransmissions per day,
percentage of Category A's with insurance, percentage of Category A's
without insurance, and percentage of Category C's without insurance.
 
If no data can be found for the selected date or date range, a message
will be displayed notifying the user that no data was found.
CREATOR POSTMASTER
TYPE run routine
PACKAGE INCOME VERIFICATION MATCH
