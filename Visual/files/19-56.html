XQHELP-UPDATE (56)    OPTION (19)

Name Value
NAME XQHELP-UPDATE
MENU TEXT New/Revised Help Frames
UPPERCASE MENU TEXT NEW/REVISED HELP FRAMES
DESCRIPTION 
This option will produce a fileman listing of help frames sorted by
DATE LAST UPDATED.  It is useful if you wish to view any recently
created or revised frames.  You may also sort by package prefix.
CREATOR USER,ONE
HELP FRAME XQHELP-NEWFRAMES
TYPE print
DIC {DIP} DIC(9.2,
L. 0
FLDS [XQHNEW/REVISED FRAMES]
BY [XQHNEW/REVISED FRAMES]
