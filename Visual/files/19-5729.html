MCARMED (5729)    OPTION (19)

Name Value
NAME MCARMED
X ACTION PRESENT YES
MENU TEXT Cardiology Medication File Update
UPPERCASE MENU TEXT CARDIOLOGY MEDICATION FILE UPD
DESCRIPTION 
Allows Cardiology medications to be added/entered/deleted in the Medication
file.  The user cannot enter a drug into the patient's record unless it
has been marked as a Cardiology drug in this file. All drugs marked as
Cardiology drugs must be present on the hospital formulary.
CREATOR USER,ONE
TYPE action
E ACTION PRESENT YES
EXIT ACTION D ^MCDIEDIE
ENTRY ACTION S MCARCODE="C",MCICDSS="CAR" D EN1^MCARSUP K MCARCODE
