MCGICONV (5835)    OPTION (19)

Name Value
NAME MCGICONV
X ACTION PRESENT YES
MENU TEXT Convert Old Records to Released Not Verified
UPPERCASE MENU TEXT CONVERT OLD RECORDS TO RELEASE
DESCRIPTION 
This option will convert old records or records that do not have a
released status to Release Not Verified.  It can only be used if
Release Control/Electronic Signature is turned on.  This option allows
the non-key holders to view the old records but not edit them.  Before
executing this option, turn on Release Control/Electronic Signature then
go into the Enter/Edit option and select the records that you want to remain as
Draft status.
CREATOR USER,ONE
TYPE run routine
E ACTION PRESENT YES
EXIT ACTION D ^MCDIEDIE
ENTRY ACTION S MCFILE=699,MCARCODE="G"
ROUTINE CONV^MCESCONV(699)
