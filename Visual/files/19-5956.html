MCESSUPONCATH (5956)    OPTION (19)

Name Value
NAME MCESSUPONCATH
X ACTION PRESENT YES
MENU TEXT Allow Printing of Superseded Reports
UPPERCASE MENU TEXT ALLOW PRINTING OF SUPERSEDED R
DESCRIPTION 
This option is a switch that will allow or disallow the non-managers to 
view superseded records from the print options. By switching this option
to on, superseded records can be viewed from the print option.  By
switching this option to off, superseded records will be removed from
the print option.
CREATOR USER,ONE
TYPE run routine
E ACTION PRESENT YES
EXIT ACTION D ^MCDIEDIE
ENTRY ACTION S MCPRO="CATH",MCICDSS="CAR"
ROUTINE SUPON^MCESMFDV
