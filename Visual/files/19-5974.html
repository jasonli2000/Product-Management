NAME MCPFTMANAGER
HEADER PRESENT? YES
MENU TEXT Release Control Options (PFT)
UPPERCASE MENU TEXT RELEASE CONTROL OPTIONS (PFT)
LOCK MCMANAGER
DESCRIPTION 
This option provides managers with options to turn on Release Control,
convert non-statuses to Released Not Verified, view superseded records
or reports that are marked for deletion, and to produce a report of
statuses. It also allows non-managers to print superseded records.
