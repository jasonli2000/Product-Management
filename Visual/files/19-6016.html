MCWKLDD (6016)    OPTION (19)

Name Value
NAME MCWKLDD
X ACTION PRESENT YES
MENU TEXT Workload Reporting Device Selection
UPPERCASE MENU TEXT WORKLOAD REPORTING DEVICE SELE
DESCRIPTION 
This option provides a mechanism for the user to select a printing device
or devices to receive message transmissions (Patient ID, Provider ID,
Patient SSN, CPT Code, and date/time).  The message transmission occurs at
the time the Medicine procedure is signed and released and only when the
Workload reporting option is turned on.
CREATOR USER,ONE
TYPE run routine
EXIT ACTION D ^MCDIEDIE
ROUTINE WLDEV^MCWORKLD
