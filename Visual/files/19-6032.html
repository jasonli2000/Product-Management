NAME SROMM
X ACTION PRESENT YES
MENU TEXT Morbidity & Mortality Reports
UPPERCASE MENU TEXT MORBIDITY & MORTALITY REPORTS
DESCRIPTION 
This option generates the Morbidity and Mortality reports to be used by
the Chief of Surgery.  This option includes the Mortality Report and
Perioperative Occurrences Report.
