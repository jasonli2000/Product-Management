XUTLOOPBACK (61)    OPTION (19)

Name Value
NAME XUTLOOPBACK
MENU TEXT Loopback Test of Device Port
UPPERCASE MENU TEXT LOOPBACK TEST OF DEVICE PORT
DESCRIPTION 
This option will allow the testing of a terminal line with the
use of a loopback connection on the line.
A loopback connector just ties pins 2, 3 together.
CREATOR USER,ONE
TYPE action
E ACTION PRESENT YES
ENTRY ACTION D ^%ZIS Q:POP X ^%ZOSF("EOFF"),^%ZOSF("TYPE-AHEAD"),"F I=65:1:90 U IO W *I R *X:1 U IO(0) W:X>32 $C(X)" X ^%ZOSF("EON"),^%ZIS("C")
