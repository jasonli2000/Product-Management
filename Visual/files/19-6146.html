SROKEY ENTER (6146)    OPTION (19)

Name Value
NAME SROKEY ENTER
X ACTION PRESENT YES
MENU TEXT Enter Restrictions for 'Person' Fields
UPPERCASE MENU TEXT ENTER RESTRICTIONS FOR 'PERSON
DESCRIPTION 
This option is used to enter restrictions on person fields.  It will
allow you to add keys to existing entries in the PERSON FIELD
RESTRICTION file.  If the field to be restricted has not already
been entered in the PERSON FIELD RESTRICTION file, you may enter it
along with the restricting keys through this option.
