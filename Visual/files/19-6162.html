NAME SR OR HOURS
X ACTION PRESENT YES
MENU TEXT Report of Normal Operating Room Hours
UPPERCASE MENU TEXT REPORT OF NORMAL OPERATING ROO
DESCRIPTION 
This report outputs the start time and the end time of the normal
working hours for all operating rooms or for the selected operating
room for each date within the specified date range.  The total time
of the normal working day is displayed for each operating room for
each date.
