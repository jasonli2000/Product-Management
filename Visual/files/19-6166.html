SR UPDATE FILES (6166)    OPTION (19)

Name Value
NAME SR UPDATE FILES
X ACTION PRESENT YES
MENU TEXT Update Site Configurable Files
UPPERCASE MENU TEXT UPDATE SITE CONFIGURABLE FILES
DESCRIPTION 
This option is used to add, edit, or delete information in the
site configurable files contained within the Surgery package.
 
File entries that are not to be used should be made inactive and should
not be deleted from the file.
