LR7O 60-64 AUTO (6174)    OPTION (19)

Name Value
NAME LR7O 60-64 AUTO
MENU TEXT Semi-automatic Linking of file 60 to 64
UPPERCASE MENU TEXT SEMI-AUTOMATIC LINKING OF FILE
DESCRIPTION 
This option should be run first to automatically link the
LABORATORY TESTS file (#60) to WKLD CODES file (#64). It will
reduce the LIM workload in preparation for PCE and OE/RR V. 3.0
installs.
 
This option looks for potential matches in File #60 with
File #64 and allow the user to link the entries. After this option
is run, the Manual Linking of File 60 to 64 [LR7O 60-64 MANUAL]
option should be run to manually link the entries that this
option could not match.
