ONCO SUSPENSE-CASEFIND (LAB) (6199)    OPTION (19)

Name Value
NAME ONCO SUSPENSE-CASEFIND (LAB)
MENU TEXT Automatic Casefinding-Lab Search
UPPERCASE MENU TEXT AUTOMATIC CASEFINDING-LAB SEAR
DESCRIPTION 
This option calls on a lab routine LRAPONC which allows the user to select
a lab section to search (Surgical Pathology, Cytology or Autopsy). A
temporary global is created storing the 'malignancy' entries found during
the time period selected. The option then creates an entry in the
Oncology patient file along with other 'suspense' information. If the
patient is already defined in the Oncology Patient file, a check is made
to see if a primary is defined with a diagnosis date which is the same
as the date of the lab test. If it isn't the same, the patient is then
put in 'suspense' using the date of the lab test found.
