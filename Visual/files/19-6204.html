ONCO SUSPENSE-CASEFIND (PTF) (6204)    OPTION (19)

Name Value
NAME ONCO SUSPENSE-CASEFIND (PTF)
MENU TEXT Automatic Casefinding-PTF Search
UPPERCASE MENU TEXT AUTOMATIC CASEFINDING-PTF SEAR
DESCRIPTION 
This option will search the PTF file for a specified time frame and pick
out ICD9 codes from 140.0 to 239.9. It will add entries to the Oncology
Patient file if they are not already there for that episode.
CREATOR USER,ONE
TYPE run routine
ROUTINE ONCOCFP
