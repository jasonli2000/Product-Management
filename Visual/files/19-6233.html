ONCO FOLLOWUP-DELINQUENT LIST (6233)    OPTION (19)

Name Value
NAME ONCO FOLLOWUP-DELINQUENT LIST
MENU TEXT Print Delinquent (LTF) List
UPPERCASE MENU TEXT PRINT DELINQUENT (LTF) LIST
DESCRIPTION 
Prints a list of all patients whose Due Followup date is over 3 months i.e.,
who have not been seen/contacted for over 15 months. These patients are
considered to be Lost to Follow-up at that time, and printing the report
causes the STATUS field to be set to 8 Lost to Follow-up.
CREATOR USER,ONE
TYPE run routine
ROUTINE DEL^ONCOPRT
