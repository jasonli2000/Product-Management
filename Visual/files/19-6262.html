ONCO STAT-SURVIVAL SITE (6262)    OPTION (19)

Name Value
NAME ONCO STAT-SURVIVAL SITE
MENU TEXT Survival by Site
UPPERCASE MENU TEXT SURVIVAL BY SITE
DESCRIPTION 
This option will produce a life table and survival curves for the
particular sites using a pre-defined search template.  A minimum of
5 years of data must be available.
