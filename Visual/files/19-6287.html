ONCO #SITE-PURGE SUSPENSE (6287)    OPTION (19)

Name Value
NAME ONCO #SITE-PURGE SUSPENSE
MENU TEXT Purge Patient Records with No Suspense/Primaries
UPPERCASE MENU TEXT PURGE PATIENT RECORDS WITH NO
DESCRIPTION 
This option drives through the ONCOLOGY PATIENT File 
(#160) looking for records that have no suspense date 
and no associated primaries.  Upon confirmation from 
the user, it deletes these records from the file.
