Name Value
NAME ONCO #SITE-FIX EXP PAT F/U
MENU TEXT Correct Next Follow-Up for Expired Patients
UPPERCASE MENU TEXT CORRECT NEXT FOLLOW-UP FOR EXP
DESCRIPTION 
This option searches all FOLLOW-UP Multiples (#400) on the 
ONCOLOGY PATIENT File (#160) looking for expired patients.  
It then makes sure that the NEXT FOLLOW-UP METHOD Sub-field 
(#6) is set to "9" (not followed).
