ONCO #SITE-CREATE ACOS REPORT (6291)    OPTION (19)

Name Value
NAME ONCO #SITE-CREATE ACOS REPORT
MENU TEXT Create a report to preview ACOS output
UPPERCASE MENU TEXT CREATE A REPORT TO PREVIEW ACO
DESCRIPTION 
Report which allows the tumor registrar to preview the contents of the
specified accessions intended for output to the ACOS disk.  This option is
independent of the disk capture function but uses the precise database
selection criteria.
CREATOR USER,ONE
TYPE run routine
