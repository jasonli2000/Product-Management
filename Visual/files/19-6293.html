ONCO #STATE DATA DISK (6293)    OPTION (19)

Name Value
NAME ONCO #STATE DATA DISK
MENU TEXT Create State/VACCR Data Download
UPPERCASE MENU TEXT CREATE STATE/VACCR DATA DOWNLO
DESCRIPTION 
This option will allow for the creation of a disk to be 
for transmission of cancer registry information
including confidential patient identity data to the State
collecting agencies.  The length of the strings gathered
here are 1150 characters as opposed to the 850 character
ACOS data strings.

TYPE run routine
