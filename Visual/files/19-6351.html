ESP MISC BICYCLE EDIT (6351)    OPTION (19)

Name Value
NAME ESP MISC BICYCLE EDIT
X ACTION PRESENT YES
MENU TEXT Bicycle Registration Add/Edit
UPPERCASE MENU TEXT BICYCLE REGISTRATION ADD/EDIT
OUT OF ORDER MESSAGE PATCH ES*1*46
DESCRIPTION 
This submenu option allows the entry of bicycle registration information
into the registration module.  This option is provided with the package but
will not be required for use at every field station.  Local field stations
may determine whether its use is necessary.
CREATOR USER,SEVENTEEN
TYPE action
PACKAGE POLICE & SECURITY
