MENU TEXT Pet Registration Add/Edit

OUT OF ORDER MESSAGE PATCH ES*1*46
DESCRIPTION 
This is the submenu option that allows the entry of pet registration
information into the Registration Module.  This option is provided with
the package but will not be required for use at every field station.
Local field stations may determine whether its use is necessary.
