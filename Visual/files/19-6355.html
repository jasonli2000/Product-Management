ESP VEHICLE INQUIRY (6355)    OPTION (19)

Name Value
NAME ESP VEHICLE INQUIRY
X ACTION PRESENT YES
MENU TEXT Lookup Vehicle Registration
UPPERCASE MENU TEXT LOOKUP VEHICLE REGISTRATION
DESCRIPTION 
This submenu inquiry option allows the user to look up vehicle
registration information on a specific registered vehicle.  The
inquiry may be obtained by entering the vehicle's registration plate
number, decal number, or owner's name.
CREATOR POSTMASTER
TYPE run routine
PACKAGE POLICE & SECURITY
EXIT ACTION D ^ESPCLEAN
