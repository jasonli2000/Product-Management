ESP CRIMINAL STATUTE LOOKUP (6363)    OPTION (19)

Name Value
NAME ESP CRIMINAL STATUTE LOOKUP
X ACTION PRESENT YES
MENU TEXT Criminal Statute Lookup
UPPERCASE MENU TEXT CRIMINAL STATUTE LOOKUP
DESCRIPTION 
This is a submenu option that allows police officers to look up
offense statutes.  It allows them to read or print specific information
on each individual offense statute to include statute
number and the actual definition of the statute.
CREATOR POSTMASTER
TYPE inquire
PACKAGE POLICE & SECURITY
EXIT ACTION D ^ESPCLEAN
DIC {DIC} ESP(915,
DIC(0) AEMQ
FLDS [ESP STATUTE LOOKUP]
DIC {DIQ} ESP(915,
