Name Value
NAME ESP REVIEW COMPLETED OR
X ACTION PRESENT YES
MENU TEXT Review Completed Offense Report
UPPERCASE MENU TEXT REVIEW COMPLETED OFFENSE REPOR
LOCK ESP SUPERVISOR
DESCRIPTION 
This submenu option allows the designated supervisor to review an
officer's completed offense report.  If the report is completed fully,
the supervisor may close the report.  If the report needs additional
information, then a mail message will be sent to the investigating
officer informing them that the offense report needs additional work.
