ESP VEHICLE IDENTIFY (6412)    OPTION (19)

Name Value
NAME ESP VEHICLE IDENTIFY
X ACTION PRESENT YES
MENU TEXT Identify Unknown Vehicle
UPPERCASE MENU TEXT IDENTIFY UNKNOWN VEHICLE
OUT OF ORDER MESSAGE PATCH ES*1*46
DESCRIPTION 
This submenu option allows VA Police staff to run a check of
Registration records when only a limited amount of information is
known about the vehicle.  This is particularly useful when conducting
investigations where a hit & run vehicle is involved.
CREATOR USER,SEVENTEEN
TYPE print
PACKAGE POLICE & SECURITY
EXIT ACTION D ^ESPCLEAN
DIC {DIP} ESP(910.2,
L. 0
FLDS [CAPTIONED]
BY [ESP VEHICLE IDENTIFY]
