ESP PRINT CRIME REPORT (6422)    OPTION (19)

Name Value
NAME ESP PRINT CRIME REPORT
X ACTION PRESENT YES
MENU TEXT Print Crime Report
UPPERCASE MENU TEXT PRINT CRIME REPORT
LOCK ESP CHIEF
DESCRIPTION 
This option allows persons holding the ESP CHIEF key to
print a crime report for a specific time period.  You must
first run the Generate Crime Statistics option before you
can print a crime report.

TYPE run routine
