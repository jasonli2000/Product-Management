NAME ESP MISSING PATIENT REPORT-ALL
X ACTION PRESENT YES
MENU TEXT Missing Patient Report-All Cases
UPPERCASE MENU TEXT MISSING PATIENT REPORT-ALL CAS
OUT OF ORDER MESSAGE PATCH ES*1*46
DESCRIPTION 
This submenu option allows the VA Police staff to generate a report
of all Missing Patient Reaction investigations.  The option allows
them to select a date range based upon the Date/Time of Offense
field contained within the Uniform Offense Report.
