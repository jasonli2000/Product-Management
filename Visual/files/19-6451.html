QAC STATUS (6451)    OPTION (19)

Name Value
NAME QAC STATUS
MENU TEXT Open / Close / Delete Contact Record
UPPERCASE MENU TEXT OPEN / CLOSE / DELETE CONTACT
LOCK QAC EDIT
DESCRIPTION 
This option lets you reopen a Consumer Contact record that has been
closed, Close a record without seeing all the fields, and
Delete a record from the file.
Deleting a record deletes / purges it from the database.
CREATOR USER,ONE
TYPE run routine
ROUTINE QACSTAT
