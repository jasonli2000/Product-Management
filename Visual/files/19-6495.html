GMPL DE-ASSIGN LIST (6495)    OPTION (19)

Name Value
NAME GMPL DE-ASSIGN LIST
MENU TEXT Remove Selection List from User(s)
UPPERCASE MENU TEXT REMOVE SELECTION LIST FROM USE
DESCRIPTION 
This option allows a selection list to be removed from user(s) as
his/her preferred menu of problems to select from.  The Add New
Problem(s) option will no longer automatically present this menu
for the selected user(s) to choose from.
CREATOR POSTMASTER
TYPE run routine
PACKAGE PROBLEM LIST
E ACTION PRESENT YES
ENTRY ACTION S GMPLSLST=$$LIST^GMPLBLD2("")
ROUTINE USERS^GMPLBLD3("")
