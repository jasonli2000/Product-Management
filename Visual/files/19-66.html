XURELOG (66)    OPTION (19)

Name Value
NAME XURELOG
MENU TEXT Restart Session
UPPERCASE MENU TEXT RESTART SESSION
DESCRIPTION 
This option allows the user to return to the sign-on logic so that
a session can be restarted without dropping a telecommunication line.
CREATOR USER,ONE
TYPE action
E ACTION PRESENT YES
ENTRY ACTION S:'$D(XQCH) XQCH="REST" G:$L(XQCH)>2 REST^XQ12 S XQUR="REST" G XPRMP^XQ12
