NAME RMPR4 CLOSE OUT
MENU TEXT Reconcile/Close Out Purchase Card Transaction
UPPERCASE MENU TEXT RECONCILE/CLOSE OUT PURCHASE C
DESCRIPTION 
This option is used to close out purchasing transactions that used the
Purchase Card.  This option will not post to IFCAP 1358.
