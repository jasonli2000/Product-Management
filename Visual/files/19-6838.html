QAM COPY MONITOR EDIT (6838)    OPTION (19)

Name Value
NAME QAM COPY MONITOR EDIT
MENU TEXT Copy/Edit a Monitor
UPPERCASE MENU TEXT COPY/EDIT A MONITOR
DESCRIPTION 
Copy/Edit a Monitor:  Takes existing monitor and copies it to a new 
monitor.  It automatically changes the code to "Copy of ..." and the 
Title to "Copy of ...".  this field allows up to 30 characters, so the code 
and name may be truncated if it is too long.  If the Code on the 
original monitor is "XX-1" then the code for the copy would read 
"Copy of XX-1".  It is recommended that the user change the code and 
title of the new monitor once the copy is made.
CREATOR POSTMASTER
TYPE run routine
