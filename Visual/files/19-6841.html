QAM GROUP FILE EDIT (6841)    OPTION (19)

Name Value
NAME QAM GROUP FILE EDIT
MENU TEXT Group Edit
UPPERCASE MENU TEXT GROUP EDIT
DESCRIPTION 
Group Edit:  Allows user to create groups off of any DHCP file that is 
in a QAM application group.  Example:  The Ward Location file 
contains all the wards at the site.  The user can select specific wards 
from this file to create a group.
