QAM PGMR DATA ELEMENTS EDIT (6848)    OPTION (19)

Name Value
NAME QAM PGMR DATA ELEMENTS EDIT
MENU TEXT Data Element File Edit
UPPERCASE MENU TEXT DATA ELEMENT FILE EDIT
DESCRIPTION 
Data Element File Edit:  Allows the programmer to add new data
elements to the Data Element file for capture.  Note:  Data elements
can be added to a monitor at any time, but the program will not
retroactively capture this data.
