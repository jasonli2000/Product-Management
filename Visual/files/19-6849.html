QAM RATIONALE FILE EDIT (6849)    OPTION (19)

Name Value
NAME QAM RATIONALE FILE EDIT
MENU TEXT Rationale Edit
UPPERCASE MENU TEXT RATIONALE EDIT
DESCRIPTION 
Rationale Edit:  Allows the user to enter new rationales (High 
Volume, Problem Prone, High Risk, Other) or change existing 
rationales.  Also allows the user to set the explanation 
field to "Yes" or "No" to allow for an explanation of the rationale.
