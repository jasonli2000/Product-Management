Name Value
NAME QAM RPT ADHOC
MENU TEXT Ad Hoc Fall Out Report
UPPERCASE MENU TEXT AD HOC FALL OUT REPORT
DESCRIPTION 
Ad Hoc Fall Out Report:  Allows the users to design their own reports 
from data in the Fall Out file.  The data elements present in the Fall
Out file are determined by the data elements selected when the monitor
was built.  An attempt to sort/print a data element that was not entered
into the monitor's OTHER DATA TO CAPTURE will produce no data for that
element.
