QAM MANUAL AUTO ENROLL RUN (6856)    OPTION (19)

Name Value
NAME QAM MANUAL AUTO ENROLL RUN
MENU TEXT Manually Run Auto Enroll
UPPERCASE MENU TEXT MANUALLY RUN AUTO ENROLL
DESCRIPTION 
Manually Run Auto Enroll:  Allows the uer to choose run of selected 
monitors or selected services.  Example:  Medical Service can request, 
through this option, a manual run of the auto enroll for the monitors 
belonging just to Medical Service.  The option asks the user for a date 
range and when the auto enroll should be queued to run.  The user 
also has the choice of having a report printed out of when those date 
ranges will be run.
CREATOR POSTMASTER
TYPE run routine
ROUTINE QAMAUTO4
