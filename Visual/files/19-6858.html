QAM TASKED AUTO ENROLL RUN (6858)    OPTION (19)

Name Value
NAME QAM TASKED AUTO ENROLL RUN
MENU TEXT Tasked Run of Auto Enroll
UPPERCASE MENU TEXT TASKED RUN OF AUTO ENROLL
DESCRIPTION 
Tasked Run of Auto Enroll:  This option runs the active monitors in the
QA Monitor file (#743) and auto enrolls any fall outs into the Fall Out
file (#743.1).  It should be queued to run every night sometime after
midnight.
CREATOR POSTMASTER
TYPE action
E ACTION PRESENT YES
ENTRY ACTION K QAMTODAY,^UTILITY($J,"QAM MONITOR"),^("QAM SERVICE") D ^QAMAUTO0
