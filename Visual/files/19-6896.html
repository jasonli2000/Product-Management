QAOS DELETE AUTO RUN DATES (6896)    OPTION (19)

Name Value
NAME QAOS DELETE AUTO RUN DATES
MENU TEXT Auto Enrollment Run Dates Purge
UPPERCASE MENU TEXT AUTO ENROLLMENT RUN DATES PURG
DESCRIPTION 
This option is used to delete data for a range of dates from the auto
enroll run dates file.  This file contains the dates on which auto enroll
was run and the number of patients auto enrolled for that date.
