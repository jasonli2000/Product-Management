QAOS GENERATE EWS BULLETIN (6908)    OPTION (19)

Name Value
NAME QAOS GENERATE EWS BULLETIN
MENU TEXT Early Warning System Transmission
UPPERCASE MENU TEXT EARLY WARNING SYSTEM TRANSMISS
DESCRIPTION 
This option allows the user to manually generate the Early Warning
System messages for any specific date range.  The data contained in
the messages is the same as the data for screens 101.1 and 109 on
the Summary of Occurrence Screening (Semi-Annual) report.
CREATOR POSTMASTER
TYPE run routine
ROUTINE EN1^QAOSEWS0
