QAOS TASKED EWS (6910)    OPTION (19)

Name Value
NAME QAOS TASKED EWS
MENU TEXT Auto Generate Early Warning System Message
UPPERCASE MENU TEXT AUTO GENERATE EARLY WARNING SY
DESCRIPTION 
This option automatically produces the Early Warning System messagess
on a monthly schedule.  The data contained in the messagess is the same
as the data for screens 101.1 and 109 on the Summary of Occurrence
Screening (Semi-Annual) report. This option should be queued to run
once a month.
