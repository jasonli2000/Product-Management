QAOS AUDIT INQUIRY (6911)    OPTION (19)

Name Value
NAME QAOS AUDIT INQUIRY
MENU TEXT Audit File Inquiry
UPPERCASE MENU TEXT AUDIT FILE INQUIRY
DESCRIPTION 
This option allows the user to pick an Occurrence Screen record and
view the audit file entry for this occurrence.  The audit file data
includes information on who modified the record, when it was modified,
and what modification was performed.
