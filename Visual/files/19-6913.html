QAOS PURGE DELETED RECORDS (6913)    OPTION (19)

Name Value
NAME QAOS PURGE DELETED RECORDS
MENU TEXT Purge Deleted Occurrence Screen Records
UPPERCASE MENU TEXT PURGE DELETED OCCURRENCE SCREE
DESCRIPTION 
This option is used to delete the Occurrence Screen records that have
been marked as deleted.  The user is prompted for a date range and
which screens should be deleted.
