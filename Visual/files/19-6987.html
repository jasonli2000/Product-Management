SPNL CLINICAL REGISTRATION MOD (6987)    OPTION (19)

Name Value
NAME SPNL CLINICAL REGISTRATION MOD
X ACTION PRESENT YES
MENU TEXT Clinical Information
UPPERCASE MENU TEXT CLINICAL INFORMATION
DESCRIPTION 
This option allows the user to edit fields in the SCD Registry that
pertain to the ASIA, Frankel, and VA classification systems.  The Last
Physical Exam date field may also be edited here.  Use of this option is
entirely optional; it contains no required information.
CREATOR USER,ONE
TYPE run routine
EXIT ACTION I $D(IOF) W @IOF D DISP^SPNMAIN
ROUTINE MENU^SPNFMENU(3,0)
