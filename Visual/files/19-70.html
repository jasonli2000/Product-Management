XUSERINQ (70)    OPTION (19)

Name Value
NAME XUSERINQ
MENU TEXT User Inquiry
UPPERCASE MENU TEXT USER INQUIRY
DESCRIPTION 
This option displays various user attributes.  If the user is currently
signed on, it displays the job and device numbers, the sign-on time, and
what option is being executed.  Otherwise, it displays the last sign-on
time. It also displays which keys are held by this user.
CREATOR USER,ONE
HELP FRAME XUUSER-INQ
TYPE inquire
DIC {DIC} VA(200,
DIC(0) AEMQ
FLDS [XUSERINQ]
IOP HOME
DIC {DIQ} VA(200,
