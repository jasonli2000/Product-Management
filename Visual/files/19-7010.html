ABSV MERGE TIME CARDS (7010)    OPTION (19)

Name Value
NAME ABSV MERGE TIME CARDS
MENU TEXT Merge Two Time Cards
UPPERCASE MENU TEXT MERGE TWO TIME CARDS
DESCRIPTION 
This option allows the user to merge the time data from two cards into one
card.  The time data from the second card will be added to the first.  The
organizational data from the first remains intact.  The second time card
is deleted following the merge.
CREATOR USER,ONE
TYPE run routine
PACKAGE VOLUNTARY TIMEKEEPING
ROUTINE MERGE^ABSVTED1
