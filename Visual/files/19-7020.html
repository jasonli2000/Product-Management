ABSV DELETE DAILY ENTRIES (7020)    OPTION (19)

Name Value
NAME ABSV DELETE DAILY ENTRIES
MENU TEXT Delete All Daily Entries THRU One Month
UPPERCASE MENU TEXT DELETE ALL DAILY ENTRIES THRU
LOCK ABSV MGR
DESCRIPTION 
This option allows the user to delete all daily entries for a specified
month.  It will not allow deletion of either the current month nor the
immediately preceding month.
CREATOR USER,ONE
TYPE run routine
PACKAGE VOLUNTARY TIMEKEEPING
ROUTINE ABSVDDE
