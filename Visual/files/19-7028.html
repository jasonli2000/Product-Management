ABSV OCC PURGE (7028)    OPTION (19)

Name Value
NAME ABSV OCC PURGE
MENU TEXT Purge Occasional Time Sheet Entries
UPPERCASE MENU TEXT PURGE OCCASIONAL TIME SHEET EN
LOCK ABSV MGR
DESCRIPTION 
This option will purge Occasional Time Sheet entries with transmission status
as transmitted.  This option will allow only deletion of past months.  It
will not allow deletion of last, current or future months.
CREATOR USER,ONE
TYPE run routine
