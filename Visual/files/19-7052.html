ABSV LOOP BACKDATE (7052)    OPTION (19)

Name Value
NAME ABSV LOOP BACKDATE
MENU TEXT Loop and Backdate All Cards for One Month
UPPERCASE MENU TEXT LOOP AND BACKDATE ALL CARDS FO
DESCRIPTION 
This option allows the user to loop through all time cards for a selected 
month and 'stuff' the BACKDATE code into the appropriate location in the
time card.  It then resets the transmission status to READY FOR TRANSMISSION.

TYPE run routine
