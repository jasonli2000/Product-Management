ABSV MASTER RECORD DISPLAY (7062)    OPTION (19)

Name Value
NAME ABSV MASTER RECORD DISPLAY
MENU TEXT Master Record Display/Print
UPPERCASE MENU TEXT MASTER RECORD DISPLAY/PRINT
DESCRIPTION 
This option allows the user to view to the screen or to print to paper a
structured query of the information in the Volunteer Master File, for a
specific volunteer.  It displays/prints only the information from a single
station number.
CREATOR USER,ONE
TYPE run routine
