ABSV MARK OCC FOR RETRANS (7065)    OPTION (19)

Name Value
NAME ABSV MARK OCC FOR RETRANS
MENU TEXT Mark Occasional Time Sheet for Retransmission
UPPERCASE MENU TEXT MARK OCCASIONAL TIME SHEET FOR
DESCRIPTION 
This option allows the user to mark for retransmission, an Occasional Hours
Time Sheet which has been transmitted to the Austin DPC and was not processed
or rejected.
