ACKQAS VISITS BY DIAG (7073)    OPTION (19)

Name Value
NAME ACKQAS VISITS BY DIAG
MENU TEXT Visits by Diagnosis
UPPERCASE MENU TEXT VISITS BY DIAGNOSIS
DESCRIPTION 
The Visits by Diagnosis report can be printed for a date range 
you select.  You can print the report for Audiology only, Speech 
Pathology only, or both Audiology and Speech Pathology.
  
You can choose to print the report for one or all clinician(s), other 
provider(s), or student(s).
  
The report lists clinic visits for the date range sorted by ICD-9CM 
diagnostic codes.  Since the diagnostic code field is a multiple 
field, the visit shows up once under each diagnostic code entered for 
that visit.
