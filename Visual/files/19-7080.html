ACKQAS WKLD GEN MAN (7080)    OPTION (19)

Name Value
NAME ACKQAS WKLD GEN MAN
MENU TEXT Compile A&SP Capitation Data
UPPERCASE MENU TEXT COMPILE A&SP CAPITATION DATA
DESCRIPTION 
The Workload Options distributed with QUASAR V. 1.0 contained useful 
information.  Due to insufficient resources for development and 
support of the central data base, the QUASAR V. 1.0 workload rollup 
report has been discontinued.  However, because the data is vital to 
local managers, the functionality remains in QUASAR V. 2.0 and is 
available in the Capitation Report.
  
The Compile A&SP Capitation Data option is used to generate capitation 
data for a selected Division or Divisions and a selected month. If data
has previously been compiled for the selected Division or Divisions and
the selected month, you are asked if you wish to continue.  If you do so,
the previously compiled data is deleted and the information will be
recalculated.  The compilation takes place in the background, and you are
notified by an E-Mail message when the task is finished.  You can use the
Print A&SP Capitation Report to review the data.
