ACKQAS WKLD VERIFY (7081)    OPTION (19)

Name Value
NAME ACKQAS WKLD VERIFY
MENU TEXT Print A&SP Capitation Report
UPPERCASE MENU TEXT PRINT A&SP CAPITATION REPORT
DESCRIPTION 
When the Compile A&SP Capitation Data option has generated information for
selected Division(s) and selected month, you are notified by an E-Mail
message.  You can then use the Print A&SP Capitation Report to review the
data.  This option produces a three-part report for each Division which
includes demographic, diagnostic and procedure data followed by a summary
(not separated by Division) that includes demographic, diagnostic and
procedure data.
