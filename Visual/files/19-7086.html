ACKQAS PRINT COST COMPARE (7086)    OPTION (19)

Name Value
NAME ACKQAS PRINT COST COMPARE
MENU TEXT Cost Comparison Report
UPPERCASE MENU TEXT COST COMPARISON REPORT
DESCRIPTION 
The Cost Comparison Report option produces a report of all CPT-4 
procedure codes used within a selected date range and their 
associated costs. The cost linked to a CPT code is based upon 
approximate private sector cost. Cost is entered by local A&SP 
supervisors using the Enter Cost Information for Procedures option on 
the Set Up/Maintenance menu.
