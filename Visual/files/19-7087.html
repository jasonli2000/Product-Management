NAME ACKQAS SET UP MENU
MENU TEXT Set Up/Maintenance
UPPERCASE MENU TEXT SET UP/MAINTENANCE
DESCRIPTION 
There are four tasks on the Set Up/Maintenance menu which must be
completed to set up and maintain the package.  These tasks are (1) Enter
Audiology and Speech Pathology Staff, (2) Establish Local Site Parameters,
(3) Update File Data at the Direction of VACO and (4) Update CPT Modifiers
per CO Directive.
