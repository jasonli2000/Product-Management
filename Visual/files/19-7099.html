DESCRIPTION 
This menu controls all the options used to set up and maintain the
Event Capture software. This includes the functionality necessary to
create the event code sceens, create and allocate the DSS units, and
appropriately control reporting sections. Access to this menu should be
restricted to the package coordinator and his or her designees.
The ECMGR security locks this menu option.
