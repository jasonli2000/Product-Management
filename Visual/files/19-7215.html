XPD PRINT PACKAGE PATCHES (7215)    OPTION (19)

Name Value
NAME XPD PRINT PACKAGE PATCHES
MENU TEXT Display Patches for a Package
UPPERCASE MENU TEXT DISPLAY PATCHES FOR A PACKAGE
DESCRIPTION 
This option will print all patches installed for a Package. It displays
the Date Installed and who installed the patches. It optionally will
print the description of the patch. All information comes from the
Package file.
CREATOR USER,ONE
TYPE run routine
PACKAGE KIDS
ROUTINE EN1^XPDDPCK
