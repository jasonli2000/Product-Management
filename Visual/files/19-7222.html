LRAR ARCHIVE SEARCH (7222)    OPTION (19)

Name Value
NAME LRAR ARCHIVE SEARCH
MENU TEXT Search for lab data to archive
UPPERCASE MENU TEXT SEARCH FOR LAB DATA TO ARCHIVE
DESCRIPTION 
Data to be archived must be older than the beginning of the month three
months ago, on a patient not currently on a ward, and on a permanent page.
If the data is prior to the time specified above it will be purged if
unverified.
Microbiology, Anatomic Pathology, and Blood Bank data are not archived.
CREATOR USER,ONE
TYPE run routine
PACKAGE File: 9.4, IEN: 156
ROUTINE SEARCH^LRARCHIV
