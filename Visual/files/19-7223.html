LRAR ARCHIVE PURGE (7223)    OPTION (19)

Name Value
NAME LRAR ARCHIVE PURGE
MENU TEXT Purge data found in the Search option
UPPERCASE MENU TEXT PURGE DATA FOUND IN THE SEARCH
DESCRIPTION 
Data found in the Search archive option is stored in the ^LAR global.
Data is removed from the Lab Data file 63 (^LR global) based on the
data in the ^LAR global.  The ^LAR data must be written to separate media,
the Clear archive option run, the data read back from the media, then
this option may be run to complete the archive.
Note that the data is removed from both the ^LR and the ^LAR globals.
