NAME LRAR ARCHIVE 64.1
MENU TEXT Archive Wkld Data file 64.1
UPPERCASE MENU TEXT ARCHIVE WKLD DATA FILE 64.1
DESCRIPTION 
This option allows you to archive data from WKLD DATA file(#64.1).
Data in the specified date range moves to the ARCHIVED WKLD DATA
file (#64.19999).  You should NOT archive or purge this file until
the DSS extraction has been completed.  Please consult with the DSS
coordinator.
