LRAR VERIFY FILES (7252)    OPTION (19)

Name Value
NAME LRAR VERIFY FILES
MENU TEXT Verify Files for Archiving
UPPERCASE MENU TEXT VERIFY FILES FOR ARCHIVING
DESCRIPTION 
This option allows the LIM to verify the data in either the WKLD DATA
or LAB MONTHLY WORKLOADS files.  This option should
be ran BEFORE archiving any of these files.  It is VERY important to
fix the problems in the source data, or the records with bad data are not 
archived.
CREATOR USER,ONE
TYPE run routine
PACKAGE File: 9.4, IEN: 156
ROUTINE VER^LRARVER
