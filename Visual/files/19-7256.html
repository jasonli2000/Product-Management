XUCSX LOCAL XSITE RT (7256)    OPTION (19)

Name Value
NAME XUCSX LOCAL XSITE RT
MENU TEXT Local XSITE Resp. Time Report
UPPERCASE MENU TEXT LOCAL XSITE RESP. TIME REPORT
DESCRIPTION 
This is the locally run option for the cross site Response Time Report for
CS*.  It includes the number of responses (events) as well as a frequency
of distribution of responses times. Altho, this option can be run for a
single day, it is suggested that it be run for a multiple of days.
CREATOR USER,ONE
TYPE run routine
