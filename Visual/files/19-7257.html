XUCSX LOCAL XSITE SYSSTATS (7257)    OPTION (19)

Name Value
NAME XUCSX LOCAL XSITE SYSSTATS
MENU TEXT Local XSITE System Stats. Report
UPPERCASE MENU TEXT LOCAL XSITE SYSTEM STATS. REPO
DESCRIPTION 
This is a local option to run the system statistics report.  It shows
events/sec for M commands, global gets, global sets and kills, logical
reads, logical writes, disk reads, disk writes, DDP requests, RVG
requests, terminal charachers input, terminal characters out, number of
jopbs, and number of runs contributing to the average, for all nodes
running MPM.  Altho, this option can be run for one day, it is suggest
that it be run over a peroid of time.
