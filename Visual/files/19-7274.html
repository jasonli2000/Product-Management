SCDX AMBCAR RETRANS SEL REJ (7274)    OPTION (19)

Name Value
NAME SCDX AMBCAR RETRANS SEL REJ
MENU TEXT Selective Retransmission of NPCDB Rejections
UPPERCASE MENU TEXT SELECTIVE RETRANSMISSION OF NP
LOCK SD SUPERVISOR
DESCRIPTION 
This option will present the user with a list of patients who currently
have rejections from the National Patient Care Database on file.  The user
can selectively choose which patients' encounters should be marked for
retransmission.
 
Note:  Encounters that are marked for retransmission are screened out
