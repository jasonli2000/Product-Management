PRCH CREATE DEL ORDER (7284)    OPTION (19)

Name Value
NAME PRCH CREATE DEL ORDER
X ACTION PRESENT YES
MENU TEXT Create Delivery Order From Repetitive Item List
UPPERCASE MENU TEXT CREATE DELIVERY ORDER FROM REP
DESCRIPTION 
This option is used to create delivery orders from a
repetitive item list. As each order is created, the
delivery item is deleted from the repetitive item list.

TYPE run routine
