PRCH EDIT SIMPLIFIED ORDER (7306)    OPTION (19)

Name Value
NAME PRCH EDIT SIMPLIFIED ORDER
X ACTION PRESENT YES
MENU TEXT Edit Simplified Purchase Card Order
UPPERCASE MENU TEXT EDIT SIMPLIFIED PURCHASE CARD
DESCRIPTION 
This option is intended to log enough information about the purchase card
order that the user can easily reconcile the order with the charge from
the Credit Card Vendor.
CREATOR USER,ONE
TYPE run routine
E ACTION PRESENT YES
EXIT ACTION K PRCHPC
ENTRY ACTION S PRCHPC=1 D MSG^PRCHNPO
ROUTINE EN6^PRCHE
