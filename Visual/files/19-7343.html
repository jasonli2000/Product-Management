DENTPURGE (7343)    OPTION (19)

Name Value
NAME DENTPURGE
MENU TEXT Purge Dental AMIS
UPPERCASE MENU TEXT PURGE DENTAL AMIS
OUT OF ORDER MESSAGE DAS turned off
DESCRIPTION 
This option allows the purging of dental data on Dental Treatment
file #221, Dental Fee Basis file #222, Dental Class I-VI file #223,
Dental Personnel file #224, and Dental Non Clinical Time file #226.
Five years of dental data will remain after purge.  The data is not
archived but is permanently purged from the system.
