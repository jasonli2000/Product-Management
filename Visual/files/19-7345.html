LR WKLD LOCATION (7345)    OPTION (19)

Name Value
NAME LR WKLD LOCATION
MENU TEXT Create Laboratory OOS Locations
UPPERCASE MENU TEXT CREATE LABORATORY OOS LOCATION
DESCRIPTION 
Occasion Of Service (OOS)
 
 This option is used to create new/addtional LAB OOS LOCATIONS in the
HOSPTIAL LOCATION (#44) file. This maybe required if other services are
using the Laboratory Package for patient result data entry.
 
You are able to create locations for different divisions or locations with
different Stop Codes. For example you may want to create a location for
Surgery Blood Gases with the stop code for GENERAL SURGERY (401).
 
Example:
site number=XXX
Stop code = 401
 
New location name = 'LAB DIV XXX OOS ID 401'
 
NOTE: ONCE LOCATION HAVE BEEN CREATED, THEY SHOULD NOT BE DELETED.
Extra locations do not create any system problems.
