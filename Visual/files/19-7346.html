LR WKLD ACC AREA LOCATION (7346)    OPTION (19)

Name Value
NAME LR WKLD ACC AREA LOCATION
MENU TEXT Edit ACC Area OOS Locations
UPPERCASE MENU TEXT EDIT ACC AREA OOS LOCATIONS
DESCRIPTION 
Occasion Of Service (OOS)
 
 This option is used to edit Accession Area's Laboratory OOS Location.
If the Accession area is not defined, the system will prompt with the
DEFAULT LAB OOS LOCATION.
 If the Accession Area already has a LAB OOS LOCATION, the system will
display the details about the location before you make a selection.
 
Use this option when you want to assign a different Stop Code to a given
Accession Area for NPCDB workload purposes. The DEFAULT LAB OOS LOCATION
is used if the Accession Area has no LAB OOS LOCATION defined.
 
The location you want to select must first exist it the HOSPITAL LOCATION
file. You may create new LAB OOS LOCATIONS using the [LR WKLD LOCATION]
option. You may use ?? at the Location prompt to view possible selection.
