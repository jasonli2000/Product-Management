NAME PRCHQ19
MENU TEXT Edit Mode Preference
UPPERCASE MENU TEXT EDIT MODE PREFERENCE
DESCRIPTION 
This option enables the user to specify whether editing should be done via
FileMan Input Templates or ScreenMan Forms.  The user can alternately select
to be prompted for edit mode within the editing options.
