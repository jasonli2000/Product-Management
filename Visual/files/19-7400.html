NAME SPN PRINT OUT
MENU TEXT CHART/FAM/DIENER/DUSOI Scores
UPPERCASE MENU TEXT CHART/FAM/DIENER/DUSOI SCORES
DESCRIPTION 
This report provides CHART/FAM/DIENER/DUSOI scores for a patient or group
of patients.  The acronyms are described as follows:
 
CHART   - Craig Handicap Assessment and Reporting Technique
FAM     - Functional Assessment Measure
DIENER  - Diener's Satisfaction with Life Scale
DUSOI   - Duke University Severity of Illness Index
CREATOR USER,ONE
TYPE action
DISPLAY OPTION? YES
PACKAGE SPINAL CORD DYSFUNCTION
E ACTION PRESENT YES
ENTRY ACTION D EN1^SPNPRT08("SPN PRINT OUT",4)
ROUTINE EN1^SPNPRT08("SPN PRINT OUT",3)
