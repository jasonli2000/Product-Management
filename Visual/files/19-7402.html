SPN PRINT DISCHARGE (7402)    OPTION (19)

Name Value
NAME SPN PRINT DISCHARGE
MENU TEXT SCI/SCD Discharges
UPPERCASE MENU TEXT SCI/SCD DISCHARGES
DESCRIPTION 
This report provides a list of SCD patients who have been discharged
within a user-specified date range.  The list consists of discharged
patients who are either in the SCD Registry or who have been marked as SCI
in the Patient file (i.e., field 57.4, "SPINAL CORD INJURY", has been
populated).
