RCCPC INQUIRE STATEMENT (7416)    OPTION (19)

Name Value
NAME RCCPC INQUIRE STATEMENT
MENU TEXT Inquire CCPC Patient Statement
UPPERCASE MENU TEXT INQUIRE CCPC PATIENT STATEMENT
DESCRIPTION 
This option will allow you to inquire into file 349.2.  You can inquire on
one, some or all patients.  It will list the patients' name and account
number, plus all the transaction for the patient.
