SROQIDP (7446)    OPTION (19)

Name Value
NAME SROQIDP
X ACTION PRESENT YES
MENU TEXT List of Invasive Diagnostic Procedures
UPPERCASE MENU TEXT LIST OF INVASIVE DIAGNOSTIC PR
OUT OF ORDER MESSAGE Not used after patch SR*3*175
DESCRIPTION 
This option provides a report listing the completed surgical cases that
were performed during the selected date range and that have a principal
CPT code on the list defined by Surgical Service at VHA Headquarters as
invasive diagnostic procedures.
