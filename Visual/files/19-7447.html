SROQ MISSING DATA (7447)    OPTION (19)

Name Value
NAME SROQ MISSING DATA
X ACTION PRESENT YES
MENU TEXT Key Missing Surgical Package Data
UPPERCASE MENU TEXT KEY MISSING SURGICAL PACKAGE D
DESCRIPTION 
This option generates a list of surgical cases performed within the
selected date range that are missing key information. This report includes
surgical cases with an entry in the TIME PAT IN OR field and does not
include aborted cases.
CREATOR USER,SEVENTEEN
TYPE run routine
PACKAGE SURGERY
E ACTION PRESENT YES
EXIT ACTION D EXIT^SROVAR
ENTRY ACTION D ^SROVAR
XQUIT EXECUTABLE Q
ROUTINE SROQN
