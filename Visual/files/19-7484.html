PRPF CHANGE ACCOUNT STATUS (7484)    OPTION (19)

Name Value
NAME PRPF CHANGE ACCOUNT STATUS
MENU TEXT Change Account Status (ACTIVE/INACTIVE)
UPPERCASE MENU TEXT CHANGE ACCOUNT STATUS (ACTIVE/
OUT OF ORDER MESSAGE DISABLED DUE TO PATIENT FUNDS MIGRATION
DESCRIPTION 
This option will allow the user to change the account status of an account
from ACTIVE to INACTIVE or INACTIVE to ACTIVE.  It will not allow the user
to inactivate an account when the balance of funds in the account is not
EQUAL to zero.
CREATOR USER,ONE
TYPE run routine
PACKAGE INTEGRATED PATIENT FUNDS
ROUTINE INACT^PRPFED2
