PRPF POST BAL CAR FWD (7492)    OPTION (19)

Name Value
NAME PRPF POST BAL CAR FWD
MENU TEXT Post Balance Carried Forward from Manual System
UPPERCASE MENU TEXT POST BALANCE CARRIED FORWARD F
OUT OF ORDER MESSAGE DISABLED DUE TO PATIENT FUNDS MIGRATION
DESCRIPTION 
This option allows the user to enter the Balance Carried Forward data
from a manual system.  It allows entry of both Private Source and
Gratuitous funds on the same transaction and verifies that the transaction
is in balance before posting.
CREATOR USER,ONE
TYPE run routine
PACKAGE INTEGRATED PATIENT FUNDS
