NAME PRPF VERIFY & CORRECT
MENU TEXT Verify and Correct Balance
UPPERCASE MENU TEXT VERIFY AND CORRECT BALANCE
OUT OF ORDER MESSAGE DISABLED DUE TO PATIENT FUNDS MIGRATION
DESCRIPTION 
This option allows the supervisor to review and whenappropriate, correct
the stored balances within the system.  This option should not be used 
until a complete audit of the account is accomplished and it is
certain that no tampering of the account has been found.
