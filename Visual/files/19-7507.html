PRPF OVERDUE RESTRICTION (7507)    OPTION (19)

Name Value
NAME PRPF OVERDUE RESTRICTION
MENU TEXT Overdue Restriction Search
UPPERCASE MENU TEXT OVERDUE RESTRICTION SEARCH
DESCRIPTION 
This option generates a list of all accounts which are restricted and
which indicate that the date of restriction is older that 180 days.
