ENWOMULTI (7567)    OPTION (19)

Name Value
NAME ENWOMULTI
MENU TEXT Multiple Work Order Entry
UPPERCASE MENU TEXT MULTIPLE WORK ORDER ENTRY
DESCRIPTION 
Create (and optionally close) multiple work orders for equipment. First a
new work order is entered in full and then additional equipment is
selected. Equipment can be selected by equipment category, manufacturer,
and/or model. Alternatively, equipment can be individually specified. The
computer will create new work orders for each selected equipment item by
copying the original work order. Equipment specific data on the work
orders (e.g. location, serial number, etc.) will be obtained from the
equipment file.
