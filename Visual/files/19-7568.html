ENIN-EDIT-MULTI (7568)    OPTION (19)

Name Value
NAME ENIN-EDIT-MULTI
MENU TEXT Purchase Order Group Edit
UPPERCASE MENU TEXT PURCHASE ORDER GROUP EDIT
LOCK ENEQMED
DESCRIPTION 
Edit a group of existing items in the inventory file without having to
enter common data for each item. Equipment is initially selected by
purchase order #. Therefore, the purchase order # field must already be
populated before this option can be used to edit equipment. Equipment with
the same purchase order # is selected for editing by line item.

TYPE run routine
