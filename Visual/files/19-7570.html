ENEQ-SYSTEM (7570)    OPTION (19)

Name Value
NAME ENEQ-SYSTEM
MENU TEXT Parent System/Component Hierarchy Report
UPPERCASE MENU TEXT PARENT SYSTEM/COMPONENT HIERAR
DESCRIPTION 
Prints a hierarchical list of a parent system and all of its components.
Components are indented and listed beneath their parent system. Items that
are components of a component in the specified system are included.
CREATOR USER,ONE
TYPE run routine
PACKAGE ENGINEERING
