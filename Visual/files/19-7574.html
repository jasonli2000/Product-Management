NAME ENEQLT_PREX
MENU TEXT Equip with Lockout/Tagout CLEAR but Category SET
UPPERCASE MENU TEXT EQUIP WITH LOCKOUT/TAGOUT CLEA
DESCRIPTION 
Prints list of Equipment Records (file #6914) that have the LOCKOUT
REQUIRED? clear (not set to 'YES') but belonging to an Equipment Category
(file #6911) for which the LOCKOUT REQUIRED? field is 'YES'.
 
Such records, if any exist, would represent exceptional situations.

TYPE run routine
