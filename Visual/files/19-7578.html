NAME ENPM9
MENU TEXT Reassign a Technician's PM Responsibilities
UPPERCASE MENU TEXT REASSIGN A TECHNICIAN'S PM RES
LOCK ENMGR
DESCRIPTION 
This option is used to to change the preventive maintenance (PM) schedules
by replacing one technician by a different techinican. This change can be
limited to PMs by one responsible shop or done for all responsible shops.
Both the Equipment Category and Equipment Inv. files will be updated.
