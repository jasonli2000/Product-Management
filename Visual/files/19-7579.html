PRCH CARD - INFO (7579)    OPTION (19)

Name Value
NAME PRCH CARD - INFO
MENU TEXT Purchase Card Information List
UPPERCASE MENU TEXT PURCHASE CARD INFORMATION LIST
DESCRIPTION 
This option will display card holder, purchase card name, approving
official, alternate approving official, and surrogate(s) for 
entries in the Purchase Card Information file (#440.5).
CREATOR USER,ONE
TYPE print
PACKAGE IFCAP
DIC {DIP} PRC(440.5,
L. 0
FLDS [PRCH CARD INFO]
BY [PRCH CARD INFOSORT]
DIS(0) I $P($G(^PRC(440.5,D0,2)),U,2)'="Y"
