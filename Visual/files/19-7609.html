NAME PRCAC RC RRR
MENU TEXT Reconciliation Rollup Request for RC
UPPERCASE MENU TEXT RECONCILIATION ROLLUP REQUEST
DESCRIPTION 
Use this option to send an E-mail request for a roll-up of Reimbursable
Health Third Party bills listed on the supporting Regional Counsel
system as referred from this site.  This request may take over a day to
fill as it is relying on the network mail delivery time and system
response time of this medical center and the supporting Regional Counsel.
The members of the RC RC REFERRALS Mail Group will receive four 
Reconciliation mail messages.
