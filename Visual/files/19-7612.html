LA7 ADL START/STOP (7612)    OPTION (19)

Name Value
NAME LA7 ADL START/STOP
MENU TEXT Start/Stop Auto Download Background Job
UPPERCASE MENU TEXT START/STOP AUTO DOWNLOAD BACKG
DESCRIPTION 
Option to allow user to start/restart/stop the lab messaging pacakge's
auto download background job. This option can be used to restart the auto
download job after changes have been made to the auto instrument file and
the auto download job needs to be updated to reflect the changes.
 
It can also be used to temporarily suspend auto downloading while making
changes. It will then keep a list of accessions to process and use this
list when auto downloading is resumed.
