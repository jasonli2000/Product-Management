LA7 PRINT LAB UI MESSAGE (7614)    OPTION (19)

Name Value
NAME LA7 PRINT LAB UI MESSAGE
MENU TEXT Display Lab Universal Interface Message
UPPERCASE MENU TEXT DISPLAY LAB UNIVERSAL INTERFAC
DESCRIPTION 
Option allows user to select a range of messages, up to 20, for printing
or displaying on a terminal. When displaying on terminal that supports
screen handling, the Kernel Browser is used to display the message(s).
 
Printing to a hardcopy device or mail message (P-MESSAGE) is also supported.
