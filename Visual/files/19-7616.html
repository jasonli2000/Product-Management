NAME SD53 CLEANUP/ERR COUNT
MENU TEXT Scheduling/PCE Bad Pointer Count
UPPERCASE MENU TEXT SCHEDULING/PCE BAD POINTER COU
DESCRIPTION 
This option produces a MailMan message showing the numbers of patient
appointments and dispositions with missing or incorrect outpatient
encounter pointers.  The message is sent to the user and to the ACRP team
at the Albany CIOFO.
