PRPF PURGE (7624)    OPTION (19)

Name Value
NAME PRPF PURGE
MENU TEXT Summarize & Purge Patient Funds Transactions
UPPERCASE MENU TEXT SUMMARIZE & PURGE PATIENT FUND
OUT OF ORDER MESSAGE DISABLED DUE TO PATIENT FUNDS MIGRATION
LOCK PRPF SUPERVISOR
DESCRIPTION 
This option allows a Patient Funds Supervisor to summarize Patient Funds
data for all accounts.  This summarization accumulates all transactions
through a specified date and enters a BALANCE CARRIED FORWARD transaction 
on the Patient Funds Card and removes all transactions through that date.
 
The actual purging is accomplished by the IFCAP Purgemaster during non
peak periods.
 
This option will not function if the Patient Funds Cards have not been
archived for the period requested.
CREATOR USER,ONE
TYPE run routine
PACKAGE INTEGRATED PATIENT FUNDS
ROUTINE PRPFPURG
