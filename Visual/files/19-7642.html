DGPRE PURGE CALL LOG (7642)    OPTION (19)

Name Value
NAME DGPRE PURGE CALL LOG
MENU TEXT Purge Call Log
UPPERCASE MENU TEXT PURGE CALL LOG
DESCRIPTION 
This option will purge all entries prior to a user specified date
from the PRE-REGISTRATION CALL LOG File, #41.43.  This gives the 
site some additional control over the log file in addition to the 
automatic purging of the file.
CREATOR USER,ONE
TYPE run routine
PACKAGE REGISTRATION
ROUTINE PURGE43^DGPREP4
