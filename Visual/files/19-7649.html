NAME GMRV SITE FILE MENU
MENU TEXT Vitals/Measurements Site Files Menu
UPPERCASE MENU TEXT VITALS/MEASUREMENTS SITE FILES
OUT OF ORDER MESSAGE GMRV*5*19 - shutdown Vitals CHUI
DESCRIPTION 
The ADP Coordinator can use the options contained in this menu to
enter:
1. Vitals sign site parameters in the GMRV Vital Type file (#120.51).
2. Vitals/measurements qualifiers in the GMRV Vital Qualifier file
   (#120.52).
3. Temperature and pulse default qualifers in the GMRV Category file
   (#120.53).
 
This menu also prints:
1. A table displaying all vital types and their associated categories and
   qualifiers.
