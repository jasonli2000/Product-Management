GMRV CAT/QUAL TABLE (7650)    OPTION (19)

Name Value
NAME GMRV CAT/QUAL TABLE
MENU TEXT Display Vitals Category/Qualifier/Synonym Table
UPPERCASE MENU TEXT DISPLAY VITALS CATEGORY/QUALIF
OUT OF ORDER MESSAGE GMRV*5*19 - shutdown Vitals CHUI
DESCRIPTION 
This option displays a list of categories and qualifiers associated with
individual vital types, e.g., blood pressure, temperature, pulse,
respirations, weight, circumference/girth and pulse oximetry.  Data comes
from the GMRV Vital Qualifier (#120.52) file and the GMRV Vital Category
(#120.53) file.
