ENINVDEL (7658)    OPTION (19)

Name Value
NAME ENINVDEL
MENU TEXT Delete Equipment Record
UPPERCASE MENU TEXT DELETE EQUIPMENT RECORD
DESCRIPTION 
This option can be used to delete equipment records from the Equipment
Inventory file. Any open work orders for the equipment item will
automatically be closed prior to deletion. Note that deletion of an
equipment item will also result in deletion of its equipment history.
Equipment that has been reported to Fixed Assets can not be deleted.
CREATOR USER,ONE
TYPE run routine
PACKAGE ENGINEERING
ROUTINE DEL^ENEQ4
