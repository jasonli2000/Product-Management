NAME LRRMM PRINT MAIL MESSAGE
MENU TEXT Extract Report from Mail Message
UPPERCASE MENU TEXT EXTRACT REPORT FROM MAIL MESSA
DESCRIPTION 
Local option which takes a mail message and prints only the text of the
message without headers. No formatting is added. Similiar to printing a
file from the spooler.
