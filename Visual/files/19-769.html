LA AUTO LLIST (769)    OPTION (19)

Name Value
NAME LA AUTO LLIST
MENU TEXT Set instrument to run by load list
UPPERCASE MENU TEXT SET INSTRUMENT TO RUN BY LOAD
LOCK LRSUPER
DESCRIPTION 
This function allows the user to set the selected instrument to be run
on the basis of load list numbers.  That is, the data transmitted from
the automated instrument contains the load list numbers which allow the
host computer to find who or what the data belongs to.
