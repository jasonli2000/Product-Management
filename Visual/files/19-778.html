DESCRIPTION 
 This routine allow for the review, editing, and deletion of data produced
by Microbiology Automated Instrumentation.
  After the above steps the data can be verified. The steps involved are
patterned after the standard chemistry verification procedures. Similar to
the EA option for chemistry/hematology type instrumentation. Refer
the Microbiology Auto verification guide for more information.
