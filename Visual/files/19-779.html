LA MI SPECIAL CHARCTER LOAD (779)    OPTION (19)

Name Value
NAME LA MI SPECIAL CHARCTER LOAD
MENU TEXT Load Vitek Special Characters
UPPERCASE MENU TEXT LOAD VITEK SPECIAL CHARACTERS
DESCRIPTION 
  There are some code used by Vitek instrumentation which cannot
be loaded into the file by Fileman. The character have a special meaning
to Fileman which will prevent them from being loaded into the file.
This option provides means for getting this type of data into the file.
CREATOR POSTMASTER
TYPE run routine
PACKAGE AUTOMATED LAB INSTRUMENTS
