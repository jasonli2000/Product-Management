UPPERCASE MENU TEXT PROCEDURE REASON REPORT
OUT OF ORDER MESSAGE OPTION NO LONGER SUPPORTED. USE THE ECS GUI INTERFACE.
DESCRIPTION 
This report provides a listing of Procedure Reasons associated with
Event Capture data in the Event Capture Patient file (#721).  The user
may select a date range for the file #721 records to be included in
the report.  The user may also select the Location(s) and DSS Unit(s)
on which to report.
 
The file #721 records are sorted and listed by Procedure Reason and
Procedure within each Location/DSS Unit.
