LRQUICK (784)    OPTION (19)

Name Value
NAME LRQUICK
MENU TEXT Multipurpose accessioning
UPPERCASE MENU TEXT MULTIPURPOSE ACCESSIONING
DESCRIPTION 
This function provides a means of accessioning manually requested tests
into the computer.  Different areas of the laboratory may have different
lists of their most often ordered tests to speed entry.  If one selects
a single test on set-up, the function serves as a means of batch entry
of the single test.
CREATOR POSTMASTER
TYPE run routine
PACKAGE LAB SERVICE
E ACTION PRESENT YES
ENTRY ACTION D ^LRPARAM
ROUTINE LRORD
