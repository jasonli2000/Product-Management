SCDX AMBCAR XMIT HIST FOR PAT (7840)    OPTION (19)

Name Value
NAME SCDX AMBCAR XMIT HIST FOR PAT
MENU TEXT Transmission History for Patient
UPPERCASE MENU TEXT TRANSMISSION HISTORY FOR PATIE
DESCRIPTION 
This option will produce the ACRP transmission history of all encounters
that a patient had during a given time frame.
 
Note: All transmissions of the encounter are included, even if the
transmission occured after the date range selected.  This is because the
date range limits the encounters printed, not when the encounter was
transmitted.
CREATOR USER,ONE
TYPE run routine
PACKAGE SCHEDULING
