SCDX AMBCAR PURGE ACRP FILES (7843)    OPTION (19)

Name Value
NAME SCDX AMBCAR PURGE ACRP FILES
MENU TEXT Purge Ambulatory Care Reporting files
UPPERCASE MENU TEXT PURGE AMBULATORY CARE REPORTIN
DESCRIPTION 
This option will purge the Transmitted Outpatient Encounter file (#409.73)
of all encounters that occurred during a user supplied date range.  This
will result in the deletion of corresponding entries in the Deleted
Outpatient Encounter file (#409.74), the Transmitted Outpatient Encounter
Error file (#409.75), and the ACRP Transmission History file (#409.77).
CREATOR USER,ONE
TYPE run routine
PACKAGE SCHEDULING
ROUTINE MAIN^SCDXPRG0
