PRCH PA STATS DAILY (7875)    OPTION (19)

Name Value
NAME PRCH PA STATS DAILY
MENU TEXT Purchasing Agent Statistics by Date
UPPERCASE MENU TEXT PURCHASING AGENT STATISTICS BY
DESCRIPTION 
This option prints detailed total line counts and total dollar values for
purchasing agents for a given time period.  A single PA or a range of
PAs can be specified.
CREATOR USER,ONE
TYPE print
E ACTION PRESENT YES
ENTRY ACTION S DIPCRIT=1
DIC {DIP} PRC(442,
L. 0
FLDS [PRCH TOT AMT DAILY]
BY [PRCHOUT PA STATS DAILY]
DIS(0) N NDUZ S NDUZ=$P($G(^PRC(442,D0,1)),"^",10) I $P($G(^VA(200,+NDUZ,400)),"^")'=""
DIPCRIT PRINT SEARCH/SORT CRITERIA
