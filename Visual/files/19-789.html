NAME LR SUP SUMMARY
MENU TEXT Summary list (supervisors')
UPPERCASE MENU TEXT SUMMARY LIST (SUPERVISORS')
LOCK LRSUPER
DESCRIPTION 
All results for a given accession area (or areas) for a given date
and range of accession numbers.  This report can serve as an 'audit trail'
for a specimen.  Includes information on person placing order, 
person performing test, verifying person, and dates and times 
of specimen collection and test completion.  The report can
be sorted by accession number or by patient, and can be printed in 
a long form, which includes the above mentioned information
plus the test results.
