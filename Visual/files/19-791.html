NAME LRGEN
MENU TEXT General report for selected tests
UPPERCASE MENU TEXT GENERAL REPORT FOR SELECTED TE
DESCRIPTION 
This function displays data in a highly generalized fashion.
All tests requested are displayed (up to 18 fields) in a cumulative format.
If more than 18 tests are to be displayed, an alternate format is used.
If a test has a special format (predefined in the 'Laboratory Test' file)
then that test must be requested separately.
If a specimen is specified, normal ranges are displayed if available.
Note that if a specimen is chosen for which there is no data,
no data will be displayed.
