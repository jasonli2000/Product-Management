PRCA RCDMC LESSER WITHHOLDING (7915)    OPTION (19)

Name Value
NAME PRCA RCDMC LESSER WITHHOLDING
MENU TEXT Enter Lesser DMC Withholding Amount
UPPERCASE MENU TEXT ENTER LESSER DMC WITHHOLDING A
DESCRIPTION 
Normally, when a patient is referred to DMC, the total amount owed is
withheld.  This option will allow the local sites to enter a lesser
amount.  That amount then will be deducted from the debtor monthly until
the full amount owed is collected.

TYPE run routine
