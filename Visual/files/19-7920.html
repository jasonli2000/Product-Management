Name Value
NAME ENPLMRX
MENU TEXT Transmit Project Status Reports
UPPERCASE MENU TEXT TRANSMIT PROJECT STATUS REPORT
DESCRIPTION 
This option transmits construction progress report (10-0051) data to VACO
for selected projects. All data pertient to a 10-0051 is sent. Selected
projects are automatically vaildated prior to the actual transmission. The
transmitter is a receipent of any created mail messages that are sent to
VACO.
