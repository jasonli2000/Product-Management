Name Value
NAME ENPRE LOCAL
MENU TEXT Local Data Edit
UPPERCASE MENU TEXT LOCAL DATA EDIT
DESCRIPTION 
Edit construction project fields that are of local interest, but are not
currently part of the electronic Construction Project Progress Report. If
local input template ENZPRLOCAL exists, it will used to edit the project
instead of the national ENPRLOCAL template.
