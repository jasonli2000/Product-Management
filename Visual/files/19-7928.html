NAME PRCH FINAL - BUYER
X ACTION PRESENT YES
MENU TEXT Final Charge YES - Reconciled Orders - Buyer
UPPERCASE MENU TEXT FINAL CHARGE YES - RECONCILED
DESCRIPTION 
This report lists all reconciled orders that have been 
marked as Final Change YES for the selected date range
and Credit Card holder/Card Name. (File 440.6, field 44)
CREATOR USER,ONE
TYPE print
PACKAGE IFCAP
E ACTION PRESENT YES
EXIT ACTION K CCHECK
ENTRY ACTION D ^PRCHFCY I $G(CCHECK)="####" S XQUIT="" K CCHECK Q
DIC {DIP} PRCH(440.6,
L. 0
FLDS [PRCH FINAL CHARGE]
BY [PRCH FINAL CHARGE1]
DIS(0) I $P($G(^PRCH(440.6,D0,0)),"^",4)=CCHECK
