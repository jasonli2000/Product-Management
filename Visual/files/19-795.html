LRFAST (795)    OPTION (19)

Name Value
NAME LRFAST
MENU TEXT Bypass normal data entry
UPPERCASE MENU TEXT BYPASS NORMAL DATA ENTRY
DESCRIPTION 
This option bypasses normal ordering, accessioning, and data entry.
Where the results are in hand and no information regarding the order
is in the computer, this option may be used to enter the patient's
name, the test, and the results.  Note that information such as the
person ordering the test, or alternative routing information is not
asked (the report routing is taken from MAS information in the computer).
CREATOR POSTMASTER
TYPE run routine
PACKAGE LAB SERVICE
E ACTION PRESENT YES
