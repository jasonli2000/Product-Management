ECXTRYIT (7979)    OPTION (19)

Name Value
NAME ECXTRYIT
MENU TEXT Trial for Setup Extract
UPPERCASE MENU TEXT TRIAL FOR SETUP EXTRACT
LOCK ECXMGR
DESCRIPTION 
This option uses the logic of the admission setup extract to generate
a printed report of the inpatient population on a selected day. This
report should be compared to reports from MAS to eliminate any
problems in the ADMISSION SETUP EXTRACT file (#727.82) used
to generate the admission setup extract.
CREATOR USER,SEVENTEEN
TYPE run routine
PACKAGE DSS EXTRACTS
E ACTION PRESENT YES
ENTRY ACTION W !!!,"WARNING.",!,"This is very resource intensive and should be queued to run at slack time.",!!!
