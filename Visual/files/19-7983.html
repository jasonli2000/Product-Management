ECX IV DIV EDIT (7983)    OPTION (19)

Name Value
NAME ECX IV DIV EDIT
MENU TEXT Enter/Edit IV Room Division
UPPERCASE MENU TEXT ENTER/EDIT IV ROOM DIVISION
LOCK ECXMGR
DESCRIPTION 
This option allows editing of the DIVISION field (.02) of the IV
Room file (#59.5).  The purpose of the DIVISION field is to provide
a way to tie outpatient IV data to a Medical Center Division for DSS
purposes.
CREATOR USER,SEVENTEEN
TYPE run routine
PACKAGE DSS EXTRACTS
ROUTINE ED^ECXDIVIV
