NAME LROW
MENU TEXT Lab test order
UPPERCASE MENU TEXT LAB TEST ORDER
DESCRIPTION 
This is to be used by the wards for order entry, NOT by the lab!
The purpose of this function is to allow selection among sending
the patient or specimen accompanied with the order number to the lab
where it can be accessioned, or holding the test request in the computer
until a collection list can be made and routine phlebotomy collection
by the laboratory on the wards occurs.
