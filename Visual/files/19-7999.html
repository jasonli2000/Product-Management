VAFH BATCH UPDATE (7999)    OPTION (19)

Name Value
NAME VAFH BATCH UPDATE
MENU TEXT UPDATE BATCH JOB
UPPERCASE MENU TEXT UPDATE BATCH JOB
DESCRIPTION 
This option is not intended to be run from programmer mode.  No output to
the screen is intended.  When this job is scheduled through TaskMan it
will collect and transmitt all the entries in the PIVOT file that are of
an update to patient information.  The job will build PIMS A08 HL7 messages
and broadcast them to any subscriber.

The job should be re-scheduled to run about every 10 minutes.  The task
interval will determine approximately the maximum time it takes for an
update to patient information to be broadcast in a HL7 message.
CREATOR USER,ONE
TYPE run routine
ROUTINE EN^VAFHBGJ
SCHEDULING RECOMMENDED YES
