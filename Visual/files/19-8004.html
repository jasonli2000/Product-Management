SCENI IEMM ERROR CORRECTION (8004)    OPTION (19)

Name Value
NAME SCENI IEMM ERROR CORRECTION
MENU TEXT Correct Incomplete Encounters
UPPERCASE MENU TEXT CORRECT INCOMPLETE ENCOUNTERS
DESCRIPTION 
Entry point for the Incomplete Encounter Management Module List Manager
tool.  Asks for either Patient, Clinic, or error code for selection.
Once, selection is made, all encounters with errors in the TRANSMITTED
OUTPATIENT ENCOUNTER ERROR FILE (#409.75) will be displayed.  The user
will then be able to pick an encounter for further display of the specific
errors.
