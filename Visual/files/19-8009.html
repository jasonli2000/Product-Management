SCENI OP ENCOUNTER EDIT (8009)    OPTION (19)

Name Value
NAME SCENI OP ENCOUNTER EDIT
MENU TEXT Edit Outpatient Encounter
UPPERCASE MENU TEXT EDIT OUTPATIENT ENCOUNTER
LOCK SCENI ENCOUNTER EDIT
DESCRIPTION 
Allows access to change the following  uneditable fields in the
OUTPATIENT ENCOUNTER FILE, #409.68: CLINIC STOP CODE, Field #.03;
APPOINTMENT TYPE, Field #.1; MEDICAL CENTER DIVISION, Field #.11; and the
ELIGIBILITY OF ENCOUNTER, Field #.13.
 
In order to keep the multiple data fields being stored in various files in
synch, the following fields and files are updated when this option is
used: 
   1.  Any child encounters for the parent encounter being changed are
updated as well in the OUTPATIENT ENCOUNTER FILE, #409.68. 
 
   2. The ELIGIBILITY OF VISIT Field #30 in the APPOINTMENT Sub-File
#44.001 of the HOSPITAL LOCATION File, #44 is updated to match the
encounter.
 
   3. The APPOINTMENT TYPE Field #9.5 in the APPOINTMENT Sub-File #2.98
of the PATIENT File, #2 is updated to match the encounter.
