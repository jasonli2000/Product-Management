DMSQ PURGE (8014)    OPTION (19)

Name Value
NAME DMSQ PURGE
MENU TEXT Purge SQLI Data
UPPERCASE MENU TEXT PURGE SQLI DATA
LOCK XUPROGMODE
DESCRIPTION 
This option purges the SQLI projection of VA FileMan data dictionaries.
Use this option if you have previously projected your VA FileMan data
dictionaries for use by an M-to-SQL product, but no longer wish to. All
data in the following files are purged:
 
  SQLI_COLUMN
  SQLI_DATA_TYPE
  SQLI_DOMAIN
  SQLI_ERROR_LOG
  SQLI_ERROR_TEXT
  SQLI_FOREIGN_KEY
  SQLI_KEY_FORMAT
  SQLI_OUTPUT_FORMAT
  SQLI_PRIMARY_KEY
  SQLI_SCHEMA
  SQLI_TABLE
  SQLI_TABLE_ELEMENT
 
Data is not purged from the following SQLI file, however:
 
  SQLI_KEY_WORD
 
This option requires programmer mode as well as the progmode key.
CREATOR USER,ONE
TYPE run routine
PACKAGE VA FILEMAN
