DMSQ PS TOTAL TABLE COLS (8021)    OPTION (19)

Name Value
NAME DMSQ PS TOTAL TABLE COLS
MENU TEXT Column Totals, by Table
UPPERCASE MENU TEXT COLUMN TOTALS, BY TABLE
DESCRIPTION 
This is a long listing of all tables along with a count of the number
of columns in each one.  It is sorted by the file/subfile number
associated with the table.
