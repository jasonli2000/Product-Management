NAME LRQCC
MENU TEXT Bull algorithm quality control
UPPERCASE MENU TEXT BULL ALGORITHM QUALITY CONTROL
DESCRIPTION 
Displays cumulative data from the Bull algorithm.  This is a
Hematology Quality Control option which allows the Hematology
Supervisor to monitor the MCH, MCV, and MCHC hematology indices.
These indices are grouped in "sets" of 20 values, from which the 
mean and delta values are calculated.  It has been reported that
these three indices do not vary except in extreme disease states.
Hence, the indices are an indication of the stability and precision
of the automated hematology instrument.  Before using this option
the original article by Dr. Brian Bull should be studied.
