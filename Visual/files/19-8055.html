SCRPW TREND FACILITY UNIQUES (8055)    OPTION (19)

Name Value
NAME SCRPW TREND FACILITY UNIQUES
MENU TEXT Trend of Facility Uniques by 12 Month Date Ranges
UPPERCASE MENU TEXT TREND OF FACILITY UNIQUES BY 1
DESCRIPTION 
Option for trending facility outpatient uniques by overlapping 12 month
date ranges for the past 5 years.  Data prior to 10/01/96 is derived
from the OPC file, data for dates following 10/01/96 is based on 
'checked out' OUTPATIENT ENCOUNTER file reocrds.
CREATOR USER,ONE
TYPE run routine
PACKAGE SCHEDULING
ROUTINE SCRPW6
