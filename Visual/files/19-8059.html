IB OUTPUT DAYS ON HOLD (8059)    OPTION (19)

Name Value
NAME IB OUTPUT DAYS ON HOLD
MENU TEXT Days on Hold Report
UPPERCASE MENU TEXT DAYS ON HOLD REPORT
DESCRIPTION 
This option produces the Days On Hold Report.  The report lists all
Integrated Billing charges that have been in the ON HOLD status for an
extended period of time.  Use the default to list charges that have been
on hold for longer than 60 days.
CREATOR USER,ONE
TYPE run routine
PACKAGE INTEGRATED BILLING
