IB OUTPUT HELD CHARGES/PT (8063)    OPTION (19)

Name Value
NAME IB OUTPUT HELD CHARGES/PT
MENU TEXT List Current/Past Held Charges by Pt
UPPERCASE MENU TEXT LIST CURRENT/PAST HELD CHARGES
DESCRIPTION 
This option lists all IB Actions for a patient that are currently on
hold or were on hold for a user specified date range.  The report lists IB
Action ID, Rate Type, Bill #, AR status, IB Status and information related
to corresponding Third Party Claims.
 
Note: Only those charges placed on hold since the installation of patch
      IB*2*70 will appear on this report.
