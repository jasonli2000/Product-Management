LRLEDI (8086)    OPTION (19)

Name Value
NAME LRLEDI
MENU TEXT Referral Patient Multi-purpose Accession
UPPERCASE MENU TEXT REFERRAL PATIENT MULTI-PURPOSE
DESCRIPTION 
This option is used to accession referral patients from another medical
center. The patient must already exist in the Patient file (#2) in order
to be selectable. The option will accession the patient into the REFERRAL
PATIENT file (#67) if it doesn't already exist.
 
The user will be prompted or the Unique Identifier. The user will then be
able to order tests for that patient.
CREATOR USER,ONE
TYPE run routine
