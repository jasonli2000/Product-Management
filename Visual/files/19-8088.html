NAME LR7O MAN RESULT NLT
MENU TEXT Link Result NLT Manual
UPPERCASE MENU TEXT LINK RESULT NLT MANUAL
DESCRIPTION 
 This option allows the user to manually select an Atomic test and assign
a RESULT NLT CODE to it.
 
This option can also be used to change or delete RESULT NLT CODES.
 
RESULT NLT CODES are required by the LEDI software to identify test names.
The receiving site will use the RESULT NLT CODE to link to local test
names.
