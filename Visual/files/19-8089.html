LR TAT URGENCY (8089)    OPTION (19)

Name Value
NAME LR TAT URGENCY
MENU TEXT Turnaround times By Urgency
UPPERCASE MENU TEXT TURNAROUND TIMES BY URGENCY
DESCRIPTION 
This opton generates a report of the turnaround times for selected lab
test for selected urgencies. WKLD urgencies will be included for each
normal urgency selected. The report format can be specified, i.e., summary
report of a detailed report showing the data used to compute the
turnaround times.
 
Hours for tours of duty can not be changed by the user. They are:
Regular Tour of duty :  7AM to 5Pm
Irregular Tour of duty: 5:01PM to 6:59AM, Saturaday, Sunday, and Holidays
 
The reports extracts data from the ACCESSION file (#68). Therefore if data
as been purged, no report is possible.
