RA DISPLAY IMAGPROCINFO (8107)    OPTION (19)

Name Value
NAME RA DISPLAY IMAGPROCINFO
MENU TEXT Rad/Nuc Med Procedure Information Look-Up
UPPERCASE MENU TEXT RAD/NUC MED PROCEDURE INFORMAT
DESCRIPTION 
This option will allow the user to select either inactive or active 
procedures by imaging type.  The option will display the procedure, type 
of procedure, imaging type abbreviation and the CPT code of the procedure.
If any Educational Description exists, it will also be displayed.
CREATOR USER,ONE
TYPE run routine
DISPLAY OPTION? YES
ROUTINE RAPINFO
