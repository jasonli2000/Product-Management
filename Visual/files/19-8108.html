RA UNCORRECTED REPORTS (8108)    OPTION (19)

Name Value
NAME RA UNCORRECTED REPORTS
X ACTION PRESENT YES
MENU TEXT Access Uncorrected Reports
UPPERCASE MENU TEXT ACCESS UNCORRECTED REPORTS
DESCRIPTION 
This option will allow the user to select a patient whose reports 
were amended sometime in the past.  The user may then display the  
reports as they appeared prior to amendment.
CREATOR USER,ONE
TYPE run routine
DISPLAY OPTION? YES
E ACTION PRESENT YES
EXIT ACTION K RAOPT("UNCORRECTED REPORTS")
ENTRY ACTION S RAOPT("UNCORRECTED REPORTS")=""
ROUTINE EN1^RAERRPT
