RA PROCPARENT (8120)    OPTION (19)

Name Value
NAME RA PROCPARENT
MENU TEXT Parent Procedure List
UPPERCASE MENU TEXT PARENT PROCEDURE LIST
DESCRIPTION 
The 'Parent Procedure List' option allows the user to display the data
associated with parent procedures.  This option allows the user to select
the imaging types so parent procedures of the selected imaging types are
displayed.  The user may also specify if active procedures, inactive
procedures, or both inactive and active procedures are to be displayed.
CREATOR USER,ONE
TYPE run routine
DISPLAY OPTION? YES
