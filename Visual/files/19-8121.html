RA PROCOSTEDIT (8121)    OPTION (19)

Name Value
NAME RA PROCOSTEDIT
MENU TEXT Cost of Procedure Enter/Edit
UPPERCASE MENU TEXT COST OF PROCEDURE ENTER/EDIT
DESCRIPTION 
This option is used to enter or edit the COST OF PROCEDURE field
in the file, RAD/NUC MED PROCEDURES.  The user will be asked for
an imaging type, then only procedures matching that imaging type
can have their cost edited.
