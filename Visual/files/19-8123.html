Name Value
NAME RA NM RADIOPHARM USAGE
MENU TEXT Radiopharmaceutical Usage Report
UPPERCASE MENU TEXT RADIOPHARMACEUTICAL USAGE REPO
DESCRIPTION 
This report includes all exams where radiopharmaceuticals are entered
within a selected exam date range.  It can be sorted by technologist
or by exam date/time.  Separate sections are printed for each division
and imaging type (only imaging types using radioharmaceuticals can be
included).  A summary only can be generated.
