NAME PRCAC RC ACTION MENU
MENU TEXT TP Referred Follow-up
UPPERCASE MENU TEXT TP REFERRED FOLLOW-UP
DESCRIPTION 
This option allows follow-up actions for referred TP bills to be 
electronically sent to the RC office. This option should be used to 
process pending EOB codes for partial payments, Increase referred TP
bills, request action be taken and to transmit selected AR Transactions
for referred TP bills.
