UPPERCASE MENU TEXT CATEGORY FILE (IV)
DESCRIPTION 
This option allows the user to group drugs from the IV ADDITIVES (52.6)
and/or IV SOLUTIONS (52.7) file so that management cost reports may be run
for this group. This is like a report generator as it allows the user to
customize reports to the ir needs. Examples of groupings may br
antibiotics, restricted drugs by provider, chemotherapy drugs, or any
thing else the user wants to group such as a number of drugs for drug
utilization studies.
