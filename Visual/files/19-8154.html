PRCA FMS RSC ENTER/EDIT (8154)    OPTION (19)

Name Value
NAME PRCA FMS RSC ENTER/EDIT
MENU TEXT Enter/Edit Revenue Source Codes
UPPERCASE MENU TEXT ENTER/EDIT REVENUE SOURCE CODE
LOCK PRCASVC
DESCRIPTION 
This option will allow the user owning the PRCASVC supervisor security
key to enter and edit the revenue source codes sent to FMS.  It is
important that this option be used with caution since it could cause
many of the FMS code sheets to reject if invalid revenue source codes
are entered.  Revenue source codes should only be entered and used
after they have been entered in FMS.
CREATOR USER,ONE
TYPE run routine
PACKAGE ACCOUNTS RECEIVABLE
ROUTINE ADDEDIT^RCXFMSUR
