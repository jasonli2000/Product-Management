DESCRIPTION 
This menu contains options that allow the user to set up certain interface
parameters that control the processing of HL7 messages.  The interface
adheres to the Health Level 7 (HL7) protocol and forms the basis for the
exchange of health care information between the VistA Surgery package and
any ancillary system.
