SRHL PARAMETER (8196)    OPTION (19)

Name Value
NAME SRHL PARAMETER
MENU TEXT Update Interface Parameter Field
UPPERCASE MENU TEXT UPDATE INTERFACE PARAMETER FIE
DESCRIPTION 
This option may be used to edit the site parameter that determines which
Surgery HL7 interface will be used, the interface compatible with VistA
HL7 v1.6 or the older one compatible with VistA HL7 v1.5.
 
If applications communicating with the Surgery HL7 interface must use the
interface designed for use with HL7 v1.5, enter YES.  Otherwise, enter NO
or leave this field blank.
CREATOR USER,ONE
TYPE run routine
