UPPERCASE MENU TEXT NLT MONTHLY WORKLOAD DETAIL
DESCRIPTION 
This option can be used to obtain a report of monthly compiled data stored
in 'LAB MONTHLY WORKLOAD' (#67.9) file. The LAB MONTHLY WORKLOAD file
contains workload totals by division by WKLD CODE by month. Totals by
specific days are not stored in the file.
 
This option is a FileMan driven search and print function. The format of
the report is sized to fit a 80 column printer. Because of space
constraints WKLD CODE names have been truncated to 30 characters.
 
The report grouped by division, listing WKLD CODE number, WKLD CODE name,
and the month. Sub totals and totals for each month by each WKLD CODE for
In-Patients (INPT), Out-Patient (OUTPT), Other, Manual Entry (MANUAL) and
if the WKLD CODE is billable.
 
This report is suitable for monthly comparison of workload a current
billable status.
