NAME LRMISTUF
MENU TEXT Results entry (batch)
UPPERCASE MENU TEXT RESULTS ENTRY (BATCH)
DESCRIPTION 
Results entry (batch) allows you to enter either preliminary or final data
for one field and test for multiple accession numbers in one session.  
The field choices are Bacteriology, Mycology, TB, Virology, Mycobacter-
iology, Parasite Report Remark, Gram Stain, Urine Screen, Sputum Screen,
Parasitology Smear/Prep, Mycology Smear/Prep, and Acid Fast Stain.  These
choices are dependent on the microbiology test selected and specified on
the test's edit code.  There are three levels of automation to choose from:
1) result is entered and you are not allowed to change it, 2) result
is entered and you are allowed to change it, 3) no result is entered
for you.  You must have the LRVERIFY security key to use this option.
