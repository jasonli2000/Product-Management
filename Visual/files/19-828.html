LRTESTDIQ (828)    OPTION (19)

Name Value
NAME LRTESTDIQ
X ACTION PRESENT YES
MENU TEXT Inquiry to LAB TEST file
UPPERCASE MENU TEXT INQUIRY TO LAB TEST FILE
LOCK LRSUPER
DESCRIPTION 
This function displays information from the Laboratory test file.
All defined fields that are associated with the requested test in
file 60 are displayed.  If information is missing on a particular
test, the desired information must be first added to File 60.
This option is just a "lookup" option to allow the user to inquire
about how a test is defined in the "Laboratory Test File 60."
CREATOR POSTMASTER
TYPE inquire
PACKAGE LAB SERVICE
EXIT ACTION K %,A,DIC,DL,DX,K,S,X,Y,Z
DIC {DIC} LAB(60,
DIC(0) AEMQ
