SC PCMM POSITION ASSIGN REVIEW (8373)    OPTION (19)

Name Value
NAME SC PCMM POSITION ASSIGN REVIEW
MENU TEXT Patient Team Position Assignment Review
UPPERCASE MENU TEXT PATIENT TEAM POSITION ASSIGNME
DESCRIPTION 
This option compares all PATIENT TEAM POSITION ASSIGNMENT (#404.43) file
entries with the corresponding PATIENT TEAM ASSIGNMENT (#404.42) file entries.
The comparison checks to make sure that the position assignment active 
timeframe is within the team assignment active timeframe.
 
This report lists those position assignments that fall outside the team
assignment active timeframe.
 
For example, the following situation would be listed on the report since
the team assignment is inactive but the position assignment is still
active.
 
                         Assigned              Unassigned
                           Date                   Date
                         --------              ----------
   Team Assignment      01/01/1997             10/31/1997
   Position Assignment  02/15/1997               <none>
 
 
 
Note: The PCMM software has been changed to prevent this and similar
      situations from occurring. However, older versions of PCMM did
      allow these assignment timeframe problems to occur. This report
      should help the site correct these problem assignments.
 
 
Note: Currently, this report can only be run in a 'diagnostic' mode.
      In the future, the report maybe enhanced to run in a 'fix' mode.
      The 'fix' mode option would also correct the problem assignments
      based on criteria the user specifies.
