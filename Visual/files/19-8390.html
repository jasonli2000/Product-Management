XDR RESTART MERGE PROCESS (8390)    OPTION (19)

Name Value
NAME XDR RESTART MERGE PROCESS
MENU TEXT Restart a Merge Process
UPPERCASE MENU TEXT RESTART A MERGE PROCESS
DESCRIPTION 
This option is used to schedule the restart of a merge process at the
current time, or at some point in the future.
