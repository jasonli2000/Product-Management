XDR PRELIMINARY SCAN LIST (8391)    OPTION (19)

Name Value
NAME XDR PRELIMINARY SCAN LIST
MENU TEXT List File Entries Identified in Preliminary Scan
UPPERCASE MENU TEXT LIST FILE ENTRIES IDENTIFIED I
DESCRIPTION 
This option is used to generate a list of those entries in the file which
were identified as lacking a zero node, having a bad SSN value, or missing
one or more of the identifiers in the file.
