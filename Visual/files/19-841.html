LRCENDEL (841)    OPTION (19)

Name Value
NAME LRCENDEL
MENU TEXT Delete entire order or individual tests
UPPERCASE MENU TEXT DELETE ENTIRE ORDER OR INDIVID
DESCRIPTION 
This option may be used to remove an entire order.  The reason for
deleting the order must be entered.  If results have already been
entered and verified for any part of the order, the order cannot be
deleted.
