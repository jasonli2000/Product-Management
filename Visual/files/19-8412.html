HL TASK RESTART (8412)    OPTION (19)

Name Value
NAME HL TASK RESTART
MENU TEXT Restart/Start All Links and Filers
UPPERCASE MENU TEXT RESTART/START ALL LINKS AND FI
DESCRIPTION 
This is a Taskman "startup" option used to restart all HL7 Logical Links
and the default number of inbound and outbound filers after a system
reboot (or whenever Taskman is restarted).
CREATOR USER,ONE
TYPE run routine
PACKAGE HEALTH LEVEL SEVEN
ROUTINE QUE^HLCS2
SCHEDULING RECOMMENDED STARTUP
