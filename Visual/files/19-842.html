Name Value
NAME LRSTUF
MENU TEXT Batch data entry (chem, hem, tox, etc.)
UPPERCASE MENU TEXT BATCH DATA ENTRY (CHEM, HEM, T
LOCK LRVERIFY
DESCRIPTION 
Once the set of accessions is specified, and the test to be edited is
specified, the system loops through the accessions with a minimum of
overhead.  If the "automatically stuff" option is selected, and the
text for entry is preceded by a "/", the text will not be checked with
the input transform; e.g, "/canceled" may be entered for numeric fields.
CREATOR POSTMASTER
TYPE run routine
