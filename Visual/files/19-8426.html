NAME IBCN OUTPUT INS BUFF ACTIVITY
MENU TEXT Insurance Buffer Activity
UPPERCASE MENU TEXT INSURANCE BUFFER ACTIVITY
DESCRIPTION 
This report provides a summary of the activity within the Insurance 
Buffer for a specified date range.  Counts, percentages, and average 
processing times are included for both processed and unprocessed 
entries.  The report can be printed with totals only or with subtotals
by month within the date range selected.
