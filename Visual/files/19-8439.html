DGEN ENROLLEES BY STATUS (8439)    OPTION (19)

Name Value
NAME DGEN ENROLLEES BY STATUS
MENU TEXT Enrollees by Status,Priority,Preferred Facility
UPPERCASE MENU TEXT ENROLLEES BY STATUS,PRIORITY,P
DESCRIPTION 
The report provides counts of patients by enrollment status, priority, and
preferred facility.  The user can select the statuses, priorities, and
facilities to include in the report.  An optional patient listing of those
patients included in the statistics can also be requested.
