XPAR LIST BY PARAM (8442)    OPTION (19)

Name Value
NAME XPAR LIST BY PARAM
MENU TEXT List Values for a Selected Parameter
UPPERCASE MENU TEXT LIST VALUES FOR A SELECTED PAR
DESCRIPTION 
This option prompts for a parameter (defined in the PARAMETER DEFINITION
file) and lists all value instances for that parameter.
CREATOR USER,ONE
TYPE action
Short Menu Text List Values by Parameter
E ACTION PRESENT YES
ENTRY ACTION D ALLPARS^XPARLIST
