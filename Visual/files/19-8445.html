XPAR LIST BY TEMPLATE (8445)    OPTION (19)

Name Value
NAME XPAR LIST BY TEMPLATE
MENU TEXT List Values for a Selected Template
UPPERCASE MENU TEXT LIST VALUES FOR A SELECTED TEM
DESCRIPTION 
This option prompts for a parameter template.  Depending on the definition
of the template, additional information may be prompted for and then the
parameter values defined by the template are displayed.
CREATOR USER,ONE
TYPE action
Short Menu Text List Values by Template
E ACTION PRESENT YES
ENTRY ACTION D TMPLT^XPARLIST()
