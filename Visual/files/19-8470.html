HL VIEW TRANSMISSION LOG (8470)    OPTION (19)

Name Value
NAME HL VIEW TRANSMISSION LOG
MENU TEXT View Transmission Log (TCP only)
UPPERCASE MENU TEXT VIEW TRANSMISSION LOG (TCP ONL
DESCRIPTION 
This option is used to view the Transmission Log; i.e.,
the Message Administration File entries.  It displays those
messages that have been sent or are pending. Display of
messages with HL7 errors or exceptions is not yet supported.
CREATOR USER,ONE
TYPE run routine
PACKAGE HEALTH LEVEL SEVEN
ROUTINE EN^HLCSRPT
