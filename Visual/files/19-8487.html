NAME RMPR EDT/COMP ISSUE
MENU TEXT Edit/Complete Issue From Stock
UPPERCASE MENU TEXT EDIT/COMPLETE ISSUE FROM STOCK
DESCRIPTION 
This is an option to Edit or Complete a Lab issue from stock
for items issued in the LAB.  Only the Lab Technician should run this
option.  Under this option, a technician has the ability to EDIT and
CLOSE a lab issue.  When a technician elects to close the lab issue
the Pre-determined time for an item will be posted to the patient's 2319.
