SRCODING OP REPORT (8496)    OPTION (19)

Name Value
NAME SRCODING OP REPORT
X ACTION PRESENT YES
MENU TEXT Operation/Procedure Report
UPPERCASE MENU TEXT OPERATION/PROCEDURE REPORT
DESCRIPTION 
This option is used by the coders to print the operation report on an
operation or the non-OR procedure report on a non-OR procedure.
CREATOR USER,ONE
TYPE run routine
PACKAGE SURGERY
DELEGABLE YES
E ACTION PRESENT YES
EXIT ACTION D EXIT^SROVAR
ENTRY ACTION D ^SROVAR I '$G(SRTN) D ^SROPS1 S:'$D(SRTN) XQUIT="" Q:$D(XQUIT)
XQUIT EXECUTABLE Q
ROUTINE ORPT^SROVER2
