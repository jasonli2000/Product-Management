ORCM REVIEW PROTOCOLS (8518)    OPTION (19)

Name Value
NAME ORCM REVIEW PROTOCOLS
MENU TEXT Review protocol file
UPPERCASE MENU TEXT REVIEW PROTOCOL FILE
DESCRIPTION 
This option will queue three tasks to examine the PROTOCOL (#101) file
for potential problems that may occur with the conversion of protocols to
the ORDER DIALOG (#101.41) file.
 
In addition a task will be queued to examine the PACKAGE (#9.4) file for
problems with the entries for packages that interface with CPRS.
 
A report of each search will be printed to a selected device.
