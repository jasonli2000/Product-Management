ORB3 REC PROCESSING FLAG (8538)    OPTION (19)

Name Value
NAME ORB3 REC PROCESSING FLAG
MENU TEXT Enable/Disable My Notifications
UPPERCASE MENU TEXT ENABLE/DISABLE MY NOTIFICATION
DESCRIPTION 
Code indicating processing flag for the current User and selected
notification.  Users indicate a notification should not be processed for
them - except where indicated for that notification in ORB3 REGULAR
RECIPIENTS parameter.
CREATOR USER,ONE
TYPE run routine
