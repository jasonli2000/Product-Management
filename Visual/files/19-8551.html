ORK SYSTEM ENABLE/DISABLE (8551)    OPTION (19)

Name Value
NAME ORK SYSTEM ENABLE/DISABLE
MENU TEXT Enable or Disable Order Checking System
UPPERCASE MENU TEXT ENABLE OR DISABLE ORDER CHECKI
DESCRIPTION 
Determines if any order checking will occur.  'E' or 'Enable' indicates
order checking is enabled and running.  'D' or 'Disabled' indicates order
checking is disabled and not running.  Can be set at the institution, System 
or Package level.
CREATOR USER,ONE
TYPE run routine
ROUTINE SYSEN^ORKMGR
