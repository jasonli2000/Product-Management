ORE LAB ORDERS CHECK 100=>69 (8553)    OPTION (19)

Name Value
NAME ORE LAB ORDERS CHECK 100=>69
MENU TEXT Check Lab orders from file 100 to 69
UPPERCASE MENU TEXT CHECK LAB ORDERS FROM FILE 100
DESCRIPTION 
This option checks for inconsistencies between OE/RR and Lab files
and attempts to correct them.  There is no set frequency for running this
option, but it should be run every few months to check for
inconsistencies.  This option will take a long time to run, so run it at
night or over a weekend.
