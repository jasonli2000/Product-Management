LRLLE PRO (860)    OPTION (19)

Name Value
NAME LRLLE PRO
MENU TEXT Edit the Load/Work list profile
UPPERCASE MENU TEXT EDIT THE LOAD/WORK LIST PROFIL
LOCK LRLIASON
DESCRIPTION 
This function allows the user to modify the list of what tests are
to be pulled from the accessions and be put on the load or work list.
If tests are accessioned to the area associated with the load/work list
but are not on the given profile, the tests will not appear on the
load/work list.  If a specimen is further indicated for a given test on
the profile, only tests for the indicated specimen will be put on the
load/work list.  If no specimen is indicated, any specimen with the
indicated test will be placed on the load/work list.
