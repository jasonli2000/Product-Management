ORW TOOL MENU ITEMS (8631)    OPTION (19)

Name Value
NAME ORW TOOL MENU ITEMS
MENU TEXT GUI Tool Menu Items
UPPERCASE MENU TEXT GUI TOOL MENU ITEMS
DESCRIPTION 
This option allows you to add items that will appear on the Tools menu 
in the CPRS GUI.  Items that appear on the tools menu should be
Windows executable files. Use this format: "Display Text=Executable".
For example, "Vista Terminal Emulator=KEA.EXE VISTA".
CREATOR USER,ONE
TYPE action
Short Menu Text GUI Tool Menu Items
E ACTION PRESENT YES
ENTRY ACTION D EDITPAR^XPAREDIT("ORWT TOOLS MENU")
