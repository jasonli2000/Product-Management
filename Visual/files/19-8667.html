Name Value
NAME PSJI 200
MENU TEXT Correct Changed User Names in IV Orders
UPPERCASE MENU TEXT CORRECT CHANGED USER NAMES IN
DESCRIPTION 
In Inpatient Medications version 5.0, the Pharmacy Patient file (#55)
subfile 55.01 field ENTRY CODE (#.23) was changed from free text to a
pointer to the NEW PERSON file (#200).  A new field ENTRY BY (#135) was
created. The post init of the version 5.0 install converts these
names to there pointer and stores it in the new field.  The
conversion also looks for the Free Text names in field ENTRY CODE that
don't have exact matches in the NEW PERSON file. These names are
temporarily stored in ^XTMP("PSJ NEW PERSON",NAME,DFN, ORDER NUMBER). This
option lets a pharmacist go through the bad names and choose the correct
user name.  After the correct name is chosen, the IV orders are updated
with the new pointer to that user.
