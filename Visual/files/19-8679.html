LR LOINC MAP (8679)    OPTION (19)

Name Value
NAME LR LOINC MAP
MENU TEXT Map Lab Tests to LOINC Codes
UPPERCASE MENU TEXT MAP LAB TESTS TO LOINC CODES
DESCRIPTION 
This option allows you to map Vista lab tests to LOINC codes.
This option allows you to enter the Result NLT Code for a
lab test and/or LEDI HL7 Code for a specimen so that the test
can be mapped to a LOINC code.
 
With patch LR*5.2*334, modification of NATIONAL VA LAB CODE (#64) field of
the LABORATORY TEST (#60) file, the user is able to select suffixed WKLD
CODE as test's NLT order code.
