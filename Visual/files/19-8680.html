Name Value
NAME LR LOINC TOPOGRAPHY
MENU TEXT Topography Print With/Without LEDI HL7 Codes
UPPERCASE MENU TEXT TOPOGRAPHY PRINT WITH/WITHOUT
DESCRIPTION 
This option allows you to print a list of lab tests with or without
LEDI HL7 Codes/Time aspect. The specimen must have a LEDI HL7 Code and
time aspect before the test is mapped to a LOINC code.  This list will
show the specimens that need a LEDI HL7 Code, time aspect or both.
