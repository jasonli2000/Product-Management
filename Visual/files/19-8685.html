LR LOINC PRINT 60/LOINC MAP (8685)    OPTION (19)

Name Value
NAME LR LOINC PRINT 60/LOINC MAP
MENU TEXT Print Lab Tests Mapped/Not Mapped to LOINC Codes
UPPERCASE MENU TEXT PRINT LAB TESTS MAPPED/NOT MAP
DESCRIPTION 
This option allows you to print a list of Lab Tests mapped or not mapped
to LOINC Codes.  You can use this list to determine which tests you need
to map and which tests are already mapped.

TYPE run routine
