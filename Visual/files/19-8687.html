LR LOINC HL7 SPECIMENS (8687)    OPTION (19)

Name Value
NAME LR LOINC HL7 SPECIMENS
MENU TEXT Specimen HL7 Codes Print
UPPERCASE MENU TEXT SPECIMEN HL7 CODES PRINT
DESCRIPTION 
This option allows you to print a list of SPECIMENS you can use to
enter LEDI HL7 specimens codes in the LAB TOPOGRAPHY FIELD (#61) file.
 
Those specimens preceded by a '*' indicate that the origin of these
specimens is from HL7 V2.3 Specimen source codes table 0070.
CREATOR USER,ONE
TYPE run routine
PACKAGE LAB SERVICE
ROUTINE LRLNCHL7
