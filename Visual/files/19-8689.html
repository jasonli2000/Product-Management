NAME PRCHOUT BOC REPORT
MENU TEXT BOC Report for OA&MM/Fiscal
UPPERCASE MENU TEXT BOC REPORT FOR OA&MM/FISCAL
DESCRIPTION 
This report will allow users to specify a date and name range to print any
Purchase Card transactions which include BOC numbers 2696 to 2699. It is
used by OA&MM/Fiscal to perform Supply Funds reconciliation.
