ORB3 UNVERIFIED MED ORDER (8718)    OPTION (19)

Name Value
NAME ORB3 UNVERIFIED MED ORDER
MENU TEXT Set Delay for Unverified Medication Orders
UPPERCASE MENU TEXT SET DELAY FOR UNVERIFIED MEDIC
DESCRIPTION 
Enter the number of hours delay to wait after a medication order is placed
before triggering an Unverified Medication Order notification/alert.  The
maximum number of hours is 10,000.
