LRMICROLOGIN (877)    OPTION (19)

Name Value
NAME LRMICROLOGIN
MENU TEXT Accessioning, standard (Microbiology)
UPPERCASE MENU TEXT ACCESSIONING, STANDARD (MICROB
DESCRIPTION 
This option allows you to accession manually requested tests.
It is especially useful for accessioning unusual specimens, or
accessioning specimens that require a descriptive comment.  You may also
comment about the order.  Any test or procedure may be accessioned, though
different lab sections may use lists of their most frequently ordered tests
to speed entry.  This option can also act as a batch entry log-in of a
specific test if you initially select that test.
When you are asked to select the label printer, if you enter '^', labels
will not be printed.  You will then be asked to select a label printer
with each new patient you select.  When you select a printer, that printer
will be used until you terminate that session.
CREATOR POSTMASTER
TYPE action
PACKAGE LAB SERVICE
E ACTION PRESENT YES
ENTRY ACTION S X="MICROBIOLOGY",DIC="^LAB(62.6,",DIC(0)="" D ^DIC S LRFLOG=Y D ^LRORD
