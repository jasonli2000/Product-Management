DGEN EGT IMPACT REPORT (8781)    OPTION (19)

Name Value
NAME DGEN EGT IMPACT REPORT
MENU TEXT EGT Impact Report
UPPERCASE MENU TEXT EGT IMPACT REPORT
DESCRIPTION 
This option will let the user choose to run one of four EGT Impact
Reports: EGT Preliminary Summary Impact Report, EGT Preliminary Detailed
Impact Report, EGT Actual Summary Impact Report, and EGT Actual Detailed
Impact Report. According to the criteria selected by the user, these four
reports will provide either a summarized count or the detailed information
of the patients who are either potentially or actually affected by the EGT
change.
