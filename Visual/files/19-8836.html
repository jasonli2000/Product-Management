FBAA FEE SCHEDULE RATE (8836)    OPTION (19)

Name Value
NAME FBAA FEE SCHEDULE RATE
MENU TEXT Calculate Payment Amount
UPPERCASE MENU TEXT CALCULATE PAYMENT AMOUNT
DESCRIPTION 
This option will calculate the fee schedule dollar amount for a service
(CPT Code). If the date of service is after September 1st, 1999 the
Medicare physician fee schedule (RBRVS) will be used.  If the RBRVS amount
is not greater than zero or if the date is prior to September 1999, the VA
75th Percentile fee schedule will be used to obtain an amount.
CREATOR USER,ONE
TYPE run routine
PACKAGE FEE BASIS
ROUTINE LOOKUP^FBAAFS
