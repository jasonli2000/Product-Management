SCMC PRECEPTOR CONVERSION (8847)    OPTION (19)

Name Value
NAME SCMC PRECEPTOR CONVERSION
MENU TEXT PRECEPTOR CONVERSION
UPPERCASE MENU TEXT PRECEPTOR CONVERSION
LOCK SC PCMM SETUP
DESCRIPTION 
This option allows the user to QUEUE off a job that will build the preceptor
histories.  It will be based upon what is currently in your PCMM preceptor
precepty relationships.  Before each of the histories can be set up, all of
the business rules pertaining to preceptor relationships will be checked.

This is the same job that was run as part of the post init for patch SD*5.3*177
CREATOR USER,ONE
TYPE run routine
