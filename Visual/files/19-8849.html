DG MISSING NEW PERSON FILE SSN (8849)    OPTION (19)

Name Value
NAME DG MISSING NEW PERSON FILE SSN
MENU TEXT Missing New Person File SSN Report
UPPERCASE MENU TEXT MISSING NEW PERSON FILE SSN RE
DESCRIPTION 
This option produces a list of active users without a social security
number defined in the NEW PERSON (#200) file.  A social security number is
required to access a PATIENT (#2) file if parameter prohibiting access to
your own PATIENT (#2) file record is enabled.
CREATOR USER,ONE
TYPE print
DIC {DIP} VA(200,
PG 1
L. 0
FLDS [DG MISSING SSN PRINT]
BY [DG MISSING SSN SORT]
