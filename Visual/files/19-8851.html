IVM FINANCIAL QUERY (8851)    OPTION (19)

Name Value
NAME IVM FINANCIAL QUERY
MENU TEXT IVM Financial Query
UPPERCASE MENU TEXT IVM FINANCIAL QUERY
DESCRIPTION 
This option permits financial queries to be sent to the Health Eligibility
Center (HEC).  This option is used to query the HEC for a patient's most
recent financial information.  This financial information may include
Means Test, Copay Test, and Income Screening data.
CREATOR USER,ONE
TYPE run routine
ROUTINE OPT^IVMCQ
