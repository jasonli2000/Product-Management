HL START/STOP LINK MANAGER (8852)    OPTION (19)

Name Value
NAME HL START/STOP LINK MANAGER
MENU TEXT TCP Link Manager Start/Stop
UPPERCASE MENU TEXT TCP LINK MANAGER START/STOP
DESCRIPTION 
This option is used to start/restart the Link Manager.  Under normal
conditions, the Link Manager should automatically start along with all
options flagged for special queuing.
CREATOR USER,ONE
TYPE run routine
PACKAGE HEALTH LEVEL SEVEN
DELEGABLE YES
