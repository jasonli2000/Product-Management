PRS MONTH CALENDAR (8854)    OPTION (19)

Name Value
NAME PRS MONTH CALENDAR
MENU TEXT Monthly Calendar
UPPERCASE MENU TEXT MONTHLY CALENDAR
DESCRIPTION 
This option displays a calendar for the month including the user 
specified date.  Optionally, a calendar that represents days of the
month as number of days since January 1st can be displayed below the
Gregorian calendar.  Pay periods and holidays are also shown.
CREATOR USER,ONE
TYPE run routine
Short Menu Text Calendar
ROUTINE MAIN^PRSLIB01
