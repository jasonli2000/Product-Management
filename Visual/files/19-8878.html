OOPS SCHEDULED XMIT 2162 DATA (8878)    OPTION (19)

Name Value
NAME OOPS SCHEDULED XMIT 2162 DATA
MENU TEXT Scheduled Transmit National Database (2162) Data
UPPERCASE MENU TEXT SCHEDULED TRANSMIT NATIONAL DA
DESCRIPTION 
This option will be used to transmit the 2162 data from closed ASISTS cases
to the Austin Automation Center for processing to the National Database.  It
will be a scheduled option that should be run monthly.
CREATOR USER,ONE
TYPE run routine
PACKAGE ASISTS
DELEGABLE YES
ROUTINE EN^OOPSNDB
SCHEDULING RECOMMENDED YES
KEEP FROM DELETING Yes
