Name Value
NAME SCCV CONV MAIN OPTION
MENU TEXT ACRP Database Conversion
UPPERCASE MENU TEXT ACRP DATABASE CONVERSION
DESCRIPTION 
The option provides all the actions to add, edit, cancel, view and execute
conversion specifiation templates (CST).

It also contains functionality to edit overall conversion parameters and 
deleting old scheduling files.
