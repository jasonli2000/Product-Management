NAME SPNP BASIC PATIENT INFO
MENU TEXT Basic Patient Information (132 Column)
UPPERCASE MENU TEXT BASIC PATIENT INFORMATION (132
DESCRIPTION 
This "canned" report prints basic demographic information, 1 row per
patient.  It is designed for 132-column printing/displaying.  Therefore,
if printing a hardcopy, send to a 132-column printer or use a 132-column
subtype, e.g., MYPRINTER;P-HPLASER-P17.  If displaying to screen for file
capture, at the DEVICE prompt enter 0;132;9999 without spaces.
 
The information printed consists of Patient Name, SSN, DOB, Phone, Street
Address 1, Street Address 2, City, State, and Zip Code.  Be advised that
due to space limitations, street address 3 is not included.
