ENFA CT EXP LIST (8924)    OPTION (19)

Name Value
NAME ENFA CT EXP LIST
MENU TEXT List Equipment to be Expensed by Task
UPPERCASE MENU TEXT LIST EQUIPMENT TO BE EXPENSED
DESCRIPTION 
The option generates a report of equipment that currently meets the
criteria to be expensed by the one-time task. The output is sorted by
station and then by value.  Equipment that is currently capitalized and
reported to Fixed Assets/FMS would be expensed unless it meets one or more
of the following criteria:
  SGL = 1524 (EXCESS)
  FUND = 4539 (FRANCHISE)
  ASSET VALUE greater than $99,999.99
  SYSTEM ASSET VALUE greater than $99,999.99 
  Item edited by user to Remain Capitalized
The system asset value is determined by adding the asset values of all
equipment which is part of a system and also reported to Fixed Assets/FMS.
Equipment can be placed in a system using the PARENT field.
CREATOR USER,ONE
TYPE run routine
PACKAGE ENGINEERING
E ACTION PRESENT YES
ENTRY ACTION I $G(DT)>3020725 W $C(7),!," *** This option is obsolete and it should not be used.***"
ROUTINE ENFACTL
