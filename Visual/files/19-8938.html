PSXR AUTO TRANSMIT CS (8938)    OPTION (19)

Name Value
NAME PSXR AUTO TRANSMIT CS
MENU TEXT Setup CS Auto-transmission
UPPERCASE MENU TEXT SETUP CS AUTO-TRANSMISSION
LOCK PSXAUTOX
DESCRIPTION 
This options schedules or unschedules the CMOP controlled substances data
transmission to run automatically in the background. The user is asked to
enter a re-scheduling frequency, in hours. The job automatically
re-schedules to run for the time interval selected. The job will continue
to run for the selected time intervals until it is unscheduled by the
user.
CREATOR USER,ONE
TYPE run routine
ROUTINE PSXAUTOC
