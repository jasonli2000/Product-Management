PSO SUCMOP RE-SUSLOCALCMOPPRI (8958)    OPTION (19)

Name Value
NAME PSO SUCMOP RE-SUSLOCALCMOPPRI
MENU TEXT Re-Suspend Printed Rxs with CMOP Drugs
UPPERCASE MENU TEXT RE-SUSPEND PRINTED RXS WITH CM
DESCRIPTION 
This utility will allow sites to re-suspend those Rxs that list on the 
List of Rxs with CMOP Drugs Printed From Suspense report [PSO SUCMOP PRINT
REPORT].  This utility will re-suspend to the date that it was originally
suspended to.  It should not be necessary to edit the suspended date because
all Rxs queued for CMOP transmission will be transmitted.
