Name Value
NAME LRCAPCPTI
MENU TEXT InActive WKLD CODE File CPT Codes Print
UPPERCASE MENU TEXT INACTIVE WKLD CODE FILE CPT CO
DESCRIPTION 
This option will scan the WKLD CODE (#64) file for CPT codes that are
InActive in the VA CPT code (#81) file. This option maybe useful when new
CPT Code files are installed. New CPT Code files are usually distributed
annually.  
  
 When a new CPT code file is released, it may contain codes that are no
longer allowed and are inactive in the CPT file. The Laboratory Package
PCE interface checks each CPT code in the WKLD CODE (#64) file before
sending workload to the PCE API. If the CPT code defined in WKLD CODE file
is deemed inactive in the CPT Code files, the software will enter a date
in the INACTIVE DATE field for that particular code. A message is sent to
the local G.LMI mail group informing the LIM that the CPT Code is no
longer active.  
  
 This option can be used to print all of the WKLD CODE file CPT Codes that
are INACTIVE and require a different CPT Code to be entered in order to
capture PCE Workload.
  
 The report will identify the WKLD CODE that contains the inappropriate
CPT Code. It will list the offending CPT code(s). It will indicate why the
CPT code is inappropriate. It will then give a list of LABORATORY TEST
names that are linked to this WKLD CODE. The LABORATORY TEST listed will
NOT receive PCE Workload credit.
