ACKQAS CDR PRINT (8961)    OPTION (19)

Name Value
NAME ACKQAS CDR PRINT
MENU TEXT Print A&SP Service CDR
UPPERCASE MENU TEXT PRINT A&SP SERVICE CDR
DESCRIPTION 
The Print A&SP Service CDR option prints the RCS 10-014 report for Audiology
and Speech Pathology.
  
Use this option to reprint the CDR report that you have generated using the
Generate A&SP Service CDR option.
  
If your CDR is reported by Division, you must select the Division(s) to be 
printed. You must then enter the month that you wish to print. The report
will print the CDR for the month and Divisions that have been generated.
