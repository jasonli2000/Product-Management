ACKQAS UPDATE CPT MODIFIERS (8962)    OPTION (19)

Name Value
NAME ACKQAS UPDATE CPT MODIFIERS
MENU TEXT Update CPT Modifiers per CO Directive
UPPERCASE MENU TEXT UPDATE CPT MODIFIERS PER CO DI
DESCRIPTION 
The system shall allows the user to select any Modifier already defined in
the A&SP Modifier file for edit. The system allows selection of CPT
Modifiers or a HCPCS Modifiers from the CPT Modifier File #81.3 that are
defined as a type 'C' or type 'H'.
The system prevents the user from adding to the A&SP Modifier File any
Modifier that is defined as Inactive on the CPT Modifier File #81.3.
The system also prevents the user from deleting a Modifier from the A&SP
Modifier File.
The system then prompts the user to enter the Status of the Modifier in
QUASAR. Valid entries are '1' for Active or '0' for Inactive.
If a new Modifier is selected from the CPT Modifier File the Modifier
Status shall default to Active.
If an existing Modifier is selected from the A&SP Modifier File then the
system shall display the current status of the Modifier as the default
value.
CREATOR USER,ONE
TYPE edit
PACKAGE QUASAR
E ACTION PRESENT YES
