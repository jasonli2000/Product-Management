ACKQAS DELETE VISIT (8963)    OPTION (19)

Name Value
NAME ACKQAS DELETE VISIT
MENU TEXT Delete an A&SP Clinic Visit
UPPERCASE MENU TEXT DELETE AN A&SP CLINIC VISIT
DESCRIPTION 
This option allows you to delete an A&SP Clinic Visit
that has been entered in error.

If the selected visit has been sent across to PCE then 
this function will also attempt to delete the visit from
PCE. If the PCE Deletion fails you will see an Error 
message displayed. You may then choose to continue or 
abort the deletion. If you continue, the Quasar Visit 
will be deleted but you will have to use the PCE package
to delete the same visit from PCE.
CREATOR USER,ONE
TYPE run routine
PACKAGE QUASAR
ROUTINE ACKQAS4
