ACKQAS PCE EXCEPTION REPORT (8964)    OPTION (19)

Name Value
NAME ACKQAS PCE EXCEPTION REPORT
MENU TEXT PCE Exception Report
UPPERCASE MENU TEXT PCE EXCEPTION REPORT
DESCRIPTION 
This report will list all A&SP Clinic Visits whose Visit Date falls
after the PCE Interface Start Date for the Division, that have not
been successfully updated in PCE.
 
A Visit will be included on the report if it falls into one of the
following two categories:-
1. The last time QUASAR attempted to send the Visit to PCE, the Visit
   was rejected. In this case the report will include the error 
   messages returned by the PCE Interface.
2. The Visit has been edited in Quasar but the new Visit data has not
   been sent to PCE because the PCE Interface has been switched off.
   In this case the report will show the date/time of the last QUASAR
   update versus the date/time of the last time the Visit was sent to 
   PCE.
 
To remove a Visit from the Exception report, the PCE Interface must be
activated for the Site, and for the Division in which the Visit took 
place. Then in order for QUASAR to send the new Visit data you must
edit the Visit using the Edit An Existing Visit option. At the end of
the Edit option, Quasar will attempt to send the new Visit data to 
PCE. Once the Visit data is the same in both systems, the visit will
be removed from the PCE Exception Report.
