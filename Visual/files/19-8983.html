PXRM TERM COPY (8983)    OPTION (19)

Name Value
NAME PXRM TERM COPY
MENU TEXT Copy Reminder Term
UPPERCASE MENU TEXT COPY REMINDER TERM
DESCRIPTION 
This option allows a user to copy an existing reminder term into a new
one. The new term must have a unique name.
