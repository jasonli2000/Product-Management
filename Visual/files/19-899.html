Name Value
NAME LRAPPF
MENU TEXT Entries by dates,patient & accession #
UPPERCASE MENU TEXT ENTRIES BY DATES,PATIENT & ACC
DESCRIPTION 
Used by anatomic pathology to print a list of accessions by patient
with organ/tissue data, an accession # index, or (for cytopath) a
calculation of the percent of positive diagnoses, for specified dates.
