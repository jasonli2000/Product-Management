LR COUNT ACC TESTS (902)    OPTION (19)

Name Value
NAME LR COUNT ACC TESTS
MENU TEXT Count accessioned tests
UPPERCASE MENU TEXT COUNT ACCESSIONED TESTS
LOCK LRSUPER
DESCRIPTION 
Provides a count of all verified tests by requesting area and cumulative totals
for a given accession area.  The report will list the # of accessions
and test counts by location and also give a total # of accessions
and total test count by accession area.  The option has two displays
"Counts by location" and "Counts by test."
 This option is different from the option 'LAB ACCESSION AND TEST
COUNTS' in that it only counts verified test'
CREATOR POSTMASTER
