DGRUG RAI/MDS BUILD ASCII FILE (9023)    OPTION (19)

Name Value
NAME DGRUG RAI/MDS BUILD ASCII FILE
MENU TEXT Build Patient ASCII File
UPPERCASE MENU TEXT BUILD PATIENT ASCII FILE
DESCRIPTION 
This option will create an ASCII file which will contain patient
demographic and admission data.  It will create a file based on station 
number as part of the name.  The fortmat for the file name is:
"VA"_station number_".TXT".  Therefore, if this were run at Albany NY, 
which has a station number of 500, the file name assigned would be:
VA500.TXT.  This is a deliminated file, and will be used by the vendor
for seeding their data base.
CREATOR USER,ONE
TYPE run routine
ROUTINE EN^DGRUGPRP
