RMPR SET RENTAL FLAG (9025)    OPTION (19)

Name Value
NAME RMPR SET RENTAL FLAG
X ACTION PRESENT YES
MENU TEXT Set CPT Modifier Rental Flag
UPPERCASE MENU TEXT SET CPT MODIFIER RENTAL FLAG
LOCK RMPRMANAGER
DESCRIPTION 
This option to be used by users with RMPRMANAGER key to set a Rental flag
of a particular HCPCS.  Users will only set the Rental flag for HCPCS that
are rented.
CREATOR USER,ONE
TYPE edit
PACKAGE PROSTHETICS
EXIT ACTION K RMC
DIC {DIC} RMPR(661.1,
DIC(0) AEMQZ
DIC(S) S RMC=$G(^RMPR(661.1,+Y,4)) I ($P(^RMPR(661.1,+Y,0),U,5)=1),(RMC["RR")
DIE RMPR(661.1,
DR {DIE} 30
