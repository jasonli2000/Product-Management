LR INF WARN (903)    OPTION (19)

Name Value
NAME LR INF WARN
MENU TEXT Infection warning edit
UPPERCASE MENU TEXT INFECTION WARNING EDIT
LOCK LRSUPER
DESCRIPTION 
This function enables the user to edit the infection warning for a patient.
The warning appears on all labels printed for this patient.
CREATOR POSTMASTER
TYPE action
PACKAGE LAB SERVICE
E ACTION PRESENT YES
ENTRY ACTION K DIC D ^LRDPA Q:LRDFN<1 S DA=LRDFN,DR=.091,DIE="^LR(" D ^DIE
