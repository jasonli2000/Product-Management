PRCG LOAD 410 PURGEMASTER (9042)    OPTION (19)

Name Value
NAME PRCG LOAD 410 PURGEMASTER
MENU TEXT Load IFCAP File 410 Documents into PurgeMaster
UPPERCASE MENU TEXT LOAD IFCAP FILE 410 DOCUMENTS
DESCRIPTION 
This option selects records in files 410 and 443, through a specified fiscal
year entered by the user, and creates entries for them in the IFCAP
PurgeMaster File (443.1) for purging.  Entries in files 410 and 443 that do
not have a reference to file 442 are then purged by PurgeMaster.
CREATOR USER,ONE
TYPE run routine
Short Menu Text Load 410 PurgeMaster
