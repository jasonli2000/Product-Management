XT-BLD RTN LIST (9044)    OPTION (19)

Name Value
NAME XT-BLD RTN LIST
MENU TEXT Routine Summary List
UPPERCASE MENU TEXT ROUTINE SUMMARY LIST
DESCRIPTION 
This option will create a list of routines in a build with before and
after checksums, and second line data.
 
This list can be used in a cut and paste operation when preparing a patch.
CREATOR USER,ONE
TYPE run routine
PACKAGE TOOLKIT
DELEGABLE YES
