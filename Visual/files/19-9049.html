DGRU RAI BATCH UPDATE (9049)    OPTION (19)

Name Value
NAME DGRU RAI BATCH UPDATE
MENU TEXT Background job for updating RAI/MDS COTS
UPPERCASE MENU TEXT BACKGROUND JOB FOR UPDATING RA
DESCRIPTION 
This option is not intended to be run from programmer mode.  No
output to the screen is intended.  When this job is scheduled through TaskMan
it will collect and transmitt all the entries in the PIVOT file that are a
COTS UPDATE to patient information . The job will build DGRU PATIENT A08
HL7 v2.3 messages and broadcast them to the COTS subscriber the TCP
linkage. The job should be re-scheduled to run about every 10 minutes. The
task interval will determine approximately the maximum time it takes for
an update to patient information to be broadcast in a HL7 message.
CREATOR USER,ONE
TYPE run routine

ROUTINE EN^DGRUGBJ
SCHEDULING RECOMMENDED YES
