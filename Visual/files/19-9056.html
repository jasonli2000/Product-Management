DESCRIPTION 
This report will create a transfer pricing statement summary.   This
bottom line to this overall summary yields the total number of episodes
and total amounts of those episodes per VISN/facility or groups of
VISN/facilities.
 
1. You can select one or more VISN/facilities (home or preferred
VISN/facility) for summary data to accumulate from.
 
2. You can select a date range for which those services were provided with
summary data inclusive.
 
3. There is summary data on the report for Inpatient, Outpatient,
Pharmacy, and a Grand Total of the three.
