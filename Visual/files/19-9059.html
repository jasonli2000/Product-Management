NAME IBAT PATIENT REPORT
MENU TEXT Patient Detail Report
UPPERCASE MENU TEXT PATIENT DETAIL REPORT
DESCRIPTION 
This report will create a detailed listing of a transfer pricing statement
by patient.  This report will also aid in determining whether or not a
billable third party is involved.
 
1. You will be able to select a VISN (home or preferred VISN) for detailed
data to accumulate from.
 
2. You will be able to select a date range with summary data inclusive.
 
3. Detail data will be available per patient on the report along with a
subtotal per patient and a Grand Total.
