IBCR ENTER TP NEG RATES (9062)    OPTION (19)

Name Value
NAME IBCR ENTER TP NEG RATES
MENU TEXT Transfer Pricing Neg. Rates Enter/Edit
UPPERCASE MENU TEXT TRANSFER PRICING NEG. RATES EN
LOCK IB SUPERVISOR
DESCRIPTION 
This option is used to enter/edit Transfer Pricing Negotiated rates.
Rates can be negotiated between another VA Facility or an entire network
(VISN).  This option can only be used to edit rates in charge sets that
were previously set up using this option.  The rates entered here are
stored in Charge Master for use.
