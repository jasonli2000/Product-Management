Name Value
NAME PSO RX SEARCH
MENU TEXT Search For Mismatched Patient Orders
UPPERCASE MENU TEXT SEARCH FOR MISMATCHED PATIENT
DESCRIPTION 
This option will be used to queue a report to look for Rxs that may have
been created for the wrong patient as reported in NCH-0800-41306.
