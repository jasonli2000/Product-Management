LRAC STATUS (910)    OPTION (19)

Name Value
NAME LRAC STATUS
MENU TEXT Cumulative device status
UPPERCASE MENU TEXT CUMULATIVE DEVICE STATUS
DESCRIPTION 
This option is used to get the status of the cumulative for a
particular device.  When cumulative is abnormally stopped, this
display will give last printed location and patient for all
devices used to print cumulative.
CREATOR POSTMASTER
TYPE run routine
PACKAGE LAB SERVICE
ROUTINE STA^LRACK
