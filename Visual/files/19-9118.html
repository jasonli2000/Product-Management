SPNL FUTURE APPTS (9118)    OPTION (19)

Name Value
NAME SPNL FUTURE APPTS
MENU TEXT Patients with Future Appointments
UPPERCASE MENU TEXT PATIENTS WITH FUTURE APPOINTME
DESCRIPTION 
This report lists patients having future appointments within a user
specified date range.  A prompt allows you to select patients in the SCD
Registry or patients not in the SCD Registry but with a Spinal Cord Injury
(as determined from the patient file), or you can select both. This report
can be of great assistance in keeping your Registry up to date.
