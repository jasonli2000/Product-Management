DGRU DATA SEED (9122)    OPTION (19)

Name Value
NAME DGRU DATA SEED
MENU TEXT Seed Accu-Max Database
UPPERCASE MENU TEXT SEED ACCU-MAX DATABASE
DESCRIPTION 
This option will create an Admit (A01) HL7 message for every patient in
an RAI/MDS Ward assigned to the selected division.  The purpose of this
option is to do the initial seeding of the Accu-Max database with
patients currently in RAI/MDS Wards.  This should only be done once
for each division, and the Accu-Max database should have an empty
patient database for the selected division when run.
CREATOR USER,ONE
TYPE run routine
DISPLAY OPTION? YES
ROUTINE DGRUGDS
