NAME ACKQAS EC PROC STATS
MENU TEXT Statistics by Event Capture Procedure
UPPERCASE MENU TEXT STATISTICS BY EVENT CAPTURE PR
DESCRIPTION 
The Event Capture Statistics by Procedure report can be printed for a date
range you select.  You can print the report for Audiology only, Speech
Pathology only, or both Audiology and Speech Pathology.
You can choose to print the report for one or all clinician(s), other 
provider(s), or student(s).
 
The report lists clinic visits for the date range sorted by Event Capture 
procedure codes.  Since the EC Procedure Code field is a multiple field,
the visit shows up once under each EC Procedure Code entered for that
visit.
