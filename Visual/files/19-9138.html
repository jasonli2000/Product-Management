RCTOP UPDATE REF/REV CODE (9138)    OPTION (19)

Name Value
NAME RCTOP UPDATE REF/REV CODE
MENU TEXT Update Refund/Refund Reversal Codes
UPPERCASE MENU TEXT UPDATE REFUND/REFUND REVERSAL
DESCRIPTION 
This option is used to enter the 1 character refund and 1 character
reversal code to be placed on Refund and Refund Reversal documents.  This
code will be updated each calendar year and will be provided by TOP.
