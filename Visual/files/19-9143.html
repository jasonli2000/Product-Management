RMPO HO PAT TEMPLATE UPDATE (9143)    OPTION (19)

Name Value
NAME RMPO HO PAT TEMPLATE UPDATE
MENU TEXT Home Oxygen Patient Template Update
UPPERCASE MENU TEXT HOME OXYGEN PATIENT TEMPLATE U
LOCK RMPRSUPERVISOR
DESCRIPTION 
This is a utility for updating VENDOR, HCPCS, FCP, ITEM, and UNIT COST of
all Home Oxygen patients in a particular station at once.  Only the
patient template is changed not the monthly billing.  If the monthly
billing is already been created for a certain month, changes on the
template will not affect the billing cycle.
CREATOR USER,ONE
TYPE run routine
ROUTINE START^RMPOVDC
