PRCP SS QTY DISCREPANCIES (9146)    OPTION (19)

Name Value
NAME PRCP SS QTY DISCREPANCIES
MENU TEXT Supply Station Quantity Discrepancy
UPPERCASE MENU TEXT SUPPLY STATION QUANTITY DISCRE
DESCRIPTION 
This option will produce a report of all items whose on hand quantity
differs from the on hand quantity last reported by the supply station.
The option is only meant to be run by secondaries linked by supply
stations.
CREATOR USER,ONE
TYPE run routine
ROUTINE PRCPRDIS
