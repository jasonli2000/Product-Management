PRCP UPDATE SUPPLY STA ITEMS (9150)    OPTION (19)

Name Value
NAME PRCP UPDATE SUPPLY STA ITEMS
MENU TEXT Supply Station Item Update
UPPERCASE MENU TEXT SUPPLY STATION ITEM UPDATE
DESCRIPTION 
This option will build transactions for supply station secondaries, that
will be sent to its associated supply station.  Transactions will be
created for all items in the selected inventory point whose normal stock
level is greater than zero.  All transactions will be flagged as either
item adds or item edits, as specified by the user of this option.
CREATOR USER,ONE
TYPE run routine
ROUTINE INIT^PRCPHLFM
