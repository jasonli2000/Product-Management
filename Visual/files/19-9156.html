SPN ICD CODE SEARCH (9156)    OPTION (19)

Name Value
NAME SPN ICD CODE SEARCH
MENU TEXT ICD9 Code Search
UPPERCASE MENU TEXT ICD9 CODE SEARCH
DESCRIPTION 
This option allows users to find patients in or out of the SCD Registry
who have just one particular ICD9 code, or who have several particular
ICD9 codes, or fall in a range of ICD9 codes.  The report searches the
patients in the PTF file (#45) according to user-specified admission
dates, and will include patients who have any of the ICD9 codes that were
inputted.
