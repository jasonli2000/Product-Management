MAGD FIX DICOM FILE (9224)    OPTION (19)

Name Value
NAME MAGD FIX DICOM FILE
MENU TEXT Correct RAD-DICOM File Entries
UPPERCASE MENU TEXT CORRECT RAD-DICOM FILE ENTRIES
DESCRIPTION 
This entry should be given to the Radiology technologist.  It will allow
corrections to be made on DICOM image files which failed during
the downloading processes.  The routine, MAGDIR1, used to download DICOM
files directly from Radiology modalities (CT, MRI, etc.) sometime will 
fail to match exactly on the patients name, ssn or Radiology case number.
A reference to the files which failed the matching will be written
to the MAGD(2006.575 global --- (please note that only a reference to  
the file is being made; the actual DICOM file will remain in the
IMAGE_IN DOS directory on the DICOM Image server.)  
The MAGDLB9 routine will use the "F" cross  reference to loop
through the entries needing corrections.  This  cross reference is
on the Unique Study Identification number assigned  by the Radiology
modalitiy.
CREATOR USER,ONE
TYPE run routine
ROUTINE START^MAGDLB9
