TIU TEMPLATE USER DELETE (9238)    OPTION (19)

Name Value
NAME TIU TEMPLATE USER DELETE
MENU TEXT Delete user's TIU templates.
UPPERCASE MENU TEXT DELETE USER'S TIU TEMPLATES.
DESCRIPTION 
This option calls code to remove all TIU templates for a user, except
those who are not orphans (i.e., also used by some other user).  It relies
on Kernel's XU USER TERMINATE option to set the variable XUIFN in order 
to identify the terminated user.
CREATOR USER,ONE
TYPE action
PACKAGE TEXT INTEGRATION UTILITIES
E ACTION PRESENT YES
ENTRY ACTION D KUSER^TIUSRVT3
