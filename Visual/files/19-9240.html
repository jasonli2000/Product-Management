TIU TEMPLATE USER DELETE PARAM (9240)    OPTION (19)

Name Value
NAME TIU TEMPLATE USER DELETE PARAM
MENU TEXT Edit auto template cleanup parameter.
UPPERCASE MENU TEXT EDIT AUTO TEMPLATE CLEANUP PAR
DESCRIPTION 
This option accesses the TIU TEMPLATE USER AUTO DELETE parameter and
toggles the entry on or off, providing for automatic deletion of
non-shared TIU templates for a user upon termination - or skipping any
such automatic action.
