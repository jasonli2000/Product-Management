TIU TEMPLATE DELETE TERM ALL (9242)    OPTION (19)

Name Value
NAME TIU TEMPLATE DELETE TERM ALL
MENU TEXT Delete templates for ALL terminated users.
UPPERCASE MENU TEXT DELETE TEMPLATES FOR ALL TERMI
DESCRIPTION 
This option allows a CAC or other manager to remove all non-shared TIU
templates for all users who have been terminated.  It orders through the
files recursively and then deletes those that have not been shared.
CREATOR USER,ONE
TYPE action
PACKAGE TEXT INTEGRATION UTILITIES
E ACTION PRESENT YES
ENTRY ACTION D CTRL^TIUSRVT4
