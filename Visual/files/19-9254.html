LR LOINC EXTRACT LAB NAMES (9254)    OPTION (19)

Name Value
NAME LR LOINC EXTRACT LAB NAMES
MENU TEXT Extract VistA Lab Test Names
UPPERCASE MENU TEXT EXTRACT VISTA LAB TEST NAMES
DESCRIPTION 
This option allows you to capture the VistA lab tests.
Copy and save the capture data into a text file. Using WordPad, edit the
text file removing extraneous data to ensure the capture only contains the
lab tests. Use RELMA to find the proper LOINC codes for all your lab
tests, and then use the VistA LOINC mapping options to map in VistA.
