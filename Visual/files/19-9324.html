RMPR VERIFY/REPAIR PC NUMBER (9324)    OPTION (19)

Name Value
NAME RMPR VERIFY/REPAIR PC NUMBER
MENU TEXT VERIFY/REPAIR PURCHASE CARD NUMBER
UPPERCASE MENU TEXT VERIFY/REPAIR PURCHASE CARD NU
DESCRIPTION 
This option was created temporarily to be used by sites that has problem
of reconciling Purchase Card order.  When Citi Bank replaces the old
Credit Card number,  Oracle data that comes back has a new Credit Card
number, thus prosthetics reconciliation doesn't work.  In order for
prostehtics user to reconcile, oracle data must be changed back to the old
Credit Card  number. This option also prints a report that shows all the
Purchase Order by 16 digit credit card number entered based on the date 
range.  This option will be removed,  once IFCAP released a patch to allow
reconciliation of purchase card  order with a different credit card
number.
