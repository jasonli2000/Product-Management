MENU TEXT Set Creatinine Date Range for Contrst Media Orders
UPPERCASE MENU TEXT SET CREATININE DATE RANGE FOR
DESCRIPTION 
Use this option to set the number of days back in time for the patient's
most recent  lab serum creatinine results when Imaging procedures using
contrast media are ordered.  This value is used in the Biochem Abnormality
for Contrast Media order check.
