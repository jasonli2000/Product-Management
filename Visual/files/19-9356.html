FBUC EXTENSION (9356)    OPTION (19)

Name Value
NAME FBUC EXTENSION
MENU TEXT Extension for Incomplete Mill Bill (1725) Claim
UPPERCASE MENU TEXT EXTENSION FOR INCOMPLETE MILL
LOCK FBAASUPERVISOR
DESCRIPTION 
This option is used by a supervisor to enter an extension for an
incomplete claim submitted under the provisions of 38 U.S.C. 1725 (Mill
Bill).  The normal expiration date for these claims is calculated as 31
days after the request information letter was sent.  However, if an
extension has been entered for a claim then the extension date will be
used as the expiration date if it is later than the calculated date.
