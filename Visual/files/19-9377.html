NAME LR LOINC HISTORICAL MAP MENU
MENU TEXT LOINC Historical Mapping Menu
UPPERCASE MENU TEXT LOINC HISTORICAL MAPPING MENU
LOCK XUPROGMODE
DESCRIPTION 
This menu contains the software required to perform historical LOINC 
mapping of CH subscripted verified results. The software will use 
existing LOINC mapping configuration defined in WKLD CODE (#64) file to 
determine LOINC code assigned a given dataname. The look back mapping 
process extends to the past 5-6 years of verified 'CH' subscripted data.
