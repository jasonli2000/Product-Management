MAGJ SCHED RECENT LIST COMPILE (9398)    OPTION (19)

Name Value
NAME MAGJ SCHED RECENT LIST COMPILE
MENU TEXT VISTARAD RECENT LIST COMPILE -- TMSCHEDULE
UPPERCASE MENU TEXT VISTARAD RECENT LIST COMPILE -
DESCRIPTION 
Use this option to SCHEDULE a TaskMan job to Compile VistaRad RECENT Exams
in the background.  For proper VistaRad functionality, this job should be
scheduled to start at Startup, and run once every 2 to 3 hours.  After
defining this Scheduled Option in Taskman, be sure to let VistaRad know
what the interval between compiles is (update the VistaRad Site Parameters
field RECENT BKGND COMPILE INTERVAL).
