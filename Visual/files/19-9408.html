GMRC IFC BACKGROUND STARTUP (9408)    OPTION (19)

Name Value
NAME GMRC IFC BACKGROUND STARTUP
MENU TEXT IFC Background Startup
UPPERCASE MENU TEXT IFC BACKGROUND STARTUP
DESCRIPTION 
This menu option is not designed for user interaction.  It is defined to
be called upon system startup and when taskman unexpectedly stops. This
option starts the background error processor for Inter-facility Consults
(IFC).
CREATOR USER,ONE
TYPE run routine
PACKAGE CONSULT/REQUEST TRACKING
ROUTINE EN^GMRCIBKG
