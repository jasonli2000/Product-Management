NAME IBCEM MESSAGES WITHOUT REVIEW
MENU TEXT EDI Messages Not Reviewed Report
UPPERCASE MENU TEXT EDI MESSAGES NOT REVIEWED REPO
DESCRIPTION 
This option allows for the display of all EDI return messages that were
filed without needing a review based on the text entries in the message
screen text file (#361.3).  The report can be run for a user-selected
date range on date the message was received at the site and may be
sorted by message text that caused the message to not need a review or
by bill #.
