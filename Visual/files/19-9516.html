IVMB SEED HEC MAIL GROUP SETUP (9516)    OPTION (19)

Name Value
NAME IVMB SEED HEC MAIL GROUP SETUP
MENU TEXT Setup Mail Group for Seed HEC Notifications
UPPERCASE MENU TEXT SETUP MAIL GROUP FOR SEED HEC
DESCRIPTION 
This option allows the authorized user to add members to the IVMB HEC MPI 
NOTIFICATION mail group that receives messages on the status of sending 
and receipt of Seed messages to the HEC.  This option should be run as a 
Post Install to patch DGEN*5.3*408 and prior to executing the option IVMB 
SEED HEC WITH MPI.
CREATOR USER,ONE
TYPE action
E ACTION PRESENT YES
