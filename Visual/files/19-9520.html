Name Value
NAME RMPR VIEW 2319 READ ONLY
X ACTION PRESENT YES
MENU TEXT View Prosthetics Item Transactions
UPPERCASE MENU TEXT VIEW PROSTHETICS ITEM TRANSACT
DESCRIPTION 
This option views or prints the patient's Item Transactions and Home
Oxygen Items issued to a particular patient.  It is mainly intended for
non-prosthetics users.  It can be given to any user who has the need to 
see the prosthetics patient's transactions or items issued.
