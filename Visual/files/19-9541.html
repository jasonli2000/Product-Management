DESCRIPTION 
This option is to be used only if you are using the VistaRad component of 
the Imaging package.  
 
In the option, you will be asked to select one Imaging type and an
examination status, then you will be prompted to enter/edit the value of
VISTARAD CATEGORY.  The option will continue to ask for another
examination status, until you are done with that Imaging type.
