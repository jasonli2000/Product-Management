GMTS OBJ ENTER/EDIT (9572)    OPTION (19)

Name Value
NAME GMTS OBJ ENTER/EDIT
MENU TEXT Create/Modify Health Summary Object
UPPERCASE MENU TEXT CREATE/MODIFY HEALTH SUMMARY O
DESCRIPTION 
This option allows the user to create or modify a Health 
Summary Object.  Objects are Health Summaries that may have 
their headers suppressed so they may be inserted into another
document.
CREATOR USER,ONE
TYPE run routine
ROUTINE OBJ^GMTSOBJ
