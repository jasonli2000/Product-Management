GMTS OBJ EXPORT (9577)    OPTION (19)

Name Value
NAME GMTS OBJ EXPORT
MENU TEXT Export a Health Summary Object
UPPERCASE MENU TEXT EXPORT A HEALTH SUMMARY OBJECT
DESCRIPTION 
This option will allow a user to export a Health Summary 
Object, and its corresponding Health Summary Type to a 
Mailman message which can be sent to other accounts/sites
for import.
