SPNL INPATIENT OUTCOMES (9585)    OPTION (19)

Name Value
NAME SPNL INPATIENT OUTCOMES
X ACTION PRESENT YES
MENU TEXT Inpatient Rehabilitation Outcomes
UPPERCASE MENU TEXT INPATIENT REHABILITATION OUTCO
DESCRIPTION 
This option controls viewing, editing, and creation of Outcomes records
for inpatient rehabilitation episodes of care.  An episode of care
consists of a series of outcomes records with the same care start date and
the same care end date.
