SPNLOLD (9593)    OPTION (19)

Name Value
NAME SPNLOLD
MENU TEXT Edit Non-conforming Outcome
UPPERCASE MENU TEXT EDIT NON-CONFORMING OUTCOME
DESCRIPTION 
This option is used to edit older outcome records, i.e., those outcomes
not on file prior to adoption of the "episode of care" clinical model,
introduced in patch SPN*2*19.  Accordingly, this option is restricted to
those records only.
