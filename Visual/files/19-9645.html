MAG ENTERPRISE (9645)    OPTION (19)

Name Value
NAME MAG ENTERPRISE
MENU TEXT Ad hoc Enterprise Site Report
UPPERCASE MENU TEXT AD HOC ENTERPRISE SITE REPORT
DESCRIPTION 
This utility will create an Enterprise covering a period of activity
specified by the user.  The report will be sent in a VistA Mail message
to each of the members of the MAG SERVER mail group.
CREATOR USER,ONE
TYPE run routine
ROUTINE AHOPT^MAGQE5
