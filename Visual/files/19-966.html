NAME LRUPAC
MENU TEXT Lab accession and test counts
UPPERCASE MENU TEXT LAB ACCESSION AND TEST COUNTS
DESCRIPTION 
Lists counts of each type of specimen and counts of each test from one date to another in the Accession Area file.  If Autopsy, Surgical pathology, Cytopathology, or EM is selected only accession counts are given.
  NOTE: This option will count all accession for the time period selected.
It will produce a total ordered test count. Use the option 'COUNT ACCESSION
TESTS' option to determine the total number verified.
