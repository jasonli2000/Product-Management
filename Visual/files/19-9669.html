LBRY LOCAL VENDOR ADD (9669)    OPTION (19)

Name Value
NAME LBRY LOCAL VENDOR ADD
MENU TEXT Add a local vendor (COREFLS)
UPPERCASE MENU TEXT ADD A LOCAL VENDOR (COREFLS)
OUT OF ORDER MESSAGE CoreFLS Disabled (LBR*2.5*12)
LOCK LBRY LOCAL VENDOR
DESCRIPTION 
Use this option to add or update CoreFLS vendors from the national
database unto your LOCAL VENDOR (#392.31) File.  The LOCAL VENDOR
(#392.31) File will be used to return vendor information.  The need to go
over the network for a CoreFLS vendor will be drastically reduced.
CREATOR USER,ONE
TYPE run routine
ROUTINE STAND^LBRCSL
