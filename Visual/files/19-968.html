LRAPC (968)    OPTION (19)

Name Value
NAME LRAPC
MENU TEXT Anatomic pathology topography counts
UPPERCASE MENU TEXT ANATOMIC PATHOLOGY TOPOGRAPHY
DESCRIPTION 
Prints counts of organ/tissues (SNOMED topography) specified by code from 
one date to another.  Also shows # of patients, # of accessions, and # of
organ/tissues for the time specified.
 More than one topography can be requested.  For each topography 
there will be a count and percentage of the total organ/tissues within
the time period.
CREATOR POSTMASTER
TYPE run routine
PACKAGE LAB SERVICE
ROUTINE LRAPC
