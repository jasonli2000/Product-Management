RCDPE MARK 0-BAL EFT MATCHED (9845)    OPTION (19)

Name Value
NAME RCDPE MARK 0-BAL EFT MATCHED
MENU TEXT Mark 0-Balance EFT Matched
UPPERCASE MENU TEXT MARK 0-BALANCE EFT MATCHED
DESCRIPTION 
This option allows the user to select an EFT detail record that has a
0-balance payment and mark it as matched to a paper EOB.  This will
remove it from the EFT UNMATCHED AGING REPORT.
CREATOR USER,ONE
TYPE run routine
