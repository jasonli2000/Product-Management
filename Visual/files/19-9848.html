RCDP SUMMARY 215 REPORT (9848)    OPTION (19)

Name Value
NAME RCDP SUMMARY 215 REPORT
MENU TEXT Summary SF215 Report
UPPERCASE MENU TEXT SUMMARY SF215 REPORT
DESCRIPTION 
This option will print the summary 215 report for a selected deposit.
It consists of an SF215 for each receipt in the deposit and a grand
totals page summarizing all the receipt totals.
CREATOR USER,ONE
TYPE run routine
