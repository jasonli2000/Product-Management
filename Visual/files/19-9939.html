AWCM ROLL UP NATIONAL DATA (9939)    OPTION (19)

Name Value
NAME AWCM ROLL UP NATIONAL DATA
MENU TEXT Roll Up Data to National Database
UPPERCASE MENU TEXT ROLL UP DATA TO NATIONAL DATAB
DESCRIPTION 
This option rolls up data to the National CPRS response time database in 
Silver Springs, MD.  It should be run as a tasked job only, at 1205 
am nightly.
