RA ALLOC (187)    SECURITY KEY (19.1)

Name Value
NAME RA ALLOC
DESCRIPTION 
The RA ALLOC key overrides location access security entered for each
Radiology/Nuclear Medicine user through Personnel Classification.  Owners
of the RA ALLOC will have expanded access to Imaging Locations, Imaging
Types, and Divisions.  In the case of most workload reports, this means
they will be able to select from a list of all Divisions and Imaging types
to include on the report.  In the case of various edit and ordering
functions, it means they will be able to select from all locations within
the Imaging Type to which they are currently signed on through the 'Select
sign-on location:' prompt.
