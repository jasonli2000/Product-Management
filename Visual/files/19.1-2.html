XUMGR (2)    SECURITY KEY (19.1)

Name Value
NAME XUMGR
DESCRIPTION 
This key is for users who need to act as site managment staff.
This gives the user access to see information that is normally
only avalible to the user that created it.
This is a partial list of its use:
1. It allows its holders to create "Routine"-type Options in the
OPTION file with bracket syntax ([UCI]) for UCI-switching.
2. To see the list of all spool file entrys.
