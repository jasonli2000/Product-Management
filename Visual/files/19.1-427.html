XUSHOWSSN (427)    SECURITY KEY (19.1)

Name Value
NAME XUSHOWSSN
DESCRIPTION 
This security key will allow the user to enter all 9 digits of a New 
Person's Social Security Number(SSN) for look up on the NEW PERSON(#200) 
file.  Any user who does not hold this key, will not be allowed to use a 9
digit SSN lookup on the NEW PERSON(#200) file.
