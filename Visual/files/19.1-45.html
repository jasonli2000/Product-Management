ENEDCLWO (45)    SECURITY KEY (19.1)

Name Value
NAME ENEDCLWO
DESCRIPTION 
Enables holders to edit closed work orders. Edit of a closed work
order automatically deletes the DATE COMPLETE. This should be
re-entered by the user if it's appropriate to do so. If subject work
order has been charged to a piece of equipment, the work order will
be removed from the Equipment History. It will be re-posted when
the DATE COMPLETE is re-entered.
