XUSIGNON (1)    BULLETIN (3.6)

Name Value
NAME XUSIGNON
SUBJECT USER SIGNING ON
DESCRIPTION 
This bulletin is triggered whenever someone signs on thru 'XUS'
MESSAGE 
User |2| has signed on at Device |3|.
PARAMETER
  1. PARAMETER:   1
    DESCRIPTION:   
    User number ('DUZ') of user signing on.
  2. PARAMETER:   2
    DESCRIPTION:   
    Name of user
  3. PARAMETER:   3
    DESCRIPTION:   
    Device at which sign-on occurred.
