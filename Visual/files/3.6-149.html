PRCP_NO_REFILL (149)    BULLETIN (3.6)

Name Value
NAME PRCP_NO_REFILL
SUBJECT Notification of ordered items without a refill amount
RETENTION DAYS 15
DESCRIPTION 
This bulletin alerts staff that a posted distribution order includes items
that were not marked as refilled by the supply station.  It is expected
that staff will verify the correct refill amount for these items by
creating another order of emergency or call-in to correct discrepancies
resulting in both the primary and secondary inventory points.
MESSAGE 
GIP did not get refill information on the following items in order |1|
to update |2| or |3|.
 
Please determine the correct quantities received and enter an emergency
order to correct the counts in the primary and secondary inventory points.
PARAMETER
  1. PARAMETER:   1
    DESCRIPTION:   
    Distribution Order Number
  2. PARAMETER:   2
    DESCRIPTION:   
    Primary Inventory Point
  3. PARAMETER:   3
    DESCRIPTION:   
    Secondary Inventory Point
