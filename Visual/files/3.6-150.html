PRCP_BAD_ORDER (150)    BULLETIN (3.6)

Name Value
NAME PRCP_BAD_ORDER
SUBJECT Notification of trouble with an HL7 refill transaction
RETENTION DAYS 15
DESCRIPTION 
This bulletin alerts staff of refill activity that cannot be processed
because the transaction information conflicts with system data or data
requirements.
 
Problems include: item not on the order specified, order not for the
secondary specified, amount refilled inappropriate, HL7 transaction
corrupted, order number specified is not in GIP
MESSAGE 
An HL7 item refill/post order transaction on order # |1| for the
secondary inventory point, |2|, rejected.
Please note the problem and make any corrections.
PARAMETER
  1. PARAMETER:   1
    DESCRIPTION:   
    Distribution Order Number from transaction
  2. PARAMETER:   2
    DESCRIPTION:   
    Secondary inventory point name from the HL7 transaction.
