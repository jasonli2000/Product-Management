PRCP_BAD_ACTIVITY (151)    BULLETIN (3.6)

Name Value
NAME PRCP_BAD_ACTIVITY
SUBJECT Notification of error in HL7 supply station activity transaction
RETENTION DAYS 15
DESCRIPTION 
This bulletin alerts staff of problems in HL7 transactions that update
a secondary inventory point with supply station activity.
 
Problems include: item not on this inventory point, quantity is too large
or too small, inventory point not in GIP.
MESSAGE 
An HL7 utilization transaction for |1| rejected.
PARAMETER
  1. PARAMETER:   1
    DESCRIPTION:   
    secondary inventory Point Name
