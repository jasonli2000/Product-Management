PRCP_QTY_MISMATCH (152)    BULLETIN (3.6)

Name Value
NAME PRCP_QTY_MISMATCH
SUBJECT Notification of inventory mismatches between systems
RETENTION DAYS 15
DESCRIPTION 
This bulletin will notify the secondary inventory staff of inventory
inconsistencies between the inventory point and its associated supply
station.  Staff need to investigate these discrepancies and make the
necessary adjustments.
MESSAGE 
An on-hand discrepancy with |1| exists for the
item |2|.
 
The quantity in |1| is |3|
The quantity on the supply station is |4|
 
(Information acquired from HL7 txn# |5|)
PARAMETER
  1. PARAMETER:   1
    DESCRIPTION:   
    secondary inventory point name
  2. PARAMETER:   2
    DESCRIPTION:   
    Item Name and number
  3. PARAMETER:   3
    DESCRIPTION:   
    quantity on the inventory point
  4. PARAMETER:   4
    DESCRIPTION:   
    Quantity in the Supply Station
  5. PARAMETER:   5
    DESCRIPTION:   
    The HL7 transaction which prompted the message to occur
