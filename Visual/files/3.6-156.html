NAME PRCP_BAD_ITEM_QOH
SUBJECT Notification of items in a Supply Station not in GIP
RETENTION DAYS 15
DESCRIPTION 
This item will display when GIP receives a QOH response from the supply
station containing an item not in file 441.
MESSAGE 
The supply station for |1| contains an item not in GIP.
Item: |2|.
 
(Information acquired from HL7 txn# |3|)

  1. PARAMETER:   1
    DESCRIPTION:   
    site and supply station name as received in the transaction
  2. PARAMETER:   2
    DESCRIPTION:   
    The item number and name as received in the transaction.
  3. PARAMETER:   3
    DESCRIPTION:   
    HL7 transaction number
