XM TOO MANY MESSAGES (172)    BULLETIN (3.6)

Name Value
NAME XM TOO MANY MESSAGES
SUBJECT Please Terminate Messages
RETENTION DAYS 30
DESCRIPTION 
This bulletin is sent by option XMMGR-DISK-MANY MESSAGE-MAINT, to ask
users to terminate excess messages.
MESSAGE 
You have at least |1| messages in your mailbox.
(|2| in your IN basket)

Please terminate all of your unnecessary messages.

Thank-you.
PARAMETER
  1. PARAMETER:   1
    DESCRIPTION:   
    number of messages in mailbox
  2. PARAMETER:   2
    DESCRIPTION:   
    number of messages in IN basket
PRIORITY? YES
