XDR MERGED (26)    BULLETIN (3.6)

Name Value
NAME XDR MERGED
SUBJECT NOTIFICATION OF DUPLICATES MERGED
DESCRIPTION 
This bulletin is sent by the Kernel Duplicate Resolution
software when a Verified Duplicate pair are merged together.
 
This bulletin is sent only if a package merge developer does not
have a routine that sends its own customized merge message.
MESSAGE 
|WRAP|
The following records from the |1| file have been merged.
|NOWRAP|
 
From record DA/DFN: |2|              To record DA/DFN: |3|
 
|4|                               |5|
 
|WRAP|
The FROM Record has now been merged to the TO Record.
|NOWRAP|
 
|6|
PARAMETER
  1. PARAMETER:   1
    DESCRIPTION:   
    The file from which the two records come from.
  2. PARAMETER:   2
    DESCRIPTION:   
    The From record's DA/DFN.
  3. PARAMETER:   3
    DESCRIPTION:   
    The To record's DA/DFN.
  4. PARAMETER:   4
    DESCRIPTION:   
    The internal value of the .01 field of the From Record.
  5. PARAMETER:   5
    DESCRIPTION:   
    The internal value of the .01 field of the To Record.
  6. PARAMETER:   6
    DESCRIPTION:   
    The package merge error listing header.
