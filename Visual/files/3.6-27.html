XDR ERROR (27)    BULLETIN (3.6)

Name Value
NAME XDR ERROR
SUBJECT ERROR DURING DUPLICATE CHECKING OR MERGE
DESCRIPTION 
This bulletin is sent by the Kernel Duplicate Resolution software
when an error that halts the process is encountered by a background
task.  It is only sent if there is a entry in the 'Duplicate Manager
Mail Group' in the Duplicate Resolution file.
MESSAGE 
|NOWRAP|
This is to notify you that an error was encountered while trying to
either check for duplicates or during the early merge process of
the |1| file.
 
From DA/DFN: |2|                           To DA/DFN: |3|
 
|4|                                     |5|
 
|WRAP|
|6|
PARAMETER
  1. PARAMETER:   1
    DESCRIPTION:   
    The file from which the two records come from.
  2. PARAMETER:   2
    DESCRIPTION:   
    The From record's DA/DFN.
  3. PARAMETER:   3
    DESCRIPTION:   
    The To record's DA/DFN.
  4. PARAMETER:   4
    DESCRIPTION:   
    The internal value of the .01 field of the From Record.
  5. PARAMETER:   5
    DESCRIPTION:   
    The internal value of the .01 field of the To Record.
  6. PARAMETER:   6
    DESCRIPTION:   
    The error condition that was encountered.
