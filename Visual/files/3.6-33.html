EN NEW EQUIPMENT (33)    BULLETIN (3.6)

Name Value
NAME EN NEW EQUIPMENT
SUBJECT NEW EQUIPMENT: ENTRY #|1|
DESCRIPTION 
Bulletin to alert users (generally in Engineering and A&MM Services)
that a new equipment record has been added to the AEMS/MERS database.
MESSAGE 
A new equipment record has been added to the AEMS/MERS database.
The ENTRY NUMBER is |1|.
It was entered by |2|.
The MFGR EQUIPMENT NAME is |3|.
The EQUIPMENT CATEGORY is |4|.
The SERVICE is |5|.
The STATION NUMBER is |6|.
 
|7|
PARAMETER
  1. PARAMETER:   1
    DESCRIPTION:   
    The IEN (computer generated).
  2. PARAMETER:   2
    DESCRIPTION:   
    Name of the user who entered the new record (from NEW PERSON FILE).
  3. PARAMETER:   3
    DESCRIPTION:   
    Brief description of the item.
  4. PARAMETER:   4
    DESCRIPTION:   
    The EQUIPMENT CATEGORY assigned (mainly for preventive maintenance
scheduling).
  5. PARAMETER:   5
    DESCRIPTION:   
    The owning SERVICE.  Generally (but not always) determined from
the CMR (Consolidated Memorandum of Receipt).
  6. PARAMETER:   6
    DESCRIPTION:   
    Station number.
  7. PARAMETER:   7
    DESCRIPTION:   
    Text indicating if equipment is capitalized NX and if reported to FAP.
