XUSERTERM (5)    BULLETIN (3.6)

Name Value
NAME XUSERTERM
SUBJECT USER TERMINATION
DESCRIPTION 
This bulletin is issued whenever a message cannot be delivered after the
Simple Mail Transfer Protocol "DATA" command is issued.  It indicates that
that the "Mail From:" and RCPT (recipient) commands were successfully 
issued, but that something in the header of the message was rejected,
such as duplicate message ID.  The error message returned by the remote
receiver is included in the bulletin.
MESSAGE 
USER |1| has been terminated as of |2|
The error message was '|3|'.
PARAMETER
  1. PARAMETER:   1
    DESCRIPTION:   
    Name of the remote site rejecting the message (may be intermediate relay)
  2. PARAMETER:   2
    DESCRIPTION:   
    Title and message number of message which was rejected
  3. PARAMETER:   3
    DESCRIPTION:   
    The rejection message issued by the rejecting receiver.
