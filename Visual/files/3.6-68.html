PRS UPDATE CCORG (68)    BULLETIN (3.6)

Name Value
NAME PRS UPDATE CCORG
SUBJECT Needed update PAID code: Cost Center/Organization
DESCRIPTION 
This bulletin is sent when either of the two options, Employee Leave Used and
Employee Leave Requested are run from the menu path, T&A Supervisor Menu,
Employee Reports, Employee Leave Reports.  The condition that the code for an
employees Cost Center/Organization code (field #458 in file 450) does not
resolve with a human readable description in file 454.1 will cause this 
bulletin to be sent.
MESSAGE 
An incomplete record was found while PAID user, |2| was running the
|3| report for employee, |4|.  
The missing data is a description in the PAID code file for 
the cost center/organization code |1|.  This needs to be updated.  
Correct the problem through the Update PAID Codes option. First enter 
the code |1| at the COST CENTER/ORGANIZATION CODE prompt.  Then 
update the description field.  An example follows.

     Select OPTION NAME: PRSD UPDATE PAID CODES
     Select FILE: COST CENTER/ORGANIZATION
     Select COST CENTER/ORGANIZATION CODE: |1|
     CODE:  |1|
     DESCRIPTION: <enter description here>
PARAMETER
  1. PARAMETER:   1
    DESCRIPTION:   
    This is the employees cost center organization code.  It's located in file
450 (PAID EMPLOYEE FILE) field #458.
  2. PARAMETER:   2
    DESCRIPTION:   
    This is the person who is using the option to print employee leave reports.
  3. PARAMETER:   3
    DESCRIPTION:   
    This is the report that was being run, either Employee Leave Used 
or Employee Leave Pattern.
  4. PARAMETER:   4
    DESCRIPTION:   
    This is the employee who is being reported on.  They have a Cost 
Center/Organization code that doesn't have a description in file 454.1.
