XQSERVER (8)    BULLETIN (3.6)

Name Value
NAME XQSERVER
SUBJECT Server Request Notice
DESCRIPTION 
This is the standard, or default, bulletin used by the menu system
to notify IRM staff and users about server request events.
MESSAGE 
                 |1|
  
A request for execution of a server option has been received.
  
Sender: |2|
Option name: |3|
Subject: |4|
Message #: |5|
  
Comments: |6|
   
This is the bulletin named XQSERVER.
PARAMETER
  1. PARAMETER:   1
    DESCRIPTION:   
    The date and time in human-readable form when the server request
was received.
  2. PARAMETER:   2
    DESCRIPTION:   
    The name of the sender of the server request.
  3. PARAMETER:   3
    DESCRIPTION:   
    The name of the option which was requested by Mailman.
  4. PARAMETER:   4
    DESCRIPTION:   
    The subject of the message which requested a server.
  5. PARAMETER:   5
    DESCRIPTION:   
    The internal number of the message requesting a server.
  6. PARAMETER:   6
    DESCRIPTION:   
    Comments appended to the bulleting.  These may include errors trapped
by the server software and/or the operating system, as well as general
purpose messages.
