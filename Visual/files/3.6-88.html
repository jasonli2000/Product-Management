NAME QAM MONITOR TOOL 1
SUBJECT |1|: |2|
DESCRIPTION 
This bulletin is automatically sent when:
 1) The threshold is met or exceeded.
 2) The minimum sample size is met or exceeded.
 3) The end of the time frame has been reached.
Each of these conditions may be turned on or off in the QA MONITOR
file (#743).
