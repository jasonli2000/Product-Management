SC CWT EXTRACT MONITOR (143)    MAIL GROUP (3.8)

Name Value
NAME SC CWT EXTRACT MONITOR
DESCRIPTION 
If defined, members of this mail group will receive the messages which
contain the raw data sent to Austin as a part of the Clinic Wait Time
extract.  It facilitates the occasional monotoring of this process if
indicated.
 
This mail group should not normally have members defined.  The presence of
defined members will result in extract messages being stored in the local
MESSAGE (#3.9) file.  Storage of these messages locally will not occur if
there are no local recipients.
TYPE public
COORDINATOR USER,ONE
