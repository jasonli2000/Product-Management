NAME Q-UPN.DOMAIN.EXT
FLAGS C
RELAY DOMAIN FOC-AUSTIN.DOMAIN.EXT
NOTES 
THIS DOMAIN IS USED AS A QUEUE FOR THE TRANSMISSION OF REQUESTS FOR
ADDITIONAL UNIVERSAL PRODUCT NUMBERS TO AUSTIN. OA&MM WILL MAINTAIN
A CENTRALIZED COPY OF THE UPN FILE FOR USE BY ALL SITES. THIS DOMAIN
RECEIVES MAIL MESSAGES FROM SITES WHEN THEY SCAN UPNS THAT AREN'T 
IN THE CURRENT COPY. IT WILL ALSO BE THE SOURCE OF PERIODIC UPDATES
OF THIS FILE. UPDATES WILL BE TRANSMITTED FROM AAC TO ALL SITES.
