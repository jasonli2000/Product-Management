2310 (2310)    FEDERAL SUPPLY CLASSIFICATION (441.2)

Name Value
NUMBER 2310
ABBREVIATED DESCRIPTION PASSENGER MOTOR VEHICLES
DESCRIPTION 
PASSENGER MOTOR VEHICLES.  **NOTE**  THIS CLASS INCLUDES ONLY COMPLETE
PASSENGER MOTOR VEHICLES, AND CHASSIS THEREFORE,  ANY END ITEMS, ASSEMBLIES,
PARTS, ATTACHMENTS, OR ACCESSORIES, OTHER THAN COMPLETE CHASSIS, FOR 
USE IN OR ON PASSENGER MOTOR VEHICLES ARE CLASSIFIED IN CLASSES OTHER THAN
THIS CLASS.
INCLUDES:  CARS, MOTOR, HIGHWAY; BUSES; CHASSIS, PASSENGER MOTOR VEHICLE;
HEARSES; AMBULANCES.
EXCLUDES:  AMPHIBIOUS PERSONNEL CARRIERS; ALL TRUCK CHASSIS.
GROUP 23
