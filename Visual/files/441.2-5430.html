NUMBER 5430
ABBREVIATED DESCRIPTION STORAGE TANKS
DESCRIPTION 
NOTE: TANKS WHICH ARE DESIGNED FOR INTEGRAL USE WITH OTHER EQUIPMENT WITHIN A 
COMMON HOUSING OR ON A COMMON BASE ARE EXCLUDED FROM THIS CLASS AND SHOULD
BE CLASSIFIED WITH THE EQUIPMENT WITH WHICH USED.
INCLUDES ASSEMBLED AND UNASSEMBLED TANKS FOR STORAGE ONLY; ENCLOSURES FOR 
PRESSURE AND VACUUM TANKS; PETROLEUM BULK STORAGE TANKS.
EXCLUDES BOILER DRUMS; DOMESTIC WATER STORAGE TANKS.
