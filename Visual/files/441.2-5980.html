5980 (5980)    FEDERAL SUPPLY CLASSIFICATION (441.2)

Name Value
NUMBER 5980
ABBREVIATED DESCRIPTION Optoelectronic Devices
DESCRIPTION 
Optoelectronic Devices and Associated Hardware and assemblies which
display numeric, alphanumberic, symbolic, or graphic information, emitters,
and nondisplay optoelectronic devices, such as couplers, detectors, and
switches which perform an active function in an electronic system.
