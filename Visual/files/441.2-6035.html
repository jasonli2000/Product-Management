6035 (6035)    FEDERAL SUPPLY CLASSIFICATION (441.2)

Name Value
NUMBER 6035
ABBREVIATED DESCRIPTION FIBER OPTIC LIGHT AND IMAGE DEVICES
DESCRIPTION 
FIBER OPTIC LIGHT TRANSFER AND IMAGE TRANSFER DEVICES. INCLUDED ARE 
FIBER OPTIC DEVICES USED FOR ILLUMINATION AND/OR IMAGE TRANSMISSION.
EXCLUDED FROM THIS CLASS ARE ITEMS USED FOR COMMUNICATION AND/OR DATA
TRANMISSION.
