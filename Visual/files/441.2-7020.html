7020 (7020)    FEDERAL SUPPLY CLASSIFICATION (441.2)

Name Value
NUMBER 7020
ABBREVIATED DESCRIPTION ADP, CPU, ANALOG
DESCRIPTION 
NOTE:  AN ANALOG IS A REPRESENTATION IN ONE FORM OF A PHYSICAL
CONDITION EXISTING IN ANOTHER FORM (E.G., THE LEVEL OF MERCURY
IN A TUBE REPRESENTS TEMPERATURE IN A THERMOMETER; THE ANGULAR
POSITION OF A NEEDLE REPRESENTS SPEED ON A SPEEDOMETER).  THIS
CLASS INCLUDES ONLY CPU'S THAT ACCEPT AS INPUTS THE ELECTRICAL
EQUIVALENT OF PHYSICAL CONDITIONS SUCH AS FLOW, TEMPERATURE, 
PRESSURE, ANGULAR POSITION OR VOLTAGE AND PERFORM COMPUTATIONS
BY MANIPULATING THESE ELECTRICAL EQUIVALENTS TO PRODUCE RESULTS 
FOR FURTHER USE.
