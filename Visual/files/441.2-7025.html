NUMBER 7025
ABBREVIATED DESCRIPTION ADP INPUT/OUTPUT & STRGE DEV.
DESCRIPTION 
NOT:  THIS CLASS INCLUDES DEVICES USED TO CONTROL AND TRANSFER
INFORMATION TO AND FROM A CPU.  THE INPUT DEVICE IS USED FOR 
TRANSFERRING DATA AND INSTRUCTIONS INTO A CPU.  THE OUTPUT 
DEVICE IS USED TO TRANSFER RESULTS OR PROCESSING BY THE CPU ON
TO PRINTED FORMS, PUNCHED CARDS, MAGNETIC MEDIA AND/OR COM
(COMPUTER OUTPUT TO MICROFORM).  INPUT/OUTPUT DEVICES COMBINE
THE ABOVE FUNCTIONS IN THE SAME DEVICE.  THIS CLASS ALSO INCLUDES DATA 
TRANSMISSION TERMINALS, BATCH TERMINALS, DISPLAY TERMINALS, MODEMS,
SENSORS, MULTIPLEXERS, AND CONCENTRATORS, WHICH ARE SPECIALLY 
DESIGNED OR MODIFIED TO BE USED IN CONJUNCTION WITH DIGITAL, ANALOG
OR HYBRID CPU'S.  IT ALSO INCLUDES STORAGE DEVICES WHICH ARE
DESIGNED TO BE CABLE CONNECTED FOR USE ON LINE IN WHICH DATA CAN 
BE INSERTED, RETAINED, AND RETRIEVED FOR LATER USE; AND 
FIRMWARE.
