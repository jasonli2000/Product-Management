7042 (7042)    FEDERAL SUPPLY CLASSIFICATION (441.2)

Name Value
NUMBER 7042
ABBREVIATED DESCRIPTION MINI AND MICRO CONTROL DEV.
DESCRIPTION 
INCLUDES MINI OR MICRO COMPUTERS USED AS CONTROL MECHANISMS
WHERE COMPUTER TECHNOLOGY IS ESSENTIAL IN CONTROLLING, 
MONITORING, MEASURING, AND DIRECTING PROCESSES, DEVICES, 
INSTRUMENTS OR OTHER EQUIPMENT.
EXCLUDES MINI AND MICRO COMPUTER CONTROL DEVICES WHICH ARE
DESIGNED SPECIFICALLY FOR USE IN AND/OR INTEGRAL TO HIGHER
ORDER SYSTEMS, E.G., AIRCRAFT FIRE CONTROL SYSTEMS, NUMERICALLY
CONTROLLED MACHINE TOOLS, SEQUENCE CONTROLLED PRINTING
EQUIPMENT, MOTION MEASURING INSTRUMENTS, OFFICE INFORMATION
SYSTEM EQUIPMENT; AND THE LIKE.
GROUP 70
COMMODITY CODE General
DLA ADDRESS DLA/DASS
