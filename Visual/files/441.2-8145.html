8145 (8145)    FEDERAL SUPPLY CLASSIFICATION (441.2)

Name Value
NUMBER 8145
ABBREVIATED DESCRIPTION SPECIAL SHIPPING CONT.
DESCRIPTION 
NOTE:  THIS CLASS INCLUDES ONLY REUSABLE AND REPAIRABLE CONTAINERS
SPECIALLY DESIGNED FOR SHIPPING AND STORAGE OF SPECIALIZED 
EQUIPMENT; I.E. SHIPPING AND STORAGE CONTAINERS FOR COMPONENTS
OF AIRCRAFT, SPACE VEHICLES, AUTOMOTIVE VEHICLES, SHIPS, GROUND 
COMMUNICATIONS EQUIPMENT, ETC.
INCLUDES SPECIALLY DESIGNED COMPONENTS ( NOT ELSEWHERE
CLASSIFIABLE) PECULIAR TO SPECIAL SHIPPING AND STORAGE
CONTAINERS AS DELIMITED UNDER THIS CLASS.
EXCLUDES GENERAL PURPOSE CONTAINERS; SPECIALLY DESIGNED 
CONTAINERS FOR AMMUNITION, NUCLEAR ORDNANCE, EXPLOSIVES, 
MILITARY CHEMICAL AGENTS AND GUIDED MISSILE COMPONENTS.
EXCLUDED FROM THIS CLASS ARE ITEMS FOR WHICH MORE SPECIFIC
CLASSIFICATIONS ARE SUITABLE.  THE FSC INDEXES AND STRUCTURE
WILL GOVERN THE CLASSIFICATION OF THOSE ITEMS PERMITTED
CLASSIFICATION IN A SINGLE CLASS ONLY.
