8325 (8325)    FEDERAL SUPPLY CLASSIFICATION (441.2)

Name Value
NUMBER 8325
ABBREVIATED DESCRIPTION FUR
DESCRIPTION 
NOTE:  THIS CLASS INCLUDES ONLY FUR MATERIALS IN BULK.
EXCLUDED FROM THIS CLASS ARE ITEMS THAT ARE PRECUT OR
PREFORMED, THUS LIMITING THEIR USEFULNESS TO SPECIFIC 
APPLICATIONS.  SUCH ITEMS ARE CLASSIFIED IN THE SAME CLASS
AS THE ITEM OR ITEMS FOR WHICH DESIGNED, OR IN SUCH OTHER
CLASS AS IS APPROPRIATE.
INCLUDES CROSSES; SACS; FUR PLATES; FUR MATS.
