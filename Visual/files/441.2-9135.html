9135 (9135)    FEDERAL SUPPLY CLASSIFICATION (441.2)

Name Value
NUMBER 9135
ABBREVIATED DESCRIPTION LQD PROPELLANT, CHEM BASE
DESCRIPTION 
NOTE;  THIS CLASS INCLUDES LIQUID PROPELLANTS WITH MORE
THAN 50% OR HIGHER CHEMICAL CONTENT.
INCLUDES LIQUID CHEMICALS DESIGNED EXCLUSIVELY FOR USE AS
PROPELLANT FUELS AND OXIDIZERS; LIQUID PROPELLANTS, BULK;
LIQUID PROPELLANTS, PREDETERMINED TO SPECIFIC QUANTITY AND
QUALITY, PACKAGED IN REUSABLE CONTAINERS.
EXCLUDES NULTIPLE-USE CHEMICALS, SUCH AS NITROGEN, TECHNICAL,
AMMONIA, TECHNICAL; FURFURYL ALCOHOL, TECHNICAL; ANILINE, 
TECHNICAL; SOLID PROPELLANTS; LIQUID PROPELLANTS, ENCASED
IN CONSUMABLE CONTAINERS, FOR INERTION INTO MISSILE PROPULSION
SYSTEMS AS INTEGRAL PARTS.
GROUP 91
COMMODITY CODE General
