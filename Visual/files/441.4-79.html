Name Value
CODE 2P
SCREENING CODE ADVICE CODE
DESCRIPTION ITEM REQUIRED IN ONE CONTINUOUS LENGTH
LONG DESCRIPTION 
Item required in one continuous length as expressed in cc 25-29 and
  the unit of issue in cc 23-24.  If requirement exceeds the unit pack
  length, multiples of the unit pack are acceptable.
