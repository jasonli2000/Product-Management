HCT (4)    DELTA CHECKS (62.1)

Name Value
NAME HCT
XECUTABLE CODE X ^LAB(62.1,1,1) Q:'$D(LRHGB)!'$D(X) S X1=3*LRHGB K LRHGB IF (X<(X1-2))!(X>(X1+2)) W !,*7,"==> CHECK HCT AND HGB VALUES <==",!
DESCRIPTION 
Checks if the hematocrit value is 3 times the hemoglobin + or - 2.
Could be used on the test HCT (must also have the HGB delta check setup
on the test HGB).  Will not be checked if HGB delta check was not done
first (ie. HGB should be before HCT in the CBC panel) or if no value
is entered for HCT.  ABS VALUE delta check is also done.
