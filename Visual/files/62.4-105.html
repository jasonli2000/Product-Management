INTERFACE NOTES 
ASTRA 4,6,8/8E,IDEAL have the same output format.
The program, LASTRA, must have consecutive entries in the CHEM TEST fields
and will not permit the processing of calculated results (ANION GAP, etc).
The program, LAASTRA, allows calculated results.  Param 3 requires the
chemistry code (01A for Na, 06A for Glucose, etc) or
description (01 for Anion Gap, 03 for Osmolality, etc.).
