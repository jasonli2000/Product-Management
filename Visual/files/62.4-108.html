INTERFACE NOTES 
WBC count is internal numbers 1 - 27.
There can be 3 rows of keys displayed, location is by internal number.
ROW 1 (1 - 9)
ROW 2 (10 - 18)
ROW 3 (19 - 27)
PARAM 1 can change the value of V.
The keys '-' (MINUS), '!' (Comment), '?' (show values) are reserved functions.
PARAM 2 is set to 1 if the test is included in the total count, or to 2
if not included in the total count e.g. NRBC set to 2, SEGS set to 1.
PARAM 3 stores the character on the keyboard used for entry.
RBC MORPHOLOGY is internal numbers 31 - 58, layed out the same as the WBC
 (31 - 39), (40 - 48), (49 - 58), The '\' is used to display the WBC values.
