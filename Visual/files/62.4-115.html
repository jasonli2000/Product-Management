INTERFACE NOTES 
These are notes for the American dade PARAMAX.
The test are NUMBERED in the file with the same number that the 
paramax uses to indcate the test. (ie. 1=albumin, 36=chloride).
Cross link by IDE for manually entered accession numbers, or ID if
you are matching the barcode # to the accession #.
If you are matching the last 3 numbers of the barcode to the accession
number use ID,LOG=+$E(ID,$L(ID)-2,$L(ID))
