INTERFACE NOTES 
Uses Param 3 to identify test, which must match the test name that is
transmitted.  Param 3 should not contatin any spaces.  Exp. BILI D
should be BILID in Param 3.  The program, LASPEC, is used by Abbott's
software version M2.0.
 
Param 2 is used to determine the number of decimal places the test
will be reported with.  No entry in Param 2 will round off to a whole
number.  Example:  NA with Param 2 blank or Param 2 as 0 will report
143.7000 as 144.  K with Param 2 as 1 will report 3.6800 as 3.7.
 
The accession number may be entered in the sample ID field along with
the patient's name.  Example: 132  or DOE, JOHN 132.
The instrument may also be run by tray/cup.
