INTERFACE NOTES 
The program LATDX1 is used with TDX instruments that have V10.1 of
Abbott's software which allows the input of a specimen identifier.
This allows running the instrument to be run by accession or by
a load list.  Param 3 should include the name of the test with no
spaces.  Example: VALPOIC ACID should be VALPOICACID.
Param 2 is used to determine the number of decimal places the result
will be reported as.  If Param 2 is not entered or is 0 the result
will be reported as a whole number, if 1 is entered the result will
have 1 decimal place, etc.
