INTERFACE NOTES 
This instrument only transmits at 110 baud rate and since the LSI may only
be configured down to 150 baud rate, a modification must be made to the
hardware.  The internal number of the test in the auto instrument file must
match the test id #.  Example: PT - entry 1, PTT - entry 3.
