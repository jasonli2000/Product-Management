INTERFACE NOTES 
Coulter S+4,S+5,S+6 all may use the program, LACOLT5.
Instruments using DT WITH DH INTERFACE and Coulter S+ Jr. (the format 
in ^LA will not have a ',' between the test name and the results) use  
LACOLT6. 
Parameters will depend on wether a patient id # is transmitted
(this is controlled by a dip switch setting inside the Coulter's CRT).
If a patient id is transfered, WBC will have for Param 1:  S V=Y(5)
and should be cross linked by IDE.
If a patient id is not transfered, Param 1:  S V=Y(4)  ie. the subscript
value will be 1 less, and should be cross linked by ID.
The ticket flag transmission mode should be set to OFF on the instrument.
 
The COUTLER JT3 will run with either the LACOLT5 or the LACOLT6 routines.
If you are not running the EO# and BA# tests you can use the LACOLT5.
If you are running these tests you will need to use LACOLT6 as when
these tests are activated the "," is not used between the test name
and the result.
