INTERFACE NOTES 
The program LAMODU is used with the Modulus Urocomp keypads with or without
the keypad being interfaced to a urinalysis instrument.  Param 1 is used
to extract the results for the test.  The position of the results on the
data stream will be dependent on being interfaced to a Clinitek 200, 2000,
or used as a seperate workstation.
 
Coded comments can be done similar to the Biovation keypads
(explained in the INTERFACE NOTES field).
If Param 3 is not used the default is 'None$Few$Mod$Many$Profuse$Noted'
where a degree of 1 would give the result Few.
 
The field 'LOAD CHEM TESTS' should be set to 'U' if you have many comments.
This will avoid possible store errors.
