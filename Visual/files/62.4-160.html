LSI #1 (160)    AUTO INSTRUMENT (62.4)

Name Value
NAME LSI #1
PROGRAM LAB
NEW DATA S T=1 D SET
RESTART K ^LA(1),^LA("Q")
ALARM TERMINAL
  1. ALARM TERMINAL:   WORK
INTERFACE NOTES 
This is the entry to be used as the system line from the LSI to the CPU.
 
For sites running multiple LSI's for automated instruments, the
following information is needed to set the fields NEW DATA and RESTART
in the auto instrument file.  Let * equla 1, 11, 21, 31, etc.
(corresponding to the internal entry number in the auto instrument file
for the LSI).
 
NEW DATA: S T=* D SET
 
RESTART: K ^LA(*),^LA("Q")
 
For all LSI's except for #1 the RESTART field should only be:
RESTART: K ^LA(*)
 
Remember to replace * with the appropriate number in the examples.
