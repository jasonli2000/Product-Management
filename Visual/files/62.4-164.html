The keyboard urinanalysis is divided into 3 screens MAIN, CAST's, CRYSTAL's
The MAIN screen is numbers 1-27
The CAST screen is numbers 31-57
The CRYSTAL screen is numbers 61-87
To enter the value for a given test press the key assoiated with PARAM 3.
This part is the same a the second part of the keyboard diff.
