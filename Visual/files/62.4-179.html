INTERFACE NOTES 
This instrument setup will work for both unidirectional and bi directional
with the routine LAH717U.  To run bidirectional you have to fill in the
fields: HANDSHAKE RESPONSE with "D ^LAH717H" (miss the quotes)
        FILE BUILD ROUTINE: with LAH717D
 
****NOTE**** to run bidirectional the serial interface board must have
switch block 5 dip switch in position 2 (bit 1) turned on to shut off
sending the BCC.  If this is not done the instrument handshake routine
will not work.
FILE BUILD ROUTINE LAH717D
SEND TRAY/CUP LOCATION no
QUEUE BUILD no
