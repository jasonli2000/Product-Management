INTERFACE NOTES 
Setup of the MicroScan: There is a special version of the interface
code in the MicroScan to work with the VA LSI.
Set the following paramaters on the MicroScan:
Timeout >20 sec, String delimiter to zero (No delimiter),
Field delimiter to | (ascii 124), Protocall to NO ACK/NAK.
You may want to use the Single Therapy reporting.
                                      
In the Auto Instrument file: To setup for Bi-directional operation
set the HANDSHAKE field to D ^LAMSP to handle the ack handshaking.
There is only one MICRO card type and it's internal number must be
1.  The 'ORGANISM' subfile is used to look up the organism number
by the 'Card Code For Organism field'. The 'DRUG' subfile is used to
lookup the Drug abbreviation by the 'Card Code' field.
At the time the PARAM1 mumps code is executed the variable V
is set to the MIC value "^" the NCCLS code.  All the "M" record fields
are in Y(field number).  There is a subroutine DGT that will load the
MicroScan Dosage Guide Text into the COMMENT field preceded by the Drug
name.
 
X-ref to build in the MicroScan:
 The specimen information is downloaded from the VA to the MicroScan
using the abbreviation from the TOPOGRAPHY file.
The provider is sent down using the initals from the PROVIDER file.
The location is sent as found in the ACCESSION file, and service
is sent if found in the ACCESSION file.
 
Use the LAMILL routine to add accessions to the load list and enter the
panel TEST number to be done (these numbers are provided in the MicroScan
documentation), and the ISOLATE number if known. If the ISOLATE number is in
question leave the isolate blank and the MicroScan will assign.
This allows the TECH to only load the MicroScan with accession
that are prepared for the MicroScan.
 
NOTE: The MicroScan must have one of its console's active in the
interface routine before starting any downloads.
If a download is started and the MicroScan isn't in the right
mode, Then on the VA side the MicroScan instrument will have to
be restarted to clear the LA global and then the download
started again. This procedure can easily by done by using the 'Restart
Processing of Instrument data' option [LRJOB] located in the Lab Interface
Menu.
 
To use the MicroScan in a upload only mode clear the HANDSHAKE
field from the auto instrument file and set the MicroScan protocall
to XON only.
