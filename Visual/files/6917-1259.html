CARDIAC CATHETERIZATION LABORATORY DATA SYSTEM. A COMPUTER-AIDED SYSTEM
DESIGNED TO PROVIDE THE CLINICAL CATHETERIZATION LAB WITH METHOD FOR
ACQUIRING, REDUCING, PRESENTING, AND STORING PATIENT DATA. THE SYSTEM
PERFORMS A COMPLETE SPECTRUM OF HEMODYNAMIC CALCULATIONS AND PRESENTS A
WRITTEN REPORT AT THE END OF THE PROCEDURE. MAY OR MAY NOT INCLUDE A
RECORDER, WHICH IS USED TO INPUT THREE PRESSURE CHANNELS. UPON
COMPLETION OF THE PROCEDURE, ONE OR MORE COM PLETE REPORTS CAN BE
PRINTED, THE DATA IN THE REPORT IS STORED IN THE DISC MEMORY OF THE
COMPUTER. SYSTEM CAN HANDLE DATA ANALYSIS FROM UP TO FOUR CATH LABS
