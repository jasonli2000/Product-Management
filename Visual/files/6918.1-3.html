ALARM (3)    UTILITY SYSTEMS (6918.1)

Name Value
NAME ALARM
FULL NAME ALARM SYSTEMS
DESCRIPTION 
Systems that monitor a condition status and provide an audio/visual
notification of any change in that status that requires action. Examples
could be found in the following areas:
 
       Hazardous Materials
       X-Rays and Nuclear Radiation
       Biomedical Hazards
       Operating Room Gases
       Infectious Disease Controls
       Biohazard Detection and Control
       Gas Detection Alarms (Carbon Monoxide, Radon, Ozone, etc.)
