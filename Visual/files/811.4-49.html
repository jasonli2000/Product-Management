VA-COMBAT VET ELIGIBILITY (49)    REMINDER COMPUTED FINDINGS (811.4)

Name Value
NAME VA-COMBAT VET ELIGIBILITY
ROUTINE PXRMMSER
ENTRY POINT CVELIG
PRINT NAME VA-Combat Vet Eligibility
DESCRIPTION 

This computed finding will be true if the patient has ever qualified or
currently qualifies as a combat vet.
 
The possible values that can be used in a Condition are: "ELIBIGLE"
and "EXPIRED". "EXPIRED" will be returned if the veteran was not
eligible on the evaluation date. An example is:
 
I V="EXPIRED"
TYPE MULTIPLE
CLASS NATIONAL
