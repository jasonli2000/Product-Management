VA-SERVICE SEPARATION DATES (578006)    REMINDER COMPUTED FINDINGS (811.4)

Name Value
NAME VA-SERVICE SEPARATION DATES
ROUTINE PXRMMSER
ENTRY POINT DISCHDT
PRINT NAME VA-Service Separation Date(s)
DESCRIPTION 
This multiple occurrence computed finding returns service separation
information. The date of the finding is the separation date.
 
CSUB data that can be used in a Condition are: 
  "BRANCH" 
  "DISCHARGE TYPE" 
  "ENTRY DATE" 
  "SEPARATION DATE" 
  "SERVICE COMPONENT" 
  
 The default value is the separation date.
TYPE MULTIPLE
CLASS NATIONAL
