NAME VA-WH PAP SMEAR DONE
DATE CREATED 2003-07-17 00:00:00
DESCRIPTION 
No mapping necessary.  Use the new VA-WH PAP SMEAR DONE CODES 
taxonomy distributed with this reminder term. This taxonomy is 
similar to the VA-CERVICAL CANCER SCREEN taxonomy distributed to 
the field with the first reminder package distribution. The new
taxonomy should be used, instead of the VA-CERVICAL CANCER SCREEN
taxonomy, because it does not include codes such as Q0091
which represent PAP smears obtained by the clinician.
 
Use the new WH PAP SMEAR OUTSIDE health factor to document PAP
Smear results completed outside the VA.
