VA-LDL >119 (33)    REMINDER TERM (811.5)

Name Value
NAME VA-LDL >119
DATE CREATED 2001-11-28 00:00:00
DESCRIPTION 
This national reminder term represents laboratory package test results
where the LDL result is greater than or equal to 120 mg/dl. Add the 
CONDITION (e.g., I V>119) to the reminder term CONDITION field for each
finding you add.
CLASS NATIONAL
SPONSOR Office of Quality & Performance
