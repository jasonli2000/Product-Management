Name Value
NAME VA-WH PAP SMEAR ORDER HEALTH FACTOR
DATE CREATED 2003-07-17 00:00:00
DESCRIPTION 
The following health factors are distributed with this term.
    WH ORDER PAP SMEAR SCREEN HF (for use when an order menu with 
                                  a PAP Smear Screen order is used
                                  from the reminder dialog)
    WH ORDER REFER WH PROVIDER GYN CARE HF (for use when a quick order
                                  is used for WH PROVIDER GYN CARE
                                  referal in reminder dialogs)
    WH ORDER REFER GYNECOLOGIST HF (for use when a quick order      
                                  is used for GYNECOLOGY referal
                                  in reminder dialogs)
    WH ORDER REPEAT PAP HF        (for use if you want to track
                                  repeat PAP orders)
 
This reminder term represents the action taken from the dialog to
indicate the clinician selected an element that could generate an
order for PAP Smear. The health factor date will be used to
calculate a short resolution frequency, instead of using the
Order's Start date.
