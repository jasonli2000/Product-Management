NAME VA-WH HX CERVICAL CANCER/ABNORMAL PAP
DATE CREATED 2003-07-21 00:00:00
DESCRIPTION 
This reminder term is mapped to the taxonomy VA-CERVICAL CA/ABNORMAL PAP 
CODES findings. This term represents ICD9, ICD0, and CPT codes used to 
represent the patient has cervical cancer or a diagnosis for abnormal PAP.
 
This reminder term is also mapped to the computed finding VA-WH PAP 
ABNORMAL IN WH PKG.  If the Women's Health package is being used to
document PAP Smear results, this computed finding will find the most
recent PAP Smear entry that has an "Abnormal" result.

  2. FINDING ITEM:   VA-WH PAP SMEAR ABNORMAL IN WH PKG
