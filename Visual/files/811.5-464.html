NAME VA-WH HX BREAST CANCER/ABNORMAL MAM
DATE CREATED 2003-07-28 00:00:00
DESCRIPTION 
Use the previously distributed VA-BREAST TUMOR and VA-MASTECTOMY
taxonomy to indicate the patient has a history of breast cancer or
has had an abnormal Mammogram.
 
Use the new computed finding VA-WH MAMMOGRAM ABNORMAL IN WH PKG if 
the Women's Health package is used to document mammogram screen results.  
Mammogram Screen results will either be documented as "Pending", "Normal",
"Abnormal", or "Unsatisfactory".

  2. FINDING ITEM:   VA-MASTECTOMY
