VA-WH PAP SMEAR UNSATISFACTORY IN LAB/WH PKG (486)    REMINDER TERM (811.5)

Name Value
NAME VA-WH PAP SMEAR UNSATISFACTORY IN LAB/WH PKG
DATE CREATED 2003-10-24 00:00:00
DESCRIPTION 
No mapping necessary.  This term represents PAP Smear results documented
in the Laboratory or Women's Health packages. Use the new VA-WH PAP SMEAR
IN LAB PKG or VA-WH PAP SMEAR IN WH PKG computed findings distributed with
this reminder term. These computed finding looks for PAP smear results in
the Laboratory or Women's Health packages that have a result status 
representing an "Unsatisfactory" result.
 
This computed finding will only work if the Women's Health WV PROCEDURE 
TYPE file entry for PAP SMEAR has SNOMED TOPOGRAPHY and MORPHOLOGY 
codes defined.  Each code in the Women's Health WV PROCEDURE TYPE
file must have a result status defined that indicates the code either
represents a "Normal", "Abnormal" or "Unsatisfactory" result.
FINDINGS
  1. FINDING ITEM:   VA-WH PAP SMEAR IN LAB PKG
    CONDITION:   I V("UNSATISFACTORY")="T"
    INTERNAL CONDITION:   I $G(V("UNSATISFACTORY"))="T"
    V SUBSCRIPT LIST:   "UNSATISFACTORY";
  2. FINDING ITEM:   VA-WH PAP SMEAR IN WH PKG
    CONDITION:   I (V("VALUE")="Unsatisfactory")
    CONDITION CASE SENSITIVE:   NO
    INTERNAL CONDITION:   I ($G(V("VALUE"))="UNSATISFACTORY")
    V SUBSCRIPT LIST:   "VALUE";
