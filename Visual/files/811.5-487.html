VA-WH MAMMOGRAM UNSATISFACTORY IN RAD/WH PKG (487)    REMINDER TERM (811.5)

Name Value
NAME VA-WH MAMMOGRAM UNSATISFACTORY IN RAD/WH PKG
DATE CREATED 2003-07-28 00:00:00
DESCRIPTION 
This term needs to be mapped to the following Radiology procedures:
        MAMMOGRAM BILAT
        MAMMOGRAM BILAT SCREEN
        MAMMOGRAM UNILAT
 
A condition must be defined to check for unsatisfactory results.
FINDINGS
  1. FINDING ITEM:   MAMMOGRAM BILAT
    CONDITION:   I V("PDX")["UNSATISFACTORY"&(V("SDX","*")["UNSATISFACTORY")
    INTERNAL CONDITION:   I $G(V("PDX"))["UNSATISFACTORY"&($G(V("SDX","*"))["UNSATISFACTORY")
    V SUBSCRIPT LIST:   "PDX";"SDX","*";
  2. FINDING ITEM:   MAMMOGRAM UNILAT
    CONDITION:   I V("PDX")["UNSATISFACTORY"&(V("SDX","*")["UNSATISFACTORY")
    INTERNAL CONDITION:   I $G(V("PDX"))["UNSATISFACTORY"&($G(V("SDX","*"))["UNSATISFACTORY")
    V SUBSCRIPT LIST:   "PDX";"SDX","*";
  3. FINDING ITEM:   VA-WH MAMMOGRAM IN WH PKG
    CONDITION:   I (V("VALUE")="Unsatisfactory")
    CONDITION CASE SENSITIVE:   NO
    INTERNAL CONDITION:   I ($G(V("VALUE"))="UNSATISFACTORY")
    V SUBSCRIPT LIST:   "VALUE";
