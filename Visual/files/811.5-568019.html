VA-ACTIVE DUTY (568019)    REMINDER TERM (811.5)

Name Value
NAME VA-ACTIVE DUTY
DESCRIPTION 
This reminder term is included in the OEF/OIF Post-Deployment Reminder 
and in the TBI Screening Reminder in order to allow sites to screen 
active duty patients if a site so chooses.  The computed finding 
VA-PATIENT TYPE can be included in this term if a site makes a decision 
to screen active duty patients.  The CF should be added to this term with 
a condition of I V="ACTIVE DUTY".
 
If a patient is listed as active duty in the TYPE field in the patient 
file (field 391), then this finding will be true and the reminder will be 
due.
 
If a site chooses not to screen active duty personnel, then this CF 
should be left out of or removed from the reminder term.  In this 
situation, the reminder term should be empty.
 
This term was exported empty in patch PXRM*2*11 since a very limited 
number of sites screen active duty patients.
CLASS NATIONAL
SPONSOR Office of Public Health and Environmental Hazards
