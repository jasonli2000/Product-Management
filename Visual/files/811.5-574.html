Name Value
NAME VA-MHV RETINAL EXAM FOR DIABETES
DESCRIPTION 
This term represents any finding or code that demonstrates that a patient 
has had a dilated retinal examination looking for diabetic retinopathy.  
A taxonomy is exported with this term but sites may want to consider 
removing this taxonomy if the ICD codes being used by the Eye clinic at a
facility do not match what is included in the taxonomy.  Add any other 
findings (HFs or exams) that represent dilated retinal exams.
FINDINGS
  1. FINDING ITEM:   VA-MHV RETINAL EXAMINATION
  2. FINDING ITEM:   DIABETIC TELERETINAL EYE EXAM
