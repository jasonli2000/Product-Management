NAME VA-ORDER LIPID PROFILE HEALTH FACTOR
DATE CREATED 2002-01-31 00:00:00
DESCRIPTION 
Use the health factor distributed with this term or add any local health 
factor representing the order action.  Do not add orderable items to this 
reminder term (see LIPID PROFILE ORDERABLE).  This represents the date the
order was placed, not the date the order will be done in the future.  The
order placement will cause the reminder to be resolved for 1 month.  
(Alternativley, Copy this reminder and add LIPID PROFILE ORDERABLE to the 
resolution findings if you want the next due date to be calculated based 
on the future date the order is to be done.)
