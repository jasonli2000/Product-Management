Name Value
NAME VA-MH ACUTE ILLNESS
DESCRIPTION 
Some screening tests may not be possible in patients with acute illnesses.
This term represents any data element that is used to indicate that the
patient has an acute illness that prevents routine screening 
instruments from being administered to the patient.  E.g. delirium,
alcohol hallucinosis, florid psychosis, MI's and other medical
emergencies.
