VA-IMAGING FOR AAA (RAD NON-SPECIFIC PROC) (66)    REMINDER TERM (811.5)

Name Value
NAME VA-IMAGING FOR AAA (RAD NON-SPECIFIC PROC)
DESCRIPTION 
This term should contain any abdominal imaging that might have been done 
for another indication but where the evaluation of the abdominal aorta 
for AAA was completed.  This term should ONLY contain radiology procedure 
(RP) type finding since it uses a condition to find any diagnostic codes 
that represent adequate assessment of the abdominal aorta.
 
The diagnostic codes for AAA screening are included in patch RA*5.0*97 
and include the following:
1200    ABDOMINAL AORTIC ANEURYSM NOT PRESENT
1201    ABDOMINAL AORTIC ANEURYSM PRESENT
1202    DOES NOT SATISFY SCREEN FOR AAA
 
Once those diagnostic codes are in use by Radiology, any procedure that 
you have included in this reminder term will resolve the AAA screening 
reminder if either of the first 2 codes (1200, 1201) are used as the 
primary or secondary code for that procedure report.
