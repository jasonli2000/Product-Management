VA-WH PAP SMEAR UNSATISFACTORY IN LAB PKG (96)    REMINDER TERM (811.5)

Name Value
NAME VA-WH PAP SMEAR UNSATISFACTORY IN LAB PKG
DATE CREATED 2003-10-24 00:00:00
DESCRIPTION 
No mapping necessary.  This term represents PAP Smear results 
documented in the Laboratory package. Use the new VA-WH PAP SMEAR
IN LAB PKG computed finding distributed with this reminder term. This
computed finding looks for PAP smear results in the Laboratory package
that have a result status that represents an "Unsatisfactory"
result.            
 
This computed finding will only work if the Women's Health WV PROCEDURE 
TYPE file entry for PAP SMEAR has SNOMED TOPOGRAPHY and MORPHOLOGY 
codes defined.  Each code in the Women's Health WV PROCEDURE TYPE
file must have a result status defined that indicates the code either
represents a "Normal", "Abnormal" or "Unsatisfactory" result.
FINDINGS
  1. FINDING ITEM:   VA-WH PAP SMEAR IN LAB PKG
    CONDITION:   I V("RESULT STATUS")="Unsatisfactory"
    INTERNAL CONDITION:   I $G(V("RESULT STATUS"))="Unsatisfactory"
    V SUBSCRIPT LIST:   "RESULT STATUS";
