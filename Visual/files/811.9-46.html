NAME VA-PROJECT ARCH VISN CONTRACT CARE PILOT ELIGIBILITY
PRINT NAME Project ARCH Pilot Eligible
DESCRIPTION 
This reminder definition is used to identify patients who are eligible 
for the Access Received Closer to Home (ARCH) pilot program. Project ARCH
is a contract care pilot program designed to improve access to needed
health care through contracted providers that are closer to the Veteran's
residence.
