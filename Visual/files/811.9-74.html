Name Value
NAME VA-BL PNEUMOC RISK IMMUNOCOMPROMISED
PRINT NAME Immunocompromised
DESCRIPTION 
This reminder is used to determine if the patient has a diagnosis, 
chemotherapy or long term steroid use that would indicate an 
immunocompromised condition and the need for PCV13 vaccination.  The 
reminder is used in the PPSV23 reminder to determine the need for a dose 
of PPSV23 after a prior PCV13 dose.
 
Not used in branching logic in the dialogs.  Used in reminder term VA-BL 
PNEUMOC RISK IC/CH which is in the reminder VA-PNEUMOCOCCAL IMMUNIZATION 
PPSV23.
TECHNICAL DESCRIPTION 
copy of VA-PNEUMOCOCCAL IMMUNIZATION PCV13
