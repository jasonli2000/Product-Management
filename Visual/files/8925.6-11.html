DESCRIPTION 
This Status applies to Document Definitions only.  ACTIVE Status 
is intended for Document Definitions when they have been 
tested and are ready for use defining actual clinical documents 
on real patients.  A Document Definition of ACTIVE Status 
is available for use by ANYONE with menu access for entering 
documents (unless document entry for this document definition 
is restricted by parameters to certain users).  ACTIVE 
Status determines whether a Document Definition MAY be used 
for actual documents.  It does not determine whether it IS 
used for actual documents.
