MAGG PAT IMAGES (1000)    REMOTE PROCEDURE (8994)

Name Value
NAME MAGG PAT IMAGES
TAG IMAGES
ROUTINE MAGGTIG
RETURN VALUE TYPE ARRAY
AVAILABILITY RESTRICTED
DESCRIPTION 
This returns the full list of images for a patient.
INPUT PARAMETER
  1. INPUT PARAMETER:   MAGDFN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   20
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is DFN of patient.
RETURN PARAMETER DESCRIPTION 
Returns an array, (Global Array if list gets too big), of all images for
the patient.
Each list entry has image info in the format : 
 - - - - - - 
$P(1^2^3)     IEN^Image FullPath and name^Abstract FullPath and Name
$P(4)           SHORT DESCRIPTION field and desc of offline JukeBox
$P(5)           PROCEDURE/ EXAM DATE/TIME field
$P(6)           OBJECT TYPE
$P(7)           PROCEDURE field
$P(8)          display date
$P(9)          to return the PARENT DATA FILE image pointer
$p(10)        return the ABSTYPE  'M' magnetic 'W' worm  'O' offline
$p(11)        is  'A' accessable   'O' offline
$p(12^13)  Dicom Series Number  $p(12) and   Image Number  $p(13)
$p(14)        is count of images in group, 1 if single image.
