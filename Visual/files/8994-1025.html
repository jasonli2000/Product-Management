NAME MAGG PAT PHOTOS
TAG PHOTOS
ROUTINE MAGGTIG
RETURN VALUE TYPE ARRAY
AVAILABILITY RESTRICTED
DESCRIPTION 
This returns an Image list of Patient Photo iD's in the Image File.
INPUT PARAMETER
  1. INPUT PARAMETER:   MAGDFN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is the DFN of the patient in question.
RETURN PARAMETER DESCRIPTION 
Returns a list of all patient Photo on file in the Image File.
(0)  node  = 1^count of photos returned   - could be zero
           or  0^error message
 each entry in list has Imaging standard Image info format i.e.
 '^' delimited string of image info in the format
 - - - - -
$P(1^2^3)     IEN^Image FullPath and name^Abstract FullPath and Name
$P(4)           SHORT DESCRIPTION field and desc of offline JukeBox

$P(6)           OBJECT TYPE
$P(7)           PROCEDURE field
$P(8)          display date
$P(9)          to return the PARENT DATA FILE image pointer
$p(10)        return the ABSTYPE  'M' magnetic 'W' worm  'O' offline
$p(11)        is  'A' accessable   'O' offline
$p(12^13)  Dicom Series Number  $p(12) and   Image Number  $p(13)
