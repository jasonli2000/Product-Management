ORWCH SAVECOL (1054)    REMOTE PROCEDURE (8994)

Name Value
NAME ORWCH SAVECOL
TAG SAVECOL
ROUTINE ORWCH
RETURN VALUE TYPE SINGLE VALUE
DESCRIPTION 
This RPC saves the column width sizes for reports in CPRS for the user.
INPUT PARAMETER
  1. INPUT PARAMETER:   COL
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   245
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Format: ifn^C1,C2,C3,C4,C5,C6 ...
 
Where ifn is the internal number of the report in file 101.24, and the
2nd '^' piece is the columns separated by commas.
