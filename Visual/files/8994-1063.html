ORWRA IMAGING EXAMS1 (1063)    REMOTE PROCEDURE (8994)

Name Value
NAME ORWRA IMAGING EXAMS1
TAG EXAMS1
ROUTINE ORWRA
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY SUBSCRIPTION
WORD WRAP ON TRUE
DESCRIPTION 
This remote procedure call returns a list on imaging exams for a
specific patient.
INPUT PARAMETER
  1. INPUT PARAMETER:   DFN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   20
    REQUIRED:   YES
    DESCRIPTION:   
    Internal entry number of patient in the Patient file [^DPT].
RETURN PARAMETER DESCRIPTION 
The format of the returned global array is the following:

   Piece        Description         Format
   -----        -----------         ------
     1          Site                SiteName;#
     2          Exam ID:            Inverse FM date/time of Exam_-_Case IEN
     3          Exam date and time  FM date/time format
     4          Procedure name      Free text
     5          Case Number         Number
     6          Status              Free text
     7          Severity            Y or null (ie. Is it an abnormal result?)
