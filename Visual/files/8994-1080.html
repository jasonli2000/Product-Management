DENTV FILE ADMIN TIME (1080)    REMOTE PROCEDURE (8994)

Name Value
NAME DENTV FILE ADMIN TIME
TAG NON
ROUTINE DENTVRP4
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY AGREEMENT
DESCRIPTION 
This will file non-clinical admin time into file 226
INPUT PARAMETER
  1. INPUT PARAMETER:   DATA
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   25
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    DATA = p1^p2^p3^p4 where
  p1 :== duz - pointer to file 200 (also must be in file 220.5)
  p2 :== single character code (A)dmin  (f)ee basis  (R)esearch
         (E)ducation/training
  p3 :== non-clinical admin time hr.min (0.25 - 49.75)
         [in 15 minute increments]
  p4 :== station.division name (from file 225)
RETURN PARAMETER DESCRIPTION 
Returns a string 1^Record successfully added
                -1^various error messages
