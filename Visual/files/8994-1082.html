DENTV GET CATEG/CODES (1082)    REMOTE PROCEDURE (8994)

Name Value
NAME DENTV GET CATEG/CODES
TAG CATC
ROUTINE DENTVRP1
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY AGREEMENT
WORD WRAP ON TRUE
DESCRIPTION 
This works in conjunction with the DENTV GET CATEGORIES rpc.  You pass
the subcategory name and this call will return the array of cpt codes
associated with that subcategory name.
 
end description update 6-23-2003
INPUT PARAMETER
  1. INPUT PARAMETER:   TYP
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is the subcategory name from file 228 that was retrieved by using
the DENTV GET CATEGORIES rpc.
RETURN PARAMETER DESCRIPTION 
An array of cpt codes is returned that have this subcategory (fields 5 or
6 in file 228).  The global array returns data in ^TMP("DENTV",$J).  For 
a descripiton of the data return format see GETADA^DENTVRP1.
