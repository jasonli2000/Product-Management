EC DIEDON (1161)    REMOTE PROCEDURE (8994)

Name Value
NAME EC DIEDON
TAG ECDOD
ROUTINE ECUERPC2
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
WORD WRAP ON TRUE
DESCRIPTION 
This RPC returns a patient's date of death.
INPUT PARAMETER
  1. INPUT PARAMETER:   ECARY
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    DESCRIPTION:   
    The input variable ECARY contains the patient's DFN.
RETURN PARAMETER DESCRIPTION 
The RPC returns either of the following pieces of data as output.
 1. 0^Patient DFN is not defined
 2. DOD internal fileman format^message when patient died
 3. ^ [if patient is not dead]
