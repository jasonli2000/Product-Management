PSB GETIVPAR (1178)    REMOTE PROCEDURE (8994)

Name Value
NAME PSB GETIVPAR
TAG GETPAR
ROUTINE PSBPARIV
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY PUBLIC
WORD WRAP ON TRUE
VERSION 2
DESCRIPTION 
Returns the IV parameters by IV type, as defined for a ward in file 53.66.
INPUT PARAMETER
  1. INPUT PARAMETER:   PSBWARD
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   10
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    The IEN of the ward from file 53.66 or the word 'ALL'.
  2. INPUT PARAMETER:   PSBIVPT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   1
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    The IV type - 'A' for Additive
              'P' for Piggyback
              'H' for Hyperal
              'C' for Chemo
              'S' for Syringe
  3. INPUT PARAMETER:   PSBDIV
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   1
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    ^1=IEN of division as in the Ward Location file (File #42)
RETURN PARAMETER DESCRIPTION 
A delimited string with the IV type and the defined parameters.
1=Warning, 2=No Warning, 3=Invalid.
 
        ^1=IV type
        ^2=Additive
        ^3=Strength
        ^4=Bottle
        ^5=Solution
        ^6=Volume
        ^7=Infusion Rate
        ^8=Med Route
        ^9=Schedule
        ^10=Admin Time
        ^11=Remarks
        ^12=Other Print Info
        ^13=Provider
        ^14=Start Date/Time
        ^15=Stop Date/Time
        ^16=Provider Comments
