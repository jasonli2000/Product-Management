PSB PUTIVPAR (1180)    REMOTE PROCEDURE (8994)

Name Value
NAME PSB PUTIVPAR
TAG PUTPAR
ROUTINE PSBPARIV
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY PUBLIC
WORD WRAP ON TRUE
VERSION 2
DESCRIPTION 
Sets 53.66 or parameters file w/input from 53.66 if ward is not "ALL"
INPUT PARAMETER
  1. INPUT PARAMETER:   PSBWARD
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   1
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    ^1=Ward name
^2=IEN of ward as in file 53.66
  2. INPUT PARAMETER:   PSBPARS
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   1
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    IV type - "A" for Additive
          "P" for Piggyback
          "H" for Hyperal
          "C" for Chemo
          "S" for Syringe
          "S" for Syringe
 
^2-^16 Setting
     1 for Warning
     2 for Non-Verify
     3 for Invalid
  3. INPUT PARAMETER:   PSBDIV
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   1
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    ^1=IEN of division as in the Ward Location file (File #42)
RETURN PARAMETER DESCRIPTION 
The IV type - "A" for Additive
              "P" for Piggyback
              "H" for Hyperal
              "C" for Chemo
              "S" for Syringe
^2=Additive
^3=Strength
^4=Bottle
^5=Solution
^6=Volume
^7=Infusion rate
^8=Med route
^9=Schedule
^10=Admin time
^11=Remarks
^12=Other Print Info
^13=Provider
^14=Start Date/time
^15=Provider comments
