PSB MOB DRUG LIST (1185)    REMOTE PROCEDURE (8994)

Name Value
NAME PSB MOB DRUG LIST
TAG OILST
ROUTINE PSBRPCMO
RETURN VALUE TYPE ARRAY
AVAILABILITY PUBLIC
WORD WRAP ON TRUE
VERSION 2
DESCRIPTION 
Used by the BCMA/CPRS Med Order Button to return an array of drug.
INPUT PARAMETER
  1. INPUT PARAMETER:   PSBSCAN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   40
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    PSBSCAN is the value that has been scanned or entered by the user.
  2. INPUT PARAMETER:   PSBOTYP
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    Type of list:
 
    Dispensed Drugs   = "UD"
 
    IV Medications    = "IV"
 
    Orderable Item    = "OIT"
 
    Drug Class        = "VAC"
RETURN PARAMETER DESCRIPTION 
Results[0] = 1
Results[1] = -1^Invalid Medication Lookup
 
or
 
Results[0] = the number of items passed in the array
Results[1] = ^1 either DD, SOL, ADD
             ^2 IEN from drug file 50
             ^3 Drug Name
             ^4 Pharmacy Orderable Item IEN 
             ^5 Pharmacy Orderable Item Name
             ^6 CPRS Orderable Item IEN
             ^7 CPRS Orderable Item Name
