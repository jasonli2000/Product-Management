MPIF EXT PDAT REMOTE (1248)    REMOTE PROCEDURE (8994)

Name Value
NAME MPIF EXT PDAT REMOTE
TAG PATINFO
ROUTINE MPIFEXT2
RETURN VALUE TYPE ARRAY
AVAILABILITY SUBSCRIPTION
VERSION 1
DESCRIPTION 
Extended PDAT call remote.  ICN or SSN can be passed.
INPUT PARAMETER
  1. INPUT PARAMETER:   ICN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    ICN for patient requesting data on.
  2. INPUT PARAMETER:   SSN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   9
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    SSN for patient data to be returned on.
  3. INPUT PARAMETER:   RPC
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   1
    REQUIRED:   NO
    SEQUENCE NUMBER:   4
  4. INPUT PARAMETER:   EXIST
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   1
    REQUIRED:   NO
    SEQUENCE NUMBER:   5
    DESCRIPTION:   
    If there is no value or 0 is passed, then a new request for data should 
be made.  If there is a 1 passed, then if there is an existing request in 
the ^XTMP("MPIF EXT PDAT",ICN/SSN,SITE) global, then that request should 
be used to return the data -- request had already been made.
