MAG4 CP GET VISITS (1260)    REMOTE PROCEDURE (8994)

Name Value
NAME MAG4 CP GET VISITS
TAG VISITS
ROUTINE MAGGSCP
RETURN VALUE TYPE ARRAY
AVAILABILITY RESTRICTED
DESCRIPTION 
 This RPC call simply call a CP Routine that lists visits for a patient.
 Imaging prompts the user with this list when a Visit String
 is needed by CP to create a Note.
INPUT PARAMETER
  1. INPUT PARAMETER:   MAGDFN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    DFN of Patient
  2. INPUT PARAMETER:   MAGCONS
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
     Consult Number, returned in the Query to get patient CP Requests.
RETURN PARAMETER DESCRIPTION 
Array of Visits for the patient.
