EC GETVISITINFO (1374)    REMOTE PROCEDURE (8994)

Name Value
NAME EC GETVISITINFO
TAG VISINFO
ROUTINE ECUERPC2
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
WORD WRAP ON TRUE
DESCRIPTION 
This broker call returns specific EC patient visit data (location, DSS Unit,
patient ien, etc) based on a Visit Number.
INPUT PARAMETER
  1. INPUT PARAMETER:   ECARY
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    DESCRIPTION:   
    The value ECARY contains the Visit Number IEN (ECVSN), found in file (#9000010).
RETURN PARAMETER DESCRIPTION 
This call returns the following EC patient values separated by an up-arrow (^):-
  1. Location IEN
  2. DSS Unit IEN
  3. Procedure Date/Time Fileman
  4. Procedure Date/Time Human Readable
  5. Patient DFN

or (if error)
  1. 0
  2. Error Message
