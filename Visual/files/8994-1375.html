EC GETPATCH (1375)    REMOTE PROCEDURE (8994)

Name Value
NAME EC GETPATCH
TAG PATCH
ROUTINE ECUURPC
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
DESCRIPTION 
Broker call checks to see if a patch has been installed. Returns 1 if
patch is installed.
INPUT PARAMETER
  1. INPUT PARAMETER:   ECARY
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    DESCRIPTION:   
    Input a patch number, ex. EC*2.0*28.
RETURN PARAMETER DESCRIPTION 
Returns a 1 if patch has been installed, otherwise returns a 0.
