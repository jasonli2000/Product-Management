OOPS NEW PERSON DATA (1387)    REMOTE PROCEDURE (8994)

Name Value
NAME OOPS NEW PERSON DATA
TAG PER
ROUTINE OOPSGUI4
RETURN VALUE TYPE ARRAY
DESCRIPTION 
This call returns the name of new person, sex, DOB, and SSN of the individual
whose name is passed in the input parameter.
INPUT PARAMETER
  1. INPUT PARAMETER:   NAME
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   256
    DESCRIPTION:   
    This input parameter contains the name of the individual to look up and in
the New Person file.
RETURN PARAMETER DESCRIPTION 
The return parameter contains the name of new person, sex, DOB, and SSN if an
entry is found in the New Person File (#200).

The return paramter format is: 
     123456814^TURNER,JAMES^MALE^^333999888^INFORMATION SYSTEMS CENTER
