OOPS WITNESS DELETE (1411)    REMOTE PROCEDURE (8994)

Name Value
NAME OOPS WITNESS DELETE
TAG DELWITN
ROUTINE OOPSGUI2
RETURN VALUE TYPE ARRAY
AVAILABILITY PUBLIC
WORD WRAP ON FALSE
DESCRIPTION 
The broker call will remove witness information (stored in node "CA1W") from
the 2260 file.
INPUT PARAMETER
  1. INPUT PARAMETER:   INPUT
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    DESCRIPTION:   
    The INPUT parm will contain the IEN of the case in the 2260 file, and the
witness number in the format IEN^WITNESS NUMBER. The witness number is the sub
field indentification as such... 
^OOPS(2260,170,"CA1W",3,0)  3 BEING THE WITNESS NUMBER.
