OOPS REPLACE MULTIPLE (1415)    REMOTE PROCEDURE (8994)

Name Value
NAME OOPS REPLACE MULTIPLE
TAG REPLMULT
ROUTINE OOPSGUI3
RETURN VALUE TYPE ARRAY
DESCRIPTION 
This procedure is for updating entries in a multiple, in 2 stages.
Stage 1 deletes all current entries, step 2 creates new entries 
using data passed back from the GUI form.
INPUT PARAMETER
  1. INPUT PARAMETER:   SINGLE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   256
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This parameter contains the file, field, and IEN of the data to be filed in 
the format FILE^FIELD^IEN.
  2. INPUT PARAMETER:   DATA
    PARAMETER TYPE:   LIST
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    This parameter contains the data to be filed in the multiple.
