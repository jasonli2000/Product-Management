OOPS MANUAL XMIT DATA (1418)    REMOTE PROCEDURE (8994)

Name Value
NAME OOPS MANUAL XMIT DATA
TAG ENT
ROUTINE OOPSGUI7
RETURN VALUE TYPE ARRAY
DESCRIPTION 
This broker call is the front end for calling the routine to transmit
data to the AAC for DOL (CA1 and CA2s) or to transmit data to the National 
Database (NDB).
INPUT PARAMETER
  1. INPUT PARAMETER:   INPUT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   256
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This input parameter contains the date that the claims were originally
transmitted and should be re-transmitted for, the queue date and time, and
whether the data to be transmitted should be DOL or NDB data.  The format is:
TRANSDT^QUEUEDT@TIME^DOL (or NDB).
RETURN PARAMETER DESCRIPTION 
The Results parameter will contain the status message of the queue.  If not
successful, it will be 'ERROR', if successful it will be 'SUCCESSFULLY QUEUED'.
