ORRHCO ORDITM (1438)    REMOTE PROCEDURE (8994)

Name Value
NAME ORRHCO ORDITM
TAG ORDITM
ROUTINE ORRHCO
RETURN VALUE TYPE ARRAY
AVAILABILITY RESTRICTED
DESCRIPTION 
Returns a subset of orderable items for use in a long list box.
INPUT PARAMETER
  1. INPUT PARAMETER:   FROM
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    DESCRIPTION:   
    Starting name for cross reference
  2. INPUT PARAMETER:   DIR
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    Direction for $ORDER
  3. INPUT PARAMETER:   XREF
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    Identifies the SET xref to use in the orderable item file.
RETURN PARAMETER DESCRIPTION 
List in the format:  LST(n)=IEN^DisplayName
                or:  LST(n)=IEN^Mnemonic  <DisplayName>
