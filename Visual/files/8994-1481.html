RMIM CONSULT LIST (1481)    REMOTE PROCEDURE (8994)

Name Value
NAME RMIM CONSULT LIST
TAG CON
ROUTINE RMIMV
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY SUBSCRIPTION
WORD WRAP ON TRUE
DESCRIPTION 
Returns a list of consult requests for a patient within optional date range
and optional service.
INPUT PARAMETER
  1. INPUT PARAMETER:   PATIENT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    REQUIRED:   YES
    DESCRIPTION:   
    Patient DFN from File #2 (Patient File.)
  2. INPUT PARAMETER:   START DATE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    DESCRIPTION:   
    Optional start date to begin the search.
  3. INPUT PARAMETER:   STOP DATE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    DESCRIPTION:   
    Optional stop date to end the search.
  4. INPUT PARAMETER:   SERVICE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    DESCRIPTION:   
    Optional consulting service from the Request Service file (#123.5).
RETURN PARAMETER DESCRIPTION 
Array of consult requests in the form:
Consult id^date/time of request^status^consulting service^procedure
