ANRV TANRVMESSAGE (1491)    REMOTE PROCEDURE (8994)

Name Value
NAME ANRV TANRVMESSAGE
TAG SNDTXT
ROUTINE ANRVOB
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY RESTRICTED
WORD WRAP ON TRUE
DESCRIPTION 
Create's and sends completed VIST Outcomes to the national mailgroup to 
populate the national database.
INPUT PARAMETER
  1. INPUT PARAMETER:   ANRVCMD
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    1> CREATE       Initializes message.
2> APPEND       Add message text.
3> SUBJECT      Give the message a subject.
4> SENDTO       e-mail address to send message to.
5> EXECUTE      Send the message using VistA Mail.
