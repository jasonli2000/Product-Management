DSIC DDR WP FILER (1591)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIC DDR WP FILER
TAG WP
ROUTINE DSICDDR0
RETURN VALUE TYPE ARRAY
AVAILABILITY AGREEMENT
DESCRIPTION 
This will file the text into the specified word processing field.
INPUT PARAMETER
  1. INPUT PARAMETER:   FILE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   20
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is the file (or subfile) number for which the word processing field 
belongs.
  2. INPUT PARAMETER:   IENS
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   50
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    This is the appropriate Fileman DBS IENS for filing the word processing 
field
  3. INPUT PARAMETER:   FIELD
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   20
    REQUIRED:   YES
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    This is field number of the word processing field
  4. INPUT PARAMETER:   FLAGS
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   10
    REQUIRED:   NO
    SEQUENCE NUMBER:   4
    DESCRIPTION:   
    This is the FLAGS parameter for the WP^DIE call.
  5. INPUT PARAMETER:   DATA
    PARAMETER TYPE:   LIST
    MAXIMUM DATA LENGTH:   250
    REQUIRED:   YES
    SEQUENCE NUMBER:   5
    DESCRIPTION:   
    This is the text to be filed into the word processing field.
RETURN PARAMETER DESCRIPTION 
If the text is successfully filed, then return 1^
Else, return -1^error message
