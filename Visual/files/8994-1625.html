DSIC CPT GET MODIFIER (1625)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIC CPT GET MODIFIER
TAG MOD
ROUTINE DSICCPT
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY AGREEMENT
DESCRIPTION 
This returns basic information for a MODIFIER, file 81.3
INPUT PARAMETER
  1. INPUT PARAMETER:   MOD
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   5
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is the ifn or name of the MODIFIER, file 81.3
  2. INPUT PARAMETER:   CDT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   14
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    This date is optional.  It can be <null>, TODAY, NOW, or a internal 
Fileman format date.  Default value is TODAY.  This date will be used to 
check if the code was active as of that date.
RETURN PARAMETER DESCRIPTION 
If problems return -1^error message
Else return p1^p2^p3^p4^p5^p6^p7  where
   p1 = ifn to MODIFIER file 81.3
   p2 = MODIFIER .01 field value
   p3 = NAME (#.02 field)
   p4 = CODE (#.03 field - alternate 5 digit code for cpt modifiers)
   p5 = SOURCE (#.04 field - C:CPT;H:HCPCS;V:VA NATIONAL)
   p6 = EFFECTIVE DATE - from field 60 multiple
   p7 = STATUS (#.02 field from #60 multiple, 0:inactive;1:active)
