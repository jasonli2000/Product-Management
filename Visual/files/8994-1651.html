DSIR ADD DOCUMENT (1651)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIR ADD DOCUMENT
TAG NEWITEM
ROUTINE DSIROI0
RETURN VALUE TYPE ARRAY
AVAILABILITY SUBSCRIPTION
DESCRIPTION 
This RPC is used to create an entry in file 19620.1 DSIR RELEASED DOCUMENT
INPUT PARAMETER
  1. INPUT PARAMETER:   DATA ARRAY
    PARAMETER TYPE:   LIST
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Input: Array of
    (1): Pointer to VEJD ROI INSTANCE (19620) file
    (2): Internal Set of Codes value for document type
    (3): Document IEN (If applicable)
    (4): Internal Set of Codes value for document media
    (5): Caption
    (6): Document length
    (7): Date/time of document
    (8): Creator of Document: Needs to be a pointer to the NEW PERSON file
RETURN PARAMETER DESCRIPTION 
Return array consists of one node
        -1 ^ Error message
         1 ^ IEN of new entry in File 19620.1
