DSIR CHECK PREV REQ (1656)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIR CHECK PREV REQ
TAG CHECKCUR
ROUTINE DSIROI3
RETURN VALUE TYPE ARRAY
AVAILABILITY SUBSCRIPTION
DESCRIPTION 
This RPC checks for previous request for the current patient and 
requestor. It then returns an array of entries in file 19620 ROI INSTANCE.
INPUT PARAMETER
  1. INPUT PARAMETER:   PATIENT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   999
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is the DFN of the patient.
  2. INPUT PARAMETER:   REQUESTOR
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   999
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    This is the IEN of the requestor in file 19620.12 DSIR NEW REQUESTOR.
RETURN PARAMETER DESCRIPTION 
Output: Array of:
     (1): Number of requests^
     (2-n): Status^Date of request^Clerk of record for request^IEN
