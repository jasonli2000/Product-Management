DSIR GET ADDRESSES (1666)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIR GET ADDRESSES
TAG GTADDLST
ROUTINE DSIROI6
RETURN VALUE TYPE GLOBAL ARRAY
WORD WRAP ON TRUE
DESCRIPTION 
This RPC gets all address know by the RELEASE OF INFORMATION - DSSI for a 
given patient or requestor.
INPUT PARAMETER
  1. INPUT PARAMETER:   PATIENT REQUESTOR
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    RQSTR - Pointer to Patient or Requestor
  examples:
    "3;DSIR(19620.96,"
    "41;DSIR(19620.12,"
    "441;DPT("
RETURN PARAMETER DESCRIPTION 
Return Array Pieces '^' delimited:
    1. Address File Internal Number - Numeric
    2. Patient - Internal Number ; Patient Name
    3. Street Address Line One - Text 35 character max
    4. Street Address Line Two - Text 35 character max
    5. Street Address Line Three - Text 35 character max
    6. City - Text 40 character max
    7. State - Internal Number ; Abbreviation
    8. Zip Code - Numeric 5 or 9 numbers
   10. Holder - Internal Number ; File Reference")
   11. Inactive flag
   12. Phone
   13. Fax
 
  OR
   -1^No Addresses found for selected patient/requestor!
