SC DELETE ENTRY (17)    REMOTE PROCEDURE (8994)

Name Value
NAME SC DELETE ENTRY
TAG DIKC
ROUTINE SCUTBK1
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY PUBLIC
DESCRIPTION 
Allows file entry deletion via FileMan ^DIK call.
INPUT PARAMETER
  1. INPUT PARAMETER:   PARAMETERS
    PARAMETER TYPE:   LIST
    REQUIRED:   YES
    DESCRIPTION:   
    Contains subscripts that indicate global root and entry number.
RETURN PARAMETER DESCRIPTION 
Indicates whether delete operation was successful.
