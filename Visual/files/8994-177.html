ORQPT SPECIALTY PATIENTS (177)    REMOTE PROCEDURE (8994)

Name Value
NAME ORQPT SPECIALTY PATIENTS
TAG SPECPTS
ROUTINE ORQPTQ2
RETURN VALUE TYPE ARRAY
AVAILABILITY SUBSCRIPTION
DESCRIPTION 
Function returns an array of patients linked to a treating specialty.
INPUT PARAMETER
  1. INPUT PARAMETER:   SPECIALTY ID
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    REQUIRED:   YES
    DESCRIPTION:   
    The record number of the treating specialty from the Facility Treating
Specialty File (#45.7).
RETURN PARAMETER DESCRIPTION 
Array of patients linked to the treating specialty in the format: patient
id (DFN)^patient name.
