DENTV TP FILE TRANSACTIONS (1903)    REMOTE PROCEDURE (8994)

Name Value
NAME DENTV TP FILE TRANSACTIONS
TAG FILE
ROUTINE DENTVTPA
RETURN VALUE TYPE ARRAY
AVAILABILITY AGREEMENT
DESCRIPTION 
This will file transaction data types to file 228.2.  It will also create 
the DAS record (#221) if appropriate.  It will also file data to PCE if 
appropriate.
INPUT PARAMETER
  1. INPUT PARAMETER:   DATA
    PARAMETER TYPE:   LIST
    MAXIMUM DATA LENGTH:   256
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    See documentation in FILE^DENTVTPA
RETURN PARAMETER DESCRIPTION 
If all filing is successful, then the return array will contain:
  1^DAS record# <number> created
  1^<number> DES transactions successfully filed
  1^PCE data successfully filed, Visit#: <visit ien>^visit ien
 The third line may be:
  1^Some exceptions encountered while filing PCE data for visit#:
    <visit ien>^visit ien
     there may be (or may not be) additional messages
 
If unsuccessful, then return
  -1^error messages
  The first line returned will start with -1^
  Subsequent message lines may or may not start with -1^
