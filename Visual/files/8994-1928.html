DENTV GET PROVIDER SPECIALTIES (1928)    REMOTE PROCEDURE (8994)

Name Value
NAME DENTV GET PROVIDER SPECIALTIES
TAG SPEC
ROUTINE DENTVRP5
RETURN VALUE TYPE ARRAY
AVAILABILITY AGREEMENT
DESCRIPTION 
Gets a list of all active Provider Specialties from file 220.52.
RETURN PARAMETER DESCRIPTION 
Returns an array List[n] := Provider Specialty info.
Returns List[1] := -1^error message if there are no active specialties.
The format of the return array is List[n] := p1^p2 where
  p1:=Specialty name
  p2:=Code (numeric)
