DSIC CPT FIND (1955)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIC CPT FIND
TAG FIND
ROUTINE DSICCPT
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY AGREEMENT
WORD WRAP ON TRUE
DESCRIPTION 
For a lookup value, this will return a list of CPT codes matching that 
lookup value.
INPUT PARAMETER
  1. INPUT PARAMETER:   VAL
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   50
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is the lookup value for getting a list of CPT codes
  2. INPUT PARAMETER:   DATE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   14
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    This date is optional.  Default to TODAY.  Only CPT codes that were 
active as of this date will be included in the return list.
RETURN PARAMETER DESCRIPTION 
If lookup is successful, return List[n] := p1^p2^p3  for n=0,1,2,3,...
where p1 = ien to file 81   p2 = CPT code   p3 = CPT short description
 
If there are problems then return
  List[0] := -1^Problems encountered
  List[n] := additional messges
    where n = 1,2,3,4,...
    There may or may not be any additional messages
