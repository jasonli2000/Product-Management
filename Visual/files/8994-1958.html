DENTV TP GET OLD ENCOUNTERS (1958)    REMOTE PROCEDURE (8994)

Name Value
NAME DENTV TP GET OLD ENCOUNTERS
TAG ENC
ROUTINE DENTVTP5
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY AGREEMENT
WORD WRAP ON TRUE
DESCRIPTION 
Returns a list of all complete and/or terminated encounters for a 
selected patient.
INPUT PARAMETER
  1. INPUT PARAMETER:   INPUT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Input parameter is the patient DFN.
RETURN PARAMETER DESCRIPTION 
Returns global array in the format:
  ^TMP("DENT",$J,i)=p1^p2  where p1=date created
                                  p2="Complete" or "Terminated"
Will return -1^error message if invalid patient/not a dental patient,
or -1^no data for selected patient if no data.
