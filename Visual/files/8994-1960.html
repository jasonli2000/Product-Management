DENTV TP FILE RESERVED TXNS (1960)    REMOTE PROCEDURE (8994)

Name Value
NAME DENTV TP FILE RESERVED TXNS
TAG SAVE
ROUTINE DENTVTP7
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY AGREEMENT
DESCRIPTION 
Saves transaction information for a particular Dental provider and 
patient.  Reserved transactions are not filed to Dental History, DAS or 
PCE.
INPUT PARAMETER
  1. INPUT PARAMETER:   DATA
    PARAMETER TYPE:   LIST
    MAXIMUM DATA LENGTH:   500
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Contains filing information:
DATA(1) = FLAG^Provider IEN^DFN  
          where FLAG=A for add, U for update, D for delete
DATA(n) = txn data
RETURN PARAMETER DESCRIPTION 
Returns 1^successful message, or -1^error message.
