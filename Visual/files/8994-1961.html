Name Value
NAME DENTV TP GET RESERVED TXNS
TAG GET
ROUTINE DENTVTP7
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY AGREEMENT
WORD WRAP ON TRUE
DESCRIPTION 
Returns all reserved transactions for a particular Dental provider and
patient.
INPUT PARAMETER
  1. INPUT PARAMETER:   PROV
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Provider IEN from file 200.
  2. INPUT PARAMETER:   DFN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    Patient DFN from file 2.
RETURN PARAMETER DESCRIPTION 
Global array of all reserved transactions for the submitted provider and 
patient. Includes all modal data for OHA, Occlusion, Parafunctional 
Habits, and TMJ.
