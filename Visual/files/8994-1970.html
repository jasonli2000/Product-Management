OOPS SET DUAL BENEFITS DATA (1970)    REMOTE PROCEDURE (8994)

Name Value
NAME OOPS SET DUAL BENEFITS DATA
TAG DUAL
ROUTINE OOPSGUID
RETURN VALUE TYPE SINGLE VALUE
DESCRIPTION 
This broker call files data in the DUAL Node of the ASISTS Accident Reporting
File (#2260).
INPUT PARAMETER
  1. INPUT PARAMETER:   INPUT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   20
    REQUIRED:   YES
    DESCRIPTION:   
    This field contains the IEN and the form of the record the data will
be filed into.  The format is IEN^FORM.  An example is 373^DUAL.
  2. INPUT PARAMETER:   DATA
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   245
    REQUIRED:   YES
    DESCRIPTION:   
    This string contains the DUAL Benefits data.  It is in the format of:
P1 data^P2 data^P3 data^P4 data^P5 data^P6 data.
RETURN PARAMETER DESCRIPTION 
The return parameter will be Update Completed if successful, or No Changes
filed, if not.
