DVBAB EXAMS BY DATE (1977)    REMOTE PROCEDURE (8994)

Name Value
NAME DVBAB EXAMS BY DATE
TAG EXAMBYDT
ROUTINE DVBABEBD
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY RESTRICTED
DESCRIPTION 
Provides a report by date range of all AMIE/CAPRI exam requests.
INPUT PARAMETER
  1. INPUT PARAMETER:   BEGDT
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
  2. INPUT PARAMETER:   ENDDT
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
  3. INPUT PARAMETER:   DVBADLMTR
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   1
    REQUIRED:   NO
    DESCRIPTION:   
    The delimiter indicates whether the report should be produced in a comma 
delimited format (.csv file) or not. DVBADLMTR=0 means report displays in
plain text. DVBADLMTR=1 means report will be created as a .csv file
importable to Excel.
