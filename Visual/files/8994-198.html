ORQOR LIST (198)    REMOTE PROCEDURE (8994)

Name Value
NAME ORQOR LIST
TAG LIST
ROUTINE ORQOR1
RETURN VALUE TYPE ARRAY
AVAILABILITY SUBSCRIPTION
DESCRIPTION 
Returns a list of patient orders.
INPUT PARAMETER
  1. INPUT PARAMETER:   PATIENT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    REQUIRED:   YES
    DESCRIPTION:   
    Patient DFN from Patient file [#2].
  2. INPUT PARAMETER:   GROUP
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    REQUIRED:   NO
    DESCRIPTION:   
    Entry number of the desired display group from the Display Group File 
[#100.98].
  3. INPUT PARAMETER:   FLAG
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    REQUIRED:   NO
    DESCRIPTION:   
    Flag identifying the context of the orders to be retreived.
  4. INPUT PARAMETER:   STARTDATE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    REQUIRED:   NO
    DESCRIPTION:   
    The startdate for the order search in 'T' or Fileman format.
Startdate is only considered when FLAG is '1' (return all orders).
  5. INPUT PARAMETER:   STOPDATE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    REQUIRED:   NO
    DESCRIPTION:   
    The stopdate for the order search in 'T' or Fileman format.
Stopdate is only considered when FLAG is '1' (return all orders).
RETURN PARAMETER DESCRIPTION 
Array of orders in the format:
ORDER NUMBER^ORDER TEXT^START DATE^STATUS
