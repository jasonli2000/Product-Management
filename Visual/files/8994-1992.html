ORWDBA2 DELPDL (1992)    REMOTE PROCEDURE (8994)

Name Value
NAME ORWDBA2 DELPDL
TAG DELPDL
ROUTINE ORWDBA2
RETURN VALUE TYPE SINGLE VALUE
DESCRIPTION 
Delete a selected diagnosis code from a Clinician's Personal DX List. The 
personal dx list is stored in CPRS Diagnosis Provider file, file # 
5000017.
INPUT PARAMETER
  1. INPUT PARAMETER:   ORCIEN
    PARAMETER TYPE:   REFERENCE
    MAXIMUM DATA LENGTH:   255
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    The clinician's internal ID number.
  2. INPUT PARAMETER:   ORDXA
    PARAMETER TYPE:   REFERENCE
    MAXIMUM DATA LENGTH:   255
    REQUIRED:   YES
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    Array of diagnoses codes to be deleted from the clinician's personal
dx list.
