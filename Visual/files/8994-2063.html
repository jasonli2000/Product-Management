DSIR UPDATE BILL (2063)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIR UPDATE BILL
TAG UPDBILL
ROUTINE DSIRBIL
RETURN VALUE TYPE SINGLE VALUE
DESCRIPTION 
This RPC is used to create and edit the ROI Bill entries.
INPUT PARAMETER
  1. INPUT PARAMETER:   REQN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   999
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Request pointer to file 19620
  2. INPUT PARAMETER:   BILN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   9999
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    Internal Entry Number to file 19620.2 (Optional)
If null new entry in 19620.2 will be created
  3. INPUT PARAMETER:   DATA
    PARAMETER TYPE:   LIST
    MAXIMUM DATA LENGTH:   999
    REQUIRED:   YES
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    Array of data for the fields in file 19620.2 (Required)
Format of Array:
        Field Number ^ Internal Data Value 
 
        .01^12345
        .02^3030911
RETURN PARAMETER DESCRIPTION 
-1^Missing Required Input Variable(s)
-2^Request IEN Missing in file 19620
-3^Bill IEN Missing in file 19620.2
-4^Unable to add Bill
BILL IEN
