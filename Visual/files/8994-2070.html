DSIR GET BILL DOCS (2070)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIR GET BILL DOCS
TAG GETDOCS
ROUTINE DSIRBIL
RETURN VALUE TYPE GLOBAL ARRAY
WORD WRAP ON TRUE
DESCRIPTION 
This RPC is used to get a list of document types and internal number of 
documents that were previously release under the type of patient/veteran.
INPUT PARAMETER
  1. INPUT PARAMETER:   DFN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   999
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Patient Internal Number
RETURN PARAMETER DESCRIPTION 
Return Array:
             -1^Missing Patient Number!
             -2^No Documents Found!
             Document Type External^Document IEN
