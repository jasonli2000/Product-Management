DSIR BILL HISTORY (2071)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIR BILL HISTORY
TAG BILLHIST
ROUTINE DSIRBIL0
RETURN VALUE TYPE GLOBAL ARRAY
WORD WRAP ON TRUE
DESCRIPTION 
This RPC returns all bill edit history for a given bill number.
INPUT PARAMETER
  1. INPUT PARAMETER:   BILL IEN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   999
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is the internal entry number into file 19620.2.
RETURN PARAMETER DESCRIPTION 
Return Array '^' delimited
        1 - Date of Edit Internal ; External (I ; E)
        2 - Edited By (I ; E)
        3 - Field Edited (I ; E)
        4 - Old Value Internal (I ; E)
        5 - New Value Internal (I ; E)
