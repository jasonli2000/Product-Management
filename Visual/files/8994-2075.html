DSIR FEES REC RPT (2075)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIR FEES REC RPT
TAG FEESREC
ROUTINE DSIRBIR0
RETURN VALUE TYPE GLOBAL ARRAY
WORD WRAP ON TRUE
DESCRIPTION 
This RPC produces the data for the Fees Recieved report.
INPUT PARAMETER
  1. INPUT PARAMETER:   START DATE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   7
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    The internal FileMan format date to start the report.
  2. INPUT PARAMETER:   END DATE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   7
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    This is an optional date (FM format) to end the reporting period.  The 
default will be the current date the report is requested.
  3. INPUT PARAMETER:   DIVISIONS
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   999
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    This is a variable '^' delimited containing selected Divisions from which 
the user may see data.
RETURN PARAMETER DESCRIPTION 
Global Array - '^' delimited.
 
-1^Missing Start Date!
-2^No Payment Activity Found for Date Range!

     Payment Date ^ Payment Amount ^ Waived ^ Allowance ^ Division ^ 
     Division Name
