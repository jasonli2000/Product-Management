DSIR AMOUNT BILLED RPT (2077)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIR AMOUNT BILLED RPT
TAG AMTBILD
ROUTINE DSIRBIR0
RETURN VALUE TYPE GLOBAL ARRAY
WORD WRAP ON TRUE
DESCRIPTION 
This RPC returns the data for the Amount Billed report.
INPUT PARAMETER
  1. INPUT PARAMETER:   Start Date
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   7
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Pass in the FileMan format date to start the reporting period.
  2. INPUT PARAMETER:   End Date
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   7
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    Pass the FileMan format ending date for the reporting period.  This 
parameter will default to the current date.
  3. INPUT PARAMETER:   Divisions
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   512
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    This is a variable '^' delimited containing selected Divisions from which
the user may see data.
RETURN PARAMETER DESCRIPTION 
Global Array '^' delimited
 
-1^Missing Start Date!
-2^No bills entered in selected date range!
Bill IEN ^ Patient Name/FOIA ^ Requestor ^ Open Date ^ Bill Entered Date 
     ^ Current Status ^ Amount Billed ^ Division ^ Division Name
