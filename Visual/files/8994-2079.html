DSIR PRIORTY REPORT (2079)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIR PRIORTY REPORT
TAG PRIRTY
ROUTINE DSIROIR2
RETURN VALUE TYPE GLOBAL ARRAY
WORD WRAP ON TRUE
DESCRIPTION 
This RPC returns the data for the High Priority Request Report.
INPUT PARAMETER
  1. INPUT PARAMETER:   START DATE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   7
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    FileMan date to start report.
  2. INPUT PARAMETER:   END DATE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   7
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    FileMan date to report through, defaults to current day.
  3. INPUT PARAMETER:   DIVISION(S)
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   999
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    '^' delimited string of internal division numbers to use for 
multi-divisional reporting. If string is null only the division of the 
user or all if user holds the DSIR MDIV key.
RETURN PARAMETER DESCRIPTION 
Return Array
  -1^Missing Start Date!
  -2^No Records Found!
  H^Request IEN^Patient(I;E)^Date of Request(I;E)^Date First Closed(I;E)
  C ^ Request IEN ^ Comment
  C ^ Request IEN ^ Comment
  .
  .
  .
  (Note: Internal Comments field is word processing, C records may occur 0
  to infinity.)
