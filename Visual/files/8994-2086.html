DENTV GET CODING GUIDELINES (2086)    REMOTE PROCEDURE (8994)

Name Value
NAME DENTV GET CODING GUIDELINES
TAG GUIDE
ROUTINE DENTVRP2
RETURN VALUE TYPE ARRAY
AVAILABILITY AGREEMENT
WORD WRAP ON TRUE
DESCRIPTION 
Gets the admin and coding guidelines for an ADA/CPT code.
INPUT PARAMETER
  1. INPUT PARAMETER:   ADA
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   10
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    The external ADA or CPT code.ADA or CPT code
  2. INPUT PARAMETER:   ADMG
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   1
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    Set to 1 to include Admin guidelines with the Coding Guidelines.  Set to 
0 to send only Coding Guidelines.
RETURN PARAMETER DESCRIPTION 
Returns RET(n)=admin guideline
        RET(n)=coding guideline
        RET(n)=coding guideline (continued...wp field))
 
Admin and coding guidelines are independent and optional.  If an error 
occurs, the return value is -1^error message.
