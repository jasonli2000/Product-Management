MPIF REMOTE PRIMARY DFN ICN (2091)    REMOTE PROCEDURE (8994)

Name Value
NAME MPIF REMOTE PRIMARY DFN ICN
TAG PRIMARY
ROUTINE MPIFRPC3
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY PUBLIC
INACTIVE ACTIVE
VERSION 1
DESCRIPTION 
This remote procedure call will return the primary DFN and ICN for
a particular station and DFN.
INPUT PARAMETER
  1. INPUT PARAMETER:   SITE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   10
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
     This is the site's station number ... a legacy station number can be passed.
  2. INPUT PARAMETER:   DFN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   20
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    This is a patient's internal IEN in the PATIENT file (#2).
A legacy IEN can be passed along with a legacy station number.
RETURN PARAMETER DESCRIPTION 
This Remote Procedure Call will return the primary IEN (DFN) in the 
PATIENT file (#2) along with the Integration Control Number (ICN) if
available on a successful RETURN, otherwise a -1 along with a problem
type and problem text.
