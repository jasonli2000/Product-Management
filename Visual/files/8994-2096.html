ACKQROESD (2096)    REMOTE PROCEDURE (8994)

Name Value
NAME ACKQROESD
TAG STARTD
ROUTINE ACKQAG05
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
WORD WRAP ON FALSE
VERSION 3
DESCRIPTION 
This is the RPC used to setup and send a deletion of an audiogram to the 
DDc when a deletion is processed on the home system for an audiogram that 
has been sent to the DDC.  It is triggered by the mDelete.OnClick action 
from the Enter/Edit program ACKQROES3E.EXE.
INPUT PARAMETER
  1. INPUT PARAMETER:   DFN
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is the internal number of the patient in the PATIENT file (#2).
  2. INPUT PARAMETER:   IEN
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    This is the internal number of the record in the AUDIOMETRIC EXAM DATA 
file (#509850.9).
  3. INPUT PARAMETER:   STANUM
    PARAMETER TYPE:   LITERAL
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    This is the Station Number of the sending clinic
  4. INPUT PARAMETER:   USRNAM
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   32
    REQUIRED:   NO
    SEQUENCE NUMBER:   4
    DESCRIPTION:   
    This is the name of the submitting user
  5. INPUT PARAMETER:   USRSER
    PARAMETER TYPE:   LITERAL
    REQUIRED:   NO
    SEQUENCE NUMBER:   5
    DESCRIPTION:   
    This is the name of the submitting user's service.
RETURN PARAMETER DESCRIPTION 
This RPC returns a literal containing the local confirmation msg number,  
The msg number sent to the DDC, the error number, and the error msg if 
any.  It functions similarly to RPC ACKQROES, that transmits the 
audiogram to the DDC.
