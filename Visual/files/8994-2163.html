ORWRP4 HDR MODIFY (2163)    REMOTE PROCEDURE (8994)

Name Value
NAME ORWRP4 HDR MODIFY
TAG HDR
ROUTINE ORWRP4
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
WORD WRAP ON TRUE
DESCRIPTION 
This RPC looks at data returned from the HDR and makes any modifications
necessary to make the data compatible with CPRS Reports.
INPUT PARAMETER
  1. INPUT PARAMETER:   HANDLE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Remote Broker ID needed to find data in ^XTMP(HANDLE,"D",
  2. INPUT PARAMETER:   ID
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    Report ID found in field .02 of file 101.24
RETURN PARAMETER DESCRIPTION 
The return value is the success/failure flag.  In the event of an error
condition, the first node of the string is equal to the
syntax "-1^error text".
