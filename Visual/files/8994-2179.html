DGRR CANCEL SEARCH (2179)    REMOTE PROCEDURE (8994)

Name Value
NAME DGRR CANCEL SEARCH
TAG CANCEL
ROUTINE DGRRLU
RETURN VALUE TYPE SINGLE VALUE
APP PROXY ALLOWED No
DESCRIPTION 
The DGRR CANCEL SEARCH RPC will cancel a patient search in VistA.
INPUT PARAMETER
  1. INPUT PARAMETER:   JOB
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   99
    REQUIRED:   YES
    DESCRIPTION:   
    JOB = A unique JOB ID used to cancel a VistA search.
