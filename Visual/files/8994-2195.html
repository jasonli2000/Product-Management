XUS KAAJEE LOGOUT (2195)    REMOTE PROCEDURE (8994)

Name Value
NAME XUS KAAJEE LOGOUT
TAG SIGNOFF
ROUTINE XUSKAAJ
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
DESCRIPTION 
This RPC calls the LOUT^XUSCLEAN tag to mark a KAAJEE-signed-on user's
entry in the sign-on log as signed off.
INPUT PARAMETER
  1. INPUT PARAMETER:   SIGNON-LOG-DA
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   1
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    The DA (IEN) of the user's signon log entry.
RETURN PARAMETER DESCRIPTION 
Returns 1. The return value has no significance.
