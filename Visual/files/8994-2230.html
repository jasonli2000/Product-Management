DSIR ADD/DEL SENSITIVE PROV (2230)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIR ADD/DEL SENSITIVE PROV
TAG ADDPROV
ROUTINE DSIROI8
RETURN VALUE TYPE SINGLE VALUE
DESCRIPTION 
This RPC will add or delete entries in file 19620.94 (DSIR SENSITIVE 
PROVIDERS)
INPUT PARAMETER
  1. INPUT PARAMETER:   PROV
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   99
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is the internal entry number of the New Person file (#200) of the 
person to be flagged as a sensitive provider in the ROI package.
  2. INPUT PARAMETER:   DEL
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   1
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    This is a flag that if true will delete the record pointed to by 
parameter 1 from file 19620.94.
RETURN PARAMETER DESCRIPTION 
-1 ^ Missing Provider Pointer!
@ - Record deleted
PROV - Internal Number to file 200 of provider marked as sensitive
