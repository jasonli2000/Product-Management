MAG3 TIU NEW (2418)    REMOTE PROCEDURE (8994)

Name Value
NAME MAG3 TIU NEW
TAG NEW
ROUTINE MAGGNTI1
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
DESCRIPTION 
RPC Call to create a New Note 
  and Optionally :  
           Electronically Sign the Note.
           Administratively Close the Note.
           Add Text to the Note.
INPUT PARAMETER
  1. INPUT PARAMETER:   MAGDFN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   60
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    DFN of the Patient.
  2. INPUT PARAMETER:   MAGTITLE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   60
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    IEN of TIU DOCUMENT DEFINITION entry in file ^TIU(8925.1
  3. INPUT PARAMETER:   MAGADCL
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   10
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
      1 = Mark this Note as Administratively Closed
  otherwise it will be Un-Signed
  4. INPUT PARAMETER:   MAGMODE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   NO
    SEQUENCE NUMBER:   4
    DESCRIPTION:   
    Mode of Admin Closure:
 "S" = Scanned Document
 "M" = Manual closure
  5. INPUT PARAMETER:   MAGES
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   60
    REQUIRED:   NO
    SEQUENCE NUMBER:   5
    DESCRIPTION:   
     The encrypted Electronic Signature
  6. INPUT PARAMETER:   MAGESBY
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   90
    REQUIRED:   NO
    SEQUENCE NUMBER:   6
    DESCRIPTION:   
     The DUZ of the Signer (Defaults to DUZ)
  7. INPUT PARAMETER:   MAGLOC
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   60
    REQUIRED:   NO
    SEQUENCE NUMBER:   7
    DESCRIPTION:   
    IEN in Hospital Location File #44, Global ^SC(
  8. INPUT PARAMETER:   MAGTEXT
    PARAMETER TYPE:   LIST
    MAXIMUM DATA LENGTH:   3200
    REQUIRED:   NO
    SEQUENCE NUMBER:   8
    DESCRIPTION:   
    Array of Text to add to the New Note.
  9. INPUT PARAMETER:   MAGDATE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   40
    REQUIRED:   NO
    SEQUENCE NUMBER:   8
    DESCRIPTION:   
    Date of the Note. For New Notes.
  10. INPUT PARAMETER:   MAGCNSLT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   NO
    SEQUENCE NUMBER:   9
    DESCRIPTION:   
    DA of Consult to Link this note to.
RETURN PARAMETER DESCRIPTION 
If success : result = "1^Note was created."
If success and Administratively Closed.
             result = "1^Note was created. Administrative Closure."
If success and Signed
             result = "1^Note was created.  Signed."
  
If error:   result = "0^<error message>"
