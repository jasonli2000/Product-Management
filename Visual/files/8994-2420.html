MAG3 TIU CREATE ADDENDUM (2420)    REMOTE PROCEDURE (8994)

Name Value
NAME MAG3 TIU CREATE ADDENDUM
TAG NEWADD
ROUTINE MAGGNTI1
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
DESCRIPTION 
Creates a TIU Addendum to a document.
  RPC call to create an Addendum to a Note
  and Optionally :  
           Electronically Sign the Addendum
           Administratively Close the Addendum
           Add Text to the Addendum
INPUT PARAMETER
  1. INPUT PARAMETER:   MAGDFN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    DFN of the Patient.
  2. INPUT PARAMETER:   MAGTIUDA
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   60
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    This is the IEN of the Note in file ^TIU(8925
  3. INPUT PARAMETER:   MAGADCL
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   60
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
     1 = Mark this Note as Administratively Closed
 otherwise it will be Un-Signed
  4. INPUT PARAMETER:   MAGMODE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   NO
    SEQUENCE NUMBER:   4
    DESCRIPTION:   
    Mode of Admin Closure:
"S" = Scanned Document
"M" = Manual closure
  5. INPUT PARAMETER:   MAGES
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   60
    REQUIRED:   NO
    SEQUENCE NUMBER:   5
    DESCRIPTION:   
    The encrypted Electronic Signature
  6. INPUT PARAMETER:   MAGESBY
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   90
    REQUIRED:   NO
    SEQUENCE NUMBER:   6
    DESCRIPTION:   
     The DUZ of the Signer (Defaults to DUZ)
  7. INPUT PARAMETER:   MAGTEXT
    PARAMETER TYPE:   LIST
    MAXIMUM DATA LENGTH:   3200
    REQUIRED:   NO
    SEQUENCE NUMBER:   7
    DESCRIPTION:   
    Array of Text to add to the New Note.
  8. INPUT PARAMETER:   MAGDATE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   40
    REQUIRED:   NO
    SEQUENCE NUMBER:   7
    DESCRIPTION:   
    The Date of the New TIU Note/Addendum
RETURN PARAMETER DESCRIPTION 
If success :
    result = "<TIU IEN>^Addendum was created."
If success and Administratively Closed.
    result ="<TIU IEN>^Addendum was created. Administrative Closure."
If success and signed.
    result ="<TIU IEN>^Addendum was created.  Signed."
  
If error: 
    result ="0^<error message>"
