MAG4 VERSION STATUS (2423)    REMOTE PROCEDURE (8994)

Name Value
NAME MAG4 VERSION STATUS
TAG VERSTAT
ROUTINE MAGGTU4
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
DESCRIPTION 
RPC to evaluate the status of an Imaging Version.
INPUT PARAMETER
  1. INPUT PARAMETER:   MAGVER
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   100
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    The Imaging Version to be checked.  In the format
   MAG*3.0*59   or   3.0.59
RETURN PARAMETER DESCRIPTION 
String
        0^No Install record for KIDs Build : MAGVER
        1^Unknown Release Status for : MAGVER
        2^Alpha/Beta Version
        3^Released Version
