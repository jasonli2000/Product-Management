DENTV UPDATE PCE (2435)    REMOTE PROCEDURE (8994)

Name Value
NAME DENTV UPDATE PCE
TAG UPD
ROUTINE DENTVTPE
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY AGREEMENT
WORD WRAP ON TRUE
DESCRIPTION 
Deletes procedures, diagnosis and providers from PCE when users delete 
transactions from DRM Plus.
INPUT PARAMETER
  1. INPUT PARAMETER:   DATA
    PARAMETER TYPE:   LIST
    MAXIMUM DATA LENGTH:   256
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Contains a list of transaction ids deleted from DRM Plus in the format:
DATA(txnid).
RETURN PARAMETER DESCRIPTION 
Returns array -1^error or list of visits that were updated:
RET(n)=$START^visit ien^visit date (external format)
RET(n)=OK message or the following:
RET(n)=visit ien^code^data to prompt user for information.
 
 code^data can be the following to prompt user for primary dx or provider:
 visit ien^"POV"^dx ien^dx external^^desc
 visit ien^"PRV"^provider ien(file 200)^provider name
