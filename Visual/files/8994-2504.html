DENTV MONITOR POLL (2504)    REMOTE PROCEDURE (8994)

Name Value
NAME DENTV MONITOR POLL
TAG POLL
ROUTINE DENTVM1
RETURN VALUE TYPE ARRAY
AVAILABILITY AGREEMENT
DESCRIPTION 
See if the monitor background job has finished and if so, get the
results from the DENTAL PATIENT (#220) file.
INPUT PARAMETER
  1. INPUT PARAMETER:   DFN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   20
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Contains the patient pointer (DFN).
RETURN PARAMETER DESCRIPTION 
Returns -1^error message or array with Fluoride Monitor in the following 
format:
  results(1)=fluoride monitor value^other monitor (future enhancement)
  results(2)=F$1st ien^first code^date^2nd ien^second code^date
  results(n)=other monitor data (future)
