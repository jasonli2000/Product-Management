Routine for setting the requested lab type for a given request.

    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   10
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:

  ROI = REQUEST IEN
  2. INPUT PARAMETER:   TYP
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   2
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    INPUT PARAMETER
    TYP = SELECTED LAB TYPE - A NUMERIC VALUE >= 0
RETURN PARAMETER DESCRIPTION

    Request IEN (same as ROI)
  OR
    -1^Invalid request number ROI
