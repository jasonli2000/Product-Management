DSIV NO INSURANCE (2546)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIV NO INSURANCE
TAG NOI
ROUTINE DSIVIC2
RETURN VALUE TYPE ARRAY
AVAILABILITY AGREEMENT
DESCRIPTION 
Allows ICB to set the NO COVERAGE VERIFICATION DATE in the Billing
 Patient (#354) file.
INPUT PARAMETER
  1. INPUT PARAMETER:   DFN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Contains the patient ien.
  2. INPUT PARAMETER:   DATA
    PARAMETER TYPE:   LIST
    MAXIMUM DATA LENGTH:   80
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    Contains the field and value to update for ICB file 354.  The format is
 DATA(field)=value.  Example is DATA(60)=3060328.
RETURN PARAMETER DESCRIPTION 
Return RET(1)=DFN^Data filed successfully
 
    or RET(n)=-1^error message
       RET(n)=additional error text -OPTIONAL (from DSICDDR0)
