DSIV GET EIV REPORT (2557)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIV GET EIV REPORT
TAG IIV
ROUTINE DSIVIC3
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY AGREEMENT
WORD WRAP ON TRUE
DESCRIPTION 
Returns data from the IIV RESPONSE file (365) based on a buffer ien.
INPUT PARAMETER
  1. INPUT PARAMETER:   BUFFER
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Insurance Buffer IEN (file 355.33).
RETURN PARAMETER DESCRIPTION 
Returns -1^error message if no buffer ien, or if buffer ien is not in the 
IIV Response file.  Or array(n)=File# (or subfile#)^field#^data for all 
fields in file 365.  If an eIV error exists, the full IIV RESPONSE 
(#365.15) file error data is returned following an error indicator line: 
$$ERROR$$.
