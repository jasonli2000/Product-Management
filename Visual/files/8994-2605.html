DSIF PAY ALL PAYMENTS DISPLAY (2605)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIF PAY ALL PAYMENTS DISPLAY
TAG EN1
ROUTINE DSIFPAY2
RETURN VALUE TYPE ARRAY
AVAILABILITY AGREEMENT
INACTIVE ACTIVE
WORD WRAP ON FALSE
VERSION 3
DESCRIPTION 
Input DFN and Vendor IEN, to get a list of existing payments.
Optional parameters are Date of service (FM date) and CPT IEN.
INPUT PARAMETER
  1. INPUT PARAMETER:   DFN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   20
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
  2. INPUT PARAMETER:   VENDOR IEN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   25
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
  3. INPUT PARAMETER:   DATE OF SERVICE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   7
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    Date of service FM format (optional) 
**Unless you enter a CPT code, then it is required!
  4. INPUT PARAMETER:   CPT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   12
    REQUIRED:   NO
    SEQUENCE NUMBER:   4
    DESCRIPTION:   
    CPT code, external value.
