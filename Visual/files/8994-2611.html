DSIF BATCH LIST BY STATUS (2611)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIF BATCH LIST BY STATUS
TAG LISTSTAT
ROUTINE DSIFBAT2
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY AGREEMENT
WORD WRAP ON TRUE
VERSION 3
DESCRIPTION 
Returns a list of batches sorted by entered status (O^C)  [open and 
closed]  Screens to users who opened and supervisors
INPUT PARAMETER
  1. INPUT PARAMETER:   STATUS
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   20
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Status to search: 
'C' FOR CLERK CLOSED;'S' FOR SUPERVISOR CLOSED;'O' FOR OPEN;'T' FOR 
TRANSMITTED;'V' FOR VOUCHERED;'P' FOR FORWARDED TO PRICER;'A' FOR
ASSIGNED PRICE;'R' FOR REVIEWED AFTER PRICER;'F' FOR CENTRAL FEE ACCEPTED
 
Delimited by "^", multiple status can be entered: "C^S^R"
  2. INPUT PARAMETER:   FEE PROGRAM
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   3
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
  3. INPUT PARAMETER:   STARTDT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   7
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    Start date to display batches, defaults to all batches if not entered
  4. INPUT PARAMETER:   ENDDATE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   7
    REQUIRED:   NO
    SEQUENCE NUMBER:   4
    DESCRIPTION:   
    Enter the end of the date rand\ge to display default to all dates.
RETURN PARAMETER DESCRIPTION 
Output:  ^TMP($J,"DSIFBAT2",0)="-1^error message"  
 
 ^TMP($J,"DSIFBAT2",Status,Batch)=Batch IEN;Batch number^Obligation 
number^Fee program(Int;ext)^Status^User who opened^Lines allowed;Lines 
currently in batch^FCP
 
When dates from and to are entered, the following criteria is used:
Transmitted/Verified Batches  - Date Transmitted
Supervisor Closed             - Date Supervisor Closed 
Closed Batches                - Date Clerk Closed
Open Batches (and all other)  - Date opened
