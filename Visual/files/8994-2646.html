NAME DSIF INP CENSUS REPORT
TAG CENSUS
ROUTINE DSIFRPT
RETURN VALUE TYPE GLOBAL ARRAY
WORD WRAP ON TRUE
DESCRIPTION 
This is an RPC version of the FBCNHCEN routine to produce the Fee Basis 
Census Report.
INPUT PARAMETER
  1. INPUT PARAMETER:   FBDT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   7
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Census Date in FileMan format.
RETURN PARAMETER DESCRIPTION 
Return Array
  D ^ Veteran Name ^ DOB ^ Veteran ID ^ PSA ^ Authorization From
  V ^ Vendor Name ^ Vendor ID ^ Chain # ^ Part Code (External) ^ Street
    Address ^ 1 = Awaiting Austin Approval ^ 1 = Vendor in Delete 
    Status ^ Street Address 2 ^ City ^ State (Abbr) ^ Zip Code ^ Phone
    Number
  -1 ^ Invalid Input!
  -1 ^ No Data Found!
