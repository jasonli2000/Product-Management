DSIF INP HOSPITAL COST RPT (2647)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIF INP HOSPITAL COST RPT
TAG COSTRPT
ROUTINE DSIFRPT0
RETURN VALUE TYPE GLOBAL ARRAY
WORD WRAP ON TRUE
DESCRIPTION 
This is an RPC version of the CIVIL HOSPITAL COST REPORT as found in 
routines FBCHCR anc FBCHCR1.
INPUT PARAMETER
  1. INPUT PARAMETER:   BEGDATE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   7
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Begining Date (Required : FileMan format)
  2. INPUT PARAMETER:   ENDDATE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   7
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    Ending Date (Required : FileMan format)
  3. INPUT PARAMETER:   FBRT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   1
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    Report Type (Optional : D = Detailed, S = Summary, Default = Summary)
RETURN PARAMETER DESCRIPTION 
Return Array
   -1 ^ Invalid Input!
   -1 ^ No payments found within specified timeframe!
   D ^ PATIENT NAME ^ SSN ^ ASSOCIATED 7078 ^ AMOUNT PAID ^ 1 = ANCILLARY
      PAYMENT ^ FINAL DRG ^ LENGTH OF STAY
   SL ^ TREATING SPECIALTY ^ LENGTH OF STAY ^ NUMBER OF CASES ^ AVG AMOUNT
      PAID
   ST ^ TOTAL CASES ^ AVG AMOUNT PAID ^ AVG LENGTH OF STAY
   TA ^ TOTAL ANCILLARY PAYMENTS ^ AVG AMOUNT PAID
      
   (First Piece: D = Detail Line, SL = Summary Line, ST = Summary Totals,
    TA = Total Ancillary)
