DSIF INP ADD ROC (2662)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIF INP ADD ROC
TAG ADD
ROUTINE DSIFROC
RETURN VALUE TYPE SINGLE VALUE
DESCRIPTION 
This RPC duplicates the logic of the FBCH ADD ROC input template for file 
161.5.
INPUT PARAMETER
  1. INPUT PARAMETER:   IEN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   99
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Internal Entry Number of Contact/Fee Notification (Required - Pointer to 
file 162.2/161.5)
  2. INPUT PARAMETER:   DTOCT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   12
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    Date & Time of Contact (Required - FileMan Date/Time)
  3. INPUT PARAMETER:   NARR
    PARAMETER TYPE:   LIST
    MAXIMUM DATA LENGTH:   999
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    Narrative (Optional - Word Processing Array)
RETURN PARAMETER DESCRIPTION 
Return Values
        -1 ^ Invalid Entry Number!
        -1 ^ Missing Date of Contact!
        1 = Contact Added
