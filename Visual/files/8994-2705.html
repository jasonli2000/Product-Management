NAME ORAMX CONSULT
TAG CONSULT
ROUTINE ORAMX
RETURN VALUE TYPE ARRAY
AVAILABILITY PUBLIC

DESCRIPTION 
Send DFN and the name of the Consult Service (from the ORAM CONSULT 
REQUEST SERVICE parameter). Returns pending and active consults which meet
those criteria.
