MAG DICOM GET RAD FILM (2749)    REMOTE PROCEDURE (8994)

Name Value
NAME MAG DICOM GET RAD FILM
TAG FILM
ROUTINE MAGDRPCB
RETURN VALUE TYPE ARRAY
AVAILABILITY RESTRICTED
INACTIVE ACTIVE
VERSION 3
DESCRIPTION 
This RPC is used to find an entry in the FILM SIZE file (#78.4).
INPUT PARAMETER
  1. INPUT PARAMETER:   FILM
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is the name of the film size to be used in the lookup process.
RETURN PARAMETER DESCRIPTION 
RESULTS(1)= -1 (entry not found) or 1 (entry found)
RESULTS(2)= ien^film size name
