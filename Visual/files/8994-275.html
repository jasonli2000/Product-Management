ORQQLR SEARCH RANGE INPT (275)    REMOTE PROCEDURE (8994)

Name Value
NAME ORQQLR SEARCH RANGE INPT
TAG SRIN
ROUTINE ORQQLR
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY SUBSCRIPTION
DESCRIPTION 
Returns the date search range in number of days (e.g. 2) to begin the 
search before today. For example, a value of 2 would indicate to limit the
search between two days and today. Limited to inpatients.
INPUT PARAMETER
  1. INPUT PARAMETER:   PATIENT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   32
    REQUIRED:   YES
    DESCRIPTION:   
    Identifier from Patient file [#2].
RETURN PARAMETER DESCRIPTION 
Date search range in number of days (e.g. 2) to begin the search before
today. For example, a value of 2 would indicate to limit the search
between two days and today.  Limited to inpatients.
