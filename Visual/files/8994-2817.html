AVAILABILITY AGREEMENT
INACTIVE ACTIVE
APP PROXY ALLOWED Yes
DESCRIPTION 
Returns Discharge to Community report information.
INPUT PARAMETER
  1. INPUT PARAMETER:   ICNLIST
    PARAMETER TYPE:   LIST
    DESCRIPTION:   
    Patient's ICN in list format (ICN^ICN^ICN^ICN^ICN...)
  2. INPUT PARAMETER:   FDATE
    PARAMETER TYPE:   LITERAL
    DESCRIPTION:   
    Date to begin search with.
  3. INPUT PARAMETER:   TDATE
    PARAMETER TYPE:   LITERAL
    DESCRIPTION:   
    Date to end search at.
RETURN PARAMETER DESCRIPTION

End of line marker
