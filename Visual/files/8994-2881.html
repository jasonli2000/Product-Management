DSIR GET AOD (2881)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIR GET AOD
TAG ACCOFDIS
ROUTINE DSIROIR
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY PUBLIC
WORD WRAP ON TRUE
DESCRIPTION 
This routine returns the data nescessary for the patient disclosure
history report.
 
  Required fields for this report:
 
  From 19620      FIELD   From 19620.1    FIELD
  ---------------------   ---------------------
  Requestor Address .81   DocumentCaption   .05
  Patient Address   .82   DocumentDate      .07
  Current Status  10.05
  Date Received   10.06
  Date Closed     10.07
  Clerk Assigned    .03
  Comment           .31
INPUT PARAMETER
  1. INPUT PARAMETER:   AODIEN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   20
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    AODIEN = The request IEN for which you want an Accounting of Disclosure.
RETURN PARAMETER DESCRIPTION 
Return Array
               -1^Missing Request IEN!
               -2^Request not found for this IEN: ###.
      The return array will contain strings formatted in groups (one per 
request) as follows:
               
$$AOD$$^IEN^DateRecieved^Requestor^Reason^RequestorType^Status
        $$COMMENT1$$^Comment line 1 text
        $$COMMENT2$$^Comment line 2 text
        $$COMMENT3$$^Comment line 3 text
        $$DOCUMENT$$^Document Caption^Document Date 
        $$DOCUMENT$$^Document Caption^Document Date^Document Type^
                     if paper "P" else ""                             
            (one line like the above for each document)
        $$PAPER$$^one or more optional records for paper documents.
