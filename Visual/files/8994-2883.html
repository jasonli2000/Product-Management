DSIR DELIVERY TYPE SUMMARY RPT (2883)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIR DELIVERY TYPE SUMMARY RPT
TAG DELIVRY
ROUTINE DSIROIR2
RETURN VALUE TYPE GLOBAL ARRAY
WORD WRAP ON TRUE
DESCRIPTION 
This RPC returns an array for the Delivery Type Summary Report. The 
optional date parameters work off of the date closed only.
INPUT PARAMETER
  1. INPUT PARAMETER:   SDATE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   7
    REQUIRED:   NO
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    SDATE - Optional - Report start date in Fileman format.
  2. INPUT PARAMETER:   EDATE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   7
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    EDATE - Optional - Report end date in Fileman format.
  3. INPUT PARAMETER:   DIVL
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   150
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    DIVL  - Optional - List of selected divisions - NULL means all divisions.
RETURN PARAMETER DESCRIPTION 
RETURN ARRAY 
  ##^DESCRIPTION    Where ## = a positive integer count >= Zero
 
  OR
 
  -1^Error Message
