DSIV XPAR GET ALL FOR ENT (2892)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIV XPAR GET ALL FOR ENT
TAG GETALL
ROUTINE DSIVXPR
RETURN VALUE TYPE ARRAY
AVAILABILITY AGREEMENT
DESCRIPTION 
Returns values for all instances and values for an entity/parameter.
INPUT PARAMETER
  1. INPUT PARAMETER:   DATA
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   240
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
     DATA = "ARR(1)~ARR(2)~ARR(3)~ARR(4)~ARR(5)~ARR(6)"
     e.g. "SYS~DSIV PAGE SETUP~~~~"
  ARR(1) = entity     ARR(2) = param name    ARR(3) = instance
  ARR(4) = value      ARR(5) = new instance value
  ARR(6) = format for GETLST, Default = "Q"
RETURN PARAMETER DESCRIPTION 
Returns array of parameter values based on the ARR(6) value:
  default= "Q" - quick,    #)=internal instance^internal value
           "E" - external, #)=external instance^external value
           "B" - both,     #,"N")=internal instance^external instance
                           #,"V")=internal value^external value
           "N" - external instance)=internal value^external value
        some of those pieces may be <null> depends upon ARR(6)
On error, return RET(1)=-1^error message
