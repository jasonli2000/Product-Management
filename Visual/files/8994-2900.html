MAGQ FINDC (2900)    REMOTE PROCEDURE (8994)

Name Value
NAME MAGQ FINDC
TAG FINDC
ROUTINE MAGQBUT5
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
INACTIVE ACTIVE
DESCRIPTION 
By placing carat delimited values in the find value parameter
the subscriber can perform a compound index lookup on an 
extended primary key.
INPUT PARAMETER
  1. INPUT PARAMETER:   FNUM
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    The file number of the content table.
  2. INPUT PARAMETER:   IENS
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    The IENS of the main level or subfile level of the content table.
  3. INPUT PARAMETER:   FLAGS
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    The FIND1^DIC lookup options.
  4. INPUT PARAMETER:   FNDVALUE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   60
    REQUIRED:   YES
    SEQUENCE NUMBER:   4
    DESCRIPTION:   
    The carat delimited lookup.  Each piece is the same as 
the subscript in the index.
  5. INPUT PARAMETER:   XREF
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   NO
    SEQUENCE NUMBER:   5
    DESCRIPTION:   
    The subscriber specified cross reference for the lookup.
  6. INPUT PARAMETER:   SCREEN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   NO
    SEQUENCE NUMBER:   6
    DESCRIPTION:   
    The subscriber specified DIC("S") logic for the lookup.
RETURN PARAMETER DESCRIPTION 
The Internal entry number of the lookup value.
