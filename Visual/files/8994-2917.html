NAME VPR GET PATIENT DATA
TAG GET
ROUTINE VPRD
RETURN VALUE TYPE GLOBAL ARRAY

DESCRIPTION 
This RPC retrieves the requested data from VistA, and returns it in
^TMP("VPR",$J,n) as XML.
