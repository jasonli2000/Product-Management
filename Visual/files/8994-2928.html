NAME DSIROI1 QUEUELBL QUEUE A LABEL
TAG QUEUELBL
ROUTINE DSIROI1
RETURN VALUE TYPE SINGLE VALUE
DESCRIPTION 
This routine adds a label to an individual clerk's queue for batch 
printing.
INPUT PARAMETER
  1. INPUT PARAMETER:   CIEN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   20
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    CIEN is the clerk IEN.  This points at file 200.
  2. INPUT PARAMETER:   RIEN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   20
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    RIEN is the requestor IEN.  This points to file 19620.12.
  3. INPUT PARAMETER:   AIEN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   20
    REQUIRED:   YES
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    AIEN is the address IEN.  This points at file 19620.92.
RETURN PARAMETER DESCRIPTION 
RETURNS 
   ##^LABEL QUEUED FOR PRINTING
   where ## = ien in label file (19620.14)
 
  or -1^error message
