NUPA SET PU AND ALT (2939)    REMOTE PROCEDURE (8994)

Name Value
NAME NUPA SET PU AND ALT
TAG SL
ROUTINE NUPABCL2
RETURN VALUE TYPE SINGLE VALUE
WORD WRAP ON TRUE
DESCRIPTION 
Sets Pressure Ulcer and Skin Alteration data.  Deletes old info for this 
careplan and inserts from the reassessment.
INPUT PARAMETER
  1. INPUT PARAMETER:   ALT
    PARAMETER TYPE:   LIST
    MAXIMUM DATA LENGTH:   999
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
     String List.
  
 Piece 1 = "A" for alterations or "U" for pressure ulcers
 Piece 2 = CarePlan DA
 Piece 3 = Text field 1 from the reassessment
 Piece 4 = Text field 2 from the reassessment
 Piece 5 = Text field 3 from the reassessment
 Piece 6 = Comments number of Lines
 Piece 7 = Comments Text
 Piece 8 = A 1 if the alteration is healed
RETURN PARAMETER DESCRIPTION 
1 if successful, 0 if not.
