NUPA WP GET (2940)    REMOTE PROCEDURE (8994)

Name Value
NAME NUPA WP GET
TAG WPGET
ROUTINE NUPABCL
RETURN VALUE TYPE GLOBAL ARRAY
WORD WRAP ON TRUE
DESCRIPTION 
Returns data from a Word Processing field.
INPUT PARAMETER
  1. INPUT PARAMETER:   FILE
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    File to get the data from, in ^FILE(subscript, format.
  2. INPUT PARAMETER:   IEN
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    Line number for the data to be put into, in number_"," format
such as "1,".
  3. INPUT PARAMETER:   NODE
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    Node in the global to pull the data from, such as node 2.
RETURN PARAMETER DESCRIPTION 
Returns the text of the word processing field.
