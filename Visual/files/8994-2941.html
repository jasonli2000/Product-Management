Name Value
NAME NUPA WP SET
TAG WPSET
ROUTINE NUPABCL
RETURN VALUE TYPE SINGLE VALUE
WORD WRAP ON TRUE
DESCRIPTION 
Sets data into a word processing field.
INPUT PARAMETER
  1. INPUT PARAMETER:   FILE
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    File to insert the data into, in ^FILE(subscript, format.
  2. INPUT PARAMETER:   LINE
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    Line number for the data to be put into, such as 1, 2, or 3.
  3. INPUT PARAMETER:   DATA
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    Data (text) to insert into the global.
RETURN PARAMETER DESCRIPTION 
Returns a 1 when done.
