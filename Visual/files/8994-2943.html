NUPA USER CLASS (2943)    REMOTE PROCEDURE (8994)

Name Value
NAME NUPA USER CLASS
TAG UC
ROUTINE NUPABCL1
RETURN VALUE TYPE SINGLE VALUE
WORD WRAP ON TRUE
DESCRIPTION 
Checks if a user is in a certain TIU User Class.  Useful to determine if 
the nurse doing the assessment is a student.
INPUT PARAMETER
  1. INPUT PARAMETER:   CLASS
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Class to check if the user is in, such as STUDENT.
RETURN PARAMETER DESCRIPTION 
Returns a 1 if the user is in a certain user class, or a 0 otherwise.
