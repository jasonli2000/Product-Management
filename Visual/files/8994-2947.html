NUPA LIST2 (2947)    REMOTE PROCEDURE (8994)

Name Value
NAME NUPA LIST2
TAG LIST2
ROUTINE NUPABCL
RETURN VALUE TYPE GLOBAL ARRAY
WORD WRAP ON TRUE
DESCRIPTION 
List of all entries from a file including other fields.
INPUT PARAMETER
  1. INPUT PARAMETER:   FILE
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    File number to list data from.
  2. INPUT PARAMETER:   FILEDS
    PARAMETER TYPE:   LITERAL
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    Other fields in format FILE^Field 1^Field 2 etc.
  3. INPUT PARAMETER:   SCREEN
    PARAMETER TYPE:   LITERAL
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    Fileman screen for the lookup.
RETURN PARAMETER DESCRIPTION 
List of all data returned in ^TMP($J).  
 
This example returns all entries from file 1927.2, along with field 4.
D LIST2^NUPABCL(.R,"^NUPA(1927.2,","1927.2^4","")

1. .01 field
2. IEN
