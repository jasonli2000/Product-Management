NUPA FILE (2951)    REMOTE PROCEDURE (8994)

Name Value
NAME NUPA FILE
TAG FILE
ROUTINE NUPABCL
RETURN VALUE TYPE SINGLE VALUE
WORD WRAP ON TRUE
DESCRIPTION 
Files data into Fileman files.
INPUT PARAMETER
  1. INPUT PARAMETER:   DIE
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    The file to save data to, in regular fileman format.
  2. INPUT PARAMETER:   IEN
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    The DA variable for the DIE call, in regular fileman format.
  3. INPUT PARAMETER:   FIELD
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    The field number to file data into, in numeric format.
  4. INPUT PARAMETER:   VALUE
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   4
    DESCRIPTION:   
    The value to save to the field.
  5. INPUT PARAMETER:   SLASHES
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   5
    DESCRIPTION:   
    The number of slashes for the DIE call, typically 3 or 4.
RETURN PARAMETER DESCRIPTION 
Returns a 1 only.
