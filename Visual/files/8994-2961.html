NUPA DP COMMENT HISTORY (2961)    REMOTE PROCEDURE (8994)

Name Value
NAME NUPA DP COMMENT HISTORY
TAG DPH
ROUTINE NUPABCL2
RETURN VALUE TYPE GLOBAL ARRAY
WORD WRAP ON TRUE
DESCRIPTION 
Pulls in the comments for Discharge Planning issues for a care plan.
INPUT PARAMETER
  1. INPUT PARAMETER:   IEN
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    The IEN of the Care Plan from file 1927.4.
  2. INPUT PARAMETER:   FLAG
    PARAMETER TYPE:   LITERAL
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    Flag to determine whether to return the latest comment for each Discharge 
Planning issue, or the entire set of Discharge Planning comments for the
Care Plan.
 
FLAG=1 - Entire set of Discharge Planning comments
FLAG=2 - Latest comment for each Discharge Planning issue
RETURN PARAMETER DESCRIPTION 
Returns a list of comments in ^TMP($J).
 
^TMP(562303312,"LIST",1)=*** Prior comments ***
                      2)=  NONE
