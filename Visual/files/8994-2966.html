NUPA GET GI DEVICES (2966)    REMOTE PROCEDURE (8994)

Name Value
NAME NUPA GET GI DEVICES
TAG GI
ROUTINE NUPABCL2
RETURN VALUE TYPE GLOBAL ARRAY
WORD WRAP ON TRUE
DESCRIPTION 
Returns a list of all GI devices applicable to this care plan.
INPUT PARAMETER
  1. INPUT PARAMETER:   IEN
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    The IEN of the care plan.
RETURN PARAMETER DESCRIPTION 
A list of devices in format:
 
^TMP(563955298,1)=1^Colostomy bag^
               2)=2^G/J tube^
               3)=3^PEG^NOV 19, 2009@15:00
 
Piece 1: The IEN from file 1927.403
Piece 2: The type of device
Piece 3: Date inserted
Piece 4: Date removed
