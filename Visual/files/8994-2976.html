DSIROIR PRL PAT REQUEST LIST (2976)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIROIR PRL PAT REQUEST LIST
TAG PRL
ROUTINE DSIROIR
RETURN VALUE TYPE GLOBAL ARRAY
WORD WRAP ON TRUE
VERSION 8.2
DESCRIPTION 
This RPC returns a list of all internal entry numbers from file 19620 for 
a specified patient.
INPUT PARAMETER
  1. INPUT PARAMETER:   STDT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   7
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    This is the FileMan date the report should stop counting.  If null the 
report will count all records from the start date thru the current date.
  2. INPUT PARAMETER:   ENDT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   7
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    The last date to searc for a request IEN for the selected patient.
  3. INPUT PARAMETER:   PAT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   99
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    DFN of the selected patient.
RETURN PARAMETER DESCRIPTION 
The array contains the internal numbers of the entries in file 19620.
