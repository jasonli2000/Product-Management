DSIRRPTR GETRPT GET RPT DATA (2984)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIRRPTR GETRPT GET RPT DATA
TAG GETRPT
ROUTINE DSIRRPTR
RETURN VALUE TYPE GLOBAL ARRAY
WORD WRAP ON TRUE
DESCRIPTION 
Retrieves stored results from a specified report.
INPUT PARAMETER
  1. INPUT PARAMETER:   IEN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   20
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Internal entry number of the report in 19620.35
  2. INPUT PARAMETER:   STFM
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   99
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    This is the Internal Number from the word processing multiple in file 
19620.35 that was passed back as the last line sent in the small group 
return.
  3. INPUT PARAMETER:   RECS
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   6
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    This is the max number of records from the report text to return.
RETURN PARAMETER DESCRIPTION 
RETURNS ^ delimited array containing report data in formats that vary 
based on the report.  Report defined returns are as follows:
  
  Report/Piece/Field
  ====================================================
  REQUEST BY CLERK
  1 Division Name
  2 ROI Clerk Name
  3 Patient Name
  4 Patient SSN
  5 Type of Request
  6 External Date Received
  7 External Date Closed (If present)
 
  REQUESTS BY TYPE
  1 DIVISION 
  2 REQUESTOR TYPE
  3 STATUS
  4 CLOSED DATE
  5 DATE RECEIVED
  6 CORPORATE NAME
  7 LAST NAME
  8 FIRST NAME
  9 ADDRESS1
  10 ADDRESS2
  11 CITY
  12 STATE
  13 ZIP
  14 ADDRESS3
  15 PHONE
  16 FAX
 
  or
  -1^error message
