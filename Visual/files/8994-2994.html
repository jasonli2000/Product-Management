DSIRRPTF FOIA YEAR END REPORT (2994)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIRRPTF FOIA YEAR END REPORT
TAG FOIA
ROUTINE DSIRRPTF
RETURN VALUE TYPE GLOBAL ARRAY
WORD WRAP ON TRUE
DESCRIPTION 
This RPC will collect the data for the year end FOIA report for VA Form
0712
INPUT PARAMETER
  1. INPUT PARAMETER:   FRDT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   7
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    The FileMan Format date to start the report.  Typically this will be
October 1st of the previous year.
  2. INPUT PARAMETER:   TODT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   7
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    This is the end date for the report period.  Typically this will be
September 30th of the current year.
  3. INPUT PARAMETER:   DIV
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   20
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    This is the list of internal division for the current user or another 
division that the current user has access to.
  4. INPUT PARAMETER:   QUICK
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   1
    REQUIRED:   NO
    SEQUENCE NUMBER:   4
    DESCRIPTION:   
    0 - Calculate the report and save the results if report for a fiscal year
1 - Return the stored results of the report for a fiscal year
2 - Return the combined stored results and user add ins for a fiscal year
  5. INPUT PARAMETER:   SCHED
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   1
    REQUIRED:   NO
    SEQUENCE NUMBER:   5
    DESCRIPTION:   
    This is an optional boolean field to indicate whether of not the report
should be scheduled.
  6. INPUT PARAMETER:   ESTART
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   15
    REQUIRED:   NO
    SEQUENCE NUMBER:   6
    DESCRIPTION:   
    This is the earliest start time for a task to launch if the report is
scheduled.
RETURN PARAMETER DESCRIPTION 
RETURNS: GLOBAL ARRAY
  -1^ERROR MESSAGE
  OR
  task #^REPORT AFTER earliest report time
  OR
  The return array will have entries with 3 pieces.  The first piece will
  be the field name, the second piece will contain the value to go in the
  specified block, the third piece is a short description;
 
  Example: EXEMP1^12^EXEMPTION 1
