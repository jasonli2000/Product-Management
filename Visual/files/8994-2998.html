DSIRRPTR KCHK REPORT KILL CHK (2998)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIRRPTR KCHK REPORT KILL CHK
TAG KCHK
ROUTINE DSIRRPTR
RETURN VALUE TYPE SINGLE VALUE
DESCRIPTION 
This procedure is used to check to see if the report cleanup task needs to
be run. If yes, the DSIR REPORT KILL CHECK DATE parameter is updated and 
the cleanup task is queued.
RETURN PARAMETER DESCRIPTION 
OUTPUT IS EITHER 
  0 (ZERO) WHICH INDICATES NO ACTION NEEDED
  1 WHICH INDICATES THE REPORT CLEANUP TASK WAS SCHEDULED
