DSIF BATCH EDIT (3012)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIF BATCH EDIT
TAG EDITBAT
ROUTINE DSIFBAT4
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY AGREEMENT
WORD WRAP ON FALSE
VERSION 3.2
DESCRIPTION 
Used to edit an existing batch in Fee Basis
Returns 1^edit successfull 
or
-1^error message
INPUT PARAMETER
  1. INPUT PARAMETER:   FBIN
    PARAMETER TYPE:   LIST
    MAXIMUM DATA LENGTH:   999
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Pass in an array formatted as follows:
 
 ; FBIN(.01)=BATCH NUMBER
 ; FBIN(1)=OBLIGATION NUMBER  (DO NOT SEND STATION NUMBER WITH OBLIGATION 
"500-C09045", PASS ONLY "C09045")
 ; FBIN(3)=DATE OPENED
RETURN PARAMETER DESCRIPTION 
FBOUT=1^Edit was successful
 
OR
 
FBOUT=-1^error message
