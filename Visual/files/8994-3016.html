DENTV GET UNFILED DETAIL (3016)    REMOTE PROCEDURE (8994)

Name Value
NAME DENTV GET UNFILED DETAIL
TAG GETUN
ROUTINE DENTVTP7
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY AGREEMENT
INACTIVE ACTIVE
WORD WRAP ON TRUE
DESCRIPTION 
This procedure will take in a single IEN to return all data for the IEN 
in ^TMP("DENT",$J,N)=value.
INPUT PARAMETER
  1. INPUT PARAMETER:   IEN
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This input should be a number coinciding with an IEN for the record you 
wish to retreive.
RETURN PARAMETER DESCRIPTION 
The return global array is in format ^TMP("DENT",$J,N)=value. There will 
be M number of nodes within the return where N is the first record and M 
is the last record.
