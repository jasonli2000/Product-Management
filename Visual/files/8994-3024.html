DENTV DEL CANNED COMMENT (3024)    REMOTE PROCEDURE (8994)

Name Value
NAME DENTV DEL CANNED COMMENT
TAG DEL
ROUTINE DENTVCC
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY AGREEMENT
INACTIVE ACTIVE
DESCRIPTION 
Will delete a canned comment from the system. This will automatically 
resort the remaining records by filling in the gap (if any) left by 
removing the record.
INPUT PARAMETER
  1. INPUT PARAMETER:   CIEN
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is the IEN of the record.
RETURN PARAMETER DESCRIPTION 
Returns 1^message if successful, or -1^message if unsuccessful.
