DENTV TP RESERVED INACT CHECK (3026)    REMOTE PROCEDURE (8994)

Name Value
NAME DENTV TP RESERVED INACT CHECK
TAG CHKINA
ROUTINE DENTVTP7
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY AGREEMENT
DESCRIPTION 
This RPC will perform a check on the IEN inputted and return whether or 
not the transaction found has been inactivated (for view only) or left 
alone. The check will inactivate a reserved transaction if it is more 
than 8 days old.
INPUT PARAMETER
  1. INPUT PARAMETER:   DENTIEN
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This input value must be the IEN for the record contained in the DENTAL 
TP RESERVE (#228.7) file.
RETURN PARAMETER DESCRIPTION 
The return value will specify whether the reserved transaction has been 
inactivated due to being more than 8 days old, or whether the transaction 
is still allowed to be loaded.
