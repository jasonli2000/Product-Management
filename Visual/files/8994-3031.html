DSIF CNSLTS GET CONSULT DETAIL (3031)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIF CNSLTS GET CONSULT DETAIL
TAG DETAIL
ROUTINE DSIFCNS
RETURN VALUE TYPE ARRAY
AVAILABILITY AGREEMENT
WORD WRAP ON TRUE
DESCRIPTION 
This RPC will retrieve Consult data as discrete information to be used to 
auto-populate fields in the FBCS Authorization application.
INPUT PARAMETER
  1. INPUT PARAMETER:   DSIFCSLT
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is the Consult IEN (file #123) where the information is needed to be 
returned in a discrete format.
RETURN PARAMETER DESCRIPTION 
The return will contain the following information:
 DSIFRET(1)="-1^Error Message"
 
 -or-
 
 DSIFRET(1)=Patient^Internal;External
 DSIFRET(2)=Referral Date (System)^Internal;External
 DSIFRET(3)=Referring Provider^Internal;External
 DSIFRET(4)=Referral Date (Entered)^Internal;External
 DSIFRET(5)=Urgency^Internal;External
 DSIFRET(6)=Service Rendered as^I or O (Inpatient or Outpatient)
