KMPD USER PARAMETERS (3041)    REMOTE PROCEDURE (8994)

Name Value
NAME KMPD USER PARAMETERS
TAG USRPARAM
ROUTINE KMPDU6
RETURN VALUE TYPE ARRAY
AVAILABILITY RESTRICTED
WORD WRAP ON TRUE
DESCRIPTION 
Set/Get gui option info.
INPUT PARAMETER
  1. INPUT PARAMETER:   DUZ
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   35
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    User's DUZ.
  2. INPUT PARAMETER:   TYPE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   5
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    Type of call: 1 - get option info
              2 - set option info
  3. INPUT PARAMETER:   OPTION
    PARAMETER TYPE:   LIST
    MAXIMUM DATA LENGTH:   256
    REQUIRED:   YES
    SEQUENCE NUMBER:   3
RETURN PARAMETER DESCRIPTION 
If TYPE=1 - array containing the user's option settings in format:
                Piece 1: option name
                Piece 2: "" - no option setting - default to display
                          0 - do not display option
                          1 - display option
 
        ex: array(0)="Error List^1"
            array(1)="Global List^0"
 
 
   TYPE=2 - return <update complete>
