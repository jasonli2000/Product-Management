DGBT CLAIM DEDUCTIBLE PAID (3064)    REMOTE PROCEDURE (8994)

Name Value
NAME DGBT CLAIM DEDUCTIBLE PAID
TAG RPC
ROUTINE DGBTRDV
RETURN VALUE TYPE ARRAY
AVAILABILITY PUBLIC
WORD WRAP ON FALSE
DESCRIPTION 
THIS RPC IS USED BY BENEFICIARY TRAVEL PACKAGE TO RETRIEVE TRAVEL CLAIM 
INFORMATION ABOUT ANY TRAVEL CLAIMS FOR PATIENT.
INPUT PARAMETER
  1. INPUT PARAMETER:   PATIENT ICN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    PATIENTS ICN NUMBER FROM ^DPT(DFN).
  2. INPUT PARAMETER:   CLAIM DATE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   15
    REQUIRED:   YES
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    BENEFICIARY TRAVEL CLAIM DATE (NOT THE CURRENT DATE).
  3. INPUT PARAMETER:   DGBTRET
    PARAMETER TYPE:   REFERENCE
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    THE ARRAY THE DATA WILL BE RETURNED IN.
RETURN PARAMETER DESCRIPTION 
THE RPC WILL RETURN THE NUMBER OF CLAIMS AND AMOUNT OF DEDUCTIBLE PAID BY 
A PATIENT DURING THE CURRENT MONTH.
