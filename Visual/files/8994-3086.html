DENTV DPT GET ICN (3086)    REMOTE PROCEDURE (8994)

Name Value
NAME DENTV DPT GET ICN
TAG ICN
ROUTINE DENTVUTL
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY AGREEMENT
WORD WRAP ON TRUE
DESCRIPTION 
This RPC takes in a patient identifier and returns the patient's ICN.
INPUT PARAMETER
  1. INPUT PARAMETER:   PAT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    A patient identifier which can be their DFN, Name, or SSN.
  2. INPUT PARAMETER:   ISSSN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   1
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    A boolean flag (0,1) which indicates whether or not the PAT value is a 
SSN value.  If a value is not passed, the flag defaults to 0.
RETURN PARAMETER DESCRIPTION 
Returns a single value.
 
The format of the returned value will be as follows:
  Result = p1^p2
 
    p1 = Patient ICN (local ICN will include checksum)
    p2 = Location Flag (N: National; L: Local)
 
If an error occurs, the return will be Result = -1^message.
