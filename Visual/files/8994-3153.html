MAGV SET DGW CONFIG (3153)    REMOTE PROCEDURE (8994)

Name Value
NAME MAGV SET DGW CONFIG
TAG SETGWCFG
ROUTINE MAGVDGWP
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
DESCRIPTION 
Set DGW configuration settings in the MAGV GATEWAY CONFIGURATION
File (#2006.9191).

  1. INPUT PARAMETER:   HOSTNAME
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    NAME OF DICOM GATEWAY
  2. INPUT PARAMETER:   MAILGROUP
    PARAMETER TYPE:   LITERAL
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    EMAIL ADDRESSEE
  3. INPUT PARAMETER:   POSTOFFICE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    POST OFFICE SERVER
  4. INPUT PARAMETER:   POSTPORT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   5
    SEQUENCE NUMBER:   4
    DESCRIPTION:   
    POST OFFICE EMAIL PORT
  5. INPUT PARAMETER:   IMPORTER
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   3
    SEQUENCE NUMBER:   5
    DESCRIPTION:   
    Indicator to determine if DGW is using the IMPORTER.
Contents are YES or NO.
  6. INPUT PARAMETER:   LOCATION
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   8
    SEQUENCE NUMBER:   6
    DESCRIPTION:   
    DICOM GATEWAY LOCATION CODE
(Site Location/Station number)
RETURN PARAMETER DESCRIPTION 
SUCCESS OR ERROR CONDITION
  0 = Success
 -1 = Failure (error)
