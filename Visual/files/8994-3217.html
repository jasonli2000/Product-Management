DSIF CNSLTS DEL TRACKING (3217)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIF CNSLTS DEL TRACKING
TAG DELETE
ROUTINE DSIFCNS
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY AGREEMENT
DESCRIPTION 
This RPC will delete FBCS Consult Tracking Records (truly mark as hidden) 
so they will no longer be returned. The Consult Tracking records are not 
truly deleted as they are still being used to look for 
Comment/Significant Finding updates that may occur. After 90 days of 
inactivity, they are removed from the file.
INPUT PARAMETER
  1. INPUT PARAMETER:   DSIFCSLT
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is the IEN of the Consult you wish to delete (hide) from being 
returned.
RETURN PARAMETER DESCRIPTION 
The message returned will indicate success with a 1^Message or 
unsuccessful with a -1^Error message.
