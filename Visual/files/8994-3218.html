DSIF CNSLTS GET CONSULT (3218)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIF CNSLTS GET CONSULT
TAG GET
ROUTINE DSIFCNS
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY AGREEMENT
DESCRIPTION 
This RPC will return details for a single Consult that uses the same 
format as DSIF CNSLTS FEE CONSULTS.
INPUT PARAMETER
  1. INPUT PARAMETER:   DSIFIEN
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is the Consult IEN that the details are requested of.
RETURN PARAMETER DESCRIPTION 
The return is a single string that looks like the following:
 
DSIFRET=Consult IEN(1)^Date/time of Request[E:I] (2)^Status 
(3)^Consulting Service (4)^Procedure (5)^
 Patient Name (6)^Patient Last Initial & Last 4 SSN (7)^Patient DFN (8)
 
If an error occurs the return will be:
 
-1^Message
