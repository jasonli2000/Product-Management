DESCRIPTION 
This RPC allows for Significant Findings updates to be added to a Consult.
 
Cloned from ORQQCN SIG FIND
INPUT PARAMETER
  1. INPUT PARAMETER:   DSIFCIEN
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is the Consult IEN for the consult you wish to add a significant 
findings update to.
  2. INPUT PARAMETER:   DSIFSF
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   1
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    The flag for what type of significant finding update is being added:
 
'Y' - Significant Finding
'N' - No Significant Finding
'U' - Unknown Significant Finding
  3. INPUT PARAMETER:   DSIFCOM
    PARAMETER TYPE:   LIST
    MAXIMUM DATA LENGTH:   80
    REQUIRED:   YES
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    This is array of comments being added to the Consult along with the 
Significant Findings update.
  4. INPUT PARAMETER:   DSIFALRT
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   1
    REQUIRED:   YES
    SEQUENCE NUMBER:   4
    DESCRIPTION:   
    Flag to signal that alerts are to be sent:
 
'N' - NO
'Y' - YES
  5. INPUT PARAMETER:   DSIFALTO
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   80
    SEQUENCE NUMBER:   5
    DESCRIPTION:   
    A list of comma delimited NEW PERSON FILE (200) iens who need to be 
alerted to the significant finding update for this consult.
  6. INPUT PARAMETER:   DSIFDT
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   6
    DESCRIPTION:   
    The date of the significant finding update.
