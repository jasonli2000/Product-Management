DSIF CNSLTS DISCONTINUE (3229)    REMOTE PROCEDURE (8994)

Name Value
NAME DSIF CNSLTS DISCONTINUE
TAG DISCON
ROUTINE DSIFCNS1
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
DESCRIPTION 
This RPC allows for the discontinuing of a selected Consult.
 
Cloned from ORQQCN DISCONTINUE
INPUT PARAMETER
  1. INPUT PARAMETER:   DSIFCIEN
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is the Consult IEN that needs to be discontinued.
  2. INPUT PARAMETER:   DSIFPROV
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    Provider who Discontinued or Denied consult.
  3. INPUT PARAMETER:   DSIFDT
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    This is the date the consult was discontinued or denied.
  4. INPUT PARAMETER:   DSIFACTM
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   2
    REQUIRED:   YES
    SEQUENCE NUMBER:   4
    DESCRIPTION:   
    Flag used to determine if Consult was DISCONTINUED or DENIED.
 
'DC' - Discontinued
'DY' - Denied
  5. INPUT PARAMETER:   DSIFCOM
    PARAMETER TYPE:   LIST
    MAXIMUM DATA LENGTH:   80
    SEQUENCE NUMBER:   5
    DESCRIPTION:   
    The array of comments associated with the discontinue/deny for the 
consult.
RETURN PARAMETER DESCRIPTION 
Returns:
1^Successful
-1^Error
