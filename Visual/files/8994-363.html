PSB USERSAVE (363)    REMOTE PROCEDURE (8994)

Name Value
NAME PSB USERSAVE
TAG USRSAVE
ROUTINE PSBRPC
RETURN VALUE TYPE ARRAY
VERSION 2
DESCRIPTION 
Save the users current window settings for the next session.
INPUT PARAMETER
  1. INPUT PARAMETER:   PSBWIN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Contains the current window settings so that the next time the user 
activates the application the window will reappear in the same format.
 
Data is in the following format:
 
   Window Top;Window Left;Window Height;Window Width;Window State
  2. INPUT PARAMETER:   PSBVDL
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    VDL settings
  3. INPUT PARAMETER:   PSBUDCW
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    UD tab column settings
  4. INPUT PARAMETER:   PSBPBCW
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   4
    DESCRIPTION:   
    PB tab column settings
  5. INPUT PARAMETER:   PSBIVCW
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   5
    DESCRIPTION:   
    IV tab column settings
  6. INPUT PARAMETER:   PSBDEV
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   6
    DESCRIPTION:   
    Last print device used.
  7. INPUT PARAMETER:   PSBCSRT
    PARAMETER TYPE:   LITERAL
    SEQUENCE NUMBER:   7
  8. INPUT PARAMETER:   PSBCV1
    PARAMETER TYPE:   LITERAL
    SEQUENCE NUMBER:   8
  9. INPUT PARAMETER:   PSBCV2
    PARAMETER TYPE:   LITERAL
    SEQUENCE NUMBER:   9
  10. INPUT PARAMETER:   PSBCV3
    PARAMETER TYPE:   LITERAL
    SEQUENCE NUMBER:   10
  11. INPUT PARAMETER:   PSBCV4
    PARAMETER TYPE:   LITERAL
    SEQUENCE NUMBER:   11
  12. INPUT PARAMETER:   PSBORMODE
    PARAMETER TYPE:   LITERAL
    SEQUENCE NUMBER:   12
    DESCRIPTION:   
    Last Order mode per user upon exit of BCMA.
 
This parameter is for the last Order Mode the user was using upon exiting 
BCMA.  
 
Value
  0    Inpatient order mode
  1    Clinic order mdoe
  13. INPUT PARAMETER:   PSBCLSRCH
    PARAMETER TYPE:   LITERAL
    SEQUENCE NUMBER:   13
    DESCRIPTION:   
    Last Clinic Search text criteria entered per user.  
This parameter contains two pieces of data delimited by a "/".  First 
piece of data is the Clinic name search Prefix text and the second is the 
Clinic name contains search text.
 
Example:
 AL-BLUE TEAM/CARDIAC
RETURN PARAMETER DESCRIPTION 
Returns
 
  Results[0]='1^Saved'
  Results[0]='-1^Unable to save settings' if an error occurs in filing.
