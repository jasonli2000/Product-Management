XWB EGCHO SORT LIST (4)    REMOTE PROCEDURE (8994)

Name Value
NAME XWB EGCHO SORT LIST
TAG SRT
ROUTINE XWBZ1
RETURN VALUE TYPE ARRAY
AVAILABILITY RESTRICTED
DESCRIPTION 
Sorts a given numeric array, starting from HI or LO
It exists for support of EGcho - Broker demonstration program.

  1. INPUT PARAMETER:   DIRECTION
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   2
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    The string LO or HI
  2. INPUT PARAMETER:   ARRAY
    PARAMETER TYPE:   LIST
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    The array of numbers.  Pass using . syntax
X(12)="",X(23)="",... for example
RETURN PARAMETER DESCRIPTION 
Sorted array
