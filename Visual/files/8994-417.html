TIU LONG LIST BOILERPLATED (417)    REMOTE PROCEDURE (8994)

Name Value
NAME TIU LONG LIST BOILERPLATED
TAG LONGLIST
ROUTINE TIUSRVT1
RETURN VALUE TYPE ARRAY
AVAILABILITY RESTRICTED
WORD WRAP ON TRUE
DESCRIPTION 
Used by the GUI to supply a long list of boilerplated titles.
INPUT PARAMETER
  1. INPUT PARAMETER:   FROM
    PARAMETER TYPE:   LITERAL
    DESCRIPTION:   
    This is the reference title from which the longlist is scrolling.
  2. INPUT PARAMETER:   DIR
    PARAMETER TYPE:   LITERAL
    DESCRIPTION:   
    This is the direction in which the longlist is scrolling from the
reference title.
RETURN PARAMETER DESCRIPTION 
This is an array of the 44 nearest titles to that indicate by the user in
the direction passed by the longlist component.
