TIU GET SITE PARAMETERS (425)    REMOTE PROCEDURE (8994)

Name Value
NAME TIU GET SITE PARAMETERS
TAG SITEPARM
ROUTINE TIUSRVP1
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY SUBSCRIPTION
DESCRIPTION 
This RPC returns the TIU Parameters for the Division the user is logged in 
to.
RETURN PARAMETER DESCRIPTION 
The call returns a '^'-delimited string, with the following pieces:

Y = INSTITUTION [1P] ^ ENABLE ELECTRONIC SIGNATURE [2S] ^ CHARACTERS PER LINE
==> [3N] ^ GRACE PERIOD FOR PURGE [4N] ^ GRACE PERIOD FOR SIGNATURE [5N] ^
==> OPTIMIZE LIST BUILDING FOR [6S] ^ SUPPRESS REVIEW NOTES PROMPT [7S] ^
==> DEFAULT PRIMARY PROVIDER [8S] ^ ASCII UPLOAD SOURCE [9S] ^ RECORD
==> HEADER SIGNAL [10F] ^ END OF MESSAGE SIGNAL [11F] ^ BEGIN REPORT TEXT
==> SIGNAL [12F] ^ MAJOR DELIMITER [13F] ^

The pieces correspond to fields .01 through .13 in the TIU PARAMETERS FILE 
(#8925.99).
