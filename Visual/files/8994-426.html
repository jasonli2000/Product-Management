TIU IS USER A PROVIDER? (426)    REMOTE PROCEDURE (8994)

Name Value
NAME TIU IS USER A PROVIDER?
TAG ISAPROV
ROUTINE TIUSRVP1
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY SUBSCRIPTION
DESCRIPTION 
This Boolean RPC returns TRUE if the user was a known provider on the date 
specified.
INPUT PARAMETER
  1. INPUT PARAMETER:   USER
    PARAMETER TYPE:   LITERAL
    REQUIRED:   NO
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Optional pointer to the NEW PERSON FILE (#200). If user is unspecified the 
current user will be assumed.
  2. INPUT PARAMETER:   DATE
    PARAMETER TYPE:   LITERAL
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    Optional FileMan formatted date (time). If none is provided the current date 
is assumed.
RETURN PARAMETER DESCRIPTION 
0=FALSE
1=TRUE
