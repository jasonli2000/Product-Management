NAME ORWDPS32 AUTH
TAG AUTH
ROUTINE ORWDPS32
RETURN VALUE TYPE SINGLE VALUE
DESCRIPTION 
Checks restrictions for entering inpatient meds.  If no restrictions, a 0
is returned.  If there is a restriction, it is returned in the format:
     1^restriction text
