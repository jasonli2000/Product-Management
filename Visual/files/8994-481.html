ORQQCN DISCONTINUE (481)    REMOTE PROCEDURE (8994)

Name Value
NAME ORQQCN DISCONTINUE
TAG DC
ROUTINE ORQQCN1
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY SUBSCRIPTION
DESCRIPTION 
Discontinue a consult or deny a consult request.
INPUT PARAMETER
  1. INPUT PARAMETER:   CONSULT ID
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    REQUIRED:   YES
    DESCRIPTION:   
    Internal file number of the consult/request.
  2. INPUT PARAMETER:   PROVIDER ID
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    REQUIRED:   YES
    DESCRIPTION:   
    Internal file number of the person discontinuing/denying the consult.
Points to NEW PERSON file (#200)
  3. INPUT PARAMETER:   DATE OF ACTION
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    REQUIRED:   YES
    DESCRIPTION:   
    Date of the discontinuance/denial of the consult.
  4. INPUT PARAMETER:   ACTION TYPE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    REQUIRED:   YES
    DESCRIPTION:   
    'DC' for discontinue, 'DY' for deny
  5. INPUT PARAMETER:   COMMENTS
    PARAMETER TYPE:   WORD PROCESSING
    REQUIRED:   YES
    DESCRIPTION:   
    Array of comments related to the discontinuance/denial of the consult.
RETURN PARAMETER DESCRIPTION 
'1^Error message' if unsuccessful
'0' if successful
