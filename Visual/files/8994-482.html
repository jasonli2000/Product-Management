ROUTINE ORQQCN1

DESCRIPTION 
Forwards a consult to a subservice of the forwarding service, as defined
in file 123.5
INPUT PARAMETER
  1. INPUT PARAMETER:   CONSULT ID
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    REQUIRED:   YES
    DESCRIPTION:   
    Consult IEN from file 123
  2. INPUT PARAMETER:   TO SERVICE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    REQUIRED:   YES
    DESCRIPTION:   
    Service to which the consult is to be forwarded.  Pointer to file 123.5
  3. INPUT PARAMETER:   FORWARDER
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    REQUIRED:   YES
    DESCRIPTION:   
    Person performing the forwarding action.  If null, defaults to user DUZ.
Points to NEW PERSON file, #200.
  4. INPUT PARAMETER:   ATTENTION OF
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    DESCRIPTION:   
    Person to whose attention the consult is being forwarded in the
destination service.  Pointer to NEW PERSON file, #200.
  5. INPUT PARAMETER:   URGENCY
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    DESCRIPTION:   
    Urgency of the request, as defined in PROTOCOL file.
  6. INPUT PARAMETER:   COMMENTS
    PARAMETER TYPE:   LIST
    DESCRIPTION:   
    Comments related to the forwarding action.
RETURN PARAMETER DESCRIPTION 
'0' if successful
'1^error message' if unsuccessful
