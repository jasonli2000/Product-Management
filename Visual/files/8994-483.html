ORQQCN SET ACT MENUS (483)    REMOTE PROCEDURE (8994)

Name Value
NAME ORQQCN SET ACT MENUS
TAG SETACTM
ROUTINE ORQQCN1
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY SUBSCRIPTION
DESCRIPTION 
Based on the IEN of the consult passed in, returns a string representing
various facets of the user's access level for that consult and service.
This allows dynamic enabling/disabling of GUI menus based on the user's
ability to act on that particular consult.
INPUT PARAMETER
  1. INPUT PARAMETER:   CONSULT ID
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    REQUIRED:   YES
    DESCRIPTION:   
    Internal file number of the selected consult.
RETURN PARAMETER DESCRIPTION 
 Piece 1 :    1: equivalent to GMRCACTM USER REVIEW SCREEN - simple
                 actions
              2: equivalent to GMRCACTM SERVICE ACTION menu  - all actions
                 possible to a service user
              3: equivalent to GMRCACTM TRK MEDICINE CONSULTS menu - all
                 actions that the user of the medicine package can take.
  
 Piece 2 :    field in file 123.5 (REQUEST SERVICES) that gave the user
              update authority (ex.  Update user w/o Notification)
  
 Piece 3 :    Service in file 123.5 (REQUEST SERVICES) that gave the user
              update authority (ex. CARDIOLOGY,NEUROLOGY)
  
 Piece 4 :    contains a 1 if user is allowed to associate medicine
              results with a consult procedure request
  
 Piece 5 :    contains a 1 if user can disassociate a medicine result that
              was incorrectly associated with a consult procedure request
  
 Piece 6 :    contains a 1 if user is allowed to EDIT and RESUBMIT a
              canceled request
  
 Piece 7 :    0 - Item is not a Clinical Procedure
              1 - Item is a Clinical Procedure, but no instrument results
                  will be associated with it.
              2 - Item is a Clinical Procedure, but instrument results are
                  not yet available for interpretation.
              3 - Item is a Clinical Procedure, instrument results are
                  ready for interpretation, but interpretation has not
                  yet been completed.
              4 - Item is a Clinical Procedure, and instrument results
                  interpretation is complete.
