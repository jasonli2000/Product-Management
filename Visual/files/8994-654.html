ORWRA DEFAULT EXAM SETTINGS (654)    REMOTE PROCEDURE (8994)

Name Value
NAME ORWRA DEFAULT EXAM SETTINGS
TAG GETDEF
ROUTINE ORWRA
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
DESCRIPTION 
This RPC returns the default settings for the display of imaging exams on
the reports tab.
RETURN PARAMETER DESCRIPTION 
The format of the returned string is as follows (pieces separated by '^')
 
  Piece         Description
  -----         -----------
    1           Beginning date for search
    2           End date for search
    3           Maximum number of exams
