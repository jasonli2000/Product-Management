ORWTIU WINPRINT NOTE (675)    REMOTE PROCEDURE (8994)

Name Value
NAME ORWTIU WINPRINT NOTE
TAG PRINTW
ROUTINE ORWTIU
RETURN VALUE TYPE GLOBAL ARRAY
WORD WRAP ON TRUE
DESCRIPTION 
Returns a formatted global of a TIU document for output to a Windows print
device.
INPUT PARAMETER
  1. INPUT PARAMETER:   Note IEN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   20
    REQUIRED:   YES
    DESCRIPTION:   
    IEN of document in file 8925.
  2. INPUT PARAMETER:   Chart Copy?
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   2
    DESCRIPTION:   
    If true, print chart copy, else print work copy.
RETURN PARAMETER DESCRIPTION 
Formatted global containing the text of the document.
