ORWRP2 HS SUBITEMS (680)    REMOTE PROCEDURE (8994)

Name Value
NAME ORWRP2 HS SUBITEMS
TAG SUBITEM
ROUTINE ORWRP2
RETURN VALUE TYPE ARRAY
AVAILABILITY RESTRICTED
DESCRIPTION 
This RPC expands a Laboratory Test panel to all it's sub-components for
selection in the ADHOC Health Summary.
INPUT PARAMETER
  1. INPUT PARAMETER:   TEST
    PARAMETER TYPE:   LITERAL
    DESCRIPTION:   
    This is the pointer value to the Laboratory Test (#60) file of the panel
to be expanded.
RETURN PARAMETER DESCRIPTION 
The array returned is a list of all the components of a Laboratory test
panel.  Each array element has the following data:
 Y(i)=test ptr^zero node of ^LAB(60,ifn,0)
