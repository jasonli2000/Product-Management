ORQQPXRM REMINDERS APPLICABLE (719)    REMOTE PROCEDURE (8994)

Name Value
NAME ORQQPXRM REMINDERS APPLICABLE
TAG APPL
ROUTINE ORQQPXRM
RETURN VALUE TYPE ARRAY
AVAILABILITY SUBSCRIPTION
WORD WRAP ON TRUE
DESCRIPTION 
Returns a list of clinical reminders due/applicable or not applicable to
the patient.
INPUT PARAMETER
  1. INPUT PARAMETER:   PATIENT ID
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Patient identifier from the patient file [#2]
  2. INPUT PARAMETER:   HOSPITAL LOCATION
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   16
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    Patients location from HOSPITAL LOCATION [#44]. May be null.
RETURN PARAMETER DESCRIPTION 
List of reminders in format:
 IEN^PRINT NAME^DUE DATE/TIME^LAST OCCURRENCE DATE/TIME^PRIORITY^DUE
   
 DUE=0 - Applicable
 DUE=1 - Due
