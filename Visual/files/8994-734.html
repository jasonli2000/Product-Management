ORWPCE MH TEST AUTHORIZED (734)    REMOTE PROCEDURE (8994)

Name Value
NAME ORWPCE MH TEST AUTHORIZED
TAG MHATHRZD
ROUTINE ORWPCE2
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
WORD WRAP ON TRUE
DESCRIPTION 
Indicates if a given mental health test can be given by the given user.
INPUT PARAMETER
  1. INPUT PARAMETER:   TEST
    PARAMETER TYPE:   LITERAL
    DESCRIPTION:   
    MENTAL HEALTH TEST NAME
  2. INPUT PARAMETER:   USER
    PARAMETER TYPE:   LITERAL
    DESCRIPTION:   
    The user giving the test.
RETURN PARAMETER DESCRIPTION 
Returns a 1 (TEST AUTHORIZED) or a 0 (TEST NOT AUTHORIZED).
