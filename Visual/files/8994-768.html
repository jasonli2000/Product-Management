TIU DIV AND CLASS INFO (768)    REMOTE PROCEDURE (8994)

Name Value
NAME TIU DIV AND CLASS INFO
TAG USERINFO
ROUTINE TIUSRVT1
RETURN VALUE TYPE ARRAY
AVAILABILITY RESTRICTED
WORD WRAP ON TRUE
DESCRIPTION 
Returns a list of Divisions and User Classes for a specific User.
INPUT PARAMETER
  1. INPUT PARAMETER:   USER
RETURN PARAMETER DESCRIPTION 
Array in the following format:
D^Division IEN
C^User Class IEN
First Division should be DUZ(2).
