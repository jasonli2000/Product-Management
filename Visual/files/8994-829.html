ORWORB KILL EXPIR OI ALERT (829)    REMOTE PROCEDURE (8994)

Name Value
NAME ORWORB KILL EXPIR OI ALERT
TAG KILEXOI
ROUTINE ORWORB
RETURN VALUE TYPE SINGLE VALUE
DESCRIPTION 
Evaluate expiring flagged orderable item orders. If none remain, kill
current alert for current user.  Kill for other users if alert so defined.
INPUT PARAMETER
  1. INPUT PARAMETER:   Patient DFN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   32
    REQUIRED:   YES
    DESCRIPTION:   
    Patient DFN
  2. INPUT PARAMETER:   Alert type
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   8
    REQUIRED:   YES
    DESCRIPTION:   
    Pointer to type of alert in the OE/RR NOTIFICATIONS file (100.9). This can
either be of type FLAGGED OI EXPIRING - INPT (#64) or FLAGGED OI EXPIRING
- OUTPT (#65).
