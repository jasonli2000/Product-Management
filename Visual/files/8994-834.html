ORWTIU IDNOTES INSTALLED (834)    REMOTE PROCEDURE (8994)

Name Value
NAME ORWTIU IDNOTES INSTALLED
TAG IDNOTES
ROUTINE ORWTIU
RETURN VALUE TYPE SINGLE VALUE
DESCRIPTION 
Returns "1" if TIU*1.0*100 is installed, "0" otherwise.  This lets the
CPRS GUI enable or disable the Interdisciplinary Notes functionality until
the server is ready.
RETURN PARAMETER DESCRIPTION 
See description.
