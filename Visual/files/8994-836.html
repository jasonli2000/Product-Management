ORQQPX SET FOLDERS (836)    REMOTE PROCEDURE (8994)

Name Value
NAME ORQQPX SET FOLDERS
TAG SETFLDRS
ROUTINE ORQQPX
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
WORD WRAP ON TRUE
DESCRIPTION 
Sets the value of the ORQQPX REMINDER FOLDERS parameter for the
current user.
INPUT PARAMETER
  1. INPUT PARAMETER:   ORFLDRS
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    String containing a one character code for each folder visible in the
reminder tree structure of the CPRS GUI.

Visible Folder Codes are:
  D=Due
  A=Applicable
  N=Not Applicable
  E=Evaluated
  O=Other
