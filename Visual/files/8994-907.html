DESCRIPTION

FILE #720.3.

    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    DESCRIPTION:

The return variable contains the following data separated by up-arrows.

  2. synonym

  4. associated clinic
  5. Procedure reason indicator
  6. Status
