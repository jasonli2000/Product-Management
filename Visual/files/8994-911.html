EC GETUSRDSSUNIT (911)    REMOTE PROCEDURE (8994)

Name Value
NAME EC GETUSRDSSUNIT
TAG USRUNT
ROUTINE ECUERPC
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY RESTRICTED
WORD WRAP ON TRUE
DESCRIPTION 
Returns an array of DSS units for which the user has access.
INPUT PARAMETER
  1. INPUT PARAMETER:   ECARY
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    DESCRIPTION:   
    The input variable ECARY contains a value from the location or New Person file.
   1. ECL   - Location IEN, #4(if define gives User's DSS units for a location)
   2. ECDUZ - New Person IEN, #200 (if define gives list of DSS Units available
              to user)
RETURN PARAMETER DESCRIPTION 
Returns an array of DSS Units. Data pieces separated by an up-arrow as follows:-
   1. IEN of file 724
   2. Name of DSS Unit
   3. Send to PCE Flag
