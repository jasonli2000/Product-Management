EC GETECSPROCS (913)    REMOTE PROCEDURE (8994)

Name Value
NAME EC GETECSPROCS
TAG PROC
ROUTINE ECUERPC
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY RESTRICTED
WORD WRAP ON TRUE
DESCRIPTION 
Returns an array of procedures for an Event Code screen (file #720.3). Event
code screens are based on location, DSS unit and Category.
INPUT PARAMETER
  1. INPUT PARAMETER:   ECARY
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    DESCRIPTION:   
    Input variable ECARY contains the following values separated by up-arrow.
  1.  ECL  - Location IEN
  2.  ECD  - DSS Unit IEN
  3.  ECC  - Category IEN
RETURN PARAMETER DESCRIPTION 
Returns an array of procedures. Data pieces as follows:-
   1.  EC National Number SPACE Procedure Name SPACE [Synonym]
   2.  Procedure Code
   3.  CPT Code
   4.  Default volume
   5.  Event code screen IEN
