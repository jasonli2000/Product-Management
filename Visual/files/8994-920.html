EC GETPROVIDER (920)    REMOTE PROCEDURE (8994)

Name Value
NAME EC GETPROVIDER
TAG PRVDER
ROUTINE ECUERPC
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY AGREEMENT
WORD WRAP ON TRUE
DESCRIPTION 
Returns an array of valid providers based on a procedure date.
INPUT PARAMETER
  1. INPUT PARAMETER:   ECARY
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    DESCRIPTION:   
    Input variable ECARY contains the procedure date.
RETURN PARAMETER DESCRIPTION 
Returns an array of active providers. Data pieces are as follows:-
  1. IEN of file 200
  2. Provider Name
  3. Occupation
  4. Speciality
  5. Subspeciality
