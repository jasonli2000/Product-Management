EC GETPATELIG (923)    REMOTE PROCEDURE (8994)

Name Value
NAME EC GETPATELIG
TAG ELIG
ROUTINE ECUERPC
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY AGREEMENT
WORD WRAP ON TRUE
DESCRIPTION 
Returns a list of patient eligibilities.
INPUT PARAMETER
  1. INPUT PARAMETER:   ECARY
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    DESCRIPTION:   
    Input variable, ECARY contains the patient IEN (#2).
RETURN PARAMETER DESCRIPTION 
Returns an array of patient eligibilities as follows:-
  1. Primary/secondary Elig flag (Flag 1-primary, 0-secondary)
  2. Eligibility ien
  3. Eligibility description
