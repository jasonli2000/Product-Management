EC GETDSSUNIT (926)    REMOTE PROCEDURE (8994)

Name Value
NAME EC GETDSSUNIT
TAG DSSUNT
ROUTINE ECUMRPC1
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY AGREEMENT
WORD WRAP ON TRUE
DESCRIPTION 
Returns array with active and/or inactive DSS units from file 724.
INPUT PARAMETER
  1. INPUT PARAMETER:   ECARY
    PARAMETER TYPE:   LITERAL
    DESCRIPTION:   
    Optional input variable ECARY containing a status of:
   STAT   - Active or inactive DSS Units (optional)
            A-ctive (default), I-nactive, B-oth
RETURN PARAMETER DESCRIPTION 
The results array contains data for DSS units as follows:-
    1.  IEN of DSS Unit
    2.  Name of DSS Unit
    3.  Service
    4.  Medical Speciality
    5.  Cost Center
    6.  Unit Number
    7.  Inactive Flag
    8.  Associated Stop Code
    9.  Category flag
    10. Default date entry
    11. Send to PCE Flag
