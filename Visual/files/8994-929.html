EC DSSCATCHECK (929)    REMOTE PROCEDURE (8994)

Name Value
NAME EC DSSCATCHECK
TAG CATCHK
ROUTINE ECUMRPC1
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
WORD WRAP ON TRUE
DESCRIPTION 
Checks whether category is used in an Event Code Screen.
INPUT PARAMETER
  1. INPUT PARAMETER:   ECARY
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    DESCRIPTION:   
    Variable ECARY contains the DSS Unit ien (file #724) that will be checked to 
determine if category exist.
RETURN PARAMETER DESCRIPTION 
Returns a value of 1 or 0 indicating whether category is used in Event Code 
Screen, 1-Yes or 0-No.
