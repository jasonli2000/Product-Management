MAGGLISTPROC (940)    REMOTE PROCEDURE (8994)

Name Value
NAME MAGGLISTPROC
TAG LIST
ROUTINE MAGGTMC
RETURN VALUE TYPE ARRAY
AVAILABILITY RESTRICTED
DESCRIPTION 
Return a list of Medicine procedures/subspecialities.
List can be subset of whole if site uses Imaging Capture Security Keys to 
limit the users access to only Medicine Procedures 
that they have a MAGCAP ... Key for.
INPUT PARAMETER
  1. INPUT PARAMETER:   MAGGZY
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   10
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Not using this input for any processing in Version 2.5
It might be used for a future version.
Kept in the parameter list, for backward compatibility.
---
If MAGGZY=1, then add procedure PRINT NAME (full name) in output.
RETURN PARAMETER DESCRIPTION 
Returns list of:  NAME       PRINT NAME  ^     GLOBAL ^   IEN
   i.e.     "ECG        ELECTROCARDIOGRAM^MCAR(691.5^2"
