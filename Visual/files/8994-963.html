NAME MAGGACTION LOG
TAG LOGACT
ROUTINE MAGGTU6
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
DESCRIPTION 
Call from Delphi to log an action performed on the image. 
Actions are logged the IMAGE ACCESS LOG file  ^MAG(2006.95
INPUT PARAMETER
  1. INPUT PARAMETER:   DATA
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   245
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    '^' delimited string contains the following:
'^' piece     description
   1   ACCESS TYPE -  A|B|C|D|E
   2   USER        -  DUZ
   3   IMAGE       -  MAGIEN
   4   USER INTERFACE SOFTWARE   - STRING
   5   PATIENT     -  DFN
   6   IMAGE COUNT
   7   ADDITIONAL DATA
RETURN PARAMETER DESCRIPTION 
Returns   "1^" if success.
Returns   "0^error text"  if failure.
