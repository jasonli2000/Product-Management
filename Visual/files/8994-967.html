MAGG VERIFY ESIG (967)    REMOTE PROCEDURE (8994)

Name Value
NAME MAGG VERIFY ESIG
TAG ESIG
ROUTINE MAGQLOG
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
DESCRIPTION 
Verifies the Electronic signature of the current user.  
Uses 'DUZ' defined in the partition.
INPUT PARAMETER
  1. INPUT PARAMETER:   ESIG
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   20
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    This is the encrypted electronic signature entered by the user on the
Imaging window.
RETURN PARAMETER DESCRIPTION 
Returns   "1^message"   for success
          "0^message"   for failure or error.
