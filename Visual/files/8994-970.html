MAGG SYS WRKS DISPLAY (970)    REMOTE PROCEDURE (8994)

Name Value
NAME MAGG SYS WRKS DISPLAY
TAG WRKS
ROUTINE MAGGTSY
RETURN VALUE TYPE GLOBAL ARRAY
AVAILABILITY RESTRICTED
WORD WRAP ON TRUE
DESCRIPTION 
Utilites for Imaging system manager.
RPC Call for MAGSYS utility. List of Workstation Information.
INPUT PARAMETER
  1. INPUT PARAMETER:   ZYx
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   60
    REQUIRED:   NO
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    the Computer Name running MAGSYS.
RETURN PARAMETER DESCRIPTION 
Returns the Imaging windows workstation file entries.
each entry is an '^' delimited piece of the following info.
$p(1)  Computer Name
$p(2)  Date of Last Logon, by a user.
$p(3)  Last User DFN
$p(4)  Active or Not
$p(5) Service/Section
$p(6)  Last User Name
$p(7) Date of Display exe last used.
$p(8) Date/Time of Last Logoff.
$p(9)  Display Version IMGVWP10.EXE
$p(10)  Capture Version info TELE19N.EXE
$p(11) Date of MAGSETUP.EXE last run on this wrks.
$p(12) Date of Capture exe last used.
$p(13)  current session error count.
$p(14) Sess Ct for Wrks
$p(15)  WRKS IEN  ^MAG(2006,81,I
