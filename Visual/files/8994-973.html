MAGG IMAGE DELETE (973)    REMOTE PROCEDURE (8994)

Name Value
NAME MAGG IMAGE DELETE
TAG IMAGEDEL
ROUTINE MAGGTID
RETURN VALUE TYPE ARRAY
AVAILABILITY RESTRICTED
DESCRIPTION 
Deletes Image entry from Image file, ^MAG(2005 and Image File from disk.
INPUT PARAMETER
  1. INPUT PARAMETER:   MAGIEN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    Internal entry number in the IMAGE FM File.  ^MAG(2005
  2. INPUT PARAMETER:   MAGGRPDF
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   1
    REQUIRED:   NO
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    This is a Group Delete Flag
1 or 0
     1  =  Allow Group Delete
     0  =  Do not Allow Group Deletes.
  3. INPUT PARAMETER:   REASON
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   60
    REQUIRED:   NO
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    This is a free text description of the Reason for Deleting the Image.
RETURN PARAMETER DESCRIPTION 
  1^Image deleted
  0^<reason image wasn't deleted>
