MAGG QUE IMAGE (978)    REMOTE PROCEDURE (8994)

Name Value
NAME MAGG QUE IMAGE
TAG QUEIMAGE
ROUTINE MAGGTU7
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
DESCRIPTION 
Queues images to be copied from JukeBox to harddrive.
 (to the Image Write Directory)
INPUT PARAMETER
  1. INPUT PARAMETER:   CODE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   5
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    A Code containing A | F | B 
   if [ A  then Abstracts will be queued.
   if [ F  then Full Resolution image will be queued
   if [ B  then BIG file ( Xray 2k x 2k ) will be queued
  2. INPUT PARAMETER:   MAGIEN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   20
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    This is the IEN from Image File ^MAG(2005  to be queued.
RETURN PARAMETER DESCRIPTION 
 1^message for success
 0^message for error
