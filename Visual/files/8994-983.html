MAGG OFFLINE IMAGE ACCESSED (983)    REMOTE PROCEDURE (8994)

Name Value
NAME MAGG OFFLINE IMAGE ACCESSED
TAG MAIL
ROUTINE MAGGTU3
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
DESCRIPTION 
This sends a message when an image has been accessed from a Jukebox
platter that is offline, as determined by having an entry in 
the file ^MAGQUEUE(2006.033.
INPUT PARAMETER
  1. INPUT PARAMETER:   MAGFILE
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   90
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    The full path and file name of the Image accessed.
  2. INPUT PARAMETER:   MAGIEN
    PARAMETER TYPE:   LITERAL
    MAXIMUM DATA LENGTH:   30
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    The Internal entry number in file MAG(2005.
RETURN PARAMETER DESCRIPTION 
always returns "1^message sent"
