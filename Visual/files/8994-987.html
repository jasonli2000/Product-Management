MAGQ GET (987)    REMOTE PROCEDURE (8994)

Name Value
NAME MAGQ GET
TAG GETQUE
ROUTINE MAGQBTM
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
DESCRIPTION 
Returns parameters for the next queue process for the specified type.
INPUT PARAMETER
  1. INPUT PARAMETER:   ACTION
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    The background processor queue type
(JBTOHD,JUKEBOX,DELETE,ABSTRACT,ROUTING).
RETURN PARAMETER DESCRIPTION 
The internal entry number (IEN) of the Mag Queue file (#2006.03) of
the next queue to be processed.
