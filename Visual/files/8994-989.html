MAGQ FS UPDATE (989)    REMOTE PROCEDURE (8994)

Name Value
NAME MAGQ FS UPDATE
TAG FSUPDT
ROUTINE MAGQBUT1
RETURN VALUE TYPE SINGLE VALUE
AVAILABILITY RESTRICTED
DESCRIPTION 
Updates the online file server entries in the Network location file
with the disk size and the space available.
INPUT PARAMETER
  1. INPUT PARAMETER:   IEN
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   1
    DESCRIPTION:   
    The network location file (#2005.2) internal entry number of the record
to be updated.
  2. INPUT PARAMETER:   SPACE
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   2
    DESCRIPTION:   
    The amount of space available for copying images.
  3. INPUT PARAMETER:   SIZE
    PARAMETER TYPE:   LITERAL
    REQUIRED:   YES
    SEQUENCE NUMBER:   3
    DESCRIPTION:   
    The disk capacity of the network share.
RETURN PARAMETER DESCRIPTION 
"1^Update Complete"
