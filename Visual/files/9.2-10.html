NAME XQHELP-DISP
HEADER USING THE 'Display/Edit Help Frame' OPTION
TEXT 
This option is used to view help frames, and edit them, if the
user has [EDIT PRIVILEGES].
 
The text of the frame is displayed, with the keywords highlighted
in reverse video or brackets.  A keyword indicates that there is more
detailed information about the topic indicated.  Entering that keyword
in response to the 'Select Action' prompt will display the frame
containing the related information.
 
Other responses to the 'Select Action' prompt will enable you to
display a list of keywords and topics, switch from brackets to reverse
video and vice versa, refresh the screen, and edit it, providing
edit privileges are established.
 
[BUILDING] a new help frame can be done by responding with a new name
to the first 'HELP FRAME' prompt.
