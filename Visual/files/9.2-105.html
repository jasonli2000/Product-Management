NAME XUFILE ACCESS USE3
HEADER LEVELS OF ACCESS AUTHORITY
TEXT 
  READ ACCESS   permits display of data on file.  This allows use of the VA
                FileMan 'Print', 'Search', 'Inquire', 'Statistics', and
                'List File Attributes' options.
 
 WRITE ACCESS   permits changing data values that are currently on file via
                the 'Enter/Edit' option.  It will not permit the adding of
                new entries to the file.
 
DELETE ACCESS   permits deletion of an entire file entry with 'Enter/Edit'
                or transfer of entries with 'Transfer'.  It does not permit
                deletion of the file or any of its attribute fields.
 
 LAYGO ACCESS   permits adding a new entry to a file with the 'Enter/Edit'
                option as long as write access has also been enabled.
 
    DD ACCESS   permits data dictionary access for changing or deleting the
                structure of the file itself.  The 'Modify' and 'Utility'
                options are enabled.
 
 AUDIT ACCESS   allows the [AUDIT] of data changes provided that DD access
                has also been granted.                           [CONTINUE]
