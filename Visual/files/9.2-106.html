NAME XUFILE ACCESS USE4
HEADER HOW IS FILE ACCESS GRANTED?
TEXT 
The options for granting file access privileges fall into three functional
categories: 
 
      EDITING        to assign file access to an individual user or a group
                     of users.  One user's profile may also be duplicated or
                     copied to another user or group of users.  To simplify
                     adding files, number ranges may be specified.
 
      LISTING        to display one user's profile or a name-sorted list of
                     all user's profiles.  Conversely, the file or range of
                     files may be printed with associated users and the
                     access levels of each.
 
     RESTRICTING     to entirely limit access by user or by file, or to
                     delele a range of files for a user or group of users.
 
The options are designed to facilitate queries by user or by file.  You may
add or delete file access for one user or for many users.  Or you may begin
with the file and list users with access or restrict access.  When using the
options, there are a few [HINTS] for getting around.
