NAME XQHELP-FILE
HEADER THE HELP FRAME FILE
TEXT 
The Help Frame file is exported with the Kernel.  Some of the fields are:
 
NAME         - a short internal name with package prefix
HEADER       - a brief description of the frame to appear as a header
TEXT         - multiple lines of description information, containing
               keywords in brackets which will indicate a related frame.
RELATED      - a multiple field containing the name of the keyword
FRAME          which will invoke more help text, and the name of the
               help frame which contains this text.
INVOKING     - routines which reference this help frame
AUTHOR       - the creator of this frame (triggered)
DATE ENTERED - the date the frame was created (triggered)
EDITOR       - those users who may edit the frame
DATE UPDATED - the last date the text was changed (computed)
