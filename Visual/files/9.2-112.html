NAME XUFILE ACCESS USE4.11
HEADER QUEUING FILE ACCESS SPECIFICATIONS
TEXT 
 
When using the file access options, were you confused by the message about
queuing?  Most of the options give you the opportunity to queue when you're
finished specifying who is to be granted which files.  Queuing will send
your specifications to the Task Manager to assign to users at a later time.
 
Choose to queue at an off-peak time, like midnight, when the system isn't
busy.  The processing can be done then and you don't need to wait for the
long pause now.  If the system isn't busy, you should still queue at the
requested time of 'NOW'.  Otherwise you'll need to wait patiently for the
job to finish.
