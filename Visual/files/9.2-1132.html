ECSSDSSNUM (1132)    HELP FRAME (9.2)

Name Value
NAME ECSSDSSNUM
HEADER DSS Unit Number
TEXT 
This is the Unit Number, found on the DSS Unit file (#724).  If this field contains a value, then it must have a "C" cross reference on the DSS Unit file (#724).  

Note: The DSS Unit Number is not required if the DSS Unit IEN or the DSS Unit Name is filled in.
