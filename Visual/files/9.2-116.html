Name Value
NAME XUDOC DEVICE PROMPT*
HEADER RESPONDING TO THE DEVICE PROMPT
TEXT 
 
When prompted for a device, you may enter the name of the device or another
response as shown below:
 
     DEVICE: DVNM5         to specify the device with the name DVNM5.
 
     DEVICE: HOME          to direct output to your terminal.  Alternatively
                           enter H, h, 0, or just press return.
 
     DEVICE: P             to select the closest printer if one is defined.
                           (This may be disabled when using decservers or
                           MICOM port contention.)
 
     DEVICE: Q             to [QUEUE] your output to run in the background.
 
You may optionally specify a right margin or page length to override the
default Device file settings:
 
     DEVICE: DVNM5;80;66   to use the DVNM5 device with a right margin of
                           80 columns and page length of 66 lines.
                                                                [CONTINUE]
