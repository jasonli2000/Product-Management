XUDOC DEVICE PROMPT SPOOLING (119)    HELP FRAME (9.2)

Name Value
NAME XUDOC DEVICE PROMPT SPOOLING
HEADER SENDING OUTPUT TO THE SPOOLER
TEXT 
 
Output may be sent to the spool device with the optional specifications of
subtype, right margin, or page length.  For example:
 
     DEVICE: SPOOL                  to spool with defaults, often right
                                    margin of 132 and page length of 66.
 
     DEVICE: SPOOL;96;66            to use a margin of 96 and page length of
                                    66 lines.
 
     DEVICE: SPOOL;P-OTHER80        to use the characteristics of the
                                    P-OTHER80 subtype (with an 80 column
                                    rather than 132 column page width).
 
The user is prompted for a spool document name once the spool device has
been requested.  As a shortcut, the spool document may be named when
responding to the device prompt:
 
     DEVICE: SPOOL;MYDOC            to name the spool document MYDOC.
 
     DEVICE: SPOOL;P-OTHER80;MYDOC  to use the P-OTHER80 characteristics.
