TEXT 
 
You can use this screen to create and edit list definitions.  These criteria
are used to actively update update calls on the list as calls are created
or edited.  Creating a new definition is similar to doing queries on calls.
You may want to take existing lists, modify them and save it as a new list
definition
 
 
New Definition - Enter a new list definition
Run - Use the current list definition to produce a list of calls
Save - Save any changes to this definition
Save As - Save a new list definition
Edit a line - Change a line in the list definition
Insert a line - Insert a new line in the list definition
Delete a line - Delete a line from the list definition
Other Definition - Switch to another list's definition
View Description - Displays the definition of the list in narrative format
Edit Description - Edit non-criteria descriptions of this list
 
 
On any screen enter ? for help, ?? for other actions, or ??? for help topics.
