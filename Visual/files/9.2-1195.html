NAME FSC SPECIALIST LIST DISPLAY
TEXT 
FSC LIST DISPLAY
 
This field is a set of codes that changes the default data on all of 
your nois lists.
Valid choices are S M U P or any combination of these four characters.
 
S - will show the status of the call on all lists.
M - will show the NOIS module on all lists.
U - will show the primary specialist currently assigned to the call.
P - will show the current priority on all lists.
