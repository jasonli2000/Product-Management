                                 What's New
 
NOIS is intended to be used as a central database where all ISCs log calls
on the same system.
Calls are not owned by any single user.  Any NOIS call can be opened,
edited, or closed by anyone having the authority to edit a call.
List Manager is the primary user interface to display lists of calls and
call information.
You can define a criteria for the calls that you normally deal with.
Virtually any type of call can be searched for.
Browsing information on calls is made easy by using ad hoc displays.
Notifications can be made using MailMan or menu alerts.  You can be
notified automatically of changes to calls you are interested in.  Sites
can also be notified automatically.
NOIS data can be easily extracted to PC applications and spreadsheets.
