XQHELP-LINK (12)    HELP FRAME (9.2)

Name Value
NAME XQHELP-LINK
HEADER INVOKING HELP TEXT
TEXT 
There are several ways in which help frames can be linked to the
system, and correspondingly, several ways of invoking them.
 
 - A help frame can be linked to a menu option.  This is done by
   filling in the HELP FRAME field in the OPTION file for the
   option you wish to have help text.  Then, when the menu system
   prompts for a selection including this option, a response of
   the option name preceeded with a '?' will invoke the help text.
   The '???' response will show which options have associated
   help text.
 
 - Another way to invoke help text is through a routine.   Set XQH to
   the INTERNAL NAME or INTERNAL NUMBER of the frame and D EN^XQH.
   This can be used in place of including lengthy help text in a
   routine.
 
 - Similarly, the EXECUTABLE HELP field in the data dictionary
   can be filled in so that help text is invoked.
