                                  Screens
 
Three main List Manager screens are used in NOIS:
 
List Calls - displays a list of multiple calls from which you can view or
edit
 
View Calls - displays formatted information on multiple calls.  You can
also select calls to be edited.
 
Edit Calls - displays a single call at a time from which you can edit.
 
A menu of actions is available on each screen.
 
A menu of hidden actions is also available.  Entering ?? will display
these actions.  These actions let you scroll the display, search the list,
get printouts, and define your custom defaults.
