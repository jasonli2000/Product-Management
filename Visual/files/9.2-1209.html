                              List Manager
 
List Manager provides applications a means of scrolling documents or lists
of items.  Items can be selected from the list by using the available
actions at the bottom of the screen.  Note: this is not always intuitive -
with List Manager, you select the action, then you select the item.
 
Documentation on List Manager is found in PIMS documentation.
The actions are protocols (XQOR) that are used as menus or items that run
programs.  The advantage is that records can be displayed and several
actions are available to perform without having to navigate through
several options (where each time you would have to select the record
again).
 
List Manager provides a hidden menu of actions that are also available to
use.  NOIS uses the hidden menu to contain actions that can generally be
done on any display.  These actions include actions for scrolling the
screen or to go to a specific page of the display.
The action 'Search List' is useful for finding text within the display.
The actions 'Print List' and 'Print Screen' provide for getting printouts
or non scrolled displays.
 
NOIS includes help using ?, ??, or ???.
 
NOIS also has User Defaults and List Definition available on the hidden
menu.
 
NOIS uses two ways to quit List Manager.  QT will quit the current display
and return to any previous display.  EX will exit back to the option menu.
