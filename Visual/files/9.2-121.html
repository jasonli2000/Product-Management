^%Z routine editor
 
This is a routine editor that can be used on all current DHCP computers.  It
 is installed in the manager account as the ^%Z global by ZTMGRSET. It can 
 also be setup by D ^ZTEDIT.
 
To use this editor ZLOAD the routine you want to edit and then X ^%Z to 
start the editor.  You can exit the editor without saving your changes so be 
careful.  When a routine is filed it is filed by the name it was loaded from 
not the first line tag,  if the routine file is found then the routine name
 
added if not already there and a entry made of the date/time and duz of 
the user that filed it.  The main prompt is EDIT: and entering a ? will give 
you help. You can work in two modes, line shows a complete line and lets 
you move around it or replace to do replace/with editing.
Example:
ZL ZU X ^%Z
