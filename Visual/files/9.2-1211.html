                               NOIS Users
 
There are three types of NOIS users.  No security keys are used in the
NOIS software.
 
User (non-NOIS specialist)
A user is someone who has the NOIS option but is not entered as a NOIS
specialist.  This user can use any NOIS options but is restricted from
editing data.
Specialist
The specialist is someone who is entered in the NOIS Specialist file.  A
specialist can enter, edit, or close any NOIS call.  There are no
different privileges between support and development specialists.
Specialists can edit their own list definitions, notifications, and
defaults.
Supervisor
The supervisor is a NOIS specialist who has the field, Acts as Supervisor,
in the NOIS Specialist file set to YES.  A supervisor can edit NOIS files
(add new packages, specialists, etc.).  They can edit the Acts as
Supervisor field in the NOIS Specialist file.  They can also edit others
list definitions, notifications, and defaults.
