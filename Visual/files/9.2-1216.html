                             Saving a List
 
You can save calls to a storage list using the File (Save) action.  If the
list is public or you are the owner of the list, the calls are saved to
the same list; otherwise you will have to save them to another list.  You
can enter a new list at this time.
 
You cannot save onto other's lists or any index, active update, or
temporary lists.
