                        Removing from a List
 
Calls can be removed from the list individually by selecting the calls
 
Calls can be removed by selecting other lists.  If the calls on those
lists are on your list then the calls are removed from your list.
 
Calls can also be removed using a query statement.
 
All calls can also be removed.  This clears your list to add a new set of
calls.
 
Removing a call from a list in no way changes the information on the call.
Changes to the list are only temporary unless the list is a storage only
list and the list is saved.  The list name is displayed as modified when
changes are made.
