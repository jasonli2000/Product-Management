TEXT 
 
                          Selecting from a List
 
Similarly to adding and removing, calls can be selected individually, by
lists, or queries.
 
Calls that have been selected remain on the list; the other calls are
removed.
 
Selection is useful for filtering those calls that are of interest from a
large list.
 
Changes to the list are only temporary, unless the list is a storage-only
list and the list is saved.  The list name is displayed as modified when
changes are made.
