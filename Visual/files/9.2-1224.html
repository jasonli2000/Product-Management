NAME FSC U90 EDIT CALLS
TEXT 
 
                           Entering/Editing Calls
 
The main NOIS activity is logging calls.  The problem is entered, the
solution (if available) is entered, along with whatever other information
needed to document the call.
 
Any specialist has access to edit any NOIS call.  Anyone editing a call is
documented as making an entry.  Any calls not attended to can be easily
searched for.
