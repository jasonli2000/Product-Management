                           Reopening a Call
 
A closed or canceled call can be reopened by changing its status to open.
 
If the call was closed, the resolution information is saved as a note.
 
There is a field that contains the last reopened date.  This field is
useful for searching for any calls that have been reopened.
 
There is no status of reopen.  If any active lists uses status = open to
add to the list, reopening the call would have the call added back on
list.
