NAME FSC U100 REVIEW CALLS
TEXT 
                          Reviewing Calls
 
Calls can be displayed using either the Edit Calls or View Calls.
Both actions allow for calls to be selected from the list.  Calls don't
have to be displayed in order to be selected.  Often you know that the
calls being selected are of interest because you have added and removed
calls, so that the remaining list consists of precisely those calls you're
interested in.  You may also scroll through the list, jotting down calls
of interest.
The edit screen allows only one call at a time to be displayed.  You can
change the format of the display and that format will be used on any
subsequent displays.
The view screen allows multiple calls to be displayed.  This format is
better for text searches (the SL action), and for statistic and custom
formats such as extracts.
