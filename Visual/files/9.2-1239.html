                        Exporting to Spreadsheets
 
If you are using NOIS through a terminal emulator on your PC, you can
capture data to a text file and then import the data into tables or
spreadsheets.
 
From your list of calls use View Calls for the calls you want to capture.
Use the custom format and select extract.
You can now enter any fields that will be the headings for the columns of
the table.
 
When youre done entering the fields, you will be prompted for a delimiter.
 You can enter any character.  If the delimiter appears in the value, the
delimiter will be replaced by a space (ex. last name,first name -> last
name first name).
 
Dates and times will be displayed in the format (m/d/y hh:mm).
