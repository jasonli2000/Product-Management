NAME FSC U1008 PRINTOUTS
TEXT 
 
                               Printouts
 
NOIS will generally be used from remote locations.  Printouts will be
limited to capturing information to a PC application or using slave
printers.
 
You can use a mail message to send calls to a local location whereby the
mail message can be printed.
 
When capturing the screen display to a terminal emulator or slave printer,
use the PL or PS action on the hidden menu to print the list or screen.
 
The reports option allows you to select a report to display.  A report is
made up of a list, a format, and a sort format.
