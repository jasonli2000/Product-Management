NAME FSC U1101 MAIL, MENU ALERTS
TEXT 
                          Mail, Menu Alerts
 
Notifying using mail allows you to send a message and/or load the list
into the message.  You can thus send several calls in the same mail
message with whatever format you've composed on the display (lists, custom
formats, statistics, etc.).  You can also edit this message before sending
it.
Mail uses an email address that can be a remote domain, so that you can
have messages sent to you locally.
NOIS should be used for documenting dialog on a problem instead of
multiple discussions on mail.
Notifying with menu alerts allows you to send a simple message to someone
that will show up as an alert whenever the recipient is on the system.
You can optionally include any calls on your current list.  This allows
the recipient to process the calls and immediately take some action (read,
edit, close, call the site back, etc.).
Besides receiving a list of calls with an alert, a short message with the
reason of the notification is displayed.
