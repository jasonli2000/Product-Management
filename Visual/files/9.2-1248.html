NAME FSC U1106 SITE NOTIFICATION
TEXT 
 
                           Site Notification
 
Sites (or services when NOIS is used locally) can schedule notification by
editing the NOIS Location file.  Supervisors can edit this file.
 
A method is entered - mail or alert.
 
If an alert is entered, the user in the Site Contact field is sent menu
alerts.
 
If mail is used then it is sent to the entry in the sites Email field.
An event is also entered: edit or status change.
