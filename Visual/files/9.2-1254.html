NAME FSC U1502 LISTS
TEXT 
 
                                   Lists
 
The DL action on the hidden menu allows creating, editing, and rebuilding
lists.
 
Notifications can be included in the list definition.
 
Queries can be edited by inserting new lines, removing lines, or replacing
a line.  Its generally easier to change to a similar list that you want to
use, save it with a new name, and then edit any changes.
 
After editing a list query, you should rebuild it  This adds and removes
any calls appropriate for the new definition.
 
You can display any list definition but you can only edit active update
and storage-only lists that are public or you are the owner of.
Supervisors can edit others lists.
