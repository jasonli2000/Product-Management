NAME FSC U1503 NOTIFICATIONS
TEXT 
 
                               Notifications
 
Notifications can be applied to list definitions (active update lists
only).
 
The DL action on the hidden menu allows editing notifications.
 
You can display any list definition but you can only edit active update
and storage only lists that are public or you are the owner of.
 
Supervisors can edit others lists.
