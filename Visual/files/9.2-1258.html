NAME FSC U180 USABILITY TEST
TEXT 
                               Usability Test
 
Test the software by performing the following tasks.
Situation:
You are a field support specialist with a personal list of NOIS calls set
up to automatically update calls that you are interested in.
(Only use the List Calls option)
Tasks:
1.  Review your list of calls.
2.  Select some calls to edit.
3.  Find a call that is open -- close the call.
4.  Change the format of the call.  (for example, if the call is in a
Brief format, change it to Detailed).
5.  Return to your list of calls.
6.  View several calls in a format that displays only the fields:
specialist, package, and support status.
7.  Switch format to Detailed.
8.  Search the list for the word task.  Use the Search List action from
the hidden actions.
9.  Return to your list of calls.
10.  Remove all calls from your list.
11.  Add calls where the support status is open and the support ISC is
Salt Lake City (use a query to specify this criteria).
12.  Select only those calls about Kernel 7.1 (use a query to specify this
criteria).
13.  Switch the modified list back to your personal list of calls.
Thats it.
 
After you're done, log and close a NOIS call (on NOIS) and describe any
problems you had, places where the software was confusing, etc.  In the
resolution summary offer any suggestions, opinions, or questions you have.
