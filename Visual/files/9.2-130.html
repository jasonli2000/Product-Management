NAME XQSRV-FIELDS
HEADER Setting up a Server Option
TEXT 
     
   There are several fields in the Option File which have an effect
on server options.  Some of these fields are common to other option
types, and some a specific to servers only.  Those fields that are
[common] to other types of options are: 1. Name (.01), 2. Out of Order
Message (#2), 3. Type (#4), 4. Priority (#3.8), 5. Exit Action (#15),
6. Action (#20), and 7. Routine (#25).
   In addition, the local IRM staff have several fields available to 
control the operation of servers.  They are: 1. [Server Bulletin]
(#220), 2. [Server Action] (#221), 3. [Server Mail Group] (#222), 
4. [Server Audit] (#223), 5. [Time to Run] (#224), and 6. [Day to Run]
(#225).
