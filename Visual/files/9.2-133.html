NAME XQSRV-ACTION CODES
HEADER Server Option Action Codes
TEXT 
   
   Field #221 'Server Action' in the Option File controls the way
a server request is handled by the menu system.  These are the
four possible codes and what they mean:
  
R - Run immediately.  When the message arrives at the menu system
     run the option.  
  
Q - [Queue] the Option.  The server option should be queued to run
     at some other time, which is determined by other fields in
     the Option File.
   
N - Notify the mail group with a bulletin that a server request
     has been recieved, but don't run anything.  The option will
     be queued and then dequeued and the Taskman number will be
     reported so that the server option can be requeued easily.
   
I - Ignore all requests to this server option.  Auditing may
     still take place, however.
