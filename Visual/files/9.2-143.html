Name Value
NAME XQTREPEAT
HEADER Repeating a Template
TEXT 
 
   When you create a new menu template you are asked if you want to repeat the
template over and over when it is envoked.  If you answer 'Yes' to the
question:
 
  After you have finished using 'TEMPLAT' will you want it to repeat?  N//
 
you will see the prompt 'Again Y//' when you finish running the template.  This
feature is designed for users who repeat the same functions over and over.  If
you answer 'No' to this question then the template will run once and drop you
back into your regular menu system.
