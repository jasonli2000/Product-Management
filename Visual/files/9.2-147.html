NAME XQOO-NAME
HEADER Naming An Option/Protocol Set
TEXT 
   We are about to build a set or collection of options and/or
protocols.  The name that you are being prompted for will be the name of
this collection or set.  It should be from 1 to 8 characters in length.
Hereafter the software will recognize the set by this name.  Since you may
have more than one set at any given time, the unique name will distinguish
between multiple sets.
   If you choose a name for which a set already exsists, you will be
notified of that and asked if you want to continue building that option
set.  If you chooses to do so, you may then add or subtract options from
that exsisting set.
