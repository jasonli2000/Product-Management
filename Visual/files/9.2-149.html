NAME XQOO-MESS
HEADER The Out Of Order Message
TEXT 
     You are being asked to enter and "Out Of Order Message".  This
string of text will be used to disable the options and/or protocols that
you will begin selecting next.  Each time an end user tries to envoke an
option or protocol that has been disabled this message will be printed on
his or her screen.  The message should be from 1 to 50 characters, and
should obviously have some meaning.  A few examples might be:
 
             Lab installation in progress, try later.
 
           The Marketing Computer Will be Down Until 6PM
 
             Buzz Off, Hard Rock, This System Is Down!!
DATE ENTERED 1994-04-26 11:13:00
