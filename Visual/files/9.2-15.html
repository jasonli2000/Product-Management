NAME XQEDIT
HEADER USING THE 'Edit the Option File' OPTION
TEXT 
This is the main option in Menu Management.  It is used to create and edit
options, the building blocks of the menu system.  Options have the following
attributes:
 
NAME        - a short internal name prefixed with the package namespace
 
MENU TEXT   - an external name which appears to the user
 
DESCRIPTION - a brief text which will be displayed to the user with a 
              '???' response.
 
[HELP FRAME]- the name of a help frame which will provide extended text
              describing this option
 
Type        - a single letter indicating whether this option is an
              intermediate <M>enu option, or one of several bottom level
              type options.
