NAME XQRESTRICT-TIMES
HEADER RESPONDING TO THE 'PROHIBITED TIMES' PROMPT
TEXT 
   An option which has a 'PROHIBITED TIME' can not be accessed during the
hours specified.  It is often useful to restrict access to print and
search type options to hours when the system is not very busy.
 
   Specify a 'prohibited time' using military format, for example:
 
       0900-1600  (to prohibit access from 9AM-4PM)
       1800-0600  (to prohibit access overnight 6PM-6AM)
 
   If an option has a prohibited time, it will appear as such when 
displayed in the menu system with '??'.
   
   Responding to this prompt with an '@' will delete any PROHIBITED
TIME currently specified for any of the options chosen, leaving the
field null so that there are no prohibited times for those options.
