NAME XQRESTRICT-OOO
HEADER RESPONDING TO THE 'OUT OF ORDER' PROMPT
TEXT 
   To temporarily disable an option, the 'OUT OF ORDER' field should
have a message indicating the inavailability of the option.  This is
preferable to removing the option from its menu if it is a temporary
condition.
 
   When there is a message in the 'OUT OF ORDER' field, the text
'NOT AVAILABLE' will appear next to the option when displayed by the menu
system.  If that option is selected by the user, then the 'OUT OF ORDER'
message will be displayed.
   
   Entering an '@' in response to this prompt will delete the OUT
OF ORDER message for the options selected leaving the field null,
which removes the OUT OF ORDER restriction for these options.
