XQKEYALLOCATE-KEY (2)    HELP FRAME (9.2)

Name Value
NAME XQKEYALLOCATE-KEY
HEADER RESPONDING TO THE 'Allocate/De-Allocate security key' PROMPT
TEXT 
     
You can specify the keys you wish to allocate or de-allocate.  You can 
specify a single key, or a set of keys.
 
Respond with the name of the key(s), one at a time.  You may assign only
pre-existing keys which you already hold.
 
You may also respond with one of the following:
 
^    - To exit this option.
 
??   - To see the list of keys you've already specified.
 
???  - To see the list of keys from which you may choose, the current 
          holders of a key, the User File, or the members of a mail
          group.
 
-KEY - To take a key off the list specified preceed it with a minus sign.
