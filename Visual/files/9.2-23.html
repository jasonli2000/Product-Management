NAME XQSECMENU
HEADER PERSONAL COMMAND SPACE
TEXT 
The personal command space (also referred to as 'secondary menu options'
allows for individual users to have their own XUCOMMAND.
 
The site manager can make entries to the SECONDARY MENU OPTION field in
the user file to give a user access to one or more options at all times.
It is the responsibility of the site manager to ascertain that each
option assigned does not require a predecessor to set up variables.
 
Keeping in mind the ^JUMP feature, the personal command space is most
useful in attaching an option which is otherwise not part of a user's
tree without building an entirely new menu.
