NAME XQ-USER-AUTOMATIC
HEADER THE COMMON OPTIONS
TEXT 
The COMMON menu is a list of options which are available to all
users from any point in the menu system.  These options are not displayed
unless the user enters '??'.  However, they are always selectable.
To resolve ambiguity between COMMON menu options and primary options
which begin with the same letters, preceeding an option with a " (double 
quote) will make the selection from the COMMON menu.
The following are included in this menu.
 
  [CONTINUE]                           DISPLAY USER CHARACTERISTICS
  [EDIT] USER CHARACTERISTICS          USER HELP
  [HALT]                               TIME
                                     RESTART SESSION
                                     MAILMAN
 
In addition there may be options assigned to individual users which appear
as part of the COMMON menu.  These are called [PERSONAL COMMANDS].
