NAME XUCONTINUE
HEADER HOW TO USE THE 'Continue' OPTION
TEXT 
The 'Continue' option is similar to 'Halt' in that it allows a 
user to exit without having to back up through the tree of menus.
 
Halting with 'Continue' will cause the computer to remember what the
last option being executed was, and next time the user signs on,
there will be the choice of resuming processing at that option.
 
If you enter 3 letters or more of the word 'CONTINUE', the computer
will halt immediately.  Otherwise, if you enter 'CO' or 'C', the
computer will want to re-affirm that you really wish to  halt.
