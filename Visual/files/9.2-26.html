NAME XUHALT
HEADER HOW TO USE THE 'Halt' OPTION
TEXT 
The HALT option will exit you from the computer.  This is useful
if you are several levels deep in a menu and do not want to
'carriage return' back to the beginning in order to exit.
 
If you enter 'HAL' or 'HALT' you will be exited immediately.
 
If you enter 'H' or 'HA', the computer will want to re-affirm whether
you really wanted to halt.
