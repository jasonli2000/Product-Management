NAME XQXREF
HEADER USING THE 'Options By Parents/Other Utilities' OPTION
TEXT 
This option produces a listing of options cross-referenced by parents.
It also indicates whether an option is in use as a primary, secondary or
tasked menu.  It can be run by package by entering a 2-4 letter package
prefix, or can be run on the entire option file.
 
This option also allows you to inquire to who holds specified options as
a primary or secondary menu option.
 
You may also delete unreferenced options.  These are options which have no 
parents, are not held by any users as primary or secondary menus, and are
not tasked.  Each unreferenced option in the package specified is presented
with all its fields and then the choice can be made to delete or not.
