NAME XQ-USER-MENU
HEADER USING THE MENU SYSTEM
TEXT 
The menu system is the way thru which options are accessed. Menus are 
arranged in a hierarchical or tree structure.  Starting at the top
you can progress thru levels to get to all your options.
 
When you are in the menu system you will see the prompt:
 
     'Select ... Option:'
 
You can respond to this prompt by [NAMING] an option which is available at
this level.  You can also [JUMP] to options which are accessible to you
but not in the current menu by using an '^' or '^^'.
 
There is a group of options called the [COMMON MENU] which is available
at all times. 
 
If you find that you need more information to be able to respond to a
'Select ... Option' prompt, entering [QUESTION] marks will be of help.
