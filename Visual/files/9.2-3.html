At this prompt you should select users.  You may choose a single user

^     - To exit this option.
 
??    - To see the list of recipients you've already specified.
 
???   - To see the list of keys from which you may choose, the current

          a mail group.
 
-USER - To take a user off the list specified preceed it with a minus sign.
 
G.GROUP - To specify a set of users as defined by a mailgroup, preceed the
          group name with 'G.'
