NAME XQ-USER-QMARK
HEADER USING QUESTION MARKS IN THE MENU SYSTEM
TEXT 
There are different sorts of 'help' available thru entering question marks
at a 'Select ... Option' prompt:
 
 ?       - A single question mark lists all options which can be accessed
           from the current one.  It has the same effect as [AUTO-MENU].
 
 ??      - Two question marks lists all options, showing their formal
           names and [LOCKS]. 
           Also listed will be the common menu containing options
           which are always available to the user.
 
 ???     - Three question marks will show brief descriptions for each
           option in the menu.
 
 ????    - Four question marks shows extended help for this menu.
 
 ?OPTION - An option name preceeded with a '?' shows extended help for
           that option.
