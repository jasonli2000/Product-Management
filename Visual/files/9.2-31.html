NAME XQ-USER-AUTOMENU
HEADER THE AUTO-MENU SWITCH
TEXT 
The 'Auto-menu' switch, when turned on, will show all options which
can be accessed at a 'Select ... Option' prompt.  If the switch is
turned off, the same list will be displayed on entering a '?'.
 
Once a user is familiar with their menu, it is preferable to have this
switch turned off for speedier navigation thru menus.
 
The list of options may include options which are temporarily
[NOT AVAILABLE] as well as options which have [PROHIBITED TIMES].
