Name Value
NAME XQ-USER-LOCKS
HEADER THE LOCKING OF OPTIONS
TEXT 
Sometimes an option will be displayed with the words 'LOCKED:' following
it.  This will only be shown in response to '??'.
 
If an option is locked, only users possessing the appropriate key can
have access to that option.  The site manager is responsible for assigning
keys to users.
