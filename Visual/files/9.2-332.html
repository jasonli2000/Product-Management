NAME XDR IDENTIFY
HEADER DUPLICATES: FINDING THEM
TEXT 
The option 'CHECK FILE FOR DUPLICATES' will help you to find duplicate
entries in a selected file.
 
This utility searches the selected file, marking entries it considers to
be probable duplicates and entering the pair in the Duplicate Record file.
You will need to look at each of these and verify whether they are
actual duplicates or not.
 
You can choose one of two [METHODS FOR SEARCHING] the file:
basic search on the entire file, or a search using only new or edited
entries.
 
You may interrup any of these searches and continue it at another time.
If you have an incomplete (halted) search, you will be asked if you want
to [CONTINUE] it, or initiate a new one.
 
The potential duplicate pairs are placed in the Duplicate Record file.
To [LIST] them, use the 'PRINT LIST OF FILE DUPLICATES' option.
