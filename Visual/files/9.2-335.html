NAME XDR IDENTIFY CONTINUE
HEADER DUPLICATES: CONTINUING A SEARCH
TEXT 
Anytime a search has been interrupted before completion, you will be
given an opportunity to continue that search where it left off.
 
The Duplicate Resolution file is used to keep track of the last
record checked for duplicates when a search is halted.
 
This is especially useful on the basic search of a file as it is
likely that it will take more than a few nights to complete.  This
also allows the search to be halted during peak operation times and
restarted during slack times.
