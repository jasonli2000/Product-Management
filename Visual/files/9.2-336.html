TEXT 
Potential Duplicates:
 
This provides a listing of all Potential Duplicate Pairs for a selected file
in a captioned format.  These are records that have not been verified as
either Verified Duplicates or Verified Non-Duplicates.  It prints all
fields that have data from the Duplicate Record file.
 
Ready to Merge Verified Duplicates:
 
This is a listing of all Verified Duplicate Pairs for a selected file that
have a status of ready to merge.  These are record pairs that can be merged
because all package interactive merges have been completed.
 
Not Ready to Merge Verified Duplicates:
 
This is a listing of all Verified Duplicate Pairs for a selected file that
have a status of not ready to merge.  These are record pairs that can not
be merged yet due to a package interactive merge process that has not
completed.  This listing is useful to find out which packages are keeping
this duplicate pair from being merged and the appropriate department can
be encouraged to finish the interactive process.
