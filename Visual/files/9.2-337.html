XDR PURGE (337)    HELP FRAME (9.2)

Name Value
NAME XDR PURGE
HEADER PURGE DUPLICATES
TEXT 
This utility is used by the Duplicate Manager to purge the Duplicate
Record file of entries of a selected file.
 
You may purge just the Potential Duplicates, the Verified Non-Duplicates,
or both for a selected file.
 
You can only purge entries for those files that you have Fileman Access
for.
 
The actual purging is put into a Taskman Queue and done in a background
job.
