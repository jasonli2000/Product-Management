NAME XQ-USER-PROTIMES
HEADER OPTIONS WITH PROHIBITED TIMES
TEXT 
Sometimes an option will be displayed with 'prohibited times'.
What this means is that the site manager has decided that this option
can only be run during certain times of the day.
 
An option is usually assigned a prohibited time if it uses a lot of
system resource, and is best not run during peak production hours.
