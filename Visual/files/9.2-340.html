XDR ADD VERIFIED (340)    HELP FRAME (9.2)

Name Value
NAME XDR ADD VERIFIED
HEADER ADD VERIFIED DUPLICATES
TEXT 
This option allows the operator to place two records that they know
are duplicates that are not already identified as a Potential or
Verified Duplicate pair into the Duplicate Record file.
 
It will run them through the Kernel Duplicate Resolution
algorithm and record there scores.
 
The normal process of [VERIFICATION] will not occur since these are
a known duplicate pair.
 
After adding them as a Verified Duplicate pair the [MERGE] process
will begin.
