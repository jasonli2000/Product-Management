Name Value
NAME XDR AUTO MERGE
HEADER AUTOMATICALLY MERGE ALL READY TO MERGE VERIFIED DUPLICATES
TEXT 
This option is used to merge all Verified Duplicate pairs for a
selected file that have a merge status of READY TO MERGE.
 
This merges all records that were not previously merged in the
normal [MERGE] process due to special package merges that needed
to occur before the final merge could occur.
 
When this option is used there will be no interaction with the operator
and no messages written to the screen.
 
Some sites may choose to place this option in a Taskman Queue to run
in the background but it is not known what effect Fileman dictionaries
that write messages to the screen will have on this process.
 
The merge process may take quite some time depending on the amount of
information and the number of files that have to be merged.  This option
if not queued should only be run on a terminal that can be dedicated to
this task for some large amount of time.
