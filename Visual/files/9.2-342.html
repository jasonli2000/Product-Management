XDR CHECK PAIR (342)    HELP FRAME (9.2)

Name Value
NAME XDR CHECK PAIR
HEADER CHECK PAIR OF RECORDS TO SEE IF DUPLICATES
TEXT 
This utility allows the operator to select two records from a selected
file and run them through the Duplicate Resolution [ALGORITHM].
 
It will display for them on the screen or they can have it print out
to a printer a listing of the two records side by side with the field
names on the left.  It will also print the scores the records received
through the duplicate resolution algorithm.
 
This option does not add the pair to the Duplicate Record file even
if the pair received a high enough total score during the check.
If through this option you determine that they are indeed duplicates,
the operator should add them as Verified Duplicates through the
'[ADD VERIFIED DUPLICATE PAIR]' option.
