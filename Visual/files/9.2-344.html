XDR EDIT DUP RESOLUTION FILE (344)    HELP FRAME (9.2)

Name Value
NAME XDR EDIT DUP RESOLUTION FILE
HEADER EDIT DUPLICATE RESOLUTION FILE
TEXT 
This utility is used to edit the fields for a selected file entry
of the Duplicate Resolution File.
 
The Duplicate Resolution file is the control file which a package
developer would create an entry from which to allow the operator
to check a selected file for duplicates.
 
This file contains the values associated with the various
threshold percentages, and holds the scoring for the specified
duplicate tests for the selected file.  It also contains information
on the status of the duplicate search, ie., beginning date, completion
date, last record checked, total number of records checked, type of
search being run.
 
You may see the fields of the [DUPLICATE RESOLUTION FILE] by entering
the key word.
