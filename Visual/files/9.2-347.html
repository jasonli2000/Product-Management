XDR MERGE VERIFIED DUPLICATES (347)    HELP FRAME (9.2)

Name Value
NAME XDR MERGE VERIFIED DUPLICATES
HEADER MERGE READY TO MERGE VERIFIED DUPLICATES
TEXT 
This option is used to merge all duplicate pairs for a selected
file that have a merge status of Ready to Merge and a status of
Verified Duplicates.
 
These are duplicate pairs that were not merged during the
verification process due to interactive package merges that
needed to occur before the main merge could occur.
 
The style of merge (Interactive versus Non-Interactive) will be
determined by the Merge Style field in the Duplicate Resolution file.
 
This process will begin with a double check through the package file
looking to see if any new packages that affect this file merge have
been added and if any of the packages that previously did not have
record data now do.  If any new packages have been added and they
contain record data and have an interactive merge process, the
Merge Status field will be reset to Not Ready to Merge and the [MERGE]
process will not be started.
