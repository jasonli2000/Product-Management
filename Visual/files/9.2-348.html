XDR MERGE SELECTED PAIR (348)    HELP FRAME (9.2)

Name Value
NAME XDR MERGE SELECTED PAIR
HEADER MERGE SELECTED READY TO MERGE VERIFIED DUPLICATE PAIR
TEXT 
This option is used to merge a operator selected Ready to Merge
Verified Duplicate Pair.
 
Ready to Merge Verified Duplicates are pairs of duplicates that were
not merged during the verification process due to package interactive
merges that needed to occur before the main merge.
 
The operator is only allowed to select duplicate pairs that have
a Merge Status of 'Ready to Merge' and a Status of 'Verified Duplicates'.
The style of merge will depend on the Merge Style field of the
Duplicate Resolution file.
 
After selecting the pair of records to merge the first thing to occur is
the rechecking of the package file to see if any new packages that affect
record merge have been added since the day the records were verified as
duplicates.  If any new packages or if any packages that previously did
not contain record data now contain data and they have interactive
merge processes that need to occur the [MERGE] process will not be
started.
