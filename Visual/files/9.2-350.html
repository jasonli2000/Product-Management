                        Default Reports
 
Report types include: Organism, Specimen, Collection Sample, Patient,
Provider, and Location.  Each report can be selected to be printed for all
values, selected values or not printed.
 
Defaults can be set for the different report types. The defaults for
report types can be defined in the Laboratory Site file All values for a
defined report are printed. (69.9).  If default reports are selected, the
questions for entering the types of reports and criteria are not asked.
At least one type of report must be defined.
 
Defaults for criteria are not defined in file 69.9, they are defined by the
program.  Default criteria is set to report all bacteria types of
organisms, not have a length of stay restriction, not be restricted to any
antibiotic pattern, merge isolates from the same specimen, and not include
a detailed report.
