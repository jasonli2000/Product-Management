                         Specific Organisms
 
All reports can be restricted to specific organisms.  Any organisms can be
selected so long as that organisms type was also selected in the prompts to
include specific types of organisms.  The default is to include only
bacteria.
 
For example, if you wanted all reports to include Mycobacteria tuberculosis
and Coccidioides immitis then Mycobacteria and Fungus types of organisms
must have been previously selected.
 
This criteria is useful in tracking specific organisms.
