                         Report Types
 
Six report types are available: 1) organism, 2) specimen, 3) patient, 4)
physician, 5) location, 6) collection sample.  Data are sorted by these
types.  Each type the data are then sorted by organism type (bacteria,
fungus, mycobacteria, parasites, and virus) and then by organisms.
 
Each type of report can be selected to be printed.  If the report is
selected it can report all values of that type or selected values.  For
example, a report for all organisms, and a report for urine and sputum
specimens could be selected.
 
Criteria can be entered that effects all report types.  The criteria can
determine the organisms to be reported, the time range, merge criteria,
detailed information, and antibiotic patterns.
