                        General Description
 
The antibiotic trend report is used to compare counts of organisms and 
patterns of antibiotic susceptibility.  Different types of reports can be 
generated.  Reports can be sorted by organism, patient, location, 
physician, specimen, and collection sample.  For each type, the report is 
sorted by organisms.  The different types of reports can be selected by the 
user.  Defaults for these report types can be defined in the Laboratory 
Site file (69.9).  Reports can be restricted to specific types of values 
(i.e. specific patients, specific specimens, etc.).  The reports can be 
restricted to criteria that effect all types of reports.
 
The criteria and types of reports are selected by the user.  The report 
is forced to be queued - preferably to be run during off hours.  
Merges can be done so that isolates from the same patient and same 
organism, and non-conflicting antibiotic patterns will only be 
counted once.  Merges also depend on being isolated from the same 
specimen, collection sample or any sample.  A detailed report showing 
each isolate's values can be used to confirm the counts reported.
