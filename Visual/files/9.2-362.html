                         Printing Reports
 
The antibiotic trend report is process intensive and should be run during
off hours.  The report should be queued.  The report has no fixed margin
length.  Any antibiotics with abbreviations in the Antibiotic
Susceptibility file (62.06) are printed as a header.  If there are more
antibiotics that can fit on a page then the report will wrap and be
difficult to read.
 
If a report is queued or in the process of printing and needs to be
stopped, the task can be stopped using the TaskMan User option in the
User's Toolbox.
 
 
If the report is queued to a spool device then multiple copies can be
printed from the spooled report.  Separate copies could be sent to
different printers.
