NAME PRCS-2237
HEADER ENTERING REQUESTS
TEXT 
Select Item Display from the
Requestors Menu.
 
If the item isn't in the Item Master File, create a Non-Repetitive Order.
If the item is in the Item Master File and lists the warehouse as one of
the vendors, create an Issue Book Request.  If the item is in the Item 
Master File, but doesn't list the warehouse as one of the vendors, create 
a Repetitive/Non-Repetitive Order.  The use of the Repetitive Order is 
discouraged.  Create a 1358 order to request a service if the cost is
estimated and a contract is in place.
