Name Value
NAME PRCB SINGLE TRANSFER
HEADER Transfer From/To Control Point
TEXT 
This option will allow transferring of funds from control point to control
point.  The Fund Distribution file will then be updated and transactions
made ready for release.
 
You can transfer funds among Control Points if the Control Points receive 
their money from the same fund and the funds are for the same quarter.
