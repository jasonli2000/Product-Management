NAME XQHELP-ASSIGNEDITOR-U
HEADER RESPONDING TO THE 'Editor of help frame' PROMPT
TEXT 
   
At this prompt you can select users.  You can assign a single user 
or a set of users. You can use a mailgroup to specify a group of users.
Respond with the name of the user(s), one at a time.
You may also respond with one of the following:
 
^     - To exit this option.
 
??    - To see the list of editors you've already specified.
 
???   - To see the Help Frame File, editors of a particular help frame,
          the User File, or members of a mail group.
 
-USER - To take a user off the list preceed it with a minus sign.
 
G.GROUP - To specify a set of users as defined by a mailgroup, preceed the
          group name with 'G.'
