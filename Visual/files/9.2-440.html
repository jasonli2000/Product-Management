NAME GMRV-TEMPERATURE RATE HELP
HEADER Temperature
TEXT 
Answer must be numeric with a maximum of one decimal place.
 
A number that is larger than or equal to 45 will be considered as a
Farenheit temperature, and any number less than 45 will be considered as a 
Centigrade temperature.
  
A letter qualifier can be appended to the end of the temperature reading. 
If no letter is added the reading will be considered as an ORAL temperature.
The following is a list of valid letters and the types of temperature:
 
                O = ORAL        C=CORE
                R = RECTAL      T=TYMPANIC
                A = AXILLARY    S=SKIN        or
                site defined types.
   
Default: Site configurable
  
E.g.,  37R   read as  37 C RECTAL
       98.6O    read as 98.6 F ORAL
