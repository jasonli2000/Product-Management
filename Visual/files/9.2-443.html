GMRV-HEIGHT RATE HELP (443)    HELP FRAME (9.2)

Name Value
NAME GMRV-HEIGHT RATE HELP
HEADER Height
TEXT 
The entry for height must be a number appended with a qualifier to
represent the value either in the English scale or the Metric system. If
you wish to enter a height in Metric system, append 'C' or 'CM' at the end
of the number. Add 'F' or 'FT' or ' for feet and 'I' or 'IN' or '"' for
inches to represent the value in English scale.
 
An optional qualifier from the GMRV Qualifier file (#120.52) can be
appended to the end of value separated with ','.
 
Some examples of valid entries are:
 
          72I,E (for ESTIMATED)
          147C
          160CM
          5F10I
          6FT1IN
          5'9"
