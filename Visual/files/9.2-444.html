NAME GMRV-WEIGHT RATE HELP
HEADER Weight
TEXT 
The weight entered must be a number + L(or l) or K(k) + an alpha  
character for weight (optional).
   
The character L or l expresses the weight in the English scale pound,
while K or k in the Metric scale kilogram.
  
The alpha character for weight is the first character for types of weight
such as A=ACTUAL, E=ESTIMATED, D=DRY or site defined qualifiers for weight.
This character is optional. The default is ACTUAL.
   
Some examples of valid entries are:
        176l or 176LA
        80KE or 80ke
DO NOT enter a number such as 120.0 for 120.
