NURS-LOCATION-DELETE (452)    HELP FRAME (9.2)

Name Value
NAME NURS-LOCATION-DELETE
HEADER Deleting a Nursing Location
TEXT 
A NURSING LOCATION CANNOT BE DELETED!!
 
To simulate a deletion:
 
   1.  Run the WARD DEACTIVATION PATIENT UPDATE OPTION (NURSPT-WRDINA) to: a)
       remove patients from the unit of the simulated deletion and b) set the
       PATIENT CARE STATUS Field of this nursing location to inactive.
 
   2.  Set the WARD STATUS Field of the nursing location to (I) inactive.  
 
   3.  Assign the corresponding MAS location(s) for the inactivated nursing
       location to a new or already existing nursing location, remembering
       that all FILE 42 ward location entries must be associated with an
       active (WARD STATUS FIELD) nursing location, including MASONLY.
 
   4.  Delete the MAS location entry(s) for the old/original nursing
       location to be deactivated.
 
   5.  Deactivate nursing locations which are receiving the MAS locations
       referenced in step 3 (excluding MASONLY).  Reactivate these wards
       using the WARD ACTIVATION PATIENT UPDATE option (NURSPT-WRDACT). This
       process will place patients from the closed unit on their new locations
      in the Nursing System.
