Name Value
NAME NURS-PRINTER QUEUE
HEADER Selecting a printer for reports
TEXT 
[Printed] Nursing reports must be queued!!
To queue a report, execute the following steps:
 
   1.  At the DEVICE: prompt, type in a 'Q'.
 
       e.g.  DEVICE: Q<RET>
 
 
   2.  At the second DEVICE: prompt, type in the name of the printer
       to be used.
 
       e.g.  DEVICE: PRINTER<RET>
