HEADER Detailed Selection Help
TEXT 
When prompted to select the number(s) of the entry(ies) you wish to choose,
in addition to making selections you may:
    1.  Type '^R' to Relist the above menu.
    2.  Type '^P' to Print/View the patient's data previously entered.
    3.  Type '^D' to Print/View the patients demographic data.
    4.  Typing '-' will allow you to view the previous frame if such
        a screen exists.
    5.  Type an '^' to exit the application.
    6.  Type '?' for help on how to [make selections].
    7.  Type '??' to get this help screen.
    8.  Type '???' to see a definition of terms on this screen
    9.  Type <return> to go back up to a previous level of selections.
Also if you need help understanding the [selection list], please choose
the appropriate related help frame.
