TEXT 
There are three advanced features of the Text Generator that can be used
to facilitate data entry.  For further information on these features, see
the Text Generator Users Manual.
 
JUMPING:
  By typing a ^^ followed by the name of an aggregate term, the user can go
  right to that aggregate term without having to go through any of the 
  intervening frames or selection process.  If there are multiple aggregate
  terms that match the text entered after the ^^ the user will be asked to
  select the appropriate one.  If there are multiple ways to get to that
  aggregate term within a prime document, the user will be asked which path
  to use to get to it.
SCRIPTING:
  By typing an open square bracket followed by the name of a template, the
  user can select a template to use in data entry.  This template is a pre-
  selected set of data that can be selected and modified for the patient.
TRAILING COMMA NOREDISPLAY:
  If the user ends a selection list with a comma then the selection list
  will not be reviewed for that frame after it has been processed.
     E.g.  1-3,
           1@,2;appended text,3-4,
