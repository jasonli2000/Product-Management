TEXT 
At any time a piece of equipment or software is added or removed
from the Library, this should be reported as an update to the ADP
Directory.  Please include manufacturer and model number of new,
as well as deleted equipment and the type of equipment, whether
CRT, microcomputer, CD/ROM, telefacsimile, etc.  When reporting
software titles not listed in the directory, please send a
follow-up mailing of photocopies of the case and/or documentation
which shows the appropriate spelling of the title, as well as the
version number.  Use the report for your facility from option FAC
to determine what your next update should include or delete.
Please send updates via MAILMAN message to SISTRUNK,CAROL or
mail them to:
                Chief, Library Service (142D)
                Attn:  ADP Directory Updates
                VA Medical Center
                1500 E. Woodrow Wilson
                Jackson, MS  39216
If you wish to mail a printed copy of your FAC report with deletions
marked appropriately and additions written in, please be sure to
include all the information requested.
