TEXT 
The Automated Data Processing (ADP) Directory in VALNET is
available on-line as a reference tool for VA librarians to
use in sharing information and ideas about computer hardware
and software available in Library Services throughout the
Agency.  It also contains information concerning the
applications which are being used and/or developed by libraries
in VALNET.  Choose Library Facility Overview option ([FAC])
to obtain an overview of the different types of equipment as
well as software titles which are available at a particular
VALNET facility.  Library ADP Hardware in VALNET ([HRD]) and
Library ADP Software/Applications ([SOFT]) options will be
used to obtain a list of all libraries which own a particular
brand of equipment or a certain software product.  [UPDATES]
should be sent to SISTRUNK,CAROL via MAILMAN message or U. S.
MAIL.
