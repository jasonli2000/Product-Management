XQHELP-DISP-EP (47)    HELP FRAME (9.2)

Name Value
NAME XQHELP-DISP-EP
HEADER WHO CAN EDIT A HELP FRAME
TEXT 
A help frame is editable under one of the following three conditions:
 
 - The user holds the XUAUTHOR key, enabling edit of all frames.
 
 - The user is the author of the frame being displayed.
 
 - The user is one of the editors of the frame being displayed.
   Editors can be assigned by the author with the option
   [ASSIGN EDITOR].
 
If a user has edit privileges for a frame, then she/he can enter
'^E' at the 'Select action' prompt to edit a frame. '^EN', '^EH',
'^ER', 'ET', are also valid responses to edit individual fields in 
a frame.  These will cause a switch to [EDIT MODE].
