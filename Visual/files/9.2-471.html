Name Value
NAME LBRY ADP HARDWARE
HEADER ADP HARDWARE
TEXT 
There are ten designated types of equipment which display when
the Library ADP Hardware in VALNET option (HRD) is selected.
At the prompt to Select Equipment Category, enter the number
which corresponds with the [TYPE] of equipment you wish to search.
At the next prompt, Select Library ADP XXX, you can obtain a list
of available entries by entering <?><CR> or typing the first few
characters of a brand name, i.e., APP for APPLE or IBM for all
IBM models.  Some lists may display without any numbers; enter
the first few characters to choose a model to view.  For numbered
lists, choose the corresponding number.  When the match is exact,
the report will display after a brief pause.  The report lists
all VALNET facilities which own the particular model you selected.
