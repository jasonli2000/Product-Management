NAME LBRY FACILITY ADP OVERVIEW
HEADER Facility ADP Overview
TEXT 
In order to obtain a complete overview of hardware and software
presently listed in the ADP Directory for any given facility,
choose the Library ADP Facility Overview option (FAC).  At
the Select VALNET Facility prompt, enter the correct five-letter
code for the facility you wish to search (i.e. MSJAC).  The
resulting report will list all equipment and software, with
applications, reported by that facility.  The information may
be used to determine if a facility should [UPDATE] their entry,
for comparison purposes between facilities, or for interaction
with another facility regarding advantages/disadvantages of
various equipment and software.
