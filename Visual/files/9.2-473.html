HEADER ADP SOFTWARE REPORT
TEXT 
A list of libraries owning particular software titles may be obtained
by choosing the Library ADP Software/Applications (SOFT) option.  At
the prompt to select Library ADP Software Title, there are two choices:
 
1)   Enter a <?><CR> for a list of software titles owned in VALNET.
When the title has been located, it is possible to ^ (up-arrow) at
the bottom of the screen.  Then enter the software title using the
same spelling and/or format as the list displayed.
 
2)   Enter the first few characters of any title at the prompt.  If the
item is not included in the file using the spelling or format you entered,
The system will respond with '??'.  For multiple matches, a list of
possible choices will be displayed.  Enter the number which corresponds
with the title you wish to view.  A single match will automatically
display the report after a brief pause.
 
The report lists all facilities which own the particular software with
its applications/uses.  Be aware that some facilities did not report
any applications or uses of a particular software title, therefore
none will display for those facilities.
