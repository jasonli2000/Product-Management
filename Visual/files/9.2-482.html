When searching for a title, an ISSN number may pull up more than one
title.  The ISSN does not always change when a journal changes its
name.
 
If you use the first four digits of an ISSN only, for example, 0002,
you will get a numbered list of all the titles in the database whose
ISSN's start with 0002.  You can choose from that list.
 
**HINT**  Using the first six digits (with the hyphen) of the ISSN,
for example, 0002-46, is often sufficient to find the precise title.
