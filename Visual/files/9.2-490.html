TEXT 
The Library ULP Reports [ULP] option is the online VALNET Union
List of Periodicals and consists of three parts:
 
  (1) Library ULP Bibliographic Record [BIB] -- which presents
      the full bibliographic record for a title.
  (2) VALNET Directory [DIR] -- which gives addresses and other
      information for VALNET libraries.
  (3) Library ULP Holdings [HOLD] -- which presents VALNET holdings
      for a title along with an abbreviated bibliographic record.
 
**HINT** If you want to move from BIB to HOLD, or vice versa, for
the same title, select the other 3-letter code at any option
prompt to move to the other option; then use  SPACEBAR-RETURN
to pull up the same title.
