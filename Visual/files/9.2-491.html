Name Value
NAME LBRY SERIALS CHECK-IN
HEADER SERIALS CHECK-IN
TEXT 
Use E(ID NUM) to (E)dit journal date, volume, and issue/part/
supplement, as well as entry type, [ISSUE STATUS], and receipt
date of a copy of an issue.
  
A JOURNAL DATE may be altered through editing but may not be deleted
because removal would cause errors in the database.  If there is an
unneeded or erroneous JOURNAL DATE already created which you cannot
edit into a legitimate JOURNAL DATE to be used soon, edit it into
a JOURNAL DATE a year ahead and change the ISSUE STATUS to NOT
RECEIVED for each copy for that JOURNAL DATE.  It will then display
chronologically at the end of the ID NUM listing and thus serve as a
reminder of an available JOURNAL DATE for future use after
appropriate editing.
