NAME LBRY SERIALS RETURN
HEADER SERIALS RETURN CHOICES
TEXT 
Enter a number which is displayed under the heading "ID NUM" to
return an issue.
  
Use "N" to see (N)otes about this journal.
  
To start at a particular journal date and see five previous journal
dates in sequence, enter B ("ID NUM").  Use "B" only to display
the five journal dates prior to those currently displayed.
  
"F" will display the next five journal dates.  Use F ("ID NUM") to
start at a specific journal date and see the next five journal
dates.
