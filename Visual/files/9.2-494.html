NAME LBRY RET
HEADER RETURN FROM ROUTING
TEXT 
Use Return from Routing (RET) to indicate that a journal issue
has been returned from a routing sequence and is ready to be shelved
in the Library collection.  
  
At the Select LOCAL SERIALS prompt, retrieve the title of the issue
ready for return from routing by entering one of the following:
     (1)  significant word or word fragment;
     (2)  ISSN/ISBN number; or
     (3)  NLM title abbreviation.
If a numbered list of choices appear, select the correct title by
number.
  
After choosing the appropriate ID NUM for the routed journal issue, 
the system automatically lists the journal issue as "shelved."
