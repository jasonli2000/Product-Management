HEADER LIBRARY SUPPORTING FILES
TEXT 
The creation of the supporting files is one of the initial steps
required to set up the Serials Control Module at each Medical Center.
Separate creation of these files under the Serials Management (LSM)
option ensures consistency of entries later in the Module.
  
These files are:
  [NME], for Name Entries
  [IPL], for Item Placement Location Entries
  [SUB], for Subject Setup.
  
Entries must be made in two of these files, IPL and SUB, before
they can be used elsewhere.
