NAME XUUSER-MENU
HEADER USING THE 'User Edit' OPTION
TEXT 
This menu contains options for setting up users, editing them,
and deactivating them.
 
In addition, it contains options to
 
 [PURGE LOG] of old access and verify codes
 
 [SWITCH IDENTITIES] to test another user's menu
 
 [INQUIRE ABOUT A USER] to obtain information about keys held and attributes.
