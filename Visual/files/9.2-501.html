NAME LBRY RST
HEADER REMOVE SERIAL TITLE FROM DATABASE
TEXT 
**WARNING**   **WARNING**   **WARNING**   **WARNING**   **WARNING**
  
Use this option to remove a serial title and all its associated data
from the database.  ALL data about that title, including holdings,
check-in, and copy information, will be REMOVED from the system upon
title deletion.
  
Use this option only for titles to be removed from the library's collection
for which no record is desired.  Do not use this option to indicate a name  
change or a discontinued title.  Consider instead the use of the INACTIVE
field in Library Title Setup (LTS) and the DELETION DATE and DELETION
NOTES fields in Holdings Setup (HOL).
