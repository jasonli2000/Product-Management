NAME LBRY RRS
HEADER REPRINT ROUTING SLIPS
TEXT 
The Reprint Routing Slips (RRS) option allows the reprinting of routing
slips for a specific time period.  This is useful in case of printer
malfunction.
  
Enter date and time following the prompts for DATE; use a ? to see the 
correct way to enter date and time information.  At the DEVICE prompt, 
accept the default printer number to reprint the routing slips to the 
usual printer, or insert another printer number if necessary.  Insert 
the word HOME to view the routing slips only on the screen.
