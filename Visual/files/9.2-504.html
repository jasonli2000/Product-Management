HEADER SERIALS MANAGEMENT
TEXT 
The Serials Management (LSM) option provides for:
(1) the creation of the local serials database for holdings and
purchasing information and current check-in [SET],
(2) the establishment of related or supporting files [LSF] and [MCF], and
(3) the procedures for removing a serials title from the database [RST].
  
The Hospital Service File (HSF), MUST be established before any of the
other functions in this option can be utilized.  This file may already
exist as an independent file on DHCP at the facility; if not, you will
need to create it.  This function is reached through the Medical 
Center Files [MCF] option.
  
You should establish the Library Supporting Files (LSF) NEXT.
