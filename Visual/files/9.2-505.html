HEADER SERIALS CONTROL
TEXT 
Serials Control (SER) provides the capability to Check-in [CHE] a
journal issue, Return [RET] a journal issue from routing to the
library collection, call up and view the Check-in History [HIS]
of a journal issue, Reprint a Routing Slip [RRS] if needed, and
generate reports and lists [STR] and [OSR].
  
It also provides [GENERAL] information about the package which you
should read first before proceeding with any data input.
  
All routines in this section may be done by any library staff person
responsible for day-to-day management of serials.
  
***REMINDER:  A title must be entered initially in Library Title
Setup (LTS) before it can be called up for use in Serials Control (SER).
