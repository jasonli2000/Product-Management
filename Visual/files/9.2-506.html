NAME LBRY SITE PARAMETERS2
HEADER SITE PARAMETERS2
TEXT 
At the Select LIBRARY SITE PARAMETERS LIBRARY SITE NAME prompt,
enter the already established local library site name or add a new
one.
  
The system will prompt for the following information:
  
LIBRARY SITE NAME:
     This will redisplay the name or number entered above and allow
     for the editing or deletion of the LIBRARY SITE NAME.
     Deletion is not allowed.
MULTI-DIVISION SITE: N//
     Enter <YES><CR> only if there is more than one separate division,
     such as exists in St. Louis.  Otherwise, accept the default of NO.
     If YES, there will be a prompt for DIVISION SITE NAME.
  
Choose [CONTINUE] to see additional prompts.
