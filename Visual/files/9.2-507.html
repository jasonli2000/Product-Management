Name Value
NAME LBRY ENTERING
HEADER ENTERING CHECK-IN DATES
TEXT 
At the prompt "Select JOURNAL DATE" it is possible to enter:
     Year (e.g., 1987 or 87)
     Month/Year (e.g., 1/87 or Jan 87)
     Month/Day/Year (e.g., 3/22/88 or Mar 22, 1988 or 032288)
  
In the Month/Day/Year format, the system will select the current year
automatically if none is entered.
  
Note that this field does not take words, such as Spring, Fall, etc.
If the journal is dated Fall 1987, enter the date in this field as 1987,
and enter "Fall" in the ISSUE/PART/SUPPLEMENT field.
