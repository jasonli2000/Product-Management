HEADER TITLE FILE INTEGRITY
TEXT 
The TITLE AUTHORITY FILE is a database of titles from the VA Union
List of Periodicals with selected general titles added.  Most local
titles should be available from the TITLE AUTHORITY file for inclusion
in the Library's LOCAL SERIALS file.
  
*NOTE*  All titles MUST be searched and entered using ALL capital letters.
  
***ATTENTION***   Before entering an entirely new title at the Select
TITLE AUTHORITY prompt, [SEARCH] for a title carefully using one of the
following methods:  key word, ISSN, significant word fragment, NLM
title abbreviation, or initialism or acronym as a synonym.  Try each search 
method until you make an exact match or you determine that the title you 
want is not in the data base.  This should ensure the integrity of both 
the TITLE AUTHORITY and LOCAL SERIALS files.
