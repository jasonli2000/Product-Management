Name Value
NAME LBRY SEARCH STRATEGY
HEADER SEARCH STRATEGY
TEXT 
When you enter a search strategy, the system will compare the entry
with the list of titles available in the TITLE AUTHORITY FILE (TAF) for
a possible match.  Depending on the search method used, the system will
respond with (1) an exact match, (2) a list of choices, or (3) if no
match is found, will request you to add a [NEW TITLE].
   
Once you select a title the screen will redisplay the title.  Then if
you selected a [NATIONAL] title, you will see a display of bibliographic
data which you can not edit.  Otherwise the [LOCAL] title you select
will prompt for bibliographic data.
