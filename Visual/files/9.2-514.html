Name Value
NAME LBRY TOC
HEADER TOC
TEXT 
To indicate that the Table of Contents (TOC) for a title should
be routed to a patron, enter I for (I)nsert.  At the prompt
COPY NUMBER or TOC, enter T.  The system then prompts for Select
TO ROUTE TO.  Enter the names of recipients of tables of contents
and routing order.
  
***NOTE***  A separate routing slip will print out for each
designee listed at the TO ROUTE TO prompt for (TOC) choices only.
