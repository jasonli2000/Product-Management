Name Value
NAME LBRY HISTORY VIEW
HEADER HISTORY VIEW
TEXT 
Enter a number displayed under the heading ID NUM to view the data
for a specific issue.
  
N will display the Check-in (N)otes about this journal
if any have been entered in the CHECK-IN NOTES field in Library
Title Setup (LTS); otherwise, the N will not display.
  
B will display a maximum of five journal dates prior to the first
issue displayed.  Enter B with the ID NUM to display a maximum of five
journal dates prior to the date of the issue selected.
  
F will display a maximum of five journal dates following the
last issue displayed.  Enter F with the ID NUM to display the next five  
journal dates from the issue selected.
