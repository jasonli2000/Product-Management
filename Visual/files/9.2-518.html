LBRY SERIALSCONTINUE (518)    HELP FRAME (9.2)

Name Value
NAME LBRY SERIALSCONTINUE
HEADER SERIALS CHECK-IN CONTINUED
TEXT 
Use B to display the previous five journal dates.  Enter
B(ID NUM) to start at that journal date and see five journal dates
in sequence.
  
Use F to display the next five journal dates.  Enter F(ID NUM)
to start at that journal date and see five journal dates in sequence.
B and F will display only if there are listed issues before
or after what is displayed on the immediate screen.
  
Use N to see (N)otes about this journal which were created in the
CHECK-IN NOTES field in Library Title Setup (LTS).
