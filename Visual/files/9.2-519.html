Name Value
NAME LBRY NOTESCONTINUE
HEADER NOTES CONTINUED
TEXT 
If you ordered [MULTIPLE] copies of the title and not all copies
were received at the same time for an issue date, answer the question
"Do you want to check-in x copies of (date) issue today? YES//"
with NO.
  
The system will prompt for the ID NUM of the issue to be checked-in.  It
then prompts for "DATE RECEIVED:TODAY//".  An earlier receipt date may be
input at this prompt if appropriate.  It then states:  "Enter copy number/s
to distribute separated by commas or a hyphen.  Copy number/s:EXIT//".
  
***  Be sure to input copy numbers.  A return at this prompt without a copy
number will negate the check-in for this issue.  ***
