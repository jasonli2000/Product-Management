HEADER NAME ENTRIES
TEXT 
***IMPORTANT***   This option is for listing the names of (1) the persons
and/or locations (routing designees) to which serials will be routed,
(2) requestors, and (3) donors.  Routing designees are input at the
TO ROUTE TO prompt in Copy Specific Edit (CSE); DONOR in CSE; and
REQUESTOR in Holdings Setup (HOL).
  
Enter the name as it should appear on the first line of the routing slip.
  
**  Personal names, titles, or locations are all appropriate as routing
designees, and thus can be defined as a "Name Entry".  **
 
After you enter the name, the system will prompt for Service name, which
must already be entered in the Hospital Service File (HSF), mail symbol,
address and telephone number (optional).
  
Entries in the Name Entries option can be edited but not deleted. If a
name will not be used again, you could choose to edit it to a line of
Z's or to a phrase, such as USE FOR NEXT ENTRY.
