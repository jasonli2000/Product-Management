NAME XQBUILDTREE-VER
HEADER VERIFY MODE FOR REBUILDING MENUS
TEXT 
The option which forces the rebuilding of menus on a CPU can be run in
one of two modes: automatic or verify.
 
The option will first go thru the user file, and compile a list of primary
menus of active users (those with access codes).  It will list these 
primary menus, along with how many users have each, and the most recent
date accessed.  If there is an existing tree built, the date it was
created will also print.
 
If the option is run in AUTOMATIC mode, all primary menus listed will
have a tree built. This could be time-consuming.  If run in VERIFY mode,
you will be prompted for each primary menu, one at a time, and you
can decide which you would like to build, based on last date accessed
and number of users accessing it.
