NAME LBRY EDITROUTE2
HEADER EDITING ROUTING 2
TEXT 
To assign a different routing order to a name, choose the correct
name and accept YES at the...OK?YES// prompt.  Accept the name again
at the next TO ROUTE TO prompt.  The system then displays ROUTING
ORDER with the previously assigned number as the default.  This
number may now be edited.
  
**NOTE**  When re-editing or reviewing already established
routing information for a copy, the last routing designee entered
will always appear before the double slashes at the Select
TO ROUTE TO prompt.
