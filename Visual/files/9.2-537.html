LBRY REMOVE (537)    HELP FRAME (9.2)

Name Value
NAME LBRY REMOVE
HEADER REMOVING COPIES
TEXT 
To delete a copy or TOC for a title, enter R for (R)emove.  At
the next prompt, enter the ID NUM for the copy or TOC to be removed.
It is then deleted.
  
***  If this feature is used to remove a copy or the TOC, ALL THE
ROUTING SLIPS listed will be erased.  Use the @ delete mark at a
TO ROUTE TO prompt in the (E)dit function of Copy Specific Edit (CSE)
to delete INDIVIDUAL names of recipients of a copy or a TOC.  ***
  
***Do not use this function to cancel a copy for which issues were
ROUTED TO RETURN while all those issues have not yet been returned in
the Return From Routing (RET) option in Serials Control (SER).  Instead,
use a [STOP] DATE in the (E)dit copy function to indicate this
cancellation.
