Name Value
NAME LBRY EDIT
HEADER EDITING COPIES
TEXT 
To edit data already established for a copy number or TOC for
a title, enter E for (E)dit.  At the next prompt, enter the
appropriate ID NUM from the chart.
  
The system then prompts for [GIFT], VENDOR, [PLACEMENT] OF ITEM,
CATEGORY OF COPY, [COPY] DISPOSITION, [START] DATE, and [STOP] 
DATE, allowing the data in each field to be edited.  It also prompts 
for routing names and routing order, if appropriate.
