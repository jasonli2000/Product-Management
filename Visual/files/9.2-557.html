NAME LBRY STATUS
HEADER ISSUE STATUS
TEXT 
Use a ? at the Select COPY OF ISSUE prompt to list the specific copies and
Table of Contents (TOC) for this JOURNAL DATE.  Input the correct copy
number; do not choose one which is not listed.
  
A ? at the ISSUE STATUS prompt then displays five [CHOICES]:
  
     1   ROUTED
     2   ROUTED (TO RETURN)
     3   SHELVED
     4   NOT RECEIVED
     5   TOC NOT RECEIVED
