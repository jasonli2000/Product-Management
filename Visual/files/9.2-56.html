Name Value
NAME XUKERNEL-USER-DISP
HEADER KERNEL MENU DISPLAY
TEXT 
The Kernel menu display can be recognized by the line of text following the
list of menu items which indicates how help text can be obtained with the
use of [QUESTION] marks.
 
The list of options in the [COMMON] menu is available to all users.  The
list can be displayed by entering two question marks at any Kernel menu
prompt.
