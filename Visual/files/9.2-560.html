AUTOMATIC QUEUEING:
     Required field.  A YES answer will automatically generate routing
     slips as the issues are checked in, if the issue was designated
     for routing in Copy Specific Edit (CSE).
  
     A NO response may be temporarily appropriate if you are entering 
     considerable retrospective check-in data; this will disable
     the automatic generation of routing slips for previously routed
     issues.
  
ROUTING SLIP PRINTER:
     Enter the device number for a local printer; this number is assigned
     by the local site manager for the printer port.
