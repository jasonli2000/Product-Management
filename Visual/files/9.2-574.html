NAME LBRY GAPS LIST
HEADER LOCAL GAPS LIST
TEXT 
Local Gaps List (GAPS) will print out a list by title of those
serials for which there are gaps in the holdings.  You must
have entered the gaps at the GAPS prompt in the Holdings Setup (HOL)
option.  There are no sort prompts for this report; the list will
automatically be arranged with all numeric titles first followed by
all other titles in alphabetic order.
  
Please enter [DEVICE] to see how to print the list.
