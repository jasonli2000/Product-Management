NAME LBRY MISSING R-T-R
HEADER OUTSTANDING ROUTED TO RETURN TITLES
TEXT 
The individual issues which have been routed and are to be returned to
the library will be displayed in Outstanding Routed to Return Titles (ORR).
The entries will display alphabetically by title, with copy number, 
date the serial was initially routed, the recipients to whom it was
routed, and volume and issue numbers also shown.
  
Please enter [START] for information on tailoring this report to
display a range of entries.
