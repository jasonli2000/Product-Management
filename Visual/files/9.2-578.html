NAME LBRY FOR-YOUR-INFORMATION
HEADER FOR-YOUR-INFORMATION TITLES
TEXT 
For-Your-Information Titles (FYI) will list all those titles for which
you entered any local information at the For-Your-Information prompt
under Library Title Setup (LTS).  The list will include the location,
category, and the For-Your-Information line.  There are no sort prompts
for this report.
  
Please enter [DEVICE] to see how to print the list.
