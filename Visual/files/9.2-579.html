TEXT 
The Library Monthly Statistical Report (MSR) provides a cumulated
numerical report on serial activity during a month.  The report
displays:
 
     (1)   the number of issues routed and the number of recipients
           to which they were routed;
 
     (2)   the number of Table of Contents (TOC) routed and the
           number of recipients for the TOC; and
 
     (3)   the number of titles which show any check-in
           activity during the month.
