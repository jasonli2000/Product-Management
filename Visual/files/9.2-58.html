NAME XUKERNEL-USER-SIGNON
HEADER SIGNING ONTO THE SYSTEM
TEXT 
To sign-on, you need to enter an access code and a verify code.
 
If you do not already have a verify code, just enter a carriage return
at the verify code prompt and you will be asked to enter a verify
code.
 
Every so often, as a security measure, you will be asked to change your
verify code.  The system will prompt you for a new code which you can
then enter.
 
Then, when you sign-off, you'll be reminded to remember your new verify
code for the next time you sign-on.
