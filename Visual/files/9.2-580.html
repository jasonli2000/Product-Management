LBRY START WITH (580)    HELP FRAME (9.2)

Name Value
NAME LBRY START WITH
HEADER START WITH PROMPT
TEXT 
At each START WITH prompt, enter the first letter or date that should be
printed out for a report.  At each GO TO prompt, enter the last letter
or date to be printed out for the report.
  
For example, enter <A><CR> at the "START WITH VENDOR: FIRST//"
prompt and enter <AZ><CR> at the "GO TO VENDOR: LAST//" prompt
to see those vendors whose name begins with the letter A.
  
Remember that entering just the initial letter, e.g. B, at the "GO TO"
prompt will only retrieve an item that has the one letter "B" as its
title.
  
At the "START WITH JOURNAL DATE: FIRST//" prompt, enter <1987><CR>
and enter <NOW><CR> at the "GO TO JOURNAL DATE: LAST//" prompt to see
journal issues from the beginning of 1987 to today.  Use the FIRST and
LAST defaults at each prompt to get a complete listing of all entries
available for the report.
  
Please [CONTINUE] for using the NULL feature, enter [STARTEX] for example.
