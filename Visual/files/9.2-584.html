To display a list of all the serials that begin with the letter "J"
which had missing issues in March, 1988 and were ordered from
AOBC, enter the information as shown in the following example:
  
  Start with vendor:  First// <AO><CR>
    Go to vendor:  Last//  <AP><CR>
  Start with Local Serials Titles:  First// <J><CR>
    Go to Local Serials Title:  Last// <JZ><CR>
  Start with Journal Date:  First// <3/1/88><CR>
    Go to Journal Date:  Last// <4/1/88><CR>
