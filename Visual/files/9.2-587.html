Your LOCAL SERIALS file is built from the TITLE AUTHORITY FILE (TAF)
provided with the package.  Much of the bibliographic information for 
commonly held titles has been supplied with the TAF, but it is not 100%
complete, nor will it ever be.  Periodic revisions to the TAF are 
planned in order to keep the bibliographic data as accurate as possible
while maintaining the integrity of that file.  VALNET libraries can help 
by providing timely notification of changes in titles, ISSN's, closures, 
etc., so that updates will be as accurate as possible.
