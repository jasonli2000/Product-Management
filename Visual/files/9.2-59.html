NAME XUKERNEL-USER-XUCOMMAND
HEADER KERNEL COMMON MENU
TEXT 
The common menu is the list of menu options displayed when entering
two question marks in response to a Kernel menu prompt, such as 'Select
MailMan Menu Option:'.  You will see several options which are available
for you to use:
 
     [CONTINUE] to use instead of 'halt',
 
     DISPLAY USER CHARACTERISTICS to show your sign-on characteristics,
 
     [EDIT USER CHARACTERISTICS] to edit some of these characteristics,
 
     HALT to finish your processing session,
 
     MAILMAN to use the electronic mail system, called MailMan,
 
     RESTART SESSION to restart at sign-on without dropping your line
     (if you have logged-in using a telecommunications line),
 
     TIME to display the time of day,
 
     USER HELP to see a series of help frames, this one included!
