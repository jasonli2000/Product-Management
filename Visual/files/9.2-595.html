TEXT 
Using <space bar><CR> at any option will display again the title which
you were just using.  For example, if you were editing copy data for
JAMA in Copy Specific Edit (CSE) and then went to Check-In (CHE), space
bar return at the Select TITLE prompt in CHE will pull up JAMA.  This
feature works within options, also.
