NAME LBRY JUMP
HEADER JUMP
TEXT 
1.  The jump mark ^ used alone takes you back one menu level at a time.
There may be a sequence of prompts in some sections where you do not need 
to input any answers, for example, perhaps the last 3 out of a series of 
8 prompts.  You may enter <^><CR> at the fifth prompt to go to the top of
the sequence for the next title; thus you will not waste time returning
through prompts you don't plan to use just then.
  
2.  The ^ used with a field name allows you to skip directly forward or
backward to that field prompt within an option.
  
3.  There are some places in the Serials Control Module where the use 
of the ^ is not allowed; the system will tell you when it will not 
accept it.
 
4.  You can also use the ^ with any 3-letter option code at any Select
Option prompt to move quickly from one part of the Module to another.
