To use the Serials Module only for basic check-in, you must create
your active titles in Library Title Setup (LTS) and Copy Specific
Edit (CSE).  They will then be ready for the functions under
Serials Control (SER), of which the primary one is Check-In (CHE).
Thus, you should use the options in the following order to prepare for 
check-in:
  
1.  LSP--Library Site Parameters           5.  LTS--Library Title Setup
2.  HSF--Hospital Service File             6.  CSE--Copy Specific Edit
3.  NME--Name Entries                      7.  CHE--Check-In
4.  IPL--Item Placement Location Entries
