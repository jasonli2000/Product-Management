NAME XUEDITSELF
HEADER USING THE 'Edit User Characteristics' OPTION
TEXT 
This option, which is accessible thru the Common Menu at all times,
enables the user to change certain of his/her own attributes.
 
It will first prompt for 'changing the devices terminal type'.
Only the current device's 'subtype' can be changed thru this option.
 
Then it will prompt for several fields in the New Person file.  This can
vary according to site.  The standard option will present VERIFY CODE,
INITIALS, NICKNAME, AUTO-MENU, and TYPE-AHEAD.  Any change on
AUTO-MENU will be effective immediately.
