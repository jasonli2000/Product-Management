NAME LBRY ID NUM
HEADER ID NUM
TEXT 
In this Serials Control Module, and especially in the Serials Control (SER)
options, ID NUM is used to identify and sequentially or
chronologically number items in a list; this is always used for
all the issue dates listed under a specific title and appears
in the left-hand column.
  
It is also used in Copy Specific Edit (CSE) to identify and 
list the copies designated for any one title.
  
When ID NUM is used in a display, the Choose ID NUM prompt
means that the number assigned to the item is to be used to 
pull up that line item for further action.
