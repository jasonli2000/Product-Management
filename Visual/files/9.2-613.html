TEXT 
If there was no prediction pattern associated with the title in
the Title Authority File (TAF), or if the title had to be added by 
you to the local portion of the TAF, then all journal issue dates 
for that title will have to be inserted for check-in.  Subsequent
regenerations of the TAF will add more prediction pattern data to that
file.
  
***NOTE***  There will be no prediction patterns for annuals and for
titles which do not use a month as part of the issue date.  The
prediction mechanism cannot work without a month in the issue date.
