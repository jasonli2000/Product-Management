Name Value
NAME XUSITEPARAM
HEADER THE 'Site Parameter Editing' OPTION
TEXT 
This option is used to edit the kernel site parameter file.
 
It is here that you may specify [AUTO-GENERATION] of access codes,
and the [VERIFY CODE] lifetime.
 
The log-on default parameters are also specified here.  These are used in
the case that there is no parameter specified in the user or device
file for attributes such as 'time-out', and 'number of attempts allowed'.
 
The fields that a user is allowed to edit through the XUCOMMAND option,
EDIT USER CHARACTERISTICS, can be modified by creating a [USER EDIT]
template and supplying its name in the kernel site parameters.
 
In addition there are fields for [LIMITING ACCESS] to the system.
