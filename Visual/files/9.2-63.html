XUSITEPARAM-AUTOCODE (63)    HELP FRAME (9.2)

Name Value
NAME XUSITEPARAM-AUTOCODE
HEADER AUTO-GENERATION OF ACCESS CODES
TEXT 
Access codes may be chosen by the person assigning them, or automatically
generated by the computer.
 
If they are automatically generated, the computer will assign a code
of three letters and three numbers.  You will then re-enter this code
to verify it.
 
The kernel site parameter file has a field indicating whether auto-generate
is in effect.
