NAME LBRY CSE
HEADER COPY SPECIFIC EDIT
TEXT 
Enter information about specific copies of a specific title.
  
For the first copy, the system will prompt for:
GIFT:    If the answer is YES, the system will prompt for an optional
         donor name and address, and then proceed to the prompt
         PLACEMENT OF ITEM.  If GIFT is left blank, the system will
         prompt for VENDOR.
VENDOR:  There will be a default to the current periodical procurement
         contractor.  To designate a different vendor, enter the name
         after the double slashes (//).  The system will confirm this entry
         and prompt for further information if it is a new vendor.
  
Choose [CONTINUE] to see more prompts.
