XUSITEPARAM-VERIFYLIFE (64)    HELP FRAME (9.2)

Name Value
NAME XUSITEPARAM-VERIFYLIFE
HEADER LIFETIME OF VERIFY CODE
TEXT 
The verify code should be changed periodically for the purpose of
security.  The site manager can decide how often a user must change
their verify code by setting the 'lifetime of verify code' field
in the kernel site parameters file.
 
When the time period has elapsed for changing a verify code, the user
will be prompted when they sign on to enter a new verify code.  They
may not use the same one, however they can re-enter an old one.
If the user prefers, the code which the computer has generated can
be used as the default.
