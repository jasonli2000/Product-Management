Fields which must have corresponding data in the Title Authority
File (TAF) include:
  
INDEXED:      Source must be an allowable Indexing Source; use ?
              at the Select INDEXED prompt to display the list.
PUBLISHER:    Must already be an allowable publisher; use a fragment
              of the publisher name to see if it is listed, or use ?
              to display all publishers available.
FORMERLY:     Title must be in Title Authority File (TAF).
CONTINUED BY: Title must be in Title Authority File (TAF).
  
After this step, the system will state "Editing LOCAL SERIALS file"
and prompt for local information.  Please see [LTS EXAMPLE].
