Name Value
NAME LBRY CSECONTINUE2
HEADER COPY EDIT CONTINUED2
TEXT 
COPY DISPOSITION:  This is used to note whether a journal is ROUTED,
         ROUTED AND RETURNED, or SHELVED.  Selecting the CATEGORY
         designation ADMINISTRATIVE will default to [ROUTED].  All
         others will default to SHELVED.  It is possible to override any
         default by entering the correct designation after the
         double slashes (//).
  
When multiple copies have been indicated in Library Title Setup (LTS),
the system duplicates all data for each copy in the COPIES ORDERED field.
Changes in specific copy information may then be made as necessary at
the "Do you want to [EDIT] any copy now? NO//" prompt.
