NAME XUAOLDPURGE
HEADER PURGING THE LOG OF OLD ACCESS AND VERIFY CODES
TEXT 
None
A log of all the access codes that have ever been used is kept on the
system.  This prevents a user from having an access code which used to
belong to someone else.  This is valuable for security reasons.
 
However, in time, it may be desirable to reduce the size of this file,
and make available those access codes which haven't been used in quite
a while.
 
This is done through the option which purges the log of old access codes.
You may specify the number of days worth of codes you wish to keep.
 
A log is also kept for each user of old verify codes.  This same option
purges all inactive verify codes older than the number of days specified.
