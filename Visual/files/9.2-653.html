NAME LBRY COMPLETION
HEADER CHECK-IN COMPLETION
TEXT 
After (1) you have either inserted an issue or (2) the Module has predicted 
the next issue, the system will ask "Do you want to check-in x copies of 
(date) issue today? YES//".
  
If you want to check-in that issue, enter <CR> to accept the YES
default, and the remainder of the check-in process will proceed, including
the generation of any routing slips requested.  There will then be a
prompt for the next title to be checked-in.
  
If you do not wish to check-in the x number of copies of that issue,
enter <NO><CR> to the question.  The system redisplays that title,
lists the issues, and presents the choices for action, including EXIT//.
