NAME LBRY DELETE
HEADER DELETE
TEXT 
The @ delete mark will delete a line of data previously entered.
As a safeguard the system will always ask if you really want to delete 
the data.
  
There are certain places in the Module where the system will not
allow the use of the @.  Deletion at those places would cause
errors in the system because the fields at those points are tied
to other parts of the Module.  However, editing can be done at these
fields.
