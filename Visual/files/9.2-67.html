XUSITEPARAM-USEREDIT (67)    HELP FRAME (9.2)

Name Value
NAME XUSITEPARAM-USEREDIT
HEADER WHAT FIELDS THE USERS THEMSELVES MAY EDIT
TEXT 
Users can edit certain of their own attributes through the 'Edit
User Characteristics' option in XUCOMMAND.  As a site manager, you may
vary the fields which a user is allowed to edit.
 
The default on the system is to use the fields specified in the 'EDIT
CHARACTERISTIC' input template.  These are verify code, initials, nickname,
auto menu, and type-ahead.  If you want the users at your site to be able
to edit different fields, construct an input template for the user file
containing those fields.  Supply the name of this template as the
'User Characteristic Template' in the kernel site parameter file.
