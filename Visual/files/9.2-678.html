NAME LBRY PREDICTION POP-UP
HEADER PREDICTION POP-UP
TEXT 
Enter the number of days in advance of the date on the next expected
issue that the issue is usually received in the library.  The default
is set to 5 days, so just <CR> should be entered if the issue is
expected to be received five days or less before the next issue date.
For example, if POPULAR SCIENCE issue dated August, 1988 is anticipated
to be received on July 29, 1988, enter <CR>; if it is expected about
July 20, 1988, enter <11><CR>.
