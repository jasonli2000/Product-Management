XUSITEPARAM-LIMITACCESS (68)    HELP FRAME (9.2)

Name Value
NAME XUSITEPARAM-LIMITACCESS
HEADER LIMITING ACCESS TO THE SYSTEM
TEXT 
The site manager can limit sign-on access to the system completely,
or by specifying the maximum number of jobs allowed.
 
This is done by setting the SYSTEM UNAVAILABLE field and MAXIMUM NUMBER
OF JOBS field in the KERNEL SITE PARAMETER file.  These fields are defined
per VOLUME set, and are found as multiples under this field.
 
The maximum number of jobs refers to active partitions.  This includes
caretaker and taskman generated jobs as well as standard interactive jobs.
 
If the SYSTEM UNAVAILABLE field is set to 'yes', then users may not sign
on and taskmanager will not start any jobs.
