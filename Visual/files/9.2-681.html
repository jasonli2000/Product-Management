NAME ONCO FOLLOW-UP METHOD
HEADER Follow-up Method
TEXT 
Code 0 (reported hospitalization) refers to a hospitalization at another
       hospital, NOT at your institution.
 
Code 1 (readmission) refers to hospitalization or outpatient visit at your
       institution.
 
Code 8 (other) refers to a friend or relative.
