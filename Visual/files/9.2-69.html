XQDEVICE-PASSWORDS (69)    HELP FRAME (9.2)

Name Value
NAME XQDEVICE-PASSWORDS
HEADER DEVICE PASSWORDS
TEXT 
For devices such as modems, it is often useful to have an additional level
of security.  This can be done by assigning a PASSWORD in the DEVICE file.
 
When a user attempts to log on, they will be prompted for the PASSWORD
before being prompted for ACCESS and VERIFY code.  The maximum number of
attempts will be in effect for the DEVCIE PASSWORD.
