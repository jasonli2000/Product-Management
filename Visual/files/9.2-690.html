NAME ECDSS1
HEADER National/Local Procedure Reports
TEXT 
This option allows you to print/display a report listing Event Capture 
procedures.  You can to choose to include active or inactive procedures. 
 
If you choose inactive procedures, you are prompted for a device.  The 
output includes  
 
 - Procedure name 
 - National Number 
 - CPT 
 - Inactive Date 
 
If you choose active procedures, you are prompted to choose a sort method 
(National, Local, or Both) and a device.  The output includes  
 
 - Procedure name
 - National Number
 - CPT
