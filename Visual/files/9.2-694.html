Name Value
NAME ECENTER
HEADER Event Capture Data Entry
TEXT 
This menu option accesses the data entry options of the Event Capture
software.  The four options allow the user to enter data either by
[individual] patient and procedure, or in [batch] mode where a large
number of patients, with a large number of the same procedures may be
entered. There is also a [batch by procedure] option to enter multiple
patients for a given procedure, and a [mult dates/mult procs] option which
allows the user to enter multiple procedure dates and procedures for
multiple patients.
