NAME ECDSAI
HEADER Activate/Inactivate
TEXT 
This menu option will allow you to use other options which will create or
inactivate event code screens, inactivate local categories and procedures,
and otherwise toggle to on/off other parameters of the Event Capture
software.
 
Included are the options for [Current] Locations; [Local] Categories;
[Event] Code Screens; [Inactivate] Event Code Screens; and
[Activate]/Deactivate Local Procedures.
