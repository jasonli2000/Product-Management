NAME ECDS CATEGORY
HEADER Local Category (Enter/Edit)
TEXT 
This option allows you to ENTER or EDIT, but NOT DELETE, local 
categories in the EVENT CAPTURE CATEGORY file (#726).  
 
DSS Units can be defined to use categories to group procedures during 
data entry.  Categories are facility-specific, so you can use names that 
best suit your facility.  For example, if you have a DSS Unit named 
Audiology and Speech, you might have categories named Speech Evaluation 
and Audiology Testing.  It is acceptable to have both DSS Units that DO 
use categories and DSS Units that DO NOT use categories defined at your 
facility.  For the DSS Units at your facility that do not use categories 
to group procedures, you can ignore this option.
