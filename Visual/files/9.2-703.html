ECNACT (703)    HELP FRAME (9.2)

Name Value
NAME ECNACT
HEADER Inactivate Event Code Screens
TEXT 
** You must define DSS Units, categories, procedures, and Event Code 
Screens before using this option. ** 
 
This option allows you to INACTIVATE or REACTIVATE, but NOT DELETE 
Event Code Screens.  It toggles the INACTIVE DATE field (#1) in the     
EC EVENT CODE SCREENS file (#720.3) on/off.  You are prompted for 
 
 - Location 
 - DSS Unit 
 - Category (If DSS Units use categories to group procedures) 
 - Procedure 
 
If the INACTIVE DATE field is turned OFF for the procedure you select, 
the software asks if want to INACTIVATE the procedure.  If this field 
is already turned ON for the procedure you select, the software tells 
you that the Event Code Screen for the selected procedure has a status  
of inactive and asks if you want to ACTIVATE it.
 
** PLEASE NOTE:  This option inactivates the selected procedure in 
the specified Event Code Screen ONLY; IT DOES NOT INACTIVATE THE 
PROCEDURE ITSELF. **
