NAME ECDSDEAC
HEADER Activate/Inactivate Local Procedures
TEXT 
** You must create local procedures before using this option. ** 
 
** You might want to use the National/Local Procedure Reports option on 
the Management Reports Menu to print a list of the inactive procedures 
before using this option. ** 
 
This option allows you to INACTIVATE or REACTIVATE, but NOT DELETE, 
local procedures.  It toggles the INACTIVE DATE field (#2) in the   
EC NATIONAL PROCEDURE file (#725) on/off.  You are prompted for 
a procedure.  
 
If the INACTIVE DATE field is turned OFF for the procedure you selected, 
the software asks if you want to INACTIVATE the procedure.  If this 
field is turned ON for the procedure you selected, the software asks 
if you want to REACTIVATE the procedure.
