NAME XQOPTWHO
HEADER USING THE 'Option Access By User' HELP FRAME
TEXT 
This option will produce a listing of users who have access to the
option which you specify. 
 
You will be prompted first for the OPTION you wish to trace.  You will
then be asked if you want to show menu paths.  If you answer 'YES',
the listing will include the paths that each user has to the specified
option.
 
The option will include active users only (those with an access code)
and will show the last sign-on time.
