Name Value
NAME ECDSUSR
HEADER User Access
TEXT 
This menu option leads to the options for controlling access to DSS Units
for data entry.  You may assign access to any DSS Units which have been
established to anyone within the NEW PERSON file.  It is suggested that
assignment of access to DSS Units be limited to those individuals who have
the responsibility for data input into those units.
 
Assigning access will still require the assignment of options within the
Menu Manager.  It should be remembered that for access to the DSS Units to
be complete, access must be granted through both the Menu Manager and the
Event Capture software.
 
Under this menu you will be able to [assign] and [remove] user access.
