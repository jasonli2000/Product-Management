NAME ECPAT
HEADER Enter/Edit Patient Procedures
TEXT 
** You must have locations, DSS Units, categories, and procedures defined 
before using this option. ** 
 
** If only one category is defined for the specified DSS Unit, you will 
not be prompted for category. ** 
 
** If only one procedure is defined for the specified category, you will 
not be prompted for procedure. ** 
 
** You will be prompted for exposure and service-connected indicators 
for OUTPATIENT PROCEDURES ONLY. ** 
 
** Procedures being passed to PCE must have an active Associated Clinic. **
 
This option allows you to enter patient procedures performed on an
individual patient.  You should use it when 
 
 - you have an individual patient record to update. 
 - you have a stack of patient encounter cards that you need to enter 
   in the Event Capture software. 
 
You can enter one procedure per patient.  When you finish entering data 
for the specified patient, the "Select Patient:" prompt repeats, so you 
can enter another procedure for the same patient, or select another patient 
for whom you want to enter procedure data.  To exit the option, press  
the RETURN or ENTER key to indicate that data entry is complete.
