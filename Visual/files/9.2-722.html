NAME ECCP1
HEADER Summary of Procedures and Categories (Old File)
TEXT 
** If your facility currently has Event Capture V. 1.6 installed, you
should run the Summary of Procedures and Categories (Old) option to print 
data from the pilot EVENT CODE SCREENING file (#720.2) for use in 
converting data to the current data formats. **
 
This option allows you to print/display the Event Code Screens for one 
or all DSS Units within a specified location.  If DSS Units are 
defined to group procedures by category, you can select one or all 
categories.  All procedures will print for the selected categories.
