ECBATCH PROC (723)    HELP FRAME (9.2)

Name Value
NAME ECBATCH PROC
HEADER Data Entry (Batch) by Procedure
TEXT 
** You must have Event Capture locations, DSS Units, local categories, and
local procedures, and defined before using this option. **
 
** You must define Event Code Screens before entering Event Capture data.
**
 
** You will be prompted for exposure and service-connected indicators for
OUTPATIENT PROCEDURES ONLY. **
 
** If only one category is defined for the specified DSS Unit, you will
not be prompted for category. **
 
** If only one procedure is defined for the specified category, you will
not be prompted for procedure. **
 
** Procedures being passed to PCE must have an active Associated Clinic.
**
 
This option allows you to enter a single procedure for multiple patients
as a batch entry.     
 
After you verify that the procedure information is correct, the "Select
Patient:" prompt appears.  When you finish entering data for the first
patient, the "Select Next Patient:" prompt appears.  This prompt will
repeat, allowing you to enter multiple patients, until you press the
RETURN or ENTER key to indicate that patient selection is complete. 
 
The software displays the procedure entered for the group of patients, and
asks if you want to enter them in the file.  If you respond YES, they are
processed in the background and added to the EVENT CAPTURE PATIENT file
(#721).
