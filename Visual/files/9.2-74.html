Name Value
NAME XQRESTRICT
HEADER Using the RESTRICT OPTIONS Option
TEXT 
   Several restrictions can be placed upon the use of any option
in the OPTION file.  The Site Manager or Security officer can
disallow the use of a particular option by:
  
 o Marking that option as '[OUT-OF-ORDER]' so that no one can use 
   it.
 
 o Place a [LOCK] on an option so that only those users with the
   proper key can use it.
 
 o Adjust the priority of an option so that runs at a lower or
   higher [PRIORITY].
 
 o Disallow the use of an option at certain [TIMES].
 
 o Restrict the use of an option to certain, specified [DEVICES].
 
 o Or any combination of the above restrictions on one or more
   [OPTIONS].
