NAME GMRV-CIRCUM/GIRTH RATE HELP
HEADER Circumference/Girth
TEXT 
The entry for circumference/girth must be a number appended with a qualifier
to represent the value either in the English scale or the Metric system.  If
you wish to enter a circumference/girth in metric system, append 'C' at the
end of the number.  Add 'I' for inches to represent the value in English
scale.  Some examples of valid entries are:
  
   41.25I, 41.25i, 72.25C, 72.25c
