                      Contract Flag Field Help
 
 
      The field 'Contract Flag' is an indicator for 'Contract Number'
      (.01) field.  It can be a Contract # or BOA # as appropriate. 
 
      If Contract Flag is selected to be a 'B' the 'Contract Number'
      field will only accept Alpha or Numeric characters as BOA #.
 
      If Contract Flag is selected to be a 'C' the 'Contract Number' 
      field will accept any characters as Contract #.
 
 
Exception: 
      Whenever the Source Code of the P.O. is selected to be an 'Open
      Market' the Contract Flag must be set to 'C'.   The Contract Number
      field is not allowed to accept the BOA # when Source Code is 'Open
      Market'.
