NAME XQRESTRICT-RESDEV
HEADER RESPONDING TO THE 'RESTRICT DEVICES' PROMPT
TEXT 
    You may restrict the use of a particular option to certain devices
on your system.  The RESTRICT DEVICES field in the option file, which
is the field being filled in by this question, is set to either YES
or NO.  This field is used by the system to tell the log-on software
whether or not it should look through the list of PERMITTED DEVICES
which is stored in another, multiple field.
  
    If this field is set to YES then only those devices listed in the
PERMITTED DEVICES will be permitted to use the option.  If this field
is set to NO or left blank then any devices listed in the PERMITTED
DEVICES field will be ignored and no device restrictions will be used
for the option in question.
  
    Responding to this prompt with an '@' will delete whatever is
already in this field, turning the PERMITTED DEVICE restriction off
for all of the options selected.
