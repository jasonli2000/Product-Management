Name Value
NAME ECMULT PROC
HEADER Multiple Dates/Multiple Procedures Data Entry
TEXT 
This option allows entry of multiple procedures on multiple dates for one
or more patient.  This option does not require the user to exit the option
to change dates or procedures for data entry.
 
You will be prompted to enter any number of procedure dates and then any
number of procedures.  You will then be prompted to enter patients
and information relative to that patient (I.E. Ordering Section, Agent
Orange, Associated Clinic, etc.).
