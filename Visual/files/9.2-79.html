Name Value
NAME XQSMD-OPTION
HEADER Building Option Lists for MenuMan Delegation
TEXT 
    
   You are building a list of options to be added to, or deleted
from, the users you have already specified.  You may enter these
options in several different ways:
  
   You may simply enter an option name from the Option File,
     or  NAM* to specify all options that begin with the
              characters 'NAM',
     or  OPT1-OPT2 to specify all the options that fall
              alphabetically between OPT1 and OPT2 inclusive,
     or  *    to select all delegable options,
     or  -    followed by any of the above to remove options
              you have already selected,
     or  ^    to quit and exit the program,
     or  ?    to see a brief help prompt,
     or  ??   to see this help screen again,
     or  ???   to see the list of Users and Options chosen so far,
     or  ????   to see the Option File,
     or  ?????   to learn more about Menu Management Delegation.
