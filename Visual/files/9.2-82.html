Name Value
NAME XQSMD-REMOVE
HEADER Remove all options from a user
TEXT 
   If you answer yes ('Y') to this prompt all of the options
that have been delegated to this user will be removed after
they have been transferred to a second user.  In this way
a new delegate will take the place of the former delegate
and the former delegate will no longer have any options to
manage.
   In order to remove all options from an existing delegate you
must have a Delegation [Level] in the User File that is numerically
lower than the Delegation Level of the person that you are
trying to remove from the Delegation System.  If you do not,
the existing delegate will retain the options already delegated
to her.  Delegation Level may be changed by a programmer using
FileMan.  The Delegation Level is automatically set to the number
of steps a delegate is away from the original delegation of the
Site Manager.
