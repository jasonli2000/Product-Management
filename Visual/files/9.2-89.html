NAME XUDOC DEVICE LOOKUP
HEADER DEVICE LOOKUP
TEXT 
There are several ways to select a device when using the Device Editor
options.  They are ways of responding to the following prompt:
 
     Select DEVICE NAME:
 
When you enter the name of a device, a lookup is done on the Device File
to find a device by that name.  Along with name, there are several other
device attributes that can be used for lookup:
 
     DEVICE NAME      Logical device name (.01 Field of the Device File).
 
     PORT/DEVICE $I   Physical port connection on the residing CPU.
 
     VOLUME SET/CPU   The volume set or central processing unit.
 
     SIGN-ON SYSTEM   Flag indicating whether the device can be used to
                      sign-on and access the computer system.
 
The Device File is cross-referenced which is the [TECHNICAL INFORMATION]
used to provide these four alternative lookup methods.
