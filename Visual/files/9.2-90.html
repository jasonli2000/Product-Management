NAME XUDOC DEVICE LOOKUP TECH
HEADER TECHNICAL OVERVIEW: DEVICE LOOKUP
TEXT 
The Device Name, .01 Field of the Device File, may be used for lookup.
If several names match the character string entered, a choice is presented.
 
The device $I (port number) may also be used for lookup.  The search will
find all devices on all volume sets/CPUs that use a port of this number.
 
Specifying the volume set/CPU will narrow the lookup search.  Entering
the CPU name will display a choice of devices on that CPU.  Entering the
CPU name and $I, separated by a decimal point, will shorten the list:
 
     Select DEVICE NAME: BBB.75
 
Devices are also cross-referenced by the flag which indicates whether
it is a sign-on system device.  Entering 'SYS' will display all devices
with the flag set to 'yes'.  This may be combined with the CPU and $I
as above to futher limit the lookup choices:
 
     Select DEVICE NAME: SYS.BBB.75
 
These lookup methods are possible since the Device File contains a number
of cross-references: by name, $I, Volume Set(CPU), and Sign-on/System.
