XUDOC SPOOL MGT1 (94)    HELP FRAME (9.2)

Name Value
NAME XUDOC SPOOL MGT1
HEADER MANAGING THE SPOOLER
TEXT 
You, the systems manager, will need to use the Spool Management Menu to
set system defaults and assign spooling privileges.  This menu is part of
the Systems Manager Menu.
 
System defaults (site parameters) set a maximum for the number of documents
per user, the maximum spool lines per user, and the maximum lifetime of
spool documents.  The Kernel Site Parameter File holds this data.
 
User access involves 3 New Person file attributes.  First, may this user
write to the spool device?  If not, a message to that effect will be issued
when the user attempts to use the spool device.
 
Second, when printing a spool document, may this user send output to more
than one printer?  If so, the system will continue to prompt for another
device until the response is null.
 
Third, may this user make spool documents into mail messages?  If so, the
user may treat the spool document just like a mail message, to copy, edit,
or send over the network.
