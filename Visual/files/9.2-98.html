NAME XUDOC SPOOL USE3
HEADER RETRIEVING YOUR SPOOLED DOCUMENTS
TEXT 
Now that you have created some spool documents, go to the Spooler Menu to
find out how you can retrieve your documents.  The Spooler Menu is usually
included in the Common menu, what you see when you type two question
marks at any main menu prompt.  For quick access to the Spooler Menu, enter
the first few letters of the menu text preceded by a double quote ("SP).  
 
  GIVE ACCESS TO OTHERS   Use this option to specify who may access which of
                          your spool documents.
 
                   LIST   This will display a listing of spool documents.
 
                  [PRINT]   This option allows the printing of documents.
 
                 DELETE   Each user is allocated a limited amount of space
                          for spooling.  As a general rule you should
                          delete unnecessary documents, and you will have
                          to delete some if you run out of space.
 
 [MAKE INTO MAIL MESSAGE]   This option is not generally available since
                          the MailMan storage space is limited.  See your
                          IRM representative if you need this option.
