    12. PATCH APPLICATION HISTORY:   12 SEQ #12
      DATE APPLIED:   1999-12-10 00:00:00
      APPLIED BY:   USER,ONE
      DESCRIPTION:   
      Full TCP/IP support for HL7 messaging is available with patch HL*1.6*19.
Two links are required for message transactions.  One link is used by
VISTA to send messages and receive acknowledgements to those messages.
The other link is used by a commercial off the shelf (COTS) voice
recognition unit to send messages and receive acknowledgments to those
messages.  (The message sent and the acknowledgement to that message
comprise a full transaction.)
 
Patch RA*5*12 includes protocols that now enable Radiology/Nuclear
Medicine to communicate with COTS systems using HL7 v2.3 messages over
TCP/IP. The original release of Rad/Nuc Med v5.0 supports HL7 v2.1
messages.  Patch RA*5*12 also supports the VISTA HL7 v2.3 Interface to the
PowerScribe for Radiology COTS voice recognition system.  Any one of
the following events generate VISTA HL7 messages:
1) Registration
2) Cancellation
3) Examined (i.e., images collected, usually set up in sites using the
VISTA Imaging package and/or a PACS system)
 
This patch also includes an exceptions file and a ListMan option to view
or print the HL7 exceptions and take appropriate action to resolve the
problem.
 
Please consult the Radiology/Nuclear Medicine Technical Manual for further
information concerning TCP/IP messaging.
