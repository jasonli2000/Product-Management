Name Value
NAME INCOME VERIFICATION MATCH
PREFIX IVM
SHORT DESCRIPTION IVM Software for interface with the IVM Center
DESCRIPTION 
 
The IVM package will transmit information to the IVM Center for patients
whose means tests meet the following criteria:
 
       Means test is Category A, but neither hardship or adjudicated
                                   OR
       Means test is Category C, but patient has no active insurance
 
Information transmitted includes demographics, income and next-of-kin
information as well as demographic and income data for the patient's
spouse.
 
Changes to the patient's name, SSN, DOB, sex, address, phone number, or
means test are transmitted for patients being reviewed by the IVM Center.

what is written to the screen and the interactive read of the IVM REASON
FOR NOT UPLOADING (#301.91) file within the UPDATE API.

         Verify Checksums in Transport Global
         Install Package(s)
         Restart Install of Package(s)
         Unload a Distribution

           NAME, enter IVM*2.0*148):

        message of any routines exported with this patch. It will not
