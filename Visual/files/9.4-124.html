any questions, your supporting CIOFO Customer Service Clinical 2
team will address them.

     onto your system.

    option 'Verify Checksum in Transport Global' and verify that all        
routines have the correct checksums.

    'Install Package(s)'.  Select the package 'MC*2.3*14' and proceed 
    with install.                                                           
 4) If you are a MSM site move the routines to the other servers.       
 5) On a mapped system, rebuild your map set.

      Description:
===========
 
The purpose of this patch is to modify the Medicine reports so that they
can be added to the CPRS List Manager / GUI reports tab.  This patch also
contains the fix to NOIS PUG-1297-51119 DIC('S') left defined.          
 
Medicine users may remain on-line during the installation of this patch.
No menu options need to be disabled.  If a user is printing a summary
report at the time the patch is installed, the process could bomb out, but
no data base corruption will occur.
 
Since this patch only installs eight routines installation time is minimal.

 Installation Instructions: 
 ==========================
This patch may be installed with users on the system.  Installation of
this patch will take less than 5 minutes.
 
 1.  Use the INSTALL/CHECK MESSAGE option on the PackMan menu, to
     load the KIDS Transport Distribution global.
 
 2.  From the Kernel Installation and Distribution System (KIDS) Menu,
     select the Installation menu.
 
 3.  From the Install menu, you may elect to use the following
     options (when prompted for INSTALL NAME, enter MC*2.3*20):
 
      a.  Print Transport Global
      b.  Compare Transport Global to Current System
 
 4.  Use the Install Package(s) option and select the package
     MC*2.3*20.
 
 5.  When prompted "Want to DISABLE Scheduled Options,
     Menu Options, and Protocols? YES//", respond "NO"
