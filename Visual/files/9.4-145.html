NAME CLINICAL MONITORING SYSTEM
PREFIX QAM
SHORT DESCRIPTION Allows users to build their own monitors.
DESCRIPTION 
The Clinical Monitoring System allows the users to create their own
monitors from a list of pre-defined conditions.  The monitors then run
nightly, scanning DHCP data for patients who meet the conditions of the
monitor.  The user may select data elements from a pre-defined list to be
captured when a patient falls out for a given monitor.  Programmers may
add new data elements and conditions to the system.  Users can design and
print ad hoc reports from the fall out data.
