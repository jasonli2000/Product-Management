SHORT DESCRIPTION Software to support integrated sites.
DESCRIPTION 
 The National Database Integration System (NDBI) is an application
developed for the VA medical center integrations. This package contains
the files used at the legacy and primary sites. The entire application
consists of files, options, routines located in the A7R namespace, which
comprises the utilities used to consolidate the data extracted from the
VISTA applications.
  
 The actual distribution of the application is sectioned into two builds.
One is the NDBI PRIMARY SYSTEM, consisting of all NDBI files, A7R*
routines, A7RP* options and A7RC* options. The other build is NDBI LEGACY
SYSTEM, consisting of all NDBI files, A7R* routines, A7RL* options and
A7RC* options.  
   
 The options are in the namespace of A7RC, A7RL and A7RP. The A7RC*
options are Common options used at the legacy and primary sites. They
consist of utilities that are used for the New Person and Patient files,
and our NDBI Event Log utilities. The A7RL* options are Legacy options
used to collect and process data from the legacy site only. The A7RP*
options are Primary options used to preview, merge and process the data
collected from legacy system. These A7RP* options are used on the primary
site only.

    DESCRIPTION OF ENHANCEMENTS:   
     This package contains a National Database Integration (NDBI) routine,
 A7RXDRPT which will be referenced by the Duplicate Record Merge patch
 (XT*7.3*23). It must be installed at all sites. When installed it will
 populate the PACKAGE (#9.4) file with the new entry of NDBI and the
 fields AFFECTS RECORD MERGE (#20), FILE AFFECTED (#.01) with 'PATIENT'
 and NAME OF MERGE ROUTINE (#3) with 'A7RXDRPT'.
 
 When the Duplicate Resolution options, exported by the XT*7.3*23
 patch, are run for the PATIENT (#2) file, the routine exported in this
 build will perform an environment check to determine if the site is an
 integrated facility (NDBI site).  If so, NDBI data files will be updated,
 as necessary, with corrected entries.
 
 This can be installed at any time while users are on the system and
 will take no more than a few minutes to complete.
 
 The checksum value for this routine is 73468.
