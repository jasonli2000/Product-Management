Name Value
NAME VA-HIV TRANSMISSION
PRINT NAME HIV Transmission
EDUCATIONAL OUTCOME 
The patient will describe what the HIV virus is and how it is transmitted.
EDUCATIONAL STANDARDS 
1.  Explain what the human immunodeficiency virus (HIV) is and how it is
related to AIDS.
 
2.  Discuss the ways that HIV is transmitted via blood and sexual contact.
 
3.  Explain how to clean shared needles.
 
4.  Describe safe sex and safer sex.
 
5.  Explain that HIV is not transmitted via casual contact.
